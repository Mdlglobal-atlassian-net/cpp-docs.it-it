# A.3   Using Parallel Regions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`parallel` direttiva \([parte 2,3](../../parallel/openmp/2-3-parallel-construct.md) a pagina 8\) pu√≤ essere utilizzato nei programmi in parallelo di grezzo\-grano.  Nell'esempio seguente, ogni thread nell'area parallela decide quale parte della matrice globale `x` per agire su, in base al numero di thread:  
  
```  
#pragma omp parallel shared(x, npoints) private(iam, np, ipoints)  
{  
    iam = omp_get_thread_num();  
    np =  omp_get_num_threads();  
    ipoints = npoints / np;  
    subdomain(x, iam, ipoints);  
}  
```
