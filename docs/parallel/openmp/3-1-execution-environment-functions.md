-   il **omp_get_max_threads** (funzione).  
  
-   il **omp_get_thread_num** (funzione).  
  
-   il **omp_get_num_procs** (funzione).  
  
-   il **omp_in_parallel** (funzione).  
  
-   il **omp_set_dynamic** (funzione).
