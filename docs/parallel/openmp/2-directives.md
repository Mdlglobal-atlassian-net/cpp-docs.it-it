# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le direttive sono basate su **\#pragma** direttive definite negli standard di C\+\+ e c\#.  I compilatori che supportano il OpenMP C\# e C\+\+ API includeranno un'opzione della riga di comando che attiva e consente l'interpretazione di tutte le direttive del compilatore di OpenMP.
