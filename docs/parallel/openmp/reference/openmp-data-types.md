# OpenMP Data Types
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Vengono forniti collegamenti a tipi di dati utilizzati in OpenMP API.  
  
 L'implementazione di Visual C\+\+ standard di OpenMP sono disponibili i seguenti tipi di dati.  
  
|Tipo di dati|Descrizione|  
|------------------|-----------------|  
|[omp\_lock\_t](../../../parallel/openmp/reference/omp-lock-t.md)|un tipo che utilizza lo stato di un blocco, se il blocco è disponibile o se un thread possiede un blocco.|  
|[omp\_nest\_lock\_t](../../../parallel/openmp/reference/omp-nest-lock-t.md)|Un tipo che contiene una delle seguenti informazioni su un blocco: se il blocco è disponibile e l'identità del thread proprietario del blocco e un conteggio di annidamento.|
