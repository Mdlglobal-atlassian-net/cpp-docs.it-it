# <a name="ordered-openmp-clauses"></a>ordered (clausole OpenMP)

Obbligatorio in un'operazione parallela [per](../../../parallel/openmp/reference/for-openmp.md) istruzione se un' [ordinati](../../../parallel/openmp/reference/ordered-openmp-directives.md) direttiva deve essere utilizzato nel ciclo.

## <a name="syntax"></a>Sintassi

```
ordered
```

## <a name="remarks"></a>Note

`ordered` viene applicata il [per](../../../parallel/openmp/reference/for-openmp.md) direttiva.

Per altre informazioni, vedere [2.4.1 costrutto for](../../../parallel/openmp/2-4-1-for-construct.md).
