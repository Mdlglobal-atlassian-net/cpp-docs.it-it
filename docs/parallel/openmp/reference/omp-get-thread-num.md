# omp_get_thread_num
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Restituisce il numero del thread che esegue all'interno del team di thread.  
  
## Sintassi  
  
```  
int omp_get_thread_num( );  
```
