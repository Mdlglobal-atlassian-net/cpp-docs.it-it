# <a name="ompgetthreadnum"></a>omp_get_thread_num
Restituisce il numero di thread del thread in esecuzione nel relativo team di thread.  
  
## <a name="syntax"></a>Sintassi  
  
```  
int omp_get_thread_num( );  
```  
  
## <a name="remarks"></a>Note  
 Per ulteriori informazioni, vedere [3.1.4 funzione omp_get_thread_num](../../../parallel/openmp/3-1-4-omp-get-thread-num-function.md).
