##  <a name="address_mode"></a>Enumerazione address_mode  
 Tipo di enumerazione utilizzare per indicare la modalità di indirizzo per il campionamento della trama è supportata.  
  
```  
enum address_mode;  
```  
  
##  <a name="filter_mode"></a>Enumerazione filter_mode  
 Tipo di enumerazione utilizzare per indicare la modalità di filtro per il campionamento della trama è supportata.  
  
```  
enum filter_mode;  
```
