Gli attributi del consumer OLE DB inseriscono codice, basato sui [modelli consumer OLE DB](../../data/oledb/ole-db-consumer-templates-reference.md), per creare un consumer OLE DB funzionante che esegue attivit√† quali l'apertura di tabelle, l'esecuzione di comandi e l'accesso ai dati.

|Attributo|Descrizione|
|---------------|-----------------|
|[db_accessor](db-accessor.md)|Associa le colonne in un set di righe e le associa alle mappe delle funzioni di accesso corrispondenti.|
|[db_column](db-column.md)|Associa una colonna specificata al set di righe.|
|[db_command](db-command.md)|Esegue un comando OLE DB.|
|[db_param](db-param.md)|Associa la variabile membro specificata a un parametro di input o di output.|
