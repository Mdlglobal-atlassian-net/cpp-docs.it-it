# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le risorse grafiche sono le immagini definite per l'applicazione.  Possono essere ottenute tramite il disegno a mano libera oppure tramite l'utilizzo di forme.  È possibile selezionare parti di immagini per modificarle, capovolgerle o ridimensionarle oppure creare un pennello personalizzato da utilizzare per disegnare.  È inoltre possibile definire le proprietà delle immagini, salvare e convertire le immagini in formati diversi.  
  
 Oltre a creare nuove risorse grafiche, si possono [importare immagini esistenti](../windows/how-to-import-and-export-resources.md) per modificarle e aggiungerle al progetto.  La [modifica autonoma di immagini](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md) consente di aprire e modificare immagini non incluse in un progetto.
