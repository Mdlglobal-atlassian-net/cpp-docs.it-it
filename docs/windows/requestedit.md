# requestedit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

indica che la proprietà supporta **OnRequestEdit** notifica.  
  
## Sintassi  
  
```  
  
[requestedit]  
  
```  
  
## Note  
 **requestedit** L'attributo di C\+\+ ha la stessa funzionalità di  [requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) Attributo MIDL.

 Vedere l'esempio relativo a [associabile](../windows/bindable.md) per un esempio di utilizzo  **requestedit**.
