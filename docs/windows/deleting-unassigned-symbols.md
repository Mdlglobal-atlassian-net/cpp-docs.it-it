# Deleting Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Per eliminare un simbolo non assegnato \(non usato\)  
  
1.  Nella [finestra di dialogo Simboli risorsa](../windows/resource-symbols-dialog-box.md), selezionare il simbolo che si vuole eliminare e fare clic su **Elimina**.  
  
    > [!NOTE]
    >  Prima di eliminare un simbolo non usato in un file di risorse, verificare che non venga usato in un punto del programma o da file di risorse inclusi in fase di compilazione.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle propriet√†, vedere [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
