# Filling a Bounded Area of an Image with a Color (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'editor di immagini consente mediante lo strumento **Riempimento** di riempire qualsiasi area delimitata di un'immagine con il colore di disegno o il colore di sfondo correnti.  
  
> [!TIP]
>  Il suggerimento viene visualizzato quando si passa il cursore sui pulsanti della barra degli strumenti.  che consentono di identificare facilmente la funzione di ciascun pulsante.  
  
### Per utilizzare lo strumento di riempimento  
  
1.  Selezionare lo strumento **Riempimento** sulla barra degli strumenti dell'**Editor immagini** o mediante l'opzione **Strumenti** del menu **Immagine**.  
  
2.  Se necessario, selezionare i colori per il disegno. Nella [tavolozza Colori](../windows/colors-window-image-editor-for-icons.md) fare clic con il pulsante sinistro del mouse per selezionare un colore di primo piano oppure fare clic con il pulsante destro per selezionare un colore di sfondo.  
  
3.  Spostare lo strumento **Riempimento** nell'area che si desidera riempire.  
  
4.  Fare clic con il pulsante sinistro o destro del mouse per riempire l'area rispettivamente con il colore di primo piano o di sfondo.
