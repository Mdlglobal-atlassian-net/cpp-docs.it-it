# Disabling Guides
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilizzando appositi tasti in combinazione con il mouse è possibile disabilitare l'effetto di blocco delle guide.  Il tasto ALT consente di disabilitare gli effetti di blocco della guida selezionata.  Tenendo premuto MAIUSC durante lo spostamento di una guida è possibile evitare che i controlli bloccati vengano spostati con la guida.  
  
### Per disabilitare l'effetto di blocco delle guide  
  
1.  Trascinare il controllo tenendo premuto **ALT**.  
  
### Per spostare le guide senza spostare i controlli bloccati  
  
1.  Trascinare la guida tenendo premuto **MAIUSC**.  
  
### Per disattivare le guide  
  
1.  Scegliere **Impostazioni guide** dal menu **Formato**.  
  
2.  Nella [finestra di dialogo Impostazioni guide](../mfc/guide-settings-dialog-box.md) selezionare **Nessuna** in **Guide layout**.  
  
    > [!NOTE]
    >  È possibile accedere alla finestra di dialogo **Impostazioni guide** anche facendo doppio clic sulla barra del righello.  
  
 \- oppure \-  
  
-   Scegliere **Attiva\/Disattiva guide** dal menu **Formato**.
