# Metodo HString::Attach
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Associa l'oggetto specificato di HString con l'oggetto corrente di HString.  
  
## Sintassi  
  
```  
  
void Attach(  
       HSTRING hstr  
       ) throw()  
```  
  
#### Parametri  
 `hstr`  
 Un oggetto esistente di HString.  
  
## Requisiti  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
