# Changing Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mentre ci si trova nella [finestra di dialogo Simboli risorsa](../windows/resource-symbols-dialog-box.md), è possibile modificare o eliminare i simboli esistenti che non sono già assegnati a una risorsa o un oggetto.  
  
### Per modificare un simbolo non assegnato  
  
1.  Nella casella **Nome** selezionare il simbolo non assegnato e fare clic su **Modifica**.  
  
2.  Modificare il nome o il valore del simbolo nelle caselle disponibili nella finestra di dialogo **Cambia simbolo** .  
  
    > [!NOTE]
    >  Per modificare un simbolo che *è* assegnato a una risorsa o un oggetto, è necessario usare l'editor di risorse o la finestra Proprietà.  Per altre informazioni vedere [Modifica di un simbolo o del nome di un simbolo](../windows/changing-a-symbol-or-symbol-name-id.md).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.*  
  
 Requisiti  
  
 Win32
