# Metodo HandleT::IsValid
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica se l'oggetto HandleT corrente rappresenta l'handle.  
  
## Sintassi  
  
```  
bool IsValid() const;  
```  
  
## Valore restituito  
 `true` se HandleT rappresenta l'handle, in caso contrario, `false`.  
  
## Requisiti  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
