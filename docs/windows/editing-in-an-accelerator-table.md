# <a name="editing-in-an-accelerator-table"></a>Modifica in una tabella dei tasti di scelta rapida
### <a name="to-edit-in-an-accelerator-table"></a>Per eseguire una modifica in una tabella dei tasti di scelta rapida  
  
1.  Aprire la tabella di tasti di scelta rapida facendo doppio clic sull'icona in [visualizzazione risorse](../windows/resource-view-window.md).  
  
    > [!NOTE]
    >  Se il progetto non contiene gi√† un file RC, vedere la pagina relativa alla [creazione di un nuovo file script di risorsa](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Selezionare una voce nella tabella e fare clic per attivare la modifica sul posto.  
  
3.  Eseguire la selezione dalla casella combinata a discesa oppure digitare sul posto per apportare modifiche.  
  
    -   Per [ID](id-property.md), selezionarlo dall'elenco o digitarlo per apportare modifiche.  
  
    -   Per [modificatore](../windows/accelerator-modifier-property.md)selezionare dall'elenco.  
  
    -   Per [chiave](../windows/accelerator-key-property.md), selezionarlo dall'elenco o digitarlo per apportare modifiche.  
  
    -   Per [tipo](../windows/accelerator-type-property.md), selezionare ASCII o VIRTKEY dall'elenco.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [risorse nelle applicazioni Desktop](/dotnet/framework/resources/index) nel *Guida per gli sviluppatori di .NET Framework.*  
  
 **Requisiti**  
  
 Win32
