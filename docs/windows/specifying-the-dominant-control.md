# Specifying the Dominant Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Il primo controllo selezionato è il controllo dominante.  
  
### Per specificare il controllo dominante  
  
1.  Tenendo premuto **CTRL**, fare *prima* clic sul controllo che si desidera utilizzare per determinare la dimensione o la posizione di altri controlli.  
  
     **Nota** I quadratini di ridimensionamento del controllo dominante sono pieni, mentre quelli dei controlli subordinati sono vuoti.  Ogni ulteriore operazione di ridimensionamento o allineamento è basata sul controllo dominante.  
  
### Per modificare il controllo dominante  
  
1.  Cancellare la selezione corrente facendo clic all'esterno di tutti i controlli correntemente selezionati.  
  
2.  Ripetere la procedura precedente, selezionando per primo un diverso controllo.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
