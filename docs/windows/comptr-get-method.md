# Metodo ComPtr::Get
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Recupera un puntatore a un'interfaccia associata al ComPtr.  
  
## Sintassi  
  
```  
T* Get() const;  
```  
  
## Valore restituito  
 Puntatore all'interfaccia associata a ComPtr.  
  
## Requisiti  
 **Header:** client.h  
  
 **Namespace:** Microsoft::WRL
