# Customizing or Changing Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La [Tavolozza colori](../windows/colors-window-image-editor-for-icons.md) dell'editor di immagini inizialmente visualizza 16 colori standard.  Oltre ai colori visualizzati, è possibile creare colori personalizzati.  È quindi possibile [salvare e caricare una Tavolozza colori personalizzata](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
### Per modificare i colori nella Tavolozza colori  
  
1.  Dal menu **Immagine**, scegliere **Regola colori**.  
  
2.  Nel [finestra di dialogo Regola colori](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md), definire il colore digitando i valori RGB o HSL nelle caselle di testo appropriate o scegliere un colore nella casella **Visualizzazione delle sfumature**.  
  
3.  Impostare la luminosità spostando il dispositivo di scorrimento sulla barra **Luminosità**.  
  
4.  Molti colori personalizzati sono retinati.  Se si vuole che il colore tinta unita si avvicini al colore retinato, fare doppio clic sulla casella **Colore**.  
  
     Se successivamente si vuole che il colore retinato sposti il dispositivo di scorrimento sulla barra **Luminosità** o spostare il mirino nella casella **Visualizzazione delle sfumature** nuovamente per ripristinare il dithering.  
  
5.  Fare clic su **OK** per aggiungere il nuovo colore.
