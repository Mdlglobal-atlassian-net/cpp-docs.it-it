# Applicazioni desktop di Windows (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dopo aver acquisito familiarità con l'ambiente di sviluppo integrato di Visual Studio e con le applicazioni della riga di comando, verrà illustrato come creare applicazioni Windows.  Con Visual C\+\+ è possibile creare applicazioni Windows utilizzando molte tecnologie diverse, ad esempio l'[API Windows](https://msdn.microsoft.com/en-us/library/cc433218.aspx), denominata anche API Win32, e .NET Framework.  
  
 In questa sezione verranno create due semplici applicazioni Windows utilizzando API Win32 e .NET Framework.  Verrà inoltre creato un controllo Windows Form mediante .NET Framework e infine un semplice gioco attraverso DirectX.  
  
## Prerequisiti  
 In questi argomenti si presuppone la conoscenza delle nozioni di base del linguaggio C\+\+.
