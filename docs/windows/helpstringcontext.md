# helpstringcontext
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Specifica l'ID di un argomento della Guida in un file CHM o di .hlp.  
  
## Sintassi  
  
```  
  
      [ helpstringcontext(  
   contextID  
) ]  
```  
  
#### Parametri  
 `contextID`  
 Un identificatore di contesto della guida a 32 bit nel file della Guida.  
  
## Note  
 **helpstringcontext** L'attributo di C\+\+ ha la stessa funzionalit√† di  [helpstringcontext](http://msdn.microsoft.com/library/windows/desktop/aa366858) Attributi ODL.  
  
## Esempio  
  
```  
// cpp_attr_ref_helpstringcontext.cpp  
// compile with: /LD  
#include <unknwn.h>  
[module(name="MyLib")];  
  
[   object,   
   helpstring("help string"),   
   helpstringcontext(1),   
   uuid="11111111-1111-1111-1111-111111111111"  
]  
__interface IMyI   
{  
   HRESULT xx();  
};  
```
