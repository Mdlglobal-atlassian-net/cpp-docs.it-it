# <a name="inserting-a-space-between-buttons-on-a-toolbar"></a>Inserimento di uno spazio tra i pulsanti di una barra degli strumenti

In generale, per inserire uno spazio tra i pulsanti, semplicemente trascinarli uno da altro sulla barra degli strumenti. Per rimuovere gli spazi, trascinarli nei confronti tra loro.

### <a name="to-insert-a-space-before-a-button-that-is-not-followed-by-a-space"></a>Per inserire uno spazio prima di un pulsante che non è seguito da uno spazio

1. Trascinare il pulsante a destra o verso il basso fino a quando non si sovrappone il pulsante Avanti a metà.

### <a name="to-insert-a-space-before-a-button-which-is-followed-by-a-space-and-to-retain-that-trailing-space"></a>Per inserire uno spazio prima di un pulsante di cui è seguito da uno spazio e conservare lo spazio finale

1. Trascinare il pulsante fino a quando il bordo destro o inferiore tocca il pulsante Avanti o semplicemente si sovrappone a.

### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-close-up-that-following-space"></a>Per inserire uno spazio prima di un pulsante che è seguito da uno spazio e chiudere lo spazio seguente

1. Trascinare il pulsante a destra o verso il basso fino a quando non si sovrappone il pulsante Avanti a metà.

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).
