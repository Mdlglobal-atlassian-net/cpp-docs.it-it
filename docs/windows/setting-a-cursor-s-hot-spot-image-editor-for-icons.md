# Setting a Cursor&#39;s Hot Spot (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'area sensibile di un [cursore](../mfc/icons-and-cursors-image-resources-for-display-devices-image-editor-for-icons.md) rappresenta il punto a cui fa riferimento Windows per rilevare la posizione del cursore.  Per impostazione predefinita, l'area sensibile corrisponde all'angolo superiore sinistro del cursore, identificato dalle coordinate 0,0.  Le coordinate dell'area sensibile vengono visualizzate nella proprietà Hotspot della [finestra Proprietà](../Topic/Properties%20Window.md).  
  
### Per impostare l'area sensibile di un cursore  
  
1.  Fare clic sullo strumento **Imposta area sensibile** sulla [barra degli strumenti dell'editor di immagini](../mfc/toolbar-image-editor-for-icons.md).  
  
2.  Fare clic sul pixel che si desidera impostare come area sensibile del cursore.  
  
     Le coordinate dell'area sensibile vengono visualizzate nella proprietà **Hotspot** della finestra **Proprietà**.  
  
    > [!TIP]
    >  Il suggerimento viene visualizzato quando si passa il cursore sui pulsanti della barra degli strumenti.  che consentono di identificare facilmente la funzione di ciascun pulsante.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
