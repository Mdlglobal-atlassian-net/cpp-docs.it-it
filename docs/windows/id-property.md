# ID Property
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La proprietà ID fa riferimento alle singole voci della tabella dei tasti di scelta rapida nel codice del programma.  Questa proprietà è il valore del comando ricevuto dal programma quando l'utente preme il tasto di scelta rapida o una combinazione di tasti.  Affinché non vi sia differenza tra un tasto di scelta rapida e una voce di menu, assegnare a entrambi lo stesso ID, in modo che l'ID della tabella dei tasti di scelta rapida sia uguale all'ID della risorsa menu.  
  
## Requisiti  
 Win32
