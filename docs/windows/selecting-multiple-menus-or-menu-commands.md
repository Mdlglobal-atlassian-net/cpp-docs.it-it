# Selecting Multiple Menus or Menu Commands
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile selezionare più nomi di menu o comandi di menu per eseguire operazioni di massa, ad esempio l'eliminazione o la modifica delle proprietà.  
  
### Per selezionare più comandi di menu  
  
1.  Tenendo premuto il tasto CTRL, fare clic sui menu o sui comandi del sottomenu desiderato.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.*  
  
 **Requisiti**  
  
 Win32
