# <a name="attributes-by-group"></a>Attributi per gruppo
Gli attributi di C++ sono organizzati in gruppi funzionali seguenti.  
  
|Attributo|Descrizione|  
|---------------|-----------------|  
|[Attributi COM](../windows/com-attributes.md)|Inserire codice per il supporto di numerose aree dello sviluppo COM e sviluppo in .NET Framework common language runtime.|  
|[Attributi IDL](../windows/idl-attributes.md)|Consentire la modifica del file da un file di codice sorgente senza l'utilizzo di una procedura guidata e conoscenza della struttura e la sintassi del file IDL.|  
|[Attributi del consumer OLE DB](../windows/ole-db-consumer-attributes.md)|Inserire il codice, in base il [modelli Consumer OLE DB](../data/oledb/ole-db-consumer-templates-reference.md), per creare un lavoro OLE DB consumer che esegue le attivit√†, ad esempio tabelle di apertura, l'esecuzione di comandi e l'accesso ai dati.|  
|[Attributi del compilatore](../windows/compiler-attributes.md)|Fornito dal compilatore Visual C++.|  
  
 Per informazioni su come definire attributi personalizzati per l'applicazione .NET Framework, vedere [attributi definiti dall'utente](../windows/user-defined-attributes-cpp-component-extensions.md).
