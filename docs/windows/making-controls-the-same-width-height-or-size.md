# Making Controls the Same Width, Height, or Size
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile ridimensionare un gruppo di controlli in base alla dimensione del controllo dominante.  È inoltre possibile [ridimensionare un controllo in base alle dimensioni del testo della didascalia](../mfc/sizing-individual-controls.md).  
  
### Per impostare la stessa larghezza, altezza o dimensione per i controlli  
  
1.  [Selezionare i controlli](../mfc/selecting-multiple-controls.md) che si desidera ridimensionare.  
  
     Il primo controllo selezionato della serie è il controllo dominante.  La dimensione finale dei controlli del gruppo dipende dalla dimensione del controllo dominante.  Per ulteriori informazioni sulla selezione del controllo dominante, vedere [Impostazione del controllo dominante](../mfc/specifying-the-dominant-control.md).  
  
2.  Scegliere **Rendi uguali** dal menu **Formato**, quindi scegliere uno dei seguenti comandi:  
  
    -   **Entrambe le dimensioni**  
  
    -   **Altezza**  
  
    -   **Larghezza**  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).  
  
 Requisiti
