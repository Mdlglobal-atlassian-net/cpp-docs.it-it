# <a name="editing-accelerator-tables"></a>Modifica delle tabelle dei tasti di scelta rapida
In Visual C++, è possibile modificare una tabella di tasti di scelta rapida direttamente con la modifica sul posto nel **acceleratore** editor.  
  
 Le procedure seguenti si riferiscono all'uso delle pagine delle proprietà standard, tuttavia, la modifica sul posto e il metodo di pagina di proprietà hanno lo stesso risultato. Le modifiche apportate utilizzando le pagine delle proprietà o la modifica sul posto vengono applicate immediatamente nella tabella di tasti di scelta rapida.  
  
-   [Modificare una tabella dei tasti di scelta rapida](../windows/editing-in-an-accelerator-table.md)  
  
-   [Ricerca di una voce in una tabella di tasti di scelta rapida](../windows/finding-an-entry-in-an-accelerator-table.md)  
  
-   [Aggiunta di una voce a una tabella di tasti di scelta rapida](../windows/adding-an-entry-to-an-accelerator-table.md)  
  
-   [Eliminazione di una voce da una tabella di tasti di scelta rapida](../windows/deleting-an-entry-from-an-accelerator-table.md)  
  
-   [Lo spostamento o copia di una voce di tabella di tasti di scelta rapida](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)  
  
-   [Modifica delle proprietà di più tasti di scelta rapida](../windows/changing-the-properties-of-multiple-accelerator-keys.md)  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).  
  
## <a name="requirements"></a>Requisiti  
 Win32
