# Editing Accelerator Tables
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In Visual C\+\+  è possibile modificare una tabella dei tasti di scelta rapida direttamente con la modifica sul posto nell'editor tasti di scelta rapida.  
  
 Le procedure riportate di seguito fanno riferimento all'utilizzo delle pagine delle proprietà standard. Il metodo di modifica sul posto e quello della pagina delle proprietà producono tuttavia lo stesso risultato.  Le modifiche apportate mediante le pagine delle proprietà o la modifica sul posto si riflettono immediatamente sulla tabella dei tasti di scelta rapida.  
  
-   [Modifica di una tabella di tasti di scelta rapida](../windows/editing-in-an-accelerator-table.md)  
  
-   [Ricerca di una voce in una tabella di tasti di scelta rapida](../windows/finding-an-entry-in-an-accelerator-table.md)  
  
-   [Aggiunta di una voce a una tabella di tasti di scelta rapida](../windows/adding-an-entry-to-an-accelerator-table.md)  
  
-   [Eliminazione di una voce da una tabella di tasti di scelta rapida](../windows/deleting-an-entry-from-an-accelerator-table.md)  
  
-   [Spostamento o copia di una voce da una tabella di tasti di scelta rapida](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)  
  
-   [Modifica delle proprietà di più tasti di scelta rapida](../windows/changing-the-properties-of-multiple-accelerator-keys.md)  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
