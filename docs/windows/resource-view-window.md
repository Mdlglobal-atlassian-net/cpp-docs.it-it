# Resource View Window
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Non supportato nelle edizioni Express.  
  
 Visualizzazione delle risorse visualizza i file di risorse inclusi nei progetti.  Se si espande la cartella di primo livello \(ad esempio Progetto1.rc\) vengono visualizzati i tipi di risorse all'interno di quel file RC.  Se si espande ogni tipo di risorsa, vengono visualizzate le singole risorse di quel tipo.  
  
> [!NOTE]
>  Se il progetto non contiene già un file RC, vedere la pagina relativa alla [creazione di un nuovo file script di risorsa](../windows/how-to-create-a-resource-script-file.md).  
  
> [!TIP]
>  È possibile fare clic con il pulsante destro del mouse nella finestra Visualizzazione risorse per avviare un menu di scelta rapida dei comandi.  È anche possibile fare doppio clic sulla barra del titolo per ancorare o disancorare la finestra.  Facendo clic con il pulsante destro del mouse sulla barra del titolo vengono resi disponibili comandi aggiuntivi per controllare il comportamento della finestra.  Per altre informazioni, vedere[Gestione finestre](../Topic/Customizing%20window%20layouts%20in%20Visual%20Studio.md).  
  
#### Per aprire la finestra Visualizzazione risorse  
  
1.  Fare clic su **Visualizzazione risorse** nel menu **Visualizza**.  
  
     \-oppure\-  
  
2.  Premere **CTRL\+MAIUSC\+A**.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
