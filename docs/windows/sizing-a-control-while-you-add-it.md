# Sizing a Control While You Add It
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Per ridimensionare un controllo durante l'aggiunta  
  
1.  Selezionare un controllo nella [finestra Casella degli strumenti](../Topic/Toolbox.md).  
  
2.  Posizionare il cursore \(visualizzato come mirino\) nel punto in cui si desidera che si trovi l'angolo superiore sinistro del nuovo controllo all'interno della finestra di dialogo.  
  
3.  Tenere premuto il pulsante del mouse per ancorare l'angolo superiore sinistro del controllo alla finestra di dialogo, quindi trascinare il cursore verso destra e in basso fino a ottenere la dimensione desiderata per il controllo.  
  
    > [!NOTE]
    >  È possibile ancorare uno qualsiasi dei quattro angoli del controllo disegnato.  Nella procedura illustrata è stato utilizzato l'angolo superiore sinistro solo come esempio.  
  
4.  Rilasciare il pulsante del mouse.  Il controllo verrà posizionato nella finestra di dialogo con la dimensione specificata.  
  
    > [!TIP]
    >  Per ridimensionare il controllo dopo il trascinamento nella finestra di dialogo, spostare i quadratini di ridimensionamento sul bordo del controllo.  Per ulteriori informazioni, vedere [Ridimensionamento di singoli controlli](../mfc/sizing-individual-controls.md).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
