# <a name="dialog-editor-states-guides-and-grids"></a>Stati dell'editor finestre (guide e griglie)
È possibile disporre i controlli nelle finestre di dialogo con l'editor di finestre in uno dei tre stati:  
  
-   Con le guide e margini (impostazione predefinita)  
  
-   Con la griglia del layout di  
  
-   Senza le funzionalità di blocco o allineamento  
  
 Il [barra degli strumenti finestra di dialogo Editor](../windows/showing-or-hiding-the-dialog-editor-toolbar.md) contiene pulsanti che consentono di controllare lo stato. Per modificare lo stato, fare clic sull'icona appropriata. È inoltre possibile modificare gli stati tramite le **impostazioni Guide** comando il **formato** menu.  
  
-   [Creazione e impostazione di guide e margini](creating-and-setting-guides-and-margins.md)  
  
-   [Allineamento di controlli su una guida](../windows/aligning-controls-on-a-guide.md)  
  
-   [Disabilitazione delle guide](../windows/disabling-guides.md)  
  
-   [Modifica della griglia di layout](../windows/modifying-the-layout-grid.md)  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [risorse nelle applicazioni Desktop](/dotnet/framework/resources/index) nel *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [la creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione di risorse nelle applicazioni gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).
