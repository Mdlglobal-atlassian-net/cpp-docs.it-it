# Finestra di dialogo Inserisci controllo ActiveX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa finestra di dialogo consente di [inserire controlli ActiveX nella finestra di dialogo](../mfc/viewing-and-adding-activex-controls-to-a-dialog-box.md) utilizzando l'[editor finestre](../mfc/dialog-editor.md).  
  
 **Controllo ActiveX**  
 Consente di visualizzare un elenco di controlli ActiveX.  L'inserimento di un controllo da questa finestra di dialogo non genera una classe wrapper.  Se è necessaria una classe wrapper, crearne una utilizzando [Visualizzazione classi](http://msdn.microsoft.com/it-it/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Per ulteriori informazioni, vedere [Aggiunta di una classe](../ide/adding-a-class-visual-cpp.md).  Se un controllo ActiveX non viene visualizzato in questa finestra di dialogo, provare a installare il controllo seguendo le istruzioni del fornitore.  
  
 **Percorso**  
 Consente di visualizzare il file in cui si trova il controllo ActiveX.  
  
 È possibile inserire controlli nella finestra Casella degli strumenti per semplificare l'accesso.  Per ulteriori informazioni, vedere [Finestra di dialogo Personalizza Casella degli strumenti](http://msdn.microsoft.com/it-it/bd07835f-18a8-433e-bccc-7141f65263bb).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).

 Win32
