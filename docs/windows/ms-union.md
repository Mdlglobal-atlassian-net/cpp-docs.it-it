# <a name="msunion"></a>ms_union

Controlla l'allineamento di rappresentazione dei dati di rete di unioni nonencapsulated.

## <a name="syntax"></a>Sintassi

```cpp
[ms_union]
```

## <a name="remarks"></a>Note

Il **ms_union** attributi di C++ ha la stessa funzionalit√† come la [ms_union](/windows/desktop/Midl/ms-union-attrib) attributo MIDL.

## <a name="example"></a>Esempio

Il codice seguente viene illustrata la posizione di **ms_union**:

```cpp
// cpp_attr_ref_ms_union.cpp
// compile with: /LD
#include <unknwn.h>
[object, ms_union, uuid("00000000-0000-0000-0000-000000000001")]
__interface IFireTabCtrl {
   HRESULT DisplayString([in, string] char * p1);
};

[export, switch_type(short)] union _WILLIE_UNION_TYPE  {
   [case(24)]
      float fMays;
   [case(25)]
      double dMcCovey;
   [default]
      int x;
 };

[public] typedef _WILLIE_UNION_TYPE WILLIE_UNION_TYPE;

[module(name="ATLFIRELib")];
```
