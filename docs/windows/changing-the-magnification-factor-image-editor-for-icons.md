# <a name="changing-the-magnification-factor-image-editor-for-icons"></a>Modifica del fattore di ingrandimento (Editor di immagini per le icone)

Per impostazione predefinita, l'editor di immagini consente di visualizzare la vista nel riquadro sinistro alle dimensioni effettive e la visualizzazione nel riquadro di destra con le dimensioni effettive di 6 volte. Il fattore di ingrandimento (visualizzato nella barra di stato nella parte inferiore dell'area di lavoro) è il rapporto tra le dimensioni effettive dell'immagine e le dimensioni di visualizzazione. Il fattore di predefinito è 6 e l'intervallo va da 1 a 10.

### <a name="to-change-the-magnification-factor"></a>Per modificare del fattore di ingrandimento

1. Selezionare il **Editor di immagini** riquadro cui fattore di ingrandimento che si desidera modificare.

2. Nel [sulla barra degli strumenti Editor di immagini](../windows/toolbar-image-editor-for-icons.md), fare clic sulla freccia a destra del [strumento Ingrandisci](../windows/toolbar-image-editor-for-icons.md) e selezionare il fattore di ingrandimento dal sottomenu: **1 X**, **2 X**, **6 X**, o **8 X**.

   > [!NOTE]
   > Per selezionare un fattore di ingrandimento diversi da quelli elencati nel **Ingrandisci** dello strumento, usare i tasti di scelta rapida.

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).
