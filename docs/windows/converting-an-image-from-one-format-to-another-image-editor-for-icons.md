# Converting an Image From One Format to Another (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Oltre a poter aprire immagini GIF o JPEG nell'editor di immagini salvandole poi come bitmap,  è possibile eseguire il processo opposto, ovvero aprire un file bitmap salvandolo come file GIF o JPEG.  Le immagini utilizzate non devono necessariamente essere incluse in un progetto per poter essere modificate nell'ambiente di sviluppo \(vedere [Modifica autonoma di immagini](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md)\).  
  
### Per convertire un'immagine in un formato diverso  
  
1.  Aprire l'immagine nell'editor di immagini.  
  
2.  Scegliere **Salva *nomefile* con nome** dal menu **File**.  
  
3.  Nella casella **Nome file** della finestra di dialogo **Salva file con nome** digitare il nome di file e l'estensione corrispondente al formato desiderato.  
  
4.  Scegliere **Salva**.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
