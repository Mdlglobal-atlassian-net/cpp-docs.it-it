# Displaying or Hiding the Pixel Grid (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In tutti i riquadri dell'editor di immagini con fattore di ingrandimento uguale o superiore a 4 è possibile visualizzare una griglia in pixel che delimita i singoli pixel dell'immagine.  
  
> [!TIP]
>  Il suggerimento viene visualizzato quando si passa il cursore sui pulsanti della barra degli strumenti.  che consentono di identificare facilmente la funzione di ciascun pulsante.  
  
### Per attivare o disattivare la visualizzazione della griglia in pixel  
  
1.  Scegliere **Impostazioni griglia** dal menu **Immagine**.  
  
2.  Selezionare la casella di controllo **Griglia in pixel** per visualizzare la griglia oppure deselezionare la casella per nasconderla.
