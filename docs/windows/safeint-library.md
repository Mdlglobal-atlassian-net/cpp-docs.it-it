# Libreria SafeInt
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le guide della libreria SafeInt impediscono agli overflow di Integer che potrebbero verificarsi quando l'applicazione esegue operazioni matematiche.

|Sezione|Descrizione|  
|-------------|-----------------|
