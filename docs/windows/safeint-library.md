# <a name="safeint-library"></a>Libreria SafeInt

Il **SafeInt** library consente di evitare gli overflow di interi che potrebbero verificarsi quando l'applicazione esegue operazioni matematiche. La versione più recente di questa libreria è disponibile all'indirizzo [ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt).

|Sezione|Descrizione|
|-------------|-----------------|

|[Funzioni SafeInt](../windows/safeint-functions.md)|Funzioni che possono essere usate senza creare una **SafeInt** oggetto.|
|[Classe SafeIntException](../windows/safeintexception-class.md)|Una classe di eccezioni correlate per il **SafeInt** classe.|
