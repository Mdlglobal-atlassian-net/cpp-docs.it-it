# Attributed Programming Concepts
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ include varie materiale per l'utilizzo degli attributi nei programmi.  Gli attributi, una nuova funzionalità di Visual C\+\+, sono progettati per semplificare la programmazione COM e lo sviluppo di Common Language Runtime di .NET Framework.  Quando si includono gli attributi nei file di origine, funzionamento del compilatore con le librerie a collegamento dinamico del \(DLL\) provider per inserire il codice o modificare il codice nei file oggetto compilati.  Esistono attributi da consentire la creazione di file IDL, interfacce, librerie dei tipi e di altri elementi COM.  Nell'\(IDE\) ambiente di sviluppo integrato, gli attributi sono supportati dalle procedure guidate e la Finestra Proprietà.  
  
 Mentre gli attributi eliminati parte della codifica dettagliata necessaria per scrivere gli oggetti COM, è necessario disporre di uno sfondo in [Nozioni fondamentali su COM](http://msdn.microsoft.com/library/windows/desktop/ms694363) la scelta migliore utilizzo.
