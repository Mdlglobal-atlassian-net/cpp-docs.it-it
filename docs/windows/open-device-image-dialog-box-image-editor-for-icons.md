# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>Aprire &lt;dispositivo&gt; immagine nella finestra di dialogo (C++) (Editor di immagini per le icone)

Usare questa finestra di dialogo per aprire immagini per i dispositivi nei progetti C++. Elenca immagini per i dispositivi esistenti nella risorsa corrente (immagini che fanno parte della risorsa corrente).

### <a name="current-images"></a>Immagini correnti

Elenca le immagini incluse nella risorsa. Selezionare il tipo di immagine che si desidera aprire.

## <a name="requirements"></a>Requisiti

nessuno
