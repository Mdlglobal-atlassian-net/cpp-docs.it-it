# Adding, Editing, or Deleting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mediante l'editor finestre è possibile aggiungere, ridimensionare, modificare ed eliminare i controlli delle finestre di dialogo.  È anche possibile modificare le proprietà di un controllo, ad esempio l'ID o la visualizzazione iniziale in fase di esecuzione.  
  
 Il menu di scelta rapida disponibile nell'editor finestre consente di aggiungere rapidamente controlli ActiveX registrati a una finestra di dialogo, nonché di aggiungere controlli ActiveX alla Casella degli strumenti per l'accesso rapido. Per ulteriori informazioni, vedere la [finestra di dialogo Personalizza Casella degli strumenti](http://msdn.microsoft.com/it-it/bd07835f-18a8-433e-bccc-7141f65263bb).

-   [Ridimensionamento di un controllo durante l'aggiunta](../mfc/sizing-a-control-while-you-add-it.md)  
  
-   [Aggiunta di più controlli](../mfc/adding-multiple-controls.md)  
  
-   [Aggiunta di un controllo ActiveX a una finestra di dialogo](../mfc/viewing-and-adding-activex-controls-to-a-dialog-box.md)  
  
-   [Modifica delle proprietà di uno o più controlli](../mfc/editing-control-properties.md)  
  
-   [Eliminazione di un controllo](../mfc/deleting-a-control.md)  
  
-   [Mostrare o nascondere la finestra Casella degli strumenti](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)
