# <a name="menu-editor"></a>Editor dei menu

I menu consentono di disporre i comandi in modo logico e facile da trovare. Con il **Menu** editor, è possibile creare e modificare i menu lavorando direttamente con una barra dei menu molto simile a quello dell'applicazione finita.

Per i progetti Visual C++, è possibile usare la **Menu** editor per:

- [Creare i menu](../windows/creating-a-menu.md)

- [Inserire un nuovo menu tra i menu esistenti](../windows/inserting-a-new-menu-between-existing-menus.md)

- [Creare un comando di menu](../windows/adding-commands-to-a-menu.md)

- [Creare un menu di sottomenu](../windows/creating-a-submenu.md)

- [Selezionare menu e comandi di menu](../windows/selecting-multiple-menus-or-menu-commands.md)

- [Spostare e copiare menu e comandi di menu](../windows/moving-and-copying-menus-and-menu-commands.md)

- [Creare menu a comparsa](../windows/creating-pop-up-menus.md)

- [Connettere un menu a comparsa all'applicazione](../windows/connecting-a-pop-up-menu-to-your-application.md)

- [Visualizzare una risorsa di menu come menu a comparsa](../windows/viewing-a-menu-as-a-pop-up-menu.md)

- [Associare un comando di menu a un tasto di scelta rapida](../windows/associating-a-menu-command-with-an-accelerator-key.md)

- [Associare un comando di menu a un testo della barra di stato](../windows/associating-menu-commands-with-status-bar-text-in-mfc-applications.md)

- [Assegnazione di tasti di scelta ai comandi di menu](../windows/assigning-access-keys-to-menu-commands.md)

- [Modificare le proprietà dei comandi di menu](../windows/menu-command-properties.md)

   > [!TIP]
   > Quando si usa la **Menu** editor, in molti casi, è possibile fare clic sul pulsante destro del mouse per visualizzare un menu di scelta rapida dei comandi usati frequentemente. I comandi disponibili dipendono dalla destinazione a cui fa riferimento il puntatore.

   > [!NOTE]
   > Per i programmi di classe della libreria MFC (Microsoft Foundation) e ATL, è possibile usare **creazioni guidate codice** per associare i comandi di menu al codice. Per altre informazioni, vedere [Aggiunta di un evento](../ide/adding-an-event-visual-cpp.md).

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).

## <a name="requirements"></a>Requisiti

Win32
