### <a name="to-create-a-submenu"></a>Per creare un sottomenu  
  
1.  Fare clic sul comando di menu per il quale si vuole creare un sottomenu.  
  
2.  Nella casella **Nuovo elemento** visualizzata a destra, digitare il nome del nuovo comando di menu. Questo nuovo comando verr√† visualizzato come prima voce del sottomenu.  
  
3.  Aggiungere altri comandi di menu alla voce del sottomenu.
