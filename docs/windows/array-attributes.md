# Array Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

I seguenti attributi applicati a matrici o **SAFEARRAY**s.  
  
|Attributo|Descrizione|  
|---------------|-----------------|  
|[library\_block](../windows/library-block.md)|Inserisce un costrutto nel blocco di libreria del file IDL.|  
|[satype](../windows/satype.md)|Specifica il tipo di dati della struttura di SAFEARRAY.|  
|[string](../windows/string-cpp.md)|indica che l'unidimensionale `char`,  `wchar_t`,  **byte** \(o equivalente\) la matrice o un puntatore a una matrice deve essere considerato come stringa.|
