# Evening the Spacing Between Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'editor finestre consente di ripartire i controlli uniformemente tra i controlli esterni selezionati.  
  
### Per ripartire uniformemente i controlli  
  
1.  Selezionare i controlli che si desidera riordinare.  
  
2.  Scegliere **Ripartisci uniformemente** dal menu **Formato**, quindi scegliere uno degli allineamenti relativi alla spaziatura riportati di seguito.  
  
    -   **Orizzontalmente**: consente di ripartire uniformemente i controlli tra i controlli selezionati all'estremità sinistra e all'estremità destra.  
  
    -   **Verticalmente**: consente di ripartire uniformemente i controlli tra i controlli selezionati nella posizione più alta e più bassa.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
