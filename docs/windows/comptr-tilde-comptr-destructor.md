# <a name="comptrcomptr-destructor"></a>Distruttore ComPtr::~ComPtr
Deinizializza un'istanza di **ComPtr**.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** client.h  
  
 **Spazio dei nomi:** Microsoft::WRL
