# <a name="adding-an-image-for-a-different-display-device-image-editor-for-icons"></a>Aggiunta di un'immagine per un diverso dispositivo di visualizzazione (Editor di immagini per le icone)

### <a name="to-add-an-image-for-a-different-display-device"></a>Per aggiungere un'immagine per un diverso dispositivo di visualizzazione

1. Nel **immagine** menu, fare clic su **nuova immagine del dispositivo** (o fare doppio clic nel **Editor di immagini** riquadro e scegliere **nuova immagine per dispositivo** dal menu di scelta rapida).

2. Selezionare il tipo di immagine da aggiungere. È anche possibile fare clic su **Custom** per creare un'icona con dimensioni non sono disponibile nell'elenco predefinito.

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).
