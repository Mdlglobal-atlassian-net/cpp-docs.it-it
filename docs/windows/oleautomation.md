# oleautomation
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Indica che un'interfaccia è compatibile con l'automazione.  
  
## Sintassi  
  
```  
  
[oleautomation]  
  
```  
  
## Note  
 **oleautomation** L'attributo di C\+\+ ha la stessa funzionalità di  [oleautomation](http://msdn.microsoft.com/library/windows/desktop/aa367129) Attributo MIDL.  
  
## Esempio  
 vedere gli esempi per [valore predefinito](../windows/defaultvalue.md) e  [non estensibile](../windows/nonextensible.md) per un esempio di utilizzo  **oleautomation**.
