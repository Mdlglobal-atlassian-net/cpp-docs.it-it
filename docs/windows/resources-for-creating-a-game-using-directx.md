# Risorse per la creazione di un gioco con DirectX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Per creare giochi desktop veramente coinvolgenti, utilizzare insieme Visual C\+\+ e DirectX.  
  
## Risorse  
 Visual C\+\+ è incluso in tutte le edizioni di Visual Studio.  Per informazioni su Visual Studio, vedere il sito Web di [Visual Studio](http://www.visualstudio.com/).  
  
 Per [!INCLUDE[win81](../misc/includes/win81_md.md)], DirectX è incluso in Windows SDK.  Non è necessario scaricare altro materiale.  Per ulteriori informazioni, vedere l'articolo relativo a [DirectX SDK](http://msdn.microsoft.com/library/windows/desktop/ee663275.aspx).  
  
 Per creare applicazioni DirectX per Windows 7, Windows Vista o Windows XP, è possibile scaricare la versione autonoma di [DirectX SDK](http://www.microsoft.com/download/details.aspx?displaylang=en&id=6812).  
  
 Nell'articolo relativo alla [grafica e ai giochi DirectX](http://msdn.microsoft.com/library/windows/desktop/ee663274\(v=vs.85\).aspx) sono contenuti esempi e informazioni per apprendere le nozioni fondamentali della programmazione DirectX.  
  
 In [Applicazioni Windows Win32](../windows/windows-desktop-applications-cpp.md) sono contenuti articoli che descrivono come creare un'applicazione desktop Windows di base.  
  
 In [Sviluppo di giochi](http://msdn.microsoft.com/library/windows/apps/hh452744.aspx) viene illustrato lo sviluppo dei giochi per [!INCLUDE[win8_appname_long](../build/includes/win8_appname_long_md.md)], anche se molti dei concetti sono applicabili anche ai giochi desktop.
