# <a name="creating-new-toolbars"></a>Creazione di nuove barre degli strumenti
### <a name="to-create-a-new-toolbar"></a>Per creare una nuova barra degli strumenti  
  
1.  In **risorse** consente di visualizzare, fare doppio clic sul file RC, quindi scegliere **Aggiungi risorsa** dal menu di scelta rapida. (Se si dispone di una barra degli strumenti esistente nel file RC, è possibile semplicemente fare doppio clic il **barra degli strumenti** cartella e selezionare **barra degli strumenti delle Insert** dal menu di scelta rapida.)  
  
     **Note** Se il progetto non contiene già un file RC, vedere la pagina relativa alla [creazione di un nuovo file script di risorsa](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Nel **Aggiungi risorsa** nella finestra di dialogo **barra degli strumenti** nel **tipo di risorsa** elenco, quindi fare clic su **New**.  
  
     Se un segno più (+) viene visualizzato accanto al **barra degli strumenti** il tipo di risorsa, significa che i modelli della barra degli strumenti sono disponibili. Fare clic sul segno più per espandere l'elenco dei modelli, selezionare un modello e fare clic su **New**.  
  
     \- oppure -  
  
3.  [Convertire una bitmap esistente in una barra degli strumenti](../windows/converting-bitmaps-to-toolbars.md).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [risorse nelle applicazioni Desktop](/dotnet/framework/resources/index) nel *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [la creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione di risorse nelle applicazioni gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).  
  
 Requisiti  
  
 MFC o ATL
