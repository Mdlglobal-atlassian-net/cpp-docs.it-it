# Metodo RuntimeClass::InternalAddRef
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Incrementa il conteggio dei riferimenti all'oggetto corrente di RuntimeClass.  
  
## Sintassi  
  
```  
ULONG InternalAddRef();  
```  
  
## Valore restituito  
 Conteggio dei riferimenti risultante.  
  
## Requisiti  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL
