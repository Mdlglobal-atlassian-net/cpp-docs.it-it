# <a name="factorycachefactory-data-member"></a>Membro dati FactoryCache::factory
Supporta l'infrastruttura di libreria di modelli di Windows Runtime C++ e non deve essere utilizzato direttamente dal codice.  
  
## <a name="syntax"></a>Sintassi  
  
```  
IUnknown* factory;  
```  
  
## <a name="remarks"></a>Note  
 Punta a una class factory COM o Windows Runtime.  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** Implements  
  
 **Namespace:** Microsoft::WRL::Details
