# Cropping or Extending an Entire Image (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Nella procedura riportata di seguito viene illustrato come ritagliare o estendere un'intera immagine.  
  
### Per ritagliare o estendere un'intera immagine  
  
1.  Selezionare l'intera immagine.  
  
     Se è correntemente selezionata una parte dell'immagine e si desidera selezionare l'intera immagine, fare clic in un punto qualsiasi dell'immagine al di fuori del bordo di selezione corrente.  
  
2.  Trascinare uno dei quadratini di ridimensionamento finché l'immagine non avrà assunto le dimensioni desiderate.  
  
 Quando si ridimensiona un'immagine spostando un quadratino di ridimensionamento, l'editor di immagini in genere ritaglia o allarga l'immagine.  Se invece si tiene premuto MAIUSC mentre viene spostato un quadratino di ridimensionamento, l'immagine viene [ridotta o ingrandita](../mfc/shrinking-or-stretching-an-entire-image-image-editor-for-icons.md).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
