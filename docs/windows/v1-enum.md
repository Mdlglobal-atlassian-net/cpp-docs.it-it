# <a name="v1enum"></a>v1_enum

Indica che il tipo enumerato specificato verrà trasmesso un'entità 32-bit anziché il valore predefinito di 16 bit.

## <a name="syntax"></a>Sintassi

```cpp
[v1_enum]
```

## <a name="remarks"></a>Note

Il **v1_enum** attributi di C++ ha la stessa funzionalità come la [v1_enum](http://msdn.microsoft.com/library/windows/desktop/aa367303) attributo MIDL.

## <a name="example"></a>Esempio

Il codice seguente illustra un uso **v1_enum**:

```cpp
// cpp_attr_ref_v1_enum.cpp
// compile with: /LD
[module(name="MyLibrary")];

[export, v1_enum]
enum eList {
   e1 = 1,
   e2 = 2
};
```
