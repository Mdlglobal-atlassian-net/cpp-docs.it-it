# <a name="shrinking-or-stretching-part-of-an-image-image-editor-for-icons"></a>Riduzione o ingrandimento di una parte dell'immagine (Editor di immagini per le icone)
### <a name="to-shrink-or-stretch-part-of-an-image"></a>Per ridurre o ingrandire parte di un'immagine  
  
1.  Selezionare la parte dell'immagine da ridimensionare. Per ulteriori informazioni, vedere [selezione di un'Area dell'immagine](../windows/selecting-an-area-of-an-image-image-editor-for-icons.md).  
  
2.  Trascinare uno dei quadratini di ridimensionamento fino a raggiungere le dimensioni desiderate.
