# <a name="flipping-an-image-image-editor-for-icons"></a>Capovolgimento di un'immagine (Editor di immagini per le icone)

È possibile capovolgere o ruotare un'immagine da:

- Creare un'immagine speculare dell'originale.

- È possibile capovolgere l'immagine.

- Ruotare di 90 gradi l'immagine a destra alla volta.

### <a name="to-flip-the-image-horizontally-mirror-image"></a>Capovolgere orizzontalmente l'immagine (immagine mirror)

1. Dal **immagine** menu, scegliere **Capovolgi orizzontalmente**.

### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Capovolgere verticalmente l'immagine (è possibile capovolgere)

1. Dal **immagine** menu, scegliere **Capovolgi verticalmente**.

### <a name="to-rotate-the-image-90-degrees"></a>Per ruotare l'immagine di 90 gradi

1. Dal **immagine** menu, scegliere **ruota di 90 gradi**.

   > [!NOTE]
   > È anche possibile usare la [tasti di scelta rapida (collegamento)](../windows/accelerator-keys-image-editor-for-icons.md) per tali comandi o accedere ai comandi dal menu di scelta rapida (fare clic all'esterno dell'immagine nell'editor di immagini).

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*. Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [creazione di file di risorse per le app Desktop](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Per informazioni sulla globalizzazione e localizzazione delle risorse nelle App gestite, vedere [globalizzazione e localizzazione di applicazioni .NET Framework](/dotnet/standard/globalization-localization/index).
