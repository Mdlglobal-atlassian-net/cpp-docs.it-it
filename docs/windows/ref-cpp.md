# <a name="ref-c"></a>ref (C++)
Identifica un puntatore di riferimento.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
[ref]  
  
```  
  
## <a name="remarks"></a>Note  
 Il `ref` attributo C++ ha la stessa funzionalit√† come il [ref](http://msdn.microsoft.com/library/windows/desktop/aa367153) attributo MIDL.  
  
## <a name="example"></a>Esempio  
 Il codice seguente viene illustrato come utilizzare il `ref` attributo:  
  
```  
// cpp_attr_ref_ref.cpp  
// compile with: /LD  
#include <windows.h>   
[module(name="ATLFIRELib")];  
[dispinterface, uuid("00000000-0000-0000-0000-000000000001")]  
__interface IFireTabCtrl  
{  
   [id(1), unique] char * GetFirstName([in, ref] char * pszFullName );   
};  
```
