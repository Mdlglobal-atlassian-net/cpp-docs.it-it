# <a name="activationfactoryrelease-method"></a>Metodo ActivationFactory::Release
Decrementa il conteggio dei riferimenti dell'oggetto corrente **ActivationFactory** oggetto.

```cpp

   ULONG,  
   Release  
)();  
```
