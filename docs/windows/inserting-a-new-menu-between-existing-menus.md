# Inserting a New Menu Between Existing Menus
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Per inserire un nuovo menu tra menu esistenti  
  
1.  Selezionare un nome di menu esistente e premere il tasto **INS**.  La casella **Nuovo elemento** viene inserita prima dell'elemento selezionato.  
  
     \-oppure\-  
  
2.  Fare clic con il pulsante destro del mouse sulla barra dei menu e scegliere **Inserisci nuovo** dal menu di scelta rapida.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.*  
  
 Requisiti  
  
 Win32
