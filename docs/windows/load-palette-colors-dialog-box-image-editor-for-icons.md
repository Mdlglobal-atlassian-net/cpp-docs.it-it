# <a name="load-palette-colors-dialog-box-image-editor-for-icons"></a>Finestra di dialogo Carica tavolozza (Editor di immagini per le icone)
Utilizzare questa finestra di dialogo per caricare le tavolozze di colori speciali da utilizzare nel progetto. Per ulteriori informazioni, vedere [salvataggio e caricamento di tavolozze di colori diversi](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
 **Cerca in**  
 Specifica il percorso in cui si desidera individuare un file o cartella. Fare clic sulla freccia per selezionare un altro percorso o fare clic sull'icona della cartella sulla barra degli strumenti per spostare i livelli.  
  
 **Nome del file**  
 Fornisce uno spazio in cui digitare il nome del file che si desidera aprire. Per trovare rapidamente un file che aperto in precedenza, fare clic sul nome di file nell'elenco a discesa, se disponibile.  
  
 Se si sta cercando un file, è possibile utilizzare un asterisco (*) come carattere jolly. Ad esempio, è possibile digitare \*.\* per visualizzare un elenco di tutti i file. È anche possibile digitare il percorso completo di un file, ad esempio, C:\My c:\Documenti\MiaTavolozzaColori.PAL o \\\NetworkServer\MyFolder\MyColorPalette.pal.  
  
 **File di tipo**  
 Elenca i tipi di file da visualizzare. Tavolozza (* PAL) è il tipo di file predefinito per tavolozze dei colori.
