# Finestra di dialogo Carica tavolozza (Editor di immagini per le icone)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa finestra di dialogo consente di caricare tavolozze di colori speciali da utilizzare nel progetto.  Per ulteriori informazioni, vedere [Salvataggio e caricamento di tavolozze di colori diversi](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
 **Cerca in**  
 Consente di specificare il percorso in cui si desidera cercare un file o una cartella.  Fare clic sulla freccia per selezionare un diverso percorso oppure fare clic sull'icona a forma di cartella riportata sulla barra degli strumenti per spostarsi ai livelli superiori.  
  
 **Nome file**  
 Consente di digitare il nome del file che si desidera aprire.  Per trovare rapidamente un file aperto in precedenza, selezionare il nome del file dall'elenco a discesa, se disponibile.  
  
 Per effettuare la ricerca di un file è possibile utilizzare degli asterischi \(\*\) come caratteri jolly.  Si può ad esempio digitare \*.\* per visualizzare un elenco di tutti i file.  È inoltre possibile digitare il percorso completo di un file, ad esempio C:\\Documenti\\MiaTavolozzaColori.pal oppure \\\\ServerRete\\MiaCartella\\MiaTavolozzaColori.pal.  
  
 **Tipo file**  
 Vengono elencati i tipi di file da visualizzare.  Il tipo di file predefinito per le tavolozze di colori è Tavolozza \(\*.pal\).  
  
## Requisiti
