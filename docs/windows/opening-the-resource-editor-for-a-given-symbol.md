# <a name="opening-the-resource-editor-for-a-given-symbol-c"></a>Apertura dell'Editor di risorse per un determinato simbolo (C++)

Quando si visualizzano i simboli nel [finestra di dialogo Simboli risorsa](../windows/resource-symbols-dialog-box.md), Ã¨ possibile ottenere altre informazioni sull'utilizzo di un determinato simbolo. Il **Visualizza uso** pulsante offre un modo rapido per ottenere queste informazioni.

### <a name="to-move-to-the-resource-editor-where-a-symbol-is-being-used"></a>Per spostarsi all'editor di risorse in cui viene usato un simbolo

1. Selezionare un simbolo nel **nome** finestra di **simboli risorsa** nella finestra di dialogo.

2. Nel **Used By** , selezionare il tipo di risorsa desiderato.

3. Scegliere il **Visualizza uso** pulsante.

   La risorsa viene visualizzata nella finestra dell'editor appropriato.

Per informazioni sull'aggiunta di risorse a progetti gestiti, vedi [risorse nelle App Desktop](/dotnet/framework/resources/index) nel *manuale dello sviluppatore di .NET Framework*.
