# Opening the Resource Editor for a Given Symbol
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Quando si visualizzano i simboli nella [finestra di dialogo Simboli risorsa](../windows/resource-symbols-dialog-box.md), Ã¨ possibile ottenere altre informazioni sull'utilizzo di un determinato simbolo.  Il pulsante **Visualizza uso** fornisce un modo rapido per ottenere queste informazioni.  
  
### Per spostarsi all'editor di risorse in cui viene usato un simbolo  
  
1.  Selezionare un simbolo nella casella **Nome** della finestra di dialogo **Simboli risorsa**.  
  
2.  Nella casella **Usato da** selezionare il tipo di risorsa desiderato.  
  
3.  Fare clic sul pulsante **Visualizza uso**.  
  
     La risorsa viene visualizzata nella finestra dell'editor appropriato.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.*  
  
 Requisiti  
  
 Win32
