# <a name="opening-the-resource-editor-for-a-given-symbol"></a>Apertura dell'editor di risorse per un determinato simbolo
Quando si visualizzano i simboli nel [la finestra di dialogo Simboli risorsa](../windows/resource-symbols-dialog-box.md), Ã¨ possibile ottenere ulteriori informazioni sull'utilizzo di un simbolo specifico. Il **Visualizza uso** pulsante fornisce un modo rapido per ottenere queste informazioni.  
  
### <a name="to-move-to-the-resource-editor-where-a-symbol-is-being-used"></a>Per spostarsi all'editor di risorse in cui viene usato un simbolo  
  
1.  Selezionare un simbolo nel **nome** casella della finestra di **simboli risorsa** la finestra di dialogo.  
  
2.  Nel **utilizzato da** , selezionare il tipo di risorsa desiderato.  
  
3.  Fare clic su di **Visualizza uso** pulsante.  
  
     La risorsa viene visualizzata nella finestra dell'editor appropriato.  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [risorse nelle applicazioni Desktop](/dotnet/framework/resources/index) nel *Guida per gli sviluppatori di .NET Framework.*
