# <a name="pragma"></a>pragma
Genera la stringa specificata nel file IDL generato senza l'utilizzo delle virgolette. .  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      [ pragma(  
   pragma_statement  
) ];  
```  
  
#### <a name="parameters"></a>Parametri  
 *pragma_statement*  
 Il pragma che si desidera analizzare il file IDL generato.  
  
## <a name="remarks"></a>Note  
 Il **pragma** attributo C++ ha la stessa funzionalit√† come il [pragma](http://msdn.microsoft.com/library/windows/desktop/aa367143) attributo MIDL.  
  
## <a name="example"></a>Esempio  
  
```  
// cpp_attr_ref_pragma.cpp  
// compile with: /LD  
#include "unknwn.h"  
[module(name="MyLib")];  
[pragma(pack(4))];  
  
[dispinterface, uuid("00000000-0000-0000-0000-000000000001")]  
__interface A  
{  
   [id(1)] HRESULT MyMethod ([in, satype("BSTR")] SAFEARRAY **p);  
};  
```
