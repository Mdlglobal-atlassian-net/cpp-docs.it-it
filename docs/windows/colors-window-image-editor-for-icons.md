# Finestra Colori (Editor di immagini per le icone)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Finestra colori è costituita da due parti:  
  
1.  La Tavolozza colori, ovvero una matrice di campioni di colore che rappresentano i colori che è possibile usare.  È possibile fare clic sugli esempi per selezionare i colori di primo piano e sfondo quando si usano gli strumenti di grafica.  
  
2.  L'indicatore colori, che mostra i colori di sfondo e primo piano e i selettori per il colore dello schermo e l'inversione dei colori.  
  
     ![Finestra Colori](../windows/media/vccolorswindow.gif "vcColorsWindow")  
Finestra Colori  
  
> [!NOTE]
>  Gli strumenti **Colore dello schermo** e **Inversione di colore**sono disponibili solo per icone e cursori.  
  
 È possibile usare la finestra Colori con la [barra degli strumenti Editor di immagini](../mfc/toolbar-image-editor-for-icons.md).  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
