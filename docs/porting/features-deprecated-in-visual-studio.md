# <a name="c-features-deprecated-in-visual-studio-2019-preview"></a>Funzionalità C++ deprecate in Visual Studio 2019 Preview

## <a name="support-for-windows-xp-development"></a>Supporto per lo sviluppo di Windows XP

Visual Studio 2019 non include il supporto per la creazione di codice per Windows XP con il set di strumenti v142. Il supporto per lo sviluppo di Windows XP con il set di strumenti v141 incluso in Visual Studio 2017 è disponibile come componente facoltativo del programma di installazione di Visual Studio. Per informazioni su come installare il set di strumenti v141 per la piattaforma Windows XP, vedere [Configurazione di programmi per Windows XP](../build/configuring-programs-for-windows-xp.md).
