# Creazione di una procedura guidata personalizzata
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La Creazione guidata personalizzata di Visual C\+\+ è lo strumento da utilizzare nei casi in cui è necessario creare una nuova procedura di creazione guidata personalizzata.  Il modo più semplice per creare una procedura di creazione guidata personalizzata consiste nell'utilizzare la [Creazione guidata personalizzata](../ide/custom-wizard.md).  
  
> [!TIP]
>  Il modello di procedura guidata personalizzata non sono disponibili nelle edizioni express.  
  
### Per creare una procedura guidata tramite la Creazione guidata personalizzata  
  
1.  Attenersi alla procedura illustrata nell'argomento della Guida [Creazione di un progetto con una creazione guidata applicazione Visual C\+\+](../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  Nella finestra di dialogo **Nuovo progetto**, selezionare l'icona **Creazione guidata personalizzata** nel riquadro Modelli **Visual C\+\+ &#124; Generale** per aprire la creazione guidata personalizzata.  
  
3.  Definire le [impostazioni dell'applicazione](../ide/application-settings-custom-wizard.md) tramite la Creazione guidata personalizzata.  
  
4.  Scegliere **Fine** per uscire dalla procedura guidata e aprire il nuovo progetto nell'ambiente di sviluppo.  
  
 Una volta creato il progetto, è possibile visualizzare i file generati in **Esplora soluzioni**.  Per ulteriori informazioni sui file creati per la Creazione guidata personalizzata, vedere [File creati per la procedura guidata](../ide/files-created-for-your-wizard.md).
