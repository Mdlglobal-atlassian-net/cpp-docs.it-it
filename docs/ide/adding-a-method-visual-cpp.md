# Aggiunta di un metodo (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile utilizzare l'[Aggiunta guidata metodo](../ide/add-method-wizard.md) per aggiungere un metodo a un'interfaccia nel progetto.  Se il progetto contiene una classe associata all'interfaccia, anche la classe verrà modificata.  
  
### Per aggiungere un metodo all'oggetto  
  
1.  In Visualizzazione classi espandere il nodo del progetto per visualizzare l'interfaccia cui si desidera aggiungere il metodo.  
  
    > [!NOTE]
    >  È inoltre possibile aggiungere metodi a interfacce dispatch, le quali, se il progetto non dispone di attributi, si trovano sotto al nodo della libreria.  
  
2.  Fare clic con il pulsante destro del mouse sul nome dell'interfaccia.  
  
3.  Dal menu di scelta rapida scegliere **Aggiungi**, quindi **Aggiungi metodo**.  
  
4.  Nell'Aggiunta guidata metodo fornire le informazioni per creare il metodo.  
  
5.  Specificare le eventuali impostazioni IDL \(interface definition language\) relative al metodo nella schermata [Attributi IDL](../ide/idl-attributes-add-method-wizard.md) della procedura guidata.  
  
6.  Scegliere **Fine** per aggiungere il metodo.
