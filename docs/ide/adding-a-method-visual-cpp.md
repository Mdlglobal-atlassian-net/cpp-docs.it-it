# <a name="adding-a-method--visual-c"></a>Aggiunta di un metodo (Visual C++)
È possibile utilizzare il [Aggiunta guidata metodo](../ide/add-method-wizard.md) per aggiungere un metodo su un'interfaccia nel progetto. Se il progetto contiene una classe associata all'interfaccia, la procedura guidata Modifica troppo, la classe.  
  
### <a name="to-add-a-method-to-your-object"></a>Per aggiungere un metodo all'oggetto  
  
1.  In visualizzazione classi, espandere il nodo di progetto per visualizzare l'interfaccia a cui si desidera aggiungere il metodo.  
  
    > [!NOTE]
    >  È inoltre possibile aggiungere metodi dispatch, che, a meno che il progetto con attribuito, si trovano sotto il nodo della libreria.  
  
2.  Fare clic sul nome dell'interfaccia.  
  
3.  Menu di scelta rapida, fare clic su **Aggiungi**, quindi fare clic su **Aggiungi metodo**.  
  
4.  In aggiunta guidata metodo fornire le informazioni per creare il metodo.  
  
5.  Specificare le impostazioni per questo metodo in interface definition language di [attributi IDL](../ide/idl-attributes-add-method-wizard.md) pagina della procedura guidata.  
  
6.  Fare clic su **fine** per aggiungere il metodo.
