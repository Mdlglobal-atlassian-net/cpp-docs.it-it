# <a name="adding-a-method--visual-c"></a>Aggiunta di un metodo (Visual C++)
È possibile usare l'[Aggiunta guidata metodo](../ide/add-method-wizard.md) per aggiungere un metodo a un'interfaccia del progetto. Se il progetto contiene una classe associata all'interfaccia, la procedura guidata modifica anche la classe.  
  
### <a name="to-add-a-method-to-your-object"></a>Per aggiungere un metodo all'oggetto  
  
1.  Nella Visualizzazione classi espandere il nodo di progetto per visualizzare l'interfaccia a cui si vuole aggiungere il metodo.  
  
    > [!NOTE]
    >  È anche possibile aggiungere metodi alle interfacce dispatch che, se il progetto non è attribuito, si trovano sotto il nodo della libreria.  
  
2.  Fare clic con il pulsante destro del mouse sul nome dell'interfaccia.  
  
3.  Scegliere **Aggiungi** dal menu di scelta rapida e quindi fare clic su **Aggiungi metodo**.  
  
4.  In Aggiunta guidata metodo specificare le informazioni per creare il metodo.  
  
5.  Specificare le impostazioni del linguaggio di definizione dell'interfaccia per il metodo nella pagina [Attributi IDL](../ide/idl-attributes-add-method-wizard.md) della procedura guidata.  
  
6.  Scegliere **Fine** per aggiungere il metodo.
