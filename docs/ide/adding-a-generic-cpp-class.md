# <a name="adding-a-generic-c-class"></a>Aggiunta di una classe C++ generica
È possibile aggiungere una classe C++ generica utilizzando **Visualizzazione classi**. Una classe C++ generica è una classe che definiscono o che deriva da una classe che Definisci.  
  
### <a name="to-add-a-generic-c-class-to-a-project"></a>Per aggiungere una classe C++ generica a un progetto  
  
1.  In **Visualizzazione classi**, fare clic sul progetto in cui si desidera aggiungere la nuova classe, fare clic su **Aggiungi**, quindi fare clic su **classe**.  
  
2.  Nel [Aggiungi classe](../ide/add-class-dialog-box.md) la finestra di dialogo, nel riquadro dei modelli, fare clic su **classe C++**. Fare clic su **Aggiungi** per visualizzare il [Creazione guidata classe C++ generica](../ide/generic-cpp-class-wizard.md).  
  
3.  Nella procedura guidata, specificare un nome di classe, quindi definire le impostazioni o accettare le impostazioni predefinite.  
  
4.  Per chiudere la procedura guidata e visualizzare la nuova classe C++ generica nel progetto, fare clic su **fine**.
