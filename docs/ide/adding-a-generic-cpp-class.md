# <a name="adding-a-generic-c-class"></a>Aggiunta di una classe C++ generica

È possibile aggiungere una classe C++ generica usando la **Visualizzazione classi**. Una classe C++ generica è una classe che viene definita o che viene derivata da una classe definita.

### <a name="to-add-a-generic-c-class-to-a-project"></a>Per aggiungere una classe C++ generica a un progetto

1. nella **Visualizzazione classi** fare clic con il pulsante destro del mouse sul progetto a cui aggiungere la nuova classe, fare clic su **Aggiungi** e quindi su **Classe**.

1. nella finestra di dialogo [Aggiungi classe](../ide/add-class-dialog-box.md) nel riquadro dei modelli fare clic su **Classe C++**. Fare clic su **Aggiungi** per visualizzare la [Creazione guidata classe C++ generica](../ide/generic-cpp-class-wizard.md).

1. Nella procedura guidata specificare un nome di classe, quindi definire le impostazioni o accettare le impostazioni predefinite.

1. Per chiudere la procedura guidata e visualizzare la nuova classe C++ generica nel progetto, fare clic su **Fine**.
