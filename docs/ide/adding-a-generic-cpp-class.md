# Aggiunta di una classe C++ generica
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile aggiungere una classe C\+\+ generica tramite **Visualizzazione classi**.  Una classe C\+\+ generica è una classe definita dal programmatore o derivata da una classe già definita.  
  
### Per aggiungere una classe C\+\+ generica a un progetto  
  
1.  In **Visualizzazione classi** fare clic con il pulsante destro del mouse sul progetto a cui si desidera aggiungere la nuova classe, quindi scegliere **Aggiungi** e quindi **Classe**.  
  
2.  Nel riquadro dei modelli della finestra di dialogo [Aggiungi classe](../ide/add-class-dialog-box.md) fare clic su **Classe C\+\+**.  Scegliere **Aggiungi** per visualizzare la [Creazione guidata classe C\+\+ generica](../ide/generic-cpp-class-wizard.md).  
  
3.  Nella procedura guidata immettere un nome per la classe e definire le impostazioni o accettare quelle predefinite.  
  
4.  Scegliere **Fine** per uscire dalla procedura guidata e visualizzare la nuova classe C\+\+ generica nel progetto.
