# <a name="ltexamplegt-visual-c"></a>&lt;example&gt; (Visual C++)

Il tag \<example> consente di specificare un esempio di come usare un metodo o un altro membro della libreria. Ci√≤ comporta in genera anche il ricorso al tag [\<code>](../ide/code-visual-cpp.md).

## <a name="syntax"></a>Sintassi

```
<example>description</example>
```

#### <a name="parameters"></a>Parametri

*description*<br/>
Descrizione dell'esempio di codice.

## <a name="remarks"></a>Note

Compilare con [/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) per elaborare i commenti relativi alla documentazione in un file.

## <a name="example"></a>Esempio

```
// xml_example_tag.cpp
// compile with: /clr /doc /LD
// post-build command: xdcmake xml_example_tag.dll

/// Text for class MyClass.
public ref class MyClass {
public:
   /// <summary>
   /// GetZero method
   /// </summary>
   /// <example> This sample shows how to call the GetZero method.
   /// <code>
   /// int main()
   /// {
   ///    return GetZero();
   /// }
   /// </code>
   /// </example>
   static int GetZero() {
      return 0;
   }
};
```
