# Pagina delle propriet&#224; delle risorse gestite
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Attiva le impostazioni del compilatore di risorse.  
  
 La pagina delle proprietà **Risorse gestite** sono contenute le seguenti proprietà:  
  
 **Risorsa nome logico**  
 Specifica il *nome logico* del file resources intermedio generato.  Il nome logico è il nome utilizzato per caricare la risorsa.  Se non è specificato alcun nome logico, il nome dei file di risorsa \(resx\) viene utilizzato come il nome logico.  
  
 **Nome file di output**  
 Specifica il nome del file di output finale al quale il file di risorse \(resx\) contribuisce.  
  
 **Risorse localizzate predefinite**  
 Specifica se il file .resx specificato contribuisce alle risorse predefinite o a una .dll satellite.  
  
 Per informazioni su come accedere alla pagina delle proprietà **Risorse gestite**, vedere [Procedura: specificare le proprietà dei progetti tramite Pagine delle proprietà](../misc/how-to-specify-project-properties-with-property-pages.md).
