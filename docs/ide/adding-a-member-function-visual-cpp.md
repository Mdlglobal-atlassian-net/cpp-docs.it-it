# <a name="adding-a-member-function-visual-c"></a>Aggiunta di una funzione membro (Visual C++)
In **Visualizzazione classi**, è possibile aggiungere una funzione membro a qualsiasi classe. Quando si esegue questa operazione, vengono aggiunti una dichiarazione al file di intestazione e un corpo di funzione membro stub al file di implementazione della classe, che è possibile modificare in seguito.  
  
### <a name="to-add-a-member-function-to-a-class"></a>Per aggiungere una funzione membro a una classe  
  
1.  In **Visualizzazione classi**, espandere il nodo del progetto per visualizzare le classi nel progetto. (Aprire **Visualizzazione classi**, sulla barra dei menu, scegliere **vista**, **Visualizzazione classi**.)  
  
2.  Aprire il menu di scelta rapida per la classe a cui si desidera aggiungere una funzione membro e quindi scegliere **Aggiungi**, **Aggiungi funzione**.  
  
3.  Specificare i dettagli appropriati sulla funzione membro. Per ulteriori informazioni, vedere [Aggiunta guidata funzione membro](../ide/add-member-function-wizard.md).  
  
4.  Scegliere il **fine** pulsante per generare il codice della funzione membro.
