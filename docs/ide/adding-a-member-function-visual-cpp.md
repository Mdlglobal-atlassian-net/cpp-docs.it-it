# Aggiunta di una funzione membro (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In **Visualizzazioni Classi**, è possibile aggiungere una funzione membro a qualsiasi classe.  Quando si esegue ciò, viene aggiunta una dichiarazione al file di intestazione, e un codice funzione membro stub al file di implementazione della classe, che è possibile modificare.  
  
### Per aggiungere una funzione membro a una classe  
  
1.  In **Visualizzazione classi** espandere il nodo del progetto per visualizzarne le classi. \(Per aprire **Visualizzazione classi**, sulla barra dei menu, scegliere **Visualizza**, **Visualizzazione classi**\).  
  
2.  Aprire il menu di scelta rapida per la classe per la quale si desidera aggiungere una funzione membro e quindi scegliere **Aggiungi**, **Aggiungi funzione**.  
  
3.  Fornire i dettagli appropriati sulla funzione membro.  Per ulteriori informazioni, vedere [Aggiunta guidata funzione membro](../ide/add-member-function-wizard.md).  
  
4.  Scegliere il pulsante **Fine** per generare il codice della funzione membro.
