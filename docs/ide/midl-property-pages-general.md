# <a name="midl-property-pages-general"></a>Pagine delle proprietà MIDL: Generale
La pagina delle proprietà **Generale** nella cartella **MIDL** consente di specificare le opzioni seguenti del compilatore MIDL:  
  
-   Definizioni preprocessore [(/D](http://msdn.microsoft.com/library/windows/desktop/aa367321))  
  
-   Directory di inclusione aggiuntive ([/I](http://msdn.microsoft.com/library/windows/desktop/aa367328))  
  
-   Ignora percorsi di inclusione standard ([/no_def_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347))  
  
-   MkTypLib Compatible ([/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332))  
  
-   Livello avvisi ([/W](http://msdn.microsoft.com/library/windows/desktop/aa367383))  
  
-   Considera gli avvisi come errori ([/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387))  
  
-   Non visualizzare messaggio di avvio ([/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341))  
  
-   Tipo char MIDL ([/char](http://msdn.microsoft.com/library/windows/desktop/aa367314))  
  
-   Ambiente di destinazione ([/env](http://msdn.microsoft.com/library/windows/desktop/aa367323))  
  
-   Genera proxy senza stub ([/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352))  
  
 Per informazioni sull'accesso alla pagina delle proprietà **Generale** nella cartella **MIDL**, vedere [Uso delle proprietà del progetto](../ide/working-with-project-properties.md).  
  
 Per informazioni su come accedere a livello di codice alle opzioni MIDL per i progetti C++, vedere l'oggetto <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>.
