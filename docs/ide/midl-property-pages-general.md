# Pagine delle propriet&#224; MIDL: Generale
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Nella pagina delle proprietà **Generale** della cartella **MIDL** sono specificate le seguenti opzioni del compilatore MIDL:  
  
-   Definizioni per il preprocessore [\(\/D](http://msdn.microsoft.com/library/windows/desktop/aa367321)\)  
  
-   Directory di inclusione aggiuntive \([\/I](http://msdn.microsoft.com/library/windows/desktop/aa367328)\)  
  
-   Ignora percorsi di inclusione standard \([\/no\_def\_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347)\)  
  
-   Compatibile MkTypLib \([\/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332)\)  
  
-   Livello avvisi \([\/W](http://msdn.microsoft.com/library/windows/desktop/aa367383)\)  
  
-   Considera gli avvisi come errori \([\/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387)\)  
  
-   Non visualizzare messaggio di avvio \([\/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341)\)  
  
-   Tipo char MIDL \([\/char](http://msdn.microsoft.com/library/windows/desktop/aa367314)\)  
  
-   Ambiente di destinazione \([\/env](http://msdn.microsoft.com/library/windows/desktop/aa367323)\)  
  
-   Genera proxy senza stub \([\/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352)\)  
  
 Per informazioni sull'accesso alla pagina delle proprietà **Generale** nella cartella **MIDL**, vedere [Procedura: specificare le proprietà del progetto tramite le pagine delle proprietà](../misc/how-to-specify-project-properties-with-property-pages.md).  
  
 Per informazioni sull'accesso alle opzioni MIDL a livello di codice per i progetti di C\+\+, vedere l'oggetto <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>.
