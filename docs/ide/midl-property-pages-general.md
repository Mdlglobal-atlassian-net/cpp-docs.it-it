# <a name="midl-property-pages-general"></a>Pagine delle proprietà MIDL: Generale
Il **generale** pagina delle proprietà di **MIDL** cartella consente di specificare le seguenti opzioni del compilatore MIDL:  
  
-   Le definizioni del preprocessore [(/D](http://msdn.microsoft.com/library/windows/desktop/aa367321))  
  
-   Altre directory di inclusione ([/I](http://msdn.microsoft.com/library/windows/desktop/aa367328))  
  
-   Ignora percorsi di inclusione Standard ([/no_def_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347))  
  
-   Compatibile MkTypLib ([/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332))  
  
-   Livello di avviso ([/W](http://msdn.microsoft.com/library/windows/desktop/aa367383))  
  
-   Avvisi come errori ([/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387))  
  
-   Non visualizzare messaggio di avvio ([/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341))  
  
-   Tipo Char MIDL ([/char](http://msdn.microsoft.com/library/windows/desktop/aa367314))  
  
-   Ambiente di destinazione ([/env](http://msdn.microsoft.com/library/windows/desktop/aa367323))  
  
-   Genera proxy senza stub ([/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352))  
  
 Per informazioni su come accedere il **generale** pagina delle proprietà di **MIDL** cartella, vedere [funziona con le proprietà del progetto](../ide/working-with-project-properties.md).  
  
 Per informazioni su come accedere a livello di codice alle opzioni MIDL per i progetti C++, vedere <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool> oggetto.
