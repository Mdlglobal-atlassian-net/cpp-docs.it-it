# Aggiunta di un evento (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In Visualizzazione classi è possibile aggiungere un evento mediante l'[Aggiunta guidata evento](../ide/add-event-wizard.md) solo alla classe Control nel progetto di [controllo ActiveX MFC](../mfc/reference/creating-an-mfc-activex-control.md).  Se si desidera aggiungere un evento a un altro tipo di progetto, utilizzare il pulsante **Eventi** nella [finestra Proprietà](../Topic/Properties%20Window.md).  
  
### Per aggiungere un evento al progetto di controllo ActiveX MFC  
  
1.  In Visualizzazione classi espandere il nodo del progetto per visualizzarne le classi.  
  
2.  Fare clic con il pulsante destro del mouse sulla classe di controlli del progetto.  
  
3.  Scegliere **Aggiungi** dal menu di scelta rapida, quindi **Aggiungi evento** per visualizzare l'Aggiunta guidata evento.  
  
4.  Immettere le informazioni sull'evento nelle caselle appropriate della procedura guidata.  
  
5.  Scegliere **Fine** per aggiungere l'evento al progetto.
