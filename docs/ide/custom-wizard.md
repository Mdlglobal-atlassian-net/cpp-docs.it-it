# creazione guidata personalizzata
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La Creazione guidata personalizzata di Visual C\+\+ è un particolare tipo di procedura guidata che consente di creare procedure guidate personalizzate.  A configurazione ultimata la procedura guidata è in grado di generare automaticamente i file iniziali necessari per i progetti.  
  
## Panoramica  
 In questa schermata della procedura guidata sono descritte le impostazioni dell'applicazione correnti relative al progetto Creazione guidata personalizzata in corso di creazione.  Per impostazione predefinita, il progetto creato con la procedura guidata presenta le caratteristiche indicate di seguito.  
  
-   Il nome descrittivo della procedura guidata corrisponde al nome del progetto indicato nella **Finestra di dialogo Nuovo progetto**.  
  
-   La procedura guidata è corredata di un'interfaccia utente.  
  
-   Tale interfaccia è composta da una schermata.  
  
 È possibile modificare le impostazioni predefinite nella schermata [Impostazioni applicazione](../ide/application-settings-custom-wizard.md) della procedura guidata.
