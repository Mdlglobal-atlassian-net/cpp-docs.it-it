# Pagina delle propriet&#224; dello strumento generatore di dati XML
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La pagina delle proprietà **Strumento generatore di dati XML** diventa disponibile quando si aggiunge un dataset al progetto.  
  
 La pagina delle proprietà **Strumento generatore di dati XML** contiene le seguenti proprietà.  
  
 **File di output**  
 Consente di specificare il nome del file di output da utilizzare.  
  
 **Non visualizzare messaggio di avvio**  
 Consente di non visualizzare il messaggio di avvio e i messaggi informativi.  
  
 **Linguaggio del proxy generato**  
 Stabilisce se emettere o meno il codice gestito.  
  
 Per informazioni sull'accesso alla pagina delle proprietà **Strumento generatore di dati di XML**, vedere [Procedura: specificare le proprietà dei progetti tramite Pagine delle proprietà](../misc/how-to-specify-project-properties-with-property-pages.md).  
  
 Per informazioni su come accedere a livello di codice ai membri di questo oggetto, vedere <xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>.
