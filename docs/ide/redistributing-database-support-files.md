# <a name="redistributing-database-support-files"></a>Ridistribuzione dei file di supporto del database
È possibile ridistribuire i file di supporto per DAO (Data Access Objects) e per le tecnologie di database disponibili in Microsoft Data Access SDK.  
  
## <a name="installing-dao-support-files"></a>Installazione dei file di supporto per DAO  
 Per ottenere la versione più recente di DAO, vedere l'articolo[239114: How to obtain the latest service pack for the Microsoft Jet 4.0 Database Engine](http://go.microsoft.com/fwlink/p/?linkid=198014) (Come ottenere l'ultima versione del Service Pack per il motore di database Microsoft Jet 4.0) nel sito Web del supporto tecnico Microsoft.  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>Installazione dei file di supporto di Microsoft Data Access SDK  
 Usare Mdac_typ.exe per installare il supporto per ODBC, OLE DB, ActiveX Data Objects (ADO) e Remote Data Services (RDS). Il file Mdac_typ.exe si trova in ..\WCU\MDAC28\ nel supporto di installazione di Visual Studio. È anche possibile scaricare Mdac_typ.exe dal sito Web [Area download Microsoft](http://go.microsoft.com/fwlink/p/?linkid=198015).  
  
 Per altre informazioni, nel sito [MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016) cercare "Redistributing MDAC 2.8 SP1". Se per distribuire l'applicazione si usano progetti di installazione di Visual Studio, vedere [Distribuzione e dipendenze](http://msdn.microsoft.com/en-us/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
