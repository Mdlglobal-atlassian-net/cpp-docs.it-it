# Ridistribuzione dei file di supporto del database
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile ridistribuire i file di supporto per DAO \(Data Access Objects\) e per le tecnologie di database in Microsoft Data Access SDK.  
  
## Installazione dei file di supporto DAO  
 Per ottenere la versione più recente di DAO, vedere [Articolo 239114: Come ottenere l'ultima versione del service pack per Microsoft Jet 4.0 Database Engine](http://go.microsoft.com/fwlink/?LinkId=198014) sul sito Web di Microsoft Support.  
  
## Installazione dei file di supporto dell'SDK di Microsoft Data Access  
 Per installare il supporto per ODBC, OLE DB, ADO \(ActiveX Data Objects\) e RDS \(Remote Data Services\), utilizzare Mdac\_typ.exe,  Mdac\_typ.exe si trova in ..\\WCU\\MDAC28\\ nel supporto di installazione di Visual Studio.  È inoltre possibile eseguire il download di Mdac\_typ.exe dal sito [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=198015).  
  
 Per ulteriori informazioni, sul sito [MSDN](http://go.microsoft.com/fwlink/?LinkId=198016), cercare "Redistributing MDAC 2.8 SP1".  Se si utilizzano i progetti di installazione di Visual Studio per distribuire l'applicazione, vedere [Deployment and Dependencies](http://msdn.microsoft.com/it-it/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
