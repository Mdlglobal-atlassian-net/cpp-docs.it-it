# GetProjectPath
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Restituisce il percorso della directory del progetto.  
  
## Sintassi  
  
```  
  
      function GetProjectPath(   
   oProj    
);  
```  
  
#### Parametri  
 `oProj`  
 Progetto selezionato  
  
## Valore restituito  
 Il percorso della directory del progetto.  
  
## Note  
 Chiamare questa funzione per recuperare il percorso del progetto.
