# <a name="using-visual-c-add-new-item-templates"></a>Utilizzo dei template Aggiungi nuovo elemento di Visual C++
È possibile aggiungere facilmente gli elementi che sono comuni per i progetti di Visual C++ tramite la **Aggiungi nuovo elemento** comando. Quando si utilizza il **Aggiungi nuovo elemento** comando, il `Add New Item` la finestra di dialogo viene visualizzato un elenco di modelli di elemento, che è possibile aggiungere i file appropriati al progetto.  
  
 Nella tabella seguente è un elenco alfabetico di modelli di Visual C++ Aggiungi nuovo elemento.  
  
|Modello|Descrizione|  
|--------------|-----------------|  
|File di risorse assembly (con estensione resx)|Crea un file contenente risorse CLR.|  
|File bitmap (bmp)|Crea un file bitmap Win32.|  
|File di C++ (. cpp)|Crea un file di origine C++.|  
|Diagramma classi (con estensione CD)|Crea un diagramma classi vuoto.|  
|Set di regole di analisi codice (. ruleSet)|Crea un file di impostazioni per la configurazione di analisi del codice.|  
|File di configurazione (app. config)|Crea un file di configurazione vuoto.|  
|Classe di componente|Aggiunge una classe di componente utilizzando le funzionalità di Common Language Runtime.|  
|File cursore (cur)|Crea un file cursore Win32.|  
|File di individuazione statica (disco)|Crea un file di individuazione statica, ovvero un documento XML che contiene collegamenti ad altre risorse che descrivono il servizio Web XML, consente l'individuazione di un servizio Web XML.|  
|Pagina con frame (con estensione htm)|Aggiunge un file HTML contenente più pagine HTML.|  
|File di intestazione (h)|Crea un file di intestazione C++.|  
|Pagina HTML (htm)|Crea un file HTML vuoto.|  
|File icona (ico)|Crea un file di icona Win32.|  
|Classe Installer|Aggiunge una classe che eredita il [classe Installer](https://msdn.microsoft.com/en-us/library/system.configuration.install.installer.aspx) tramite funzionalità CLR.|  
|File MIDL (. idl)|Crea un file Interface Definition Language.|  
|File di definizione moduli (def)|Crea un file di definizione di esportazione DLL.|  
|Finestra delle proprietà (con estensione props)|Crea un file finestra delle proprietà di Visual C++.|  
|Script di registrazione (con estensione RGS)|Crea un file di script di registrazione ATL.|  
|Report (con estensione rdlc)|Crea un file di report.|  
|File di risorse (RC)|Crea un file di risorse Win32.|  
|File di modello di risorse (RCT)|Crea un file di modello di risorse.|  
|Barra multifunzione (mfcribbon-ms)|Crea un file della barra multifunzione.|  
|File di risposta server (Srf)|Crea un file di risposta di server che viene usato con ATL Server.|  
|File di Script SQL (SQL)|Crea un file di script SQL. **Nota:** questo modello non è una funzionalità Professional Edition.|  
|Foglio di stile (CSS)|Aggiunge un foglio di stile CSS utilizzato per le definizioni di stile HTML.|  
|File di testo (txt)|Aggiunge un file di testo vuoto.|  
|Controllo utente|Aggiunge un controllo utente utilizzando le funzionalità di Common Language Runtime.|  
|Windows Form|Aggiunge un Windows Form utilizzando le funzionalità di Common Language Runtime.|  
|File XML (con estensione XML)|Aggiunge un file XML vuoto.|  
|File XML Schema (XSD)|Crea un file che viene utilizzato per definire uno schema per documenti XML.|  
|File XSLT (XSLT)|Crea un file utilizzato per trasformare i documenti XML.|
