# <a name="makefile-project-wizard"></a>Creazione guidata progetto MakeFile
Usare questa procedura guidata per creare una riga di comando per l'esecuzione del motore di compilazione. Dopo aver creato il progetto, è possibile visualizzare e modificare le proprietà del progetto nella [pagina delle proprietà](../ide/working-with-project-properties.md) del progetto.  
  
## <a name="overview"></a>Panoramica  
 Questa pagina di procedura guidata illustra le impostazioni di progetto correnti per il progetto che si sta creando. Per impostazione predefinita, il progetto è configurato per compilare le impostazioni per il debug e il rilascio di *ProjName*.exe.  
  
 Per modificare questi valori predefiniti, fare clic sulla scheda [Impostazioni applicazione](../ide/application-settings-makefile-project-wizard.md) nella colonna sinistra della procedura guidata e apportare le modifiche desiderate.
