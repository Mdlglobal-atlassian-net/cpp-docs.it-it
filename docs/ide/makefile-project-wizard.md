# Creazione guidata progetto MakeFile
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa procedura guidata consente di creare una riga di comando per l'esecuzione del modulo di gestione della compilazione.  Una volta creato il progetto, è possibile visualizzarne e modificarne le proprietà nella relativa [pagina delle proprietà](../ide/working-with-project-properties.md).  
  
## Panoramica  
 In questa schermata della procedura guidata vengono descritte le impostazioni correnti del progetto creato.  Per impostazione predefinita, il progetto viene impostato per compilare configurazioni delle versioni di rilascio e di debug di *Nomeprog*.exe.  
  
 Per modificare le impostazioni predefinite, fare clic sulla scheda [Impostazioni applicazione](../ide/application-settings-makefile-project-wizard.md) nella colonna sinistra della procedura guidata e apportare le modifiche desiderate.
