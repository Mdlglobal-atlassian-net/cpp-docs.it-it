# <a name="makefile-project-wizard"></a>Creazione guidata progetto MakeFile
Utilizzare questa procedura guidata per creare una riga di comando per il motore di compilazione da eseguire. Dopo aver creato il progetto, è possibile visualizzare e modificare le proprietà del progetto del progetto [pagina delle proprietà](../ide/working-with-project-properties.md).  
  
## <a name="overview"></a>Panoramica  
 Questa pagina della procedura guidata descrive le impostazioni di progetto corrente per il progetto che si sta creando. Per impostazione predefinita, il progetto è configurato per compilare debug e rilascio delle configurazioni di *ProjName*.exe.  
  
 Per modificare le impostazioni predefinite, scegliere il [le impostazioni dell'applicazione](../ide/application-settings-makefile-project-wizard.md) scheda nella colonna sinistra della procedura guidata e apportare le modifiche desiderate.
