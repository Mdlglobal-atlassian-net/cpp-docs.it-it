# <a name="general-property-page-file"></a>Pagina delle proprietà Generale (File)

Quando si seleziona un file in **Esplora soluzioni**, **generale** pagina delle proprietà nel **le proprietà di configurazione** nodo contiene le proprietà seguenti:

**Escluso dalla compilazione**  
Specifica se il file deve essere incluso nella compilazione per la configurazione corrente.

Per accedere a livello di codice a questa proprietà, vedere <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>.

**Strumento**  
Lo strumento che verrà utilizzato per generare questo file. Vedere [strumenti di compilazione personalizzato specificando](../ide/specifying-custom-build-tools.md) per ulteriori informazioni.

Per accedere a livello di codice a questa proprietà, vedere <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>.

Per informazioni su come accedere il **generale** pagina delle proprietà nel **le proprietà di configurazione** nodo, vedere [funziona con le proprietà del progetto](../ide/working-with-project-properties.md).

Per i progetti non di Windows, vedere [riferimento alla pagina proprietà di Linux C++](../linux/prop-pages-linux.md)<!-- or [C++ Cross Platform Property Page Reference](../linux/prop-pages-linux.md)-->.
