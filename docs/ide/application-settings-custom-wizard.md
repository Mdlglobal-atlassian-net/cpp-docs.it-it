# Impostazioni applicazione, Creazione guidata personalizzata
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilizzare questa schermata della Creazione guidata personalizzata per progettare una procedura guidata.  
  
 **Nome descrittivo creazione guidata**  
 Imposta il nome della procedura guidata che si sta progettando.  Tale nome viene visualizzato nel file ReadMe.txt e come titolo nel file HTML.  
  
 **Interfaccia utente**  
 Valore predefinito.  Specifica che la creazione guidata personalizzata è corredata di un'interfaccia utente.  Se si desidera che l'utente selezioni delle opzioni per specificare il codice generato dalla procedura guidata, è necessario che quest'ultima preveda un'interfaccia utente.  L'interfaccia utente è superflua quando lo scopo della procedura guidata è di generare codice modello senza opzioni utente.  
  
 **Numero di pagine**  
 Imposta il numero di pagine necessarie per la creazione guidata personalizzata.  Questa opzione è valida solo per le procedure guidate corredate di interfaccia utente.
