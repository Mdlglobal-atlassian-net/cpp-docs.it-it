# <a name="navigate-the-class-structure"></a>Esplorare la struttura delle classi

È possibile esplorare la struttura delle classi di un progetto usando **Visualizzazione classi** o **Visualizzatore oggetti**. La struttura delle classi del progetto può includere funzioni, ereditarietà e interfacce. Per altre informazioni, vedere [Visualizzazione della struttura del codice](/visualstudio/ide/viewing-the-structure-of-code).

Per passare a una dichiarazione di funzione, fare doppio clic sul nome della funzione in Visualizzazione classi. Per passare a una definizione di classe, fare doppio clic sul nome della classe in Visualizzazione classi.
