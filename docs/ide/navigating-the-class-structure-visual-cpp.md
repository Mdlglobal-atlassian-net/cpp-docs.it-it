# Esplorazione della struttura delle classi (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile esplorare la struttura delle classi di un progetto, comprese le funzioni, l'ereditarietà e le interfacce mediante [Visualizzazione classi](http://msdn.microsoft.com/it-it/8d7430a9-3e33-454c-a9e1-a85e3d2db925) o il [Visualizzatore oggetti](http://msdn.microsoft.com/it-it/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Per visualizzare una dichiarazione di funzione, fare doppio clic sul nome della funzione in Visualizzazione classi.  Per passare a una definizione di classe, fare doppio clic sul nome della classe in Visualizzazione classi.
