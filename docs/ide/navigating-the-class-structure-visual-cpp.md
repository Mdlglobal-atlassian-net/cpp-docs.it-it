# <a name="navigating-the-class-structure-visual-c"></a>Esplorazione della struttura delle classi (Visual C++)
È possibile passare una struttura di una classe di progetto, comprese le funzioni, ereditarietà e interfacce utilizzando [Visualizzazione classi](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925) o [Visualizzatore oggetti](http://msdn.microsoft.com/en-us/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Facendo doppio clic sul nome della funzione in visualizzazione classi, è possibile passare a una dichiarazione di funzione. Facendo doppio clic su un nome di classe in visualizzazione classi, è possibile passare a una definizione di classe.
