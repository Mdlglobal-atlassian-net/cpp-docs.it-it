# Pagina delle propriet&#224; Riferimenti Web
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilizzando la pagina delle proprietà **Riferimenti Web** è possibile specificare come generare la classe proxy del servizio Web XML.  La classe viene generata se si aggiunge un riferimento Web al progetto.  Per ulteriori informazioni, vedere [Aggiungi riferimento Web](http://msdn.microsoft.com/it-it/bdf05776-c591-40af-bfd7-e1e2aa1e87b5).  
  
 Nella pagina delle proprietà **Riferimenti Web** sono disponibili le seguenti proprietà:  
  
 **File di output**  
 Il nome del file che contiene la classe proxy del servizio Web XML.  
  
 **Non visualizzare messaggio di avvio**  
 Utilizzare questa opzione per non visualizzare il messaggio per lo strumento del linguaggio di descrizione dei servizi Web \(Wsdl.exe\).  
  
 **Spazio dei nomi**  
 Specifica il nome del proxy Web generato.  
  
 **Riferimenti aggiuntivi**  
 Specifica le DLL aggiuntive a cui viene fatto riferimento dalla DLL proxy.  
  
 Per informazioni sull'accesso alla pagina delle proprietà **Riferimenti Web**, vedere [Procedura: specificare le proprietà del progetto tramite le pagine delle proprietà](../misc/how-to-specify-project-properties-with-property-pages.md).
