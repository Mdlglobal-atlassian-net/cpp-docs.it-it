# Utilizzo dell&#39;ambiente di sviluppo di Visual Studio per lo sviluppo di applicazioni desktop C++
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'ambiente di sviluppo integrato \(IDE, Integrated Development Environment\) di [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] offre un set di strumenti che consentono di scrivere e modificare codice, nonché di rilevare e correggere eventuali errori.  
  
 Gli articoli in questa sezione della documentazione illustrano come creare un programma C\+\+ standard e come testarne il funzionamento tramite le funzionalità disponibili in [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  Il semplice programma creato registrerà il numero di giocatori dei diversi giochi di carte.
