# <a name="486p-32-bit-masm"></a>.486P (MASM a 32 bit)

Abilita l'assembly di tutte le istruzioni (incluso il privilegio) per il processore 80486. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> **.486P**
