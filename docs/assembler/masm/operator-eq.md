# <a name="operator-eq"></a>Operatore EQ
Restituisce true (-1) se *expression1* è uguale a *expression2*, oppure restituisce false (0) se non è presente.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 EQ expression2  
```
