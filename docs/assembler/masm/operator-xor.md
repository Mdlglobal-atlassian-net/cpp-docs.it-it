# <a name="operator-xor"></a>Operatore XOR
Restituisce il risultato di un bit per bit **XOR** operazione per *expression1* e *expression2*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 XOR expression2  
```
