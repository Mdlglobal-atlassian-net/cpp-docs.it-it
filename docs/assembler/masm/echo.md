# <a name="echo"></a>ECHO
Consente di visualizzare *messaggio* al dispositivo di output standard (per impostazione predefinita, la schermata).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>Note  
 Uguale a [% OUT](../../assembler/masm/percent-out.md).
