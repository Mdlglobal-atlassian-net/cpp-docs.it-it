# <a name="echo"></a>ECHO

Consente di visualizzare *messaggio* alla periferica di output standard (per impostazione predefinita, la schermata).

## <a name="syntax"></a>Sintassi

> ECHO *messaggio*

## <a name="remarks"></a>Note

Uguale allo [% OUT](../../assembler/masm/percent-out.md).
