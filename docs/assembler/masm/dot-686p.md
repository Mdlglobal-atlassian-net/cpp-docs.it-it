# <a name="686p-32-bit-masm"></a>.686P (MASM a 32 bit)

Abilita l'assembly di tutte le istruzioni (incluso il privilegio) per il processore Pentium Pro. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> **.686P**
