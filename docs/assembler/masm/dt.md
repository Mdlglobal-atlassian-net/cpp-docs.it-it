# <a name="dt"></a>DT
Alloca e, facoltativamente, inizializza 10 byte di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido. `DT` è un sinonimo di [TBYTE](../../assembler/masm/tbyte.md).  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
