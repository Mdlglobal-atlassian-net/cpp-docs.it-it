# <a name="dt"></a>DT

Alloca e inizializza facoltativamente 10 byte di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo ovunque un tipo è consentito. `DT` è un sinonimo di [TBYTE](../../assembler/masm/tbyte.md).

## <a name="syntax"></a>Sintassi

> [[nome]] Inizializzatore DT [[, inizializzatore]]...
