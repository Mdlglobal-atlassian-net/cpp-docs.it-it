# <a name="operator-"></a>Operatore .
Il primo operatore restituisce *espressione* oltre l'offset del *campo* la struttura o unione. Il secondo operatore restituisce valore in corrispondenza della posizione a cui puntata *registrare* oltre l'offset del *campo* la struttura o unione.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
