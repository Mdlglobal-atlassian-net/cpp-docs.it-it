# operator .
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La prima restituzione dell'operatore *espressione* più l'offset di *campo* nella struttura o unione.  Il secondo valore dell'operatore restituisce il percorso indicato da *il registro* più l'offset di *campo* nella struttura o unione.  
  
## Sintassi  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
