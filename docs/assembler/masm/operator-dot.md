# <a name="operator-"></a>Operatore .

Il primo operatore restituisce *espressione* oltre l'offset del *campo* entro la struttura o unione. Il secondo operatore restituisce valore in corrispondenza della posizione a cui punta *registrare* oltre l'offset del *campo* entro la struttura o unione.

## <a name="syntax"></a>Sintassi

> *expression* __.__ *field* \[__.__ *field*] ...
>
> __\[__*register*__]__ __.__ *field* \[__.__ *field*] ...
