# <a name="group-32-bit-masm"></a>GRUPPO (MASM a 32 bit)

Aggiungere i *segmenti* specificati al gruppo denominato *nome*. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> ⟦ *segmento* **gruppo** nome __,__ *segmento* ... ⟧
