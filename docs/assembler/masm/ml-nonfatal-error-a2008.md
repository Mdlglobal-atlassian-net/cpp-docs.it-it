# ML Nonfatal Error A2008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**errore di sintassi:**  
  
 Un token nella posizione corrente ha provocato un errore di sintassi.  
  
 Uno dei seguenti elementi può verificarsi:  
  
-   Un prefisso di punto è stato aggiunto a o è stato omesso da una direttiva.  
  
-   Una parola riservata \(ad esempio **C\#** o  **DIMENSIONE**\) è stato utilizzato come identificatore.  
  
-   Un'istruzione è stata utilizzata non è disponibile tramite la selezione del coprocessore o del processore corrente.  
  
-   Un operatore di runtime di confronto \(ad esempio `==`\) è stato utilizzato in un l'istruzione condizionale anziché un operatore relazionale \(ad esempio  [EQ](../../assembler/masm/operator-eq.md)\).  
  
-   Un'istruzione o una direttiva è stata fornita insufficienti operandi.  
  
-   Una direttiva precedente è stata utilizzata.
