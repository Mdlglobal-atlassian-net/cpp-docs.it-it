# <a name="operator-"></a>Operatore ;;
Considera *testo* come un commento in una macro che viene visualizzato solo nella definizione della macro. Non viene visualizzato l'elenco *testo* in cui la macro viene espansa.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
;;  
text  
  
```
