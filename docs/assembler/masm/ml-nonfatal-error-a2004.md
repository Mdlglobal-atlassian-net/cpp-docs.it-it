# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**conflitto del tipo del simbolo: identificatore**  
  
 [EXTERNDEF](../../assembler/masm/externdef.md) o  [ETICHETTA](../../assembler/masm/label-masm.md) la direttiva è stata utilizzata in una variabile, con un simbolo, in una struttura di dati, oppure su un'etichetta che è definita nello stesso modulo ma con un tipo diverso.
