# operator LOWWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce i bit di minimo di 16 *espressione*.  le espressioni di MASM sono valori a 64 bit.  
  
## Sintassi  
  
```  
  
LOWWORD expression  
```
