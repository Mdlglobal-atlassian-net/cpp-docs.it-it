# <a name="operator-lowword"></a>Operatore LOWWORD

Restituisce i 16 bit bassi dell' *espressione*. Le espressioni MASM sono valori a 64 bit.

## <a name="syntax"></a>Sintassi

> *Espressione* LOWWORD
