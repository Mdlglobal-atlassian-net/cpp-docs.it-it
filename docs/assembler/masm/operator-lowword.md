# <a name="operator-lowword"></a>Operatore LOWWORD
Restituisce i 16 bit bassi di *espressione*. Espressioni MASM sono valori a 64 bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
LOWWORD expression  
```
