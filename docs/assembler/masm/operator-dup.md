# <a name="operator-dup"></a>Operatore DUP

Specifica *il numero totale di* dichiarazioni di *InitialValue*.

## <a name="syntax"></a>Sintassi

> *conteggio* **duplicato** __(__ *InitialValue* ⟦ __,__ *InitialValue* ... ⟧ __)__
