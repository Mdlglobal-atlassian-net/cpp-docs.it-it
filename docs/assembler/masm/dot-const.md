# <a name="const"></a>.CONST
Se usato con [. MODELLO](../../assembler/masm/dot-model.md), viene avviato un segmento di dati costanti (con segmento nome CONST).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
.CONST  
  
```  
  
## <a name="remarks"></a>Note  
 Questo segmento include l'attributo di sola lettura.
