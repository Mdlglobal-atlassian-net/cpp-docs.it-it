# <a name="typedef-masm"></a>TYPEDEF (MASM)
Definisce un nuovo tipo chiamato *nome*, che equivale a `type`.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name TYPEDEF type  
```
