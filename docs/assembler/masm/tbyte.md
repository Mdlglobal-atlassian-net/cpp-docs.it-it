# <a name="tbyte"></a>TBYTE
Alloca e, facoltativamente, inizializza 10 byte di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido.  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] TBYTEinitializer [[, initializer]]...  
```
