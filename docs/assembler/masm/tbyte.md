# <a name="tbyte"></a>TBYTE

Alloca e inizializza facoltativamente 10 byte di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo ovunque un tipo è consentito.

## <a name="syntax"></a>Sintassi

> [[nome]] TBYTEinitializer [[, inizializzatore]]...
