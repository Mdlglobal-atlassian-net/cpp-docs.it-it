# <a name="sdword"></a>SDWORD
Alloca e, facoltativamente, consente di inizializzare una parola doppia con segno (4 byte) di spazio di archiviazione per ogni `initializer`. Può essere utilizzata anche come un identificatore di tipo in qualsiasi punto un tipo è valido.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
