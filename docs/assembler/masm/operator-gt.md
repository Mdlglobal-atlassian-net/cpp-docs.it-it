# operator GT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce true \(\- 1\) *expression1* è maggiore di *expression2*, o restituisce false \(0\) se non è.  
  
## Sintassi  
  
```  
  
expression1 GT expression2  
```
