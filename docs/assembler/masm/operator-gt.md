# <a name="operator-gt"></a>Operatore GT
Restituisce true (-1) se *expression1* è maggiore di *expression2*, oppure restituisce false (0) se non lo è.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 GT expression2  
```
