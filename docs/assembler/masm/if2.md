# IF2
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Un'istruzione IF** il blocco viene esaminato a ogni passaggio dell'assembly se  **OPZIONE: SETIF2** viene  **TRUE**.  
  
## Sintassi  
  
```  
  
IF2   
expression  
  
```  
  
## Note  
 vedere [Un'istruzione IF](../../assembler/masm/if-masm.md) per la sintassi completa.
