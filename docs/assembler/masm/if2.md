# <a name="if2"></a>IF2

**IF** blocco viene valutato a ogni passaggio di assembly, se **opzione: SETIF2** viene **TRUE**.

## <a name="syntax"></a>Sintassi

> IF2 *espressione*

## <a name="remarks"></a>Note

Visualizzare [IF](../../assembler/masm/if-masm.md) per la sintassi completa.
