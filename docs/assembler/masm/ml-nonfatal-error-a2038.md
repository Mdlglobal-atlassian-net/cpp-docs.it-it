# <a name="ml-nonfatal-error-a2038"></a>Errore ML non irreversibile A2038

**operando mancante per l'operatore (macro)**

L'assembler trovata la fine dell'elenco di parametri della macro immediatamente dopo il [!](../../assembler/masm/operator-logical-not-masm-run-time.md) oppure [ % ](../../assembler/masm/operator-percent.md) operatore.
