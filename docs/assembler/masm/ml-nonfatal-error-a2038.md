# ML Nonfatal Error A2038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**operando mancante per la macro operatore**  
  
 L'assemblatore individuata la fine dell'elenco di parametri di una macro immediatamente dopo il [\!](../../assembler/masm/operator-logical-not-masm-run-time.md) o [%](../../assembler/masm/operator-percent.md) operatore.
