# <a name="operator-le"></a>Operatore LE
Restituisce true (-1) se *expression1* è minore o uguale a *expression2*, oppure restituisce false (0) se non lo è.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 LE expression2  
```
