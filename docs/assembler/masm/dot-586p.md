# <a name="586p-32-bit-masm"></a>.586P (MASM a 32 bit)

Abilita l'assembly di tutte le istruzioni (incluso il privilegio) per il processore Pentium. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> **.586P**
