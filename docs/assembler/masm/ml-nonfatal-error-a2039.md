# ML Nonfatal Error A2039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**riga troppo lungo**  
  
 Un file di origine ha superato il limite di 512 caratteri.  
  
 Se le linee fisiche più sono concatenate con il carattere di continuazione di riga \(\\\), la riga logica risultante è ancora limitato a 512 caratteri.
