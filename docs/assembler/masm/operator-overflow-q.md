# operator OVERFLOW?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Lo stato del flag di overflow.  Utilizzato solo in [.IF](../../assembler/masm/dot-if.md),  [.WHILE](../../assembler/masm/dot-while.md), o  [.REPEAT](../../assembler/masm/dot-repeat.md) blocchi e valutato in fase di esecuzione, non al momento dell'assembly.  
  
## Sintassi  
  
```  
  
OVERFLOW?  
  
```
