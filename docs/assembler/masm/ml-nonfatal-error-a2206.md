# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**operatore mancante nell'espressione**  
  
 Un'espressione non può essere valutata perché manca un operatore.  Questo messaggio di errore può anche essere un effetto collaterale di un errore precedente del programma.  
  
 La riga seguente genera questo errore:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
