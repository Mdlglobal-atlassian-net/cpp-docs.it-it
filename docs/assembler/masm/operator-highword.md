# <a name="operator-highword"></a>Operatore HIGHWORD

Restituisce i 16 bit alti di 32 bit bassi della *espressione*. Espressioni MASM sono valori a 64 bit.

## <a name="syntax"></a>Sintassi

> Espressione HIGHWORD
