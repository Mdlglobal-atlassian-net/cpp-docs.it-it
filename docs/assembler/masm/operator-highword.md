# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce i bit di livello 16 di bit di minimo di 32 *espressione*.  le espressioni di MASM sono valori a 64 bit.  
  
## Sintassi  
  
```  
  
HIGHWORD expression  
```
