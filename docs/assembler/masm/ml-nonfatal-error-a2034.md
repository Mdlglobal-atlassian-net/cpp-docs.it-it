# ML Nonfatal Error A2034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**deve trovarsi in blocco del segmento**  
  
 Uno dei seguenti elementi è stato rilevato all'esterno di un blocco del segmento:  
  
-   un'istruzione  
  
-   Una definizione dell'etichetta  
  
-   In [CIÒ](../../assembler/masm/operator-this.md) operatore  
  
-   In [$](../../assembler/masm/dollar.md) operatore  
  
-   Una definizione della routine  
  
-   [ALLINEAR](../../assembler/masm/align-masm.md) direttiva  
  
-   [ORG](../../assembler/masm/org.md) direttiva
