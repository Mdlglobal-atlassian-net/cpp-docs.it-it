# <a name="operator-opattr"></a>Operatore OPATTR

Restituisce una parola che definisce la modalità e l'ambito del *espressione*. Il byte basso è identico al byte restituiti da [. TIPO](../../assembler/masm/operator-dot-type.md). Il primo byte contiene informazioni aggiuntive.

## <a name="syntax"></a>Sintassi

> Espressione OPATTR
