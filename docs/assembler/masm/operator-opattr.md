# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

restituisce una parola che definisce la modalità e l'ambito di *espressione*.  Il byte basso è identico al byte restituito da [.TYPE](../../assembler/masm/operator-dot-type.md).  Il byte superiore contiene informazioni aggiuntive.  
  
## Sintassi  
  
```  
  
OPATTR expression  
```
