# <a name="nolistmacro"></a>.NOLISTMACRO

Suppresses listing of macro expansions.

## <a name="syntax"></a>Sintassi

> **.NOLISTMACRO**

## <a name="remarks"></a>Note

Same as [.SALL](../../assembler/masm/dot-sall.md).
