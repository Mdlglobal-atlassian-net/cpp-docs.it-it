# <a name="nolistmacro"></a>.NOLISTMACRO
Elimina l'elenco di espansioni della macro.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
.NOLISTMACRO  
  
```  
  
## <a name="remarks"></a>Note  
 Uguale a [. SALL](../../assembler/masm/dot-sall.md).
