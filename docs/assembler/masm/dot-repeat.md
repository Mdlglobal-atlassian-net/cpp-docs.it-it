# <a name="repeat-32-bit-masm"></a>. REPEAT (MASM a 32 bit)

Genera codice che ripete l'esecuzione del blocco di *istruzioni* fino a quando la *condizione* non diventa true. [. UNTILCXZ](../../assembler/masm/dot-untilcxz.md), che diventa true quando CX è zero, può essere sostituito da [. FINO a](../../assembler/masm/dot-until.md). La *condizione* è facoltativa con **. UNTILCXZ**. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> **. Ripeti**\
> \ di *istruzioni*
> **.**  *Condizione* until
