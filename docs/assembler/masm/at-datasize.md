# <a name="datasize"></a>\@DataSize

0 per i modelli **small, Small**, **medium**e **Flat** , 1 per i modelli **Compact** e **large** e 2 per un modello **molto** **grande (** equivalenti numerici).

## <a name="syntax"></a>Sintassi

> **\@DataSize**
