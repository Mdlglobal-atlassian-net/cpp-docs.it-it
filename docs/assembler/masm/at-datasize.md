# <a name="datasize"></a>@DataSize

0 per **TINY**, **SMALL**, **MEDIUM**, e **FLAT** dei modelli, 1 per **COMPACT** e  **GRANDI** modelli e 2 per **molto grandi** modello (numeric equivalgono).

## <a name="syntax"></a>Sintassi

> @DataSize
