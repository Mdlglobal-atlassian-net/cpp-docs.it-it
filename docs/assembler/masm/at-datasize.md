# @DataSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 per **MINUSCOLO**,  **PICCOLO**,  **SUPPORTO**e  **PIANO** modelli, 1 per  **COMPATTO** e  **GRANDE** modelli e 2 per  **ENORME** modello \(numerico essere uguale a\).  
  
## Sintassi  
  
```  
  
@DataSize  
  
```
