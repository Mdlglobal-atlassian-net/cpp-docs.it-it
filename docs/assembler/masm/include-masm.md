# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Inserire il codice sorgente dal file di origine fornito da l *nome file* nel file di origine corrente durante l'assembly.  
  
## Sintassi  
  
```  
  
INCLUDE filename  
```  
  
## Note  
 *nome file* deve essere racchiuso tra parentesi acute se include una barra rovesciata, un punto e virgola, un simbolo maggiore di, un simbolo minore di., una virgoletta, o virgolette doppie.
