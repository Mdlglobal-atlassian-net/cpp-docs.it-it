# <a name="real10"></a>REAL10

Alloca ed eventualmente Inizializza un numero a virgola mobile a 10 byte per ogni *inizializzatore*.

## <a name="syntax"></a>Sintassi

> *nome* **REAL10** *inizializzatore* ⟦ __,__ *inizializzatore* ... ⟧
