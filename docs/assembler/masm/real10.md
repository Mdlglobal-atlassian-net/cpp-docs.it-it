# <a name="real10"></a>REAL10

Alloca e inizializza facoltativamente un numero a virgola mobile di 10 byte per ogni *inizializzatore*.

## <a name="syntax"></a>Sintassi

> inizializzatore REAL10 nome [[, inizializzatore]]...
