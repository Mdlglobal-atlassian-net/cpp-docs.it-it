# <a name="real10"></a>REAL10
Alloca e, facoltativamente, consente di inizializzare un numero a virgola mobile di 10 byte per ogni *inizializzatore*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
