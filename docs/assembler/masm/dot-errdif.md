# .ERRDIF]]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Genera un errore se gli elementi di testo sono diversi.  
  
## Sintassi  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Note  
 se **i** viene fornito, il confronto non rileva la distinzione tra maiuscole e minuscole.
