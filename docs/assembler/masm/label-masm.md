# <a name="label"></a>LABEL

Crea una nuova etichetta assegnando il valore del contatore di posizione corrente e il valore di *qualifiedType* specificato a *Name*.

## <a name="syntax"></a>Sintassi

> **etichetta** nome *qualifiedType*
>
> *nome* **LABEL** ⟦**near** | **Lung** | **proc**⟧ **ptr** ⟦*qualifiedType*⟧
