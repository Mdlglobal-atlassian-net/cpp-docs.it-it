# <a name="label-masm"></a>LABEL (MASM)
Crea una nuova etichetta assegnando il valore del contatore di posizione corrente e il determinato `type` a *nome*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
