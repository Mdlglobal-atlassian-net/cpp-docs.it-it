# <a name="public-masm"></a>PUBLIC (MASM)
Rende ogni variabile, l'etichetta o simbolo assoluto specificato come *nome* disponibili per tutti gli altri moduli del programma.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
