# <a name="public-masm"></a>PUBLIC (MASM)

Consente di ogni variabile, un'etichetta o simbolo assoluto specificato come *nome* disponibile per tutti gli altri moduli nel programma.

## <a name="syntax"></a>Sintassi

> Nome pubblico [[langtype]] [[, nome [[langtype]]]]...
