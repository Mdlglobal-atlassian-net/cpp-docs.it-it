# <a name="operator-shl"></a>Operatore SHL
Restituisce il risultato di spostamento di bit di *espressione* sinistro *conteggio* numero di bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression SHL count  
```
