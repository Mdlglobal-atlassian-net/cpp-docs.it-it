# <a name="ymmword"></a>YMMWORD
Utilizzato per gli operandi multimediali 256 bit con istruzioni Intel Advanced Vector estensioni AVX ().  
  
## <a name="syntax"></a>Sintassi  
  
```  
YMMWORD  
```  
  
## <a name="remarks"></a>Note  
 `YMMWORD` rappresenta lo stesso tipo `__m256` per le funzioni intrinseche AVX.
