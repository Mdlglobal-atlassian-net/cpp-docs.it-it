# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Elimina l'elenco dei simboli nella tabella di simboli e nel file del browser.  
  
## Sintassi  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Note  
 Se i nomi specificati, pertanto solo i nomi specificati vengono eliminati.  Equivale a [.XCREF](../../assembler/masm/dot-xcref.md).
