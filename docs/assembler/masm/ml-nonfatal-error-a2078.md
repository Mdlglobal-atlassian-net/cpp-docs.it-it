# <a name="ml-nonfatal-error-a2078"></a>Errore ML non irreversibile A2078
**istruzione non consente di FAR indirizzamento indiretto**  
  
 Un ciclo o salto condizionale può accettare un operando di memoria. È necessario assegnare un indirizzo relativo o un'etichetta.
