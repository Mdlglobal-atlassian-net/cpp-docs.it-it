# <a name="ml-nonfatal-error-a2107"></a>Errore ML non irreversibile A2107

**non può avere jump distante implicite o chiamata quasi etichetta**

È stato effettuato il tentativo per rendere implicita molto jump o chiamare a un'etichetta più vicino in un altro segmento.
