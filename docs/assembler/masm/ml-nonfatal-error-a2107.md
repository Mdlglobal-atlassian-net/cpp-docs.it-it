# <a name="ml-nonfatal-error-a2107"></a>Errore ML non irreversibile A2107
**non può avere jump distante implicita o chiamata quasi etichetta**  
  
 È stato effettuato un tentativo di rendere implicita molto passare o chiamare a un'etichetta in un altro segmento vicino.
