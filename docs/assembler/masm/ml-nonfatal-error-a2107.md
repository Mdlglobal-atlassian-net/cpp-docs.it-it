# ML Nonfatal Error A2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**non può avere distante il salto o chiamata implicita all'etichetta vicina**  
  
 Si è tentato di fare distante un ramo o una chiamata implicita a un'etichetta vicina in un altro segmento.
