# <a name="endp"></a>ENDP

Contrassegna la fine del *nome* della procedura precedentemente iniziata con **proc**.

## <a name="syntax"></a>Sintassi

> *nome* **ENDP**

## <a name="remarks"></a>Note

Vedere [proc](../../assembler/masm/proc.md).
