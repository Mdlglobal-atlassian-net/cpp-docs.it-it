# <a name="endp"></a>ENDP
Contrassegna la fine della procedura *nome* avviata in precedenza con **PROC**.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name  
 ENDP  
  
```  
  
## <a name="remarks"></a>Note  
 Vedere [PROC](../../assembler/masm/proc.md).
