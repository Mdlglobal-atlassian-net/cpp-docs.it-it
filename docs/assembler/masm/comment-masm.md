# <a name="comment-masm"></a>COMMENT (MASM)

Considera tutto il *testo* tra o nella stessa riga dei delimitatori come commento.

## <a name="syntax"></a>Sintassi

> *Delimitatore* di commento ⟦*testo*⟧ \
> ⟦*testo*⟧ \
> ⟦*testo*⟧ *delimitatore* ⟦*testo*⟧
