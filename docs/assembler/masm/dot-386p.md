# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Abilita il montaggio di tutte le istruzioni \(autorizzato incluso\) per il processore 80386; disabilita il montaggio delle istruzioni introdotte con i processori successivi.  
  
## Sintassi  
  
```  
  
.386P  
  
```  
  
## Note  
 Consente inoltre alle 80387 istruzioni.
