# <a name="ml-nonfatal-error-a2074"></a>Errore ML non irreversibile A2074

**etichetta è possibile accedere tramite i registri del segmento**

È stato effettuato un tentativo di accedere a un'etichetta tramite un registro del segmento che non si presuppone che per il segmento o un gruppo.
