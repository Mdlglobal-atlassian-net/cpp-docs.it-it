# ML Nonfatal Error A2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**impossibile accedere all'etichetta dai registri di segmento**  
  
 È stato effettuato un tentativo di accedere a un'etichetta da un registro di segmento che non è stato considerato al segmento o gruppo.
