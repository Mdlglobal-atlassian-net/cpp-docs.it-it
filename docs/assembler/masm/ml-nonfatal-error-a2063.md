# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**può ALLINEARSI solo a una potenza di 2: espressione**  
  
 l'espressione specificata con [ALLINEAR](../../assembler/masm/align-masm.md) la direttiva non è valida.  
  
 **ALLINEAR** l'espressione deve essere una potenza di 2 tra 2 e 256 e deve essere minore o uguale perallineamento del segmento, una struttura o un'unione, corrente.
