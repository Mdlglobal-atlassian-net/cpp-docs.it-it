# <a name="operator-ampamp-masm"></a>operatore &amp; &amp; (MASM)
Sostituisce `parameter` con il valore dell'argomento corrispondente.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
&  
parameter  
&  
  
```
