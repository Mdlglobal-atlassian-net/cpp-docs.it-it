# <a name="operator-low32"></a>Operatore LOW32
Restituisce i 32 bit bassi di *espressione*. Espressioni MASM sono valori a 64 bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
LOW32 expression  
```
