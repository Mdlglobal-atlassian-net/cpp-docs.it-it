# <a name="sqword"></a>SQWORD
Alloca e, facoltativamente, inizializza 8 byte con segno di archiviazione per ogni `initializer`. Può essere utilizzata anche come un identificatore di tipo in qualsiasi punto un tipo è valido.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
