# SQWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloca e facoltativamente inizializza 8 Signed Byte di archiviazione per ciascuno `initializer`.  È inoltre possibile utilizzare come identificatore di tipo ogni volta che un tipo è valido.  
  
## Sintassi  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
