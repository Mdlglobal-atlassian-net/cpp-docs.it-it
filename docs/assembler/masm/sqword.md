# <a name="sqword"></a>SQWORD

Alloca e inizializza facoltativamente 8 byte con segno dell'archiviazione per ogni `initializer`. Inoltre, può essere utilizzato come identificatore di tipo ovunque un tipo è consentito.

## <a name="syntax"></a>Sintassi

> inizializzatore SQWORD nome [[, inizializzatore]]...
