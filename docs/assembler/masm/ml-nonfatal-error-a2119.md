# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**il tipo di linguaggio deve essere specificato**  
  
 Una definizione o un prototipo della routine non è stato fornito un tipo di linguaggio.  
  
 Un tipo di linguaggio deve essere dichiarato in ogni definizione o prototipo della routine se un tipo di lingua predefinita non è specificato.  Un tipo di lingua predefinita è impostato utilizzando spostato a una o l'altra [.MODEL](../../assembler/masm/dot-model.md) direttiva,  **LANG OF OPZIONE**, o le opzioni della riga di comando di ML  **\/Gc** o  **\/Gd**.
