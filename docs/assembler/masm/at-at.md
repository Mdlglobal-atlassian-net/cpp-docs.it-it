# <a name=""></a>\@\@:

Definisce un'etichetta del codice riconoscibile solo tra *Label1* e *Label2*, dove *Label1* è l'inizio del codice o l'etichetta di `@@:` precedente e *Label2* è la fine del codice o l'etichetta `@@:` successiva. Vedere [\@B](at-b.md) e [\@F](at-f.md).
