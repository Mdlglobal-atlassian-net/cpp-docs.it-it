# REPEAT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

contrassegna un blocco che deve essere ripetuto *espressione* volte.  Equivale a [REPT](../../assembler/masm/rept.md).  
  
## Sintassi  
  
```  
  
   REPEAT expression  
statements  
ENDM  
```
