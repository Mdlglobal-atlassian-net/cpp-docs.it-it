# <a name="repeat"></a>RIPETIZIONI

Contrassegna un blocco che deve essere ripetuto per i tempi di *espressione* . Uguale a [REPT](rept.md).

## <a name="syntax"></a>Sintassi

> **Ripeti** *espressione*\
> \ di *istruzioni*
> **ENDM**
