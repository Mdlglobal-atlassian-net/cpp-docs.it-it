# <a name="listall"></a>.LISTALL

Avvia l'elenco di tutte le istruzioni.

## <a name="syntax"></a>Sintassi

> **.LISTALL**

## <a name="remarks"></a>Note

Equivale alla combinazione di [. ELENCO](dot-list.md), [. LISTIF](dot-listif.md), e [. LISTMACROALL](dot-listmacroall.md).
