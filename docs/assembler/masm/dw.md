# <a name="dw"></a>DW

Alloca e, facoltativamente, consente di inizializzare una parola (2 byte) di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo ovunque un tipo è consentito. `DW` è un sinonimo di [WORD](../../assembler/masm/word.md).

## <a name="syntax"></a>Sintassi

> [[nome]] Inizializzatore di data Warehouse [[, inizializzatore]]...
