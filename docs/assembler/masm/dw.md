# <a name="dw"></a>DW
Alloca e, facoltativamente, consente di inizializzare una parola (2 byte) di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido. `DW`è un sinonimo di [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
