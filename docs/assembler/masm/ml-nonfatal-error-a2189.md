# <a name="ml-nonfatal-error-a2189"></a>Errore ML non irreversibile A2189

**combinazione non valida con l'allineamento del segmento**

L'allineamento specificato il **ALIGN** o `EVEN` direttiva era maggiore allineamento segmento corrente come specificato dal **segmento** direttiva.
