# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**impossibile aprire il file aperto: nome file**  
  
 L'assemblatore non è possibile aprire un database di origine, include, o file di output.  
  
 Uno dei seguenti elementi può essere una causa:  
  
-   Il file non esiste.  
  
-   Il file è utilizzato da un altro processo.  
  
-   Il nome file non è valido.  
  
-   Un file di sola lettura mediante il nome file di output esiste già.  
  
-   l'unità corrente è completa.  
  
-   La directory corrente è la radice e è completa.  
  
-   Il dispositivo non può essere scritto su.  
  
-   l'unità non è pronta.
