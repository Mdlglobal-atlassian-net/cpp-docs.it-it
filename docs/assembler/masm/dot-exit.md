# <a name="exit"></a>.EXIT
Genera il codice di terminazione.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Note  
 Restituisce facoltativo *espressione* alla shell.
