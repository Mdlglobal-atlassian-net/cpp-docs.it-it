# <a name="exit-32-bit-masm"></a>. EXIT (MASM a 32 bit)

Genera il codice di terminazione. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> . EXIT [[espressione]]

## <a name="remarks"></a>Note

Restituisce un' *espressione* facoltativa alla Shell.
