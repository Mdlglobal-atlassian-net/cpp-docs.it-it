# <a name="ml-fatal-error-a1009"></a>Errore ML irreversibile A1009

**riga troppo lunga**

Una riga in un file di origine supera il limite di 512 caratteri.

Se più righe fisiche vengono concatenate con il carattere di continuazione di riga (\), riga logica risulta è ancora limitata a 512 caratteri.
