# <a name="operator-ptr"></a>Operatore PTR
Forza l'operatore prima di *espressione* per essere considerata come se presentasse specificato `type`. Il secondo operatore specifica un puntatore a `type`.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
