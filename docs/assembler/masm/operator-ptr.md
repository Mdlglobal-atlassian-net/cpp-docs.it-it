# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Il primo operatore impone espressione per essere considerato come se si specifica `type`.  il secondo operatore specifica un puntatore a `type`.  
  
## Sintassi  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
