# <a name="operator-ptr"></a>Operatore PTR

Forza l'operatore prima la *espressione* per essere considerata come se presentasse specificato *tipo*. Il secondo operatore specifica un puntatore alla *tipo*.

## <a name="syntax"></a>Sintassi

> *tipo di* PTR *espressione*
>
> \[*distanza*] PTR *tipo*
