# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La prima riga direttiva di insiemi *durata* e carattere *larghezza* un elenco di programma.  Se non viene fornito alcun argomento, genera un'interruzione di pagina.  la seconda direttiva incrementa il numero di sezione e reimposta il numero di pagina a 1.  
  
## Sintassi  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
