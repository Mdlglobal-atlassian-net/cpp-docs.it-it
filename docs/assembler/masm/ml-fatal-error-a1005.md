# <a name="ml-fatal-error-a1005"></a>Errore ML irreversibile A1005

**limite dell'assembler: tabella dei nomi parametro macro completa**

Troppi parametri, variabili locali o le etichette (macro) sono state definite per una macro. Si è verificato alcun altro spazio, basta la tabella di nomi di macro.

Definire più breve o meno i nomi o Rimuovi macro non necessari.
