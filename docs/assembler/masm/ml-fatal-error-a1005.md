# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**limite dell'assemblatore: macro tabella del nome del parametro completa**  
  
 Troppi parametri, locali, macro o etichette sono definiti per una macro.  È necessario più spazio nella macro tabella dei nomi.  
  
 Definire più breve o meno nomi, o rimuovere le macro non necessari.
