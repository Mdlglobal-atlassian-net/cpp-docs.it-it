# IFDIF]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Concede l'assembly se gli elementi di testo sono diversi.  
  
## Sintassi  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Note  
 se **i** viene fornito, il confronto non rileva la distinzione tra maiuscole e minuscole.  vedere [Un'istruzione IF](../../assembler/masm/if-masm.md) per la sintassi completa.
