# <a name="end-masm"></a>END (MASM)

Contrassegna la fine di un modulo e, facoltativamente, imposta il punto di ingresso del programma da *indirizzare*.

## <a name="syntax"></a>Sintassi

> **Fine** ⟦*Indirizzo*⟧
