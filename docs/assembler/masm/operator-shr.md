# <a name="operator-shr"></a>Operatore SHR

Restituisce il risultato dello spostamento dei bit del numero di bit del *conteggio* a destra dell' *espressione* .

## <a name="syntax"></a>Sintassi

> *conteggio* *espressioni* **SHR**
