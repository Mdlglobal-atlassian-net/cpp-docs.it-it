# operator SHR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce il risultato di scorrere i bit *espressione* destra *conteggio* numero di bit.  
  
## Sintassi  
  
```  
  
expression SHR count  
```
