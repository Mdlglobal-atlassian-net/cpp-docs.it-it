# <a name="operator-lroffset"></a>Operatore LROFFSET

Restituisce l'offset del *espressione*. Uguale allo **OFFSET**, ma genera un offset risolto del caricatore, che consente a Windows spostare i segmenti di codice.

## <a name="syntax"></a>Sintassi

> Espressione LROFFSET
