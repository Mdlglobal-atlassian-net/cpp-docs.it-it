# <a name="operator-lroffset"></a>Operatore LROFFSET
Restituisce l'offset di *espressione*. Uguale a **OFFSET**, ma genera un offset risolto caricatore, che consente di rilocare i segmenti di codice.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
LROFFSET expression  
```
