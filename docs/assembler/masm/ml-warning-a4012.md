# <a name="ml-warning-a4012"></a>Avviso ML A4012

**informazioni sul numeri di riga per segmento senza che la classe 'CODE'**

Si sono verificati istruzioni in un segmento che non è un nome di classe che termina con "CODE". L'assembler non ha generato le informazioni di CodeView per queste istruzioni.

CodeView non è in grado di elaborare i moduli con il codice in segmenti con i nomi delle classi che non terminano con "CODE".
