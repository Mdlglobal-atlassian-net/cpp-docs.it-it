# <a name="codesize"></a>\@codeSize

0 per i modelli **piccoli**, **piccoli**, **Compact**e **Flat** e 1 per i modelli **medium**, **large**e **huge** (equivalenti numerici).

## <a name="syntax"></a>Sintassi

> **\@codeSize**
