# <a name="codesize"></a>@CodeSize
0 per **TINY**, **SMALL**, **COMPACT**, e **FLAT** modelli e 1 per **Media**,  **GRANDE**, e **molto grandi** modelli (equivalgono numerico).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
@CodeSize  
  
```
