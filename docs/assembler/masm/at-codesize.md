# <a name="codesize"></a>@CodeSize

0 per **TINY**, **SMALL**, **COMPACT**, e **FLAT** modelli e 1 per **MEDIUM**,  **GRANDI**, e **molto grandi** modelli (numeric equivalgono).

## <a name="syntax"></a>Sintassi

> @CodeSize
