# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Contrassegna un blocco che venga ripetuto una volta per ogni carattere in `string`, con sostituire corrente del carattere  `parameter` a ogni ripetizione.  
  
## Sintassi  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## Note  
 Equivale a [IRPC](../../assembler/masm/irpc.md).
