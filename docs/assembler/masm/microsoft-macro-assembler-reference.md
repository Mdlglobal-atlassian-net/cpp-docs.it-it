# Microsoft Macro Assembler Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft Macro Assembler \(MASM\) offre numerosi vantaggi rispetto all'assembly inline.  MASM contiene un linguaggio macro che include funzionalità quali l'elaborazione di stringa di un ciclo, operazioni aritmetiche e testo.  MASM fornisce inoltre maggiore controllo sull'hardware perché supporta il set di istruzioni il 386, 486 e processori Pentium.  Utilizzando MASM si può anche ridurre l'overhead di tempo e memoria.
