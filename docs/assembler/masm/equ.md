# EQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La prima direttiva viene assegnato il valore numerico di *espressione* in *nome*.  
  
## Sintassi  
  
```  
  
      name EQU expression  
name EQU <text>  
```  
  
## Note  
 *nome* non può essere ridefinito più avanti.  
  
 la seconda direttiva assegna specificato *testo* in *nome*.  *nome* è possibile assegnare un diverso *testo* più avanti.  vedere [TEXTEQU](../../assembler/masm/textequ.md).
