# INSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Cercare la prima occorrenza di *textitem2* in *textitem1*.  
  
## Sintassi  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## Note  
 Avviare *percorso* è facoltativo.  Ogni elemento di testo può essere una stringa letterale, una costante preceduta da un oggetto `%`in caso contrario, la stringa restituita da una macro funzione.
