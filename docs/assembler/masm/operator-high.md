# <a name="operator-high"></a>Operatore HIGH
Restituisce 8 bit più significativi di 16 bit più bassi di *espressione*. Espressioni MASM sono valori a 64 bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
HIGH expression  
```
