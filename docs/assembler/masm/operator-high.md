# <a name="operator-high"></a>Operatore HIGH

Restituisce gli 8 bit massimi dei 16 bit inferiori dell' *espressione*. Le espressioni MASM sono valori a 64 bit.

## <a name="syntax"></a>Sintassi

> *Espressione* alta
