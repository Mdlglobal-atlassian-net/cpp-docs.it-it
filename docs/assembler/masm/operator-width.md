# <a name="operator-width"></a>Operatore WIDTH
Restituisce la larghezza in bit dell'oggetto corrente *recordfieldname* o *record*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
WIDTH {recordfieldname | record}  
```
