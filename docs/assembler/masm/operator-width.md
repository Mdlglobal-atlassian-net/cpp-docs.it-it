# <a name="operator-width"></a>Operatore WIDTH

Restituisce la larghezza in bit dell'oggetto corrente *recordfieldname* oppure *record*.

## <a name="syntax"></a>Sintassi

> LARGHEZZA {recordfieldname | record}
