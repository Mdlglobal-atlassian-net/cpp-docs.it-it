# operator WIDTH
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce la larghezza in bit corrente *recordfieldname* o *registrare*.  
  
## Sintassi  
  
```  
  
WIDTH {recordfieldname | record}  
```
