# <a name="fardata"></a>.FARDATA
Se usato con [. MODELLO](../../assembler/masm/dot-model.md), viene avviato un segmento di dati decisamente per dati inizializzati (nome del segmento FAR_DATA o *nome*).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
.FARDATA [[name]]  
```
