# .386
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Consente all'assembly delle istruzioni nonprivileged per il processore 80386; disabilita il montaggio delle istruzioni introdotte con i processori successivi.  
  
## Sintassi  
  
```  
  
.386  
  
```  
  
## Note  
 Consente inoltre alle 80387 istruzioni.
