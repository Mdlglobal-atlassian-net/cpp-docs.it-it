# operator THIS
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce un operando del controllo specificato di `type` di cui offset e segmento stima essere uguale al valore del contatore di posizione corrente.  
  
## Sintassi  
  
```  
  
THIS type  
```
