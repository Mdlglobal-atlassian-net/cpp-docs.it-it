# <a name="operator-this"></a>Operatore THIS
Restituisce un operando di specificato `type` i cui valori di offset e segmento sono uguali al valore del contatore di posizione corrente.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
THIS type  
```
