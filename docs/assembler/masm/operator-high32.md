# <a name="operator-high32"></a>Operatore HIGH32
Restituisce i 32 bit alti di *espressione*. Espressioni MASM sono valori a 64 bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
HIGH32 expression  
```
