# ML Warning A4004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**non è AMMETTERE A CS**  
  
 È stato effettuato un tentativo di assumere un valore per il log di CS.  Il CS è sempre impostato al segmento o al gruppo corrente.
