# <a name="operator-mod"></a>Operatore MOD
Restituisce il valore intero del resto (modulo) quando si divide *expression1* da *expression2*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 MOD expression2  
```
