# ML Fatal Error A1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**macro annidamento non corrispondente**  
  
 Una macro non termina prima della fine del file o della direttiva di terminazione [ENDM](../../assembler/masm/endm.md) è stato rilevato all'esterno di macro blocco.  
  
 Una causa di questo errore è prima del punto [.REPEAT](../../assembler/masm/dot-repeat.md) o  [.WHILE](../../assembler/masm/dot-while.md).
