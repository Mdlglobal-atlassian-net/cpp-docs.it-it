# <a name="errdef"></a>.ERRDEF

Genera un errore se il *nome* è un'etichetta, una variabile o un simbolo precedentemente definito.

## <a name="syntax"></a>Sintassi

> **.**  *Nome* ERRDEF ⟦ __,__ *messaggio*⟧
