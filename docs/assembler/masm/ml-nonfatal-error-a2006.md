# <a name="ml-nonfatal-error-a2006"></a>Errore ML non irreversibile A2006
**simbolo non definito: identificatore**  
  
 È stato effettuato un tentativo di utilizzare un simbolo che non è stato definito.  
  
 Potrebbe essersi verificato uno dei valori seguenti:  
  
-   Non è stato definito un simbolo.  
  
-   Un campo non è un membro della struttura specificata.  
  
-   È stato definito un simbolo in un file di inclusione che non è stato incluso.  
  
-   Un simbolo esterno è stato usato senza un [EXTERN](../../assembler/masm/extern-masm.md) o [EXTERNDEF](../../assembler/masm/externdef.md) direttiva.  
  
-   Un nome di simbolo è stato digitato correttamente.  
  
-   Un'etichetta codice locale è stato fatto riferimento di fuori dell'ambito.
