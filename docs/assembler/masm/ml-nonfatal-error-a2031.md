# <a name="ml-nonfatal-error-a2031"></a>Errore ML non irreversibile A2031
**deve essere registro base o indice**  
  
 È stato effettuato un tentativo di utilizzare un registro che non è un registro base o di indice in un'espressione di memoria.  
  
 Questo errore è causato ad esempio, le espressioni seguenti:  
  
```  
[ax]  
[bl]  
```
