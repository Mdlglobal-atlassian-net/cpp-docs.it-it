# ML Nonfatal Error A2031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**deve essere l'indice o il registro base**  
  
 Si è tentato di utilizzare un log che non è una base o un registro di indice in un'espressione di memoria.  
  
 Ad esempio, le espressioni seguenti viene generato questo errore:  
  
```  
[ax]  
[bl]  
```
