# <a name="qword"></a>QWORD

Alloca ed eventualmente Inizializza 8 byte di archiviazione per ogni *inizializzatore*. Può essere usato anche come identificatore di tipo in qualsiasi punto in cui un tipo è valido.

## <a name="syntax"></a>Sintassi

> ⟦*nome*⟧ **QWORD** *inizializzatore* ⟦ __,__ *inizializzatore* ... ⟧
