# <a name="real4"></a>REAL4
Alloca e, facoltativamente, consente di inizializzare un numero a virgola mobile e precisione singola (4 byte) per ogni *inizializzatore*.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```
