# <a name="struct-masm"></a>STRUCT (MASM)

Dichiara un tipo di struttura che include l'oggetto specificato *dichiarazioni di campo*. Ogni campo deve essere una definizione di dati validi. Uguale allo [STRUC](../../assembler/masm/struc.md).

## <a name="syntax"></a>Sintassi

> *nome* STRUCT [[*allineamento*]] [[, non univoco]]<br/>
> *dichiarazioni di campo*<br/>
> *nome* termina
