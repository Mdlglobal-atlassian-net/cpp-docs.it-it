# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

istruzioni e dati inizializzati non supportati nei segmenti di BSS  
  
 Si è tentato di definire dati inizializzati all'interno di una sezione di BSS.  Una sezione di BSS viene definita come una classe il cui nome è BSS.  Sono inclusi il segmento semplificato `.data?`.
