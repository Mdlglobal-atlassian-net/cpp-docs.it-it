# <a name="ml-warning-a4014"></a>Avviso ML A4014
istruzioni e dati inizializzati non è supportati in segmenti BSS  
  
 È stato effettuato un tentativo per definire i dati inizializzati all'interno di una sezione BSS.  Una sezione BSS è definita come una classe il cui nome è BSS.  Ciò include il segmento semplificato `.data?`.
