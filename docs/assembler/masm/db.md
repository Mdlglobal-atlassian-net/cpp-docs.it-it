# <a name="db"></a>DB
Alloca e, facoltativamente, consente di inizializzare un byte di spazio di archiviazione per ogni `initializer`. `DB`è un sinonimo di [BYTE](../../assembler/masm/byte-masm.md)  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Note  
 Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido.
