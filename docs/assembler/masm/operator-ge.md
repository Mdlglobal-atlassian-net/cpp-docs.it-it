# <a name="operator-ge"></a>Operatore GE
Restituisce true (-1) se *expression1* è maggiore o uguale a *expression2*, oppure restituisce false (0) se non lo è.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 GE expression2  
```
