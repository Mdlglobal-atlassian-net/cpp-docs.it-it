# <a name="data"></a>dati \@

Nome del gruppo di dati predefinito. Restituisce DGROUP per tutti i modelli tranne **Flat**. Restituisce **Flat** sotto il modello di memoria **piatta** (macro di testo).

## <a name="syntax"></a>Sintassi

> **dati \@**
