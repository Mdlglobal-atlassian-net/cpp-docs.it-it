# <a name="data"></a>@data

Il nome del gruppo di dati predefinito. Restituisce tutti i modelli, ad eccezione DGROUP **FLAT**. Viene valutata **FLAT** sotto il **FLAT** il modello di memoria (macro di testo).

## <a name="syntax"></a>Sintassi

> @data
