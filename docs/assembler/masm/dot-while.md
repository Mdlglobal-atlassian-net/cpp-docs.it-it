# <a name="while-32-bit-masm"></a>. WHILE (MASM a 32 bit)

Genera il codice che esegue il blocco di *istruzioni* mentre *Condition* rimane true. (solo MASM a 32 bit).

## <a name="syntax"></a>Sintassi

> **.**  *Condizione*\
> \ di *istruzioni*
> **.ENDW**
