# <a name="while"></a>.WHILE
Genera il codice che esegue il blocco di `statements` mentre `condition` rimane true.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
