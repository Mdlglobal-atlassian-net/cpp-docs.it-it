# .RADIX
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Imposta la radice predefinita, compreso tra 2 e 16, al valore di *espressione*.  
  
## Sintassi  
  
```  
  
.RADIX expression  
```
