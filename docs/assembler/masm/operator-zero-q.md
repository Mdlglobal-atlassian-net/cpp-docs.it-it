# <a name="operator-zero"></a>Operatore ZERO?

Stato del flag di zero. Utilizzato solo all'interno [. IF](../../assembler/masm/dot-if.md), [. Sebbene](../../assembler/masm/dot-while.md), o [. Ripetere](../../assembler/masm/dot-repeat.md) blocca e valutate in fase di esecuzione, non in fase di assembly.

## <a name="syntax"></a>Sintassi

> ZERO?
