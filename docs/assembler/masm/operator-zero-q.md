# <a name="operator-zero"></a>Operatore ZERO?

Stato del flag zero. Utilizzato solo all'interno di [. SE](../../assembler/masm/dot-if.md), [. WHILE](../../assembler/masm/dot-while.md)o [. Ripetere](../../assembler/masm/dot-repeat.md) i blocchi e valutati in fase di esecuzione, non in fase di assembly.

## <a name="syntax"></a>Sintassi

> **ZERO?**
