# <a name="listif"></a>.LISTIF

Comincia a elencare delle istruzioni nei blocchi condizionali false.

## <a name="syntax"></a>Sintassi

> .LISTIF

## <a name="remarks"></a>Note

Uguale allo [. LFCOND](../../assembler/masm/dot-lfcond.md).
