# .LISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Avvia l'elenco delle istruzioni in blocchi condizionali false.  
  
## Sintassi  
  
```  
  
.LISTIF  
  
```  
  
## Note  
 Equivale a [.LFCOND](../../assembler/masm/dot-lfcond.md).
