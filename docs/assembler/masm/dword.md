# DWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloca e facoltativamente inizializza una doppia parola \(4 byte\) di archiviazione per ciascuno `initializer`.  
  
## Sintassi  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## Note  
 Può essere utilizzato come un identificatore di tipo in un tipo è valido.
