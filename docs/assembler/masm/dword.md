# <a name="dword"></a>DWORD
Alloca e, facoltativamente, consente di inizializzare una parola doppia (4 byte) di spazio di archiviazione per ogni `initializer`.  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Note  
 Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido.
