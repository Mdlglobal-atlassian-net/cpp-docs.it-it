# <a name="operator-lt"></a>Operatore LT
Restituisce true (-1) se *expression1* è minore di *expression2*, oppure restituisce false (0) se non lo è.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
expression1 LT expression2  
```
