# <a name="sbyte-masm"></a>SBYTE (MASM)
Alloca e, facoltativamente, consente di inizializzare un signed byte di spazio di archiviazione per ogni `initializer`. Può anche essere utilizzato come identificatore di tipo in qualsiasi punto un tipo è valido.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
