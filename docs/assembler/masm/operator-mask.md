# operator MASK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

restituisce una maschera di bit in cui i bit in *recordfieldname* o *registrare* vengono impostati e tutti gli altri bit vengono cancellati.  
  
## Sintassi  
  
```  
  
MASK {recordfieldname | record}  
```
