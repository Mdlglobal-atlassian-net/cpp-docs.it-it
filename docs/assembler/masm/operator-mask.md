# <a name="operator-mask"></a>Operatore MASK
Restituisce una maschera di bit in cui i bit nel *recordfieldname* o *record* vengono impostati e che tutti gli altri bit siano deselezionate.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
MASK {recordfieldname | record}  
```
