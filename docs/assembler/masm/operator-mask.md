# <a name="operator-mask"></a>Operatore MASK

Restituisce una maschera di bit in cui i bit nel *recordfieldname* oppure *record* vengono impostate e tutti gli altri bit vengono cancellate.

## <a name="syntax"></a>Sintassi

> MASCHERA {recordfieldname | record}
