# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

imposta il tipo di *etichetta* a short.  tutto passa a *etichetta* deve essere uguale a short \(nell'intervallo tra \- 128 e \+127 byte dall'istruzione di passare a *etichetta*\).  
  
## Sintassi  
  
```  
  
SHORT label  
```
