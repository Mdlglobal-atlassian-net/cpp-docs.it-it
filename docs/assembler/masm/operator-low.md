# operator LOW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Restituisce i bit di minimo di 8 *espressione*.  le espressioni di MASM sono valori a 64 bit.  
  
## Sintassi  
  
```  
  
LOW expression
```  
