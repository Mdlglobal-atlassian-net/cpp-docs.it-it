# <a name="operator-low"></a>Operatore LOW
Restituisce gli 8 bit inferiori del *espressione*. Espressioni MASM sono valori a 64 bit.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
LOW expression  
```
