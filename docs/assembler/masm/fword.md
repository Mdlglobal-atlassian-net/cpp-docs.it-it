# <a name="fword"></a>FWORD

Alloca e inizializza facoltativamente 6 byte di spazio di archiviazione per ogni `initializer`.

## <a name="syntax"></a>Sintassi

> [[nome]] Inizializzatore FWORD [[, inizializzatore]]...

## <a name="remarks"></a>Note

Inoltre, può essere utilizzato come identificatore di tipo ovunque un tipo è consentito.
