# FWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

alloca e facoltativamente inizializza 6 byte di archiviazione per ciascuno `initializer`.  
  
## Sintassi  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## Note  
 È inoltre possibile utilizzare come identificatore di tipo ogni volta che un tipo è valido.
