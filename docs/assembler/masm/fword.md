# <a name="fword"></a>FWORD
Alloca e, facoltativamente, inizializza 6 byte di spazio di archiviazione per ogni `initializer`.  
  
## <a name="syntax"></a>Sintassi  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Note  
 Può essere utilizzata anche come un identificatore di tipo in qualsiasi punto un tipo è valido.
