# <a name="intel39s-mmx-instruction-set"></a>Set&#39;di istruzioni MMX Intel s

**Sezione specifica Microsoft**

Il compilatore C++ Microsoft consente di usare il set di istruzioni di Intel MMX (Multimedia Extension) nell'assembler inline. Le istruzioni MMX sono supportate anche dal disassembly del debugger. Il compilatore genera un messaggio di avviso se la funzione contiene istruzioni MMX, ma non contiene un'istruzione EMMS per impostare lo stato dei contenuti multimediali su vuoto. Per ulteriori informazioni, vedere il sito Web di Intel.
