# <a name="intel39s-mmx-instruction-set"></a>Intel&#39;Set di istruzioni MMX s

**Sezione specifica Microsoft**

Microsoft C++ compilatore consente di utilizzare set nell'assembler inline di istruzioni di MMX (multimedia extension) di Intel. Le istruzioni MMX sono supportate anche dal disassembly del debugger. Il compilatore genera un messaggio di avviso se la funzione contiene istruzioni MMX, ma non contiene un'istruzione EMMS per impostare lo stato dei contenuti multimediali su vuoto. Per ulteriori informazioni, vedere il sito Web di Intel.
