# Risoluzione dei problemi relativi alle eccezioni: System.Reflection.InvalidFilterCriteriaException
Eccezione generata nel metodo <xref:System.Type.FindMembers%2A> quando i criteri del filtro non sono validi per il tipo di filtro in uso.
