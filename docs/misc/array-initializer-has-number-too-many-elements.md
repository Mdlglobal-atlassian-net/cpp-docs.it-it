# L&#39;inizializzatore di matrice ha &lt;numero&gt; elementi in eccesso
L'inizializzatore di matrice contiene troppi elementi.  
  
 **ID errore:** BC30568  
  
### Per correggere l'errore  
  
-   Usare `ReDim` per modificare le dimensioni della matrice.
