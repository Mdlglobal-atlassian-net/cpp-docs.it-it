# Avviso di compilazione progetto PRJ0045
La proprietà 'Output' della regola di compilazione personalizzata 'rule' assegnata al file 'file' non è impostata. La regola verrà ignorata.  
  
 La proprietà Output non è stata definita per una regola di compilazione personalizzata. L'output dell'azione di compilazione è necessaria per le regole di compilazione personalizzate.
