# Avviso del compilatore di risorse RC4006
previsto un identificatore dopo \#undef  
  
 Il nome dell'identificatore la cui definizione deve essere rimossa non è stato specificato con la direttiva `#undef`. La direttiva `#undef` è stata ignorata.
