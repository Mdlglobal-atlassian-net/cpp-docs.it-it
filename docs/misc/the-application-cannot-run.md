# The application cannot run.
Memoria insufficiente per eseguire Visual Studio.  
  
### Per correggere l'errore  
  
1.  Verificare che il computer soddisfi i requisiti minimi per l'esecuzione di Visual Studio.  
  
2.  Chiudere altri programmi in esecuzione sul computer per liberare memoria.  
  
     \-oppure\-  
  
     Reinstallare Visual Studio.
