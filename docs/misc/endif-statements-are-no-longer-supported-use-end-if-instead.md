# Le istruzioni &#39;EndIf&#39; non sono pi&#249; supportate. Usare &#39;End If&#39;
L'istruzione `EndIf` non è più supportata.  
  
 **ID errore:** BC30826  
  
### Per correggere l'errore  
  
-   Usare `End If` per terminare le istruzioni `If`.
