# piecewise_linear_distribution::densities
Restituisce le densità archiviate per la distribuzione.  
  
## Sintassi  
  
```  
std::vector<double> densities() const;  
```  
  
## Note  
 La funzione membro restituisce i valori di densità archiviati.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
