# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
Un'eccezione <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> viene generata quando il codice tenta di accedere a un controllo che non è presente nel documento. Il controllo potrebbe essere stato rimosso dall'utente finale o da altro codice. Non vi è alcun modo di verificare la presenza del controllo prima di tentare di accedervi; se il controllo è stato rimosso la soluzione non funzionerà.
