# Risoluzione dei problemi relativi alle eccezioni: System.IdentityModel.Selectors.CardSpaceException
Quando si sono verificate uno o più eccezioni a livello del servizio CardSpace, viene generata un'eccezione <xref:System.IdentityModel.Selectors.CardSpaceException>. La causa dell'errore è indicata nel registro eventi.  
  
## Note  
 Tra le condizioni che causano questa eccezione vi è la ricorrenza di un errore durante il caricamento di un archivio o di un registro.
