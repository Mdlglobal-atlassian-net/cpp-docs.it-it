# Errore di compilazione progetto PRJ0043
La proprietà 'Output' della regola di compilazione personalizzata 'rule' assegnata al file 'file' non è valida. La proprietà contiene 'string' che restituisce 'value'.  
  
 La proprietà Output ha restituito una stringa vuota. L'output dell'azione di compilazione è necessaria per le regole di compilazione personalizzate.
