# Errore interno del compilatore: il generatore del codice ha ricevuto un input non valido
Il compilatore [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] non riesce a interpretare correttamente uno dei file scritti internamente.  
  
 **ID errore:** BC30011  
  
### Per correggere l'errore  
  
1.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
2.  Se l'errore si ripresenta, reinstallare il compilatore [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)].  
  
3.  Se l'errore persiste dopo la reinstallazione, raccogliere informazioni sulla situazione contingente e informare il Servizio supporto tecnico Microsoft.
