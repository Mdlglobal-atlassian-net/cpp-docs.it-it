# Impossibile ripristinare alcune associazioni di file predefinite. Nota: per poter eseguire questa operazione, &#232; necessario essere un amministratore o un membro del gruppo Power Users su questo computer.
Questo errore si verifica quando viene usata l'opzione \/AssociateFiles della riga di comando di devenv, ma i diritti utente correnti non consentono di accedere alla sezione HKEY\_CLASSES\_ROOT del Registro di sistema. Quando si esegue [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] in [!INCLUDE[wiprlhext](../c-runtime-library/reference/includes/wiprlhext_md.md)], Ã¨ necessario eseguire devenv come amministratore per usare l'opzione \/AssociateFiles \(devenv.exe\).  
  
### Per correggere l'errore  
  
-   Passare alle credenziali di amministratore e riprovare.
