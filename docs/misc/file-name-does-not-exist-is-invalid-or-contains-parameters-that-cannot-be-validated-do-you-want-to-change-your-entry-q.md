# Il nome file specificato non esiste, non &#232; valido o contiene parametri che non possono essere convalidati. Modificare la voce?
L'errore si verifica in genere quando si aggiunge un percorso o un nome file non corretto nella finestra di dialogo **Aggiungi programma**.  
  
### Per correggere l'errore  
  
1.  Scegliere **Sfoglia** per selezionare il file corretto.  
  
     \-oppure\-  
  
2.  Digitare nuovamente il percorso e il nome del file.
