# Operatore ^ (C++)
L'operatore `^` ha due significati in Visual C\+\+, a seconda del contesto.  Per ulteriori informazioni, vedere:  
  
-   [Operatore OR bit per bit esclusivo: ^](../cpp/bitwise-exclusive-or-operator-hat.md), per l'utilizzo in tutte le versioni di C\+\+.  
  
-   [Operatore handle a oggetto \(^\)](../windows/handle-to-object-operator-hat-cpp-component-extensions.md), per l'utilizzo in [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] e [!INCLUDE[cppcli](../build/reference/includes/cppcli_md.md)]
