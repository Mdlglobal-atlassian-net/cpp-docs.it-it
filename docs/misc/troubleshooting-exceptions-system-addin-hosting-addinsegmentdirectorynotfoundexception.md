# Risoluzione dei problemi relativi alle eccezioni: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
Quando nella struttura di directory della pipeline manca una directory di segmento, viene generata un'eccezione <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException>.  
  
## Note  
 Quando si attiva un componente aggiuntivo, il sistema crea la relativa pipeline di collegamento all'host. Come descritto in [Requisiti di sviluppo delle pipeline](http://msdn.microsoft.com/it-it/ef9fa986-e80b-43e1-868b-247f4c1d9da5), il sistema dei componenti aggiuntivi richiede che i segmenti della pipeline si trovino in una determinata struttura di directory avente i nomi di directory specificati.
