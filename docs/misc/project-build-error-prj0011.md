# Errore di compilazione progetto PRJ0011
Le risorse di sistema potrebbero non essere sufficienti. Impossibile creare un evento necessario per avviare la compilazione.  
  
 Se le risorse di sistema sono insufficienti:  
  
-   Chiudere alcune delle applicazioni in esecuzione.  
  
-   Riavviare Visual Studio per liberare le risorse.  
  
-   Riavviare il computer.
