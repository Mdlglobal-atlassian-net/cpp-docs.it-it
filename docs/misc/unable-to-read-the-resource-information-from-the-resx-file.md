# Unable to read the resource information from the resx file
Non è stato possibile leggere un file RESX.  
  
 Alla fine del messaggio verrà aggiunto, se disponibile, il motivo specificato dal sistema operativo.  Verrà inoltre aggiunto il nome del file che non è stato possibile leggere.  
  
 Alcune delle possibili cause di questo errore sono rappresentate dalla mancanza di autorizzazioni per la lettura dei file o, per i progetti Web, dall'interruzione della comunicazione con il server Web.  
  
 L'errore comporterà la mancata riuscita del processo di compilazione.
