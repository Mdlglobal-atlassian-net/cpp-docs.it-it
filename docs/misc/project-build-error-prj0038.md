# Errore di compilazione progetto PRJ0038
IIS non è stato trovato nella directory radice. Per il funzionamento dello strumento di distribuzione Web è necessario che sia installato IIS.  
  
 Per la distribuzione Web è richiesta l'installazione di IIS nel computer.
