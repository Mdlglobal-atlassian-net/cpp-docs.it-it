# Impossibile generare la risorsa binaria. Impossibile aprire il file di output &#39;file&#39;. &lt;motivo&gt;
Una copia non aggiornata del file con estensione resources non può essere eliminata dal sistema del progetto. Durante la preparazione delle risorse, il sistema del progetto trasformerà i file XML con estensione resx in file con estensione resources binari per Gestione risorse di .NET.  
  
 Il motivo specifico del sistema operativo riportato alla fine del testo del messaggio di errore fornirà informazioni utili.  
  
 **Per correggere l'errore**  
  
-   Riavviare [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Questo errore impedirà la riuscita del processo di compilazione.
