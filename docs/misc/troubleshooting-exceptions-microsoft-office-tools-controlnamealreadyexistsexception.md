# Risoluzione dei problemi relativi alle eccezioni: Microsoft.Office.Tools.ControlNameAlreadyExistsException
L'eccezione <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> viene generata quando si tenta di creare a livello di codice un controllo con lo stesso nome di un altro controllo del progetto.  
  
## Suggerimenti associati  
 **Utilizzare un nome diverso per uno dei controlli.**  
 I nomi dei controlli devono essere univoci.
