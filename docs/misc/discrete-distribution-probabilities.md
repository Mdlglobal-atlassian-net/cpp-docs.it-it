# discrete_distribution::probabilities
Restituisce le probabilità archiviate per la distribuzione.  
  
## Sintassi  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Note  
 Questa funzione membro restituisce i valori di probabilità archiviati.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
