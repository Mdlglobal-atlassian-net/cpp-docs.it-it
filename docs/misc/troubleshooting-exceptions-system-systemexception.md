# Risoluzione dei problemi relativi alle eccezioni: System.SystemException
Definisce la classe di base per eccezioni predefinite nello spazio dei nomi `System`.  
  
## Note  
 Questa classe viene fornita per consentire di distinguere le eccezioni definite dal sistema da quelle definite dalle applicazioni.
