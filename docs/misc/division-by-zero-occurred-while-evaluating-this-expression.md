# Si &#232; verificata una divisione per zero durante la valutazione dell&#39;espressione
Nella valutazione di questa espressione si è tentato di dividere un numero per zero.  
  
 **ID errore:** BC30542  
  
### Per correggere l'errore  
  
1.  Controllare i valori costanti e variabili per stabilire il motivo per cui nel denominatore dell'espressione è presente lo zero.
