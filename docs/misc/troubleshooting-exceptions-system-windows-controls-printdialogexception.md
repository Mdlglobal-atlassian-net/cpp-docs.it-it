# Risoluzione dei problemi relativi alle eccezioni: System.Windows.Controls.PrintDialogException
L'eccezione <xref:System.Windows.Controls.PrintDialogException> viene generata quando si verifica una condizione di errore durante l'apertura, l'accesso o l'utilizzo di un oggetto <xref:System.Windows.Controls.PrintDialog>.
