# Errore MSB8020 di MSBuild
Gli strumenti di compilazione per *\<piattaforma\>* \(Set di strumenti della piattaforma \= '*\<versione\>*'\) non sono stati trovati. Per compilare usando gli strumenti di compilazione di *\<versione\>*, installare gli strumenti di compilazione di *\<piattaforma\>*.
