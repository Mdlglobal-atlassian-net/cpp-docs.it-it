# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.PoisonMessageException
Quando un messaggio viene considerato non elaborabile, viene generata un'eccezione <xref:System.ServiceModel.PoisonMessageException>. Un messaggio è considerato non elaborabile se più tentativi di recapitarlo non sortiscono alcun effetto.
