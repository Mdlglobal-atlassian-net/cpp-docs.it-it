# MSBuild Error MSB3175
**MSB3175: valore non valido per '\<file\>' dell'elemento '\<attività di generazione\>'.**  
  
 Questo avviso viene generato quando il processo di compilazione non riconosce il valore di numerose proprietà di elementi di attività basati sull'enumerazione come `AssemblyType`, `DependencyType`, `FileType` o `TargetZone`.  Si applica ai manifesti dell'applicazione e a quelli di distribuzione
