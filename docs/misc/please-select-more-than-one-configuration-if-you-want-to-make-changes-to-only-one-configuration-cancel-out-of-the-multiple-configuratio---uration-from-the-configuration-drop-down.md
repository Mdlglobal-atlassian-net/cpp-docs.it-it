# Selezionare pi&#249; di una configurazione. Se si desidera modificare solo una configurazione, chiudere la finestra di dialogo Configurazioni multiple scegliendo Annulla e selezionare la configurazione desiderata dall&#39;elenco a discesa.
In genere questo errore si verifica quando viene selezionata un'unica configurazione di soluzione nella finestra di dialogo **Configurazioni multiple**.  
  
### Per correggere l'errore  
  
1.  Selezionare altre configurazioni di soluzione.  
  
     \-oppure\-  
  
2.  Se si desidera aggiornare una singola configurazione, scegliere Annulla e selezionare la configurazione di soluzione corretta dall'elenco Configurazione nella finestra di dialogo **Propriet√† soluzione**.
