# CA0505
La proprietà deprecata sarà ignorata perché è definita la proprietà superceding.  
  
 L'avviso CA505 viene generato quando sia la proprietà di analisi del codice deprecata sia quella superceding contengono valori in un file di configurazione della build o del progetto.  Per risolvere il problema, rimuovere la proprietà deprecata.
