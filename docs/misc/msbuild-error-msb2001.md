# MSBuild Error MSB2001
**L'elemento \<{0}\> non contiene l'attributo obbligatorio "{1}".**  
  
 Il progetto in corso di conversione è stato modificato, danneggiato o non è stato creato utilizzando [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
### Per correggere l'errore  
  
-   Verificare se il file di progetto è stato modificato o danneggiato.  In tal caso, aprire il progetto nella versione di [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] in cui è stato creato, salvarlo, quindi tentare nuovamente la conversione.
