# MSBuild Error MSB3186
**MSB3186: impossibile dedurre un'identità assembly per il manifesto generato dai parametri di input dell'attività.**  
  
 Questo errore si verifica quando il processo di compilazione non è in grado di dedurre un nome assembly per il manifesto dell'applicazione o di distribuzione.  Il nome assembly non viene dato esplicitamente; non vi è identità nel manifesto di base e non è specificata nemmeno l'identità del punto di ingresso.
