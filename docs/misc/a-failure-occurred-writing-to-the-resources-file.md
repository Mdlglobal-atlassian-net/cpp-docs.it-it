# A failure occurred writing to the resources file
Non è stato possibile scrivere il file di output RESOURCES.  Durante la preparazione delle risorse, i file XML con estensione RESX verranno convertiti in file RESOURCES binari per il gestore di risorse .NET.  
  
 Alla fine del messaggio verrà aggiunto, se disponibile, il motivo specificato dal sistema operativo.  Verrà inoltre aggiunto il nome del file RESX.  
  
 L'errore comporterà la mancata riuscita del processo di compilazione.
