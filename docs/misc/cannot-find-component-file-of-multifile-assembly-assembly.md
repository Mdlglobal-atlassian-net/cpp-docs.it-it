# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
Non è stato possibile trovare un file in un assembly su più file.  Un assembly su più file fa riferimento a uno o più file, quali moduli o file di risorsa.  
  
 L'assembly copiato potrebbe essere un riferimento diretto, visibile nel nodo **Riferimenti** in Esplora soluzioni, o un assembly che rappresenta una dipendenza di un riferimento diretto.  
  
 **Per correggere l'errore**  
  
-   Reinstallare il componente.  
  
     È possibile che il progetto venga compilato ma non sia eseguibile.
