# Errore di compilazione progetto PRJ0012
Le risorse di sistema potrebbero non essere sufficienti. Impossibile creare un thread necessario per avviare la compilazione.  
  
 Se le risorse di sistema sono insufficienti:  
  
-   Chiudere alcune delle applicazioni in esecuzione.  
  
-   Riavviare Visual Studio per liberare le risorse.  
  
-   Riavviare il computer.
