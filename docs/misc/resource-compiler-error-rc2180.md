# Errore del compilatore di risorse RC2180
impossibile aprire il file temporaneo  
  
 Il compilatore risorse\/Visual C\+\+ non è riuscito ad aprire un file temporaneo. È probabile che non si abbiano le autorizzazioni di scrittura per la directory o che la directory non esista. Il compilatore di risorse\/Visual C\+\+ tenta di usare questi file nella directory specificata dalla variabile di ambiente **TMP** o nella directory corrente se non è specificata.
