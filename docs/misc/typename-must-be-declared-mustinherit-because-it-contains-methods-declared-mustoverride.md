# &#39;&lt;nometipo&gt;&#39; deve essere dichiarato come &#39;MustInherit&#39; perch&#233; contiene metodi dichiarati come &#39;MustOverride&#39;
I tipi che contengono i membri `MustOverride` devono essere dichiarati come `MustInherit`.  
  
 **ID errore:** BC31411  
  
### Per correggere l'errore  
  
-   Aggiungere il modificatore `MustInherit` al tipo.
