# Package &lt;name&gt; failed to load.
Non è possibile aprire il package.  Generalmente questo errore si verifica se il package non è stato registrato correttamente, nel package manca una DLL, la DLL è danneggiata o manca una DLL proveniente da un altro componente richiesto dal package.  
  
### Per correggere l'errore  
  
1.  Reinstallare il prodotto che fornisce il package.
