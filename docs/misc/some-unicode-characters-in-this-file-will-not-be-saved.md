# Some Unicode characters in this file will not be saved.
Questo errore si verifica in genere quando un file contiene caratteri che possono essere salvati solo applicando una code page Unicode.  
  
### Per salvare il file con caratteri Unicode  
  
1.  Fare clic su **Annulla**.  
  
2.  Scegliere **Opzioni di salvataggio avanzate** dal menu **File**.  
  
3.  Scegliere la codifica appropriata per il file nella finestra di dialogo **Opzioni di salvataggio avanzate**.
