# Risoluzione dei problemi relativi alle eccezioni: System.Xml.Schema.XmlSchemaValidationException
Eccezione generata quando vengono rilevati errori o avvisi di convalida dello schema XSD \(XML Schema Definition\) in un documento XML sottoposto a convalida.
