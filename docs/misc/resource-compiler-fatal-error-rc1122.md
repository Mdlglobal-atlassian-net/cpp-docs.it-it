# Errore irreversibile del compilatore di risorse RC1122
Errore di I\/O durante la scrittura del file  
  
 Il compilatore di risorse non è riuscito a scrivere in un file.  
  
### Per risolverlo è possibile verificare le seguenti cause possibili  
  
1.  Spazio su disco insufficiente. Lo spazio disponibile deve essere almeno due volte la dimensione del file eseguibile che si sta creando.  
  
2.  Volume di sola lettura.  
  
3.  Settore danneggiato.  
  
4.  Violazione di condivisione.
