# &#39;Exit Operator&#39; non &#232; valido. Usare &#39;Return&#39; per uscire da un operatore
Un'istruzione `Exit Operator` viene visualizzata in una routine `Operator`.  
  
 Ãˆ necessario usare un'[Return Statement](../Topic/Return%20Statement%20\(Visual%20Basic\).md) per uscire da una routine `Operator`. L'[Exit Statement](../Topic/Exit%20Statement%20\(Visual%20Basic\).md) non accetta la parola chiave `Operator` e l'istruzione `End Operator` non restituisce il controllo al codice chiamante.  
  
 **ID errore:** BC33008  
  
### Per correggere l'errore  
  
-   Sostituire l'istruzione `Exit Operator` con un'istruzione `Return`.
