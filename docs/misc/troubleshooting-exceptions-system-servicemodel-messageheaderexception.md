# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.MessageHeaderException
Quando le aspettative riguardo le intestazioni di un messaggio SOAP non vengono soddisfatte al momento dell'elaborazione del messaggio, viene generata un'eccezione <xref:System.ServiceModel.MessageHeaderException>.
