# Risoluzione dei problemi relativi alle eccezioni: System.IdentityModel.Selectors.UntrustedRecipientException
Quando un utente esamina le informazioni di un'entità che sta richiedendo un token e decide di non considerare attendibile tale entità, viene generata un'eccezione <xref:System.IdentityModel.Selectors.UntrustedRecipientException>. L'utente fa clic su **Disabilita Windows CardSpace** nell'interfaccia utente di CardSpace.
