# Errore del compilatore di risorse RC2145
ID LINGUA PRINCIPALE troppo grande  
  
 L'ID LINGUA PRINCIPALE deve essere un ID lingua esadecimale valido. Per un elenco di ID lingua validi, vedere [Stringhe relative a lingue e paesi](../c-runtime-library/locale-names-languages-and-country-region-strings.md) in *Riferimenti alla libreria di runtime*.
