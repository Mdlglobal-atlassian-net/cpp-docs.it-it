# gamma_distribution::alpha
Restituisce il parametro di distribuzione alfa.  
  
## Sintassi  
  
```  
RealType alpha();  
```  
  
## Note  
 La funzione membro restituisce il valore `alpha` archiviato.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
