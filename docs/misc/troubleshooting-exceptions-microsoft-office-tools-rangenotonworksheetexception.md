# Risoluzione dei problemi relativi alle eccezioni: Microsoft.Office.Tools.RangeNotOnWorksheetException
L'eccezione <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> viene generata quando un controllo viene creato a livello di codice con un intervallo non presente nel foglio di lavoro corrispondente alla raccolta di controlli.  
  
## Suggerimenti associati  
 Accertarsi che l'intervallo specificato corrisponda al foglio di lavoro corretto.
