# Risoluzione dei problemi relativi alle eccezioni: System.IO.DirectoryNotFoundException
Un'eccezione <xref:System.IO.DirectoryNotFoundException> viene generata quando non viene trovata parte del percorso di un file o una directory.  
  
## Suggerimenti associati  
 **Verificare che la directory esista nel percorso specificato.**  
 Controllare che la directory esista nel percorso specificato.  
  
 **Quando si utilizzano i percorsi relativi, accertarsi che la directory corrente sia corretta.**  
 Se la directory corrente non è corretta, è possibile che anche il percorso non risulti corretto.
