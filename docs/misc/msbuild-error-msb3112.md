# MSBuild Error MSB3112
**MSB3112: due o più assembly hanno la stessa identità '\<assembly\>'.**  
  
 Questo avviso viene generato quando più assembly di riferimento hanno la stessa identità.  
  
### Per correggere l'errore  
  
-   Rimuovere uno degli assembly di riferimento dal progetto.
