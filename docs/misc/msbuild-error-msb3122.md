# MSBuild Error MSB3122
**MSB3122: per utilizzare l'associazione di file è necessaria l'attendibilità totale.**  
  
 La pubblicazione di un'applicazione che configura associazioni di file richiede che l'applicazione sia completamente attendibile.  
  
### Per correggere l'errore  
  
-   Attivare le impostazioni di sicurezza ClickOnce e impostare l'applicazione su un'applicazione completamente attendibile.  Per ulteriori informazioni, vedere [How to: Enable ClickOnce Security Settings](../Topic/How%20to:%20Enable%20ClickOnce%20Security%20Settings.md).
