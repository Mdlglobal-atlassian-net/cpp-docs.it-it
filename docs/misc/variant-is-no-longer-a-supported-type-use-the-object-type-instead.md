# Il tipo &#39;Variant&#39; non &#232; pi&#249; supportato. Usare il tipo &#39;Object&#39;
Il tipo di dati `Object` sostituisce `Variant`.  
  
 **ID errore:** BC30804  
  
### Per correggere l'errore  
  
-   Dichiarare l'elemento come `Object`.
