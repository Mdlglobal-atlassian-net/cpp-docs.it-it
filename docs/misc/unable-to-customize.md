# Unable to customize.
Questo errore viene generato quando si tenta di personalizzare menu o barre degli strumenti durante il debug di un'applicazione.  
  
### Per correggere l'errore  
  
-   Attendere il completamento del debug prima di eseguire operazioni di personalizzazione dei menu o delle barre degli strumenti.  
  
     \-oppure\-  
  
     Interrompere il debug, quindi eseguire le operazioni di personalizzazione dei meno o delle barre degli strumenti.
