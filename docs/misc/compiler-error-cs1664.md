# Errore del compilatore CS1664
Il buffer a dimensione fissa di lunghezza 'length' e di tipo 'type' è troppo grande  
  
 La grandezza massima di un buffer di dimensione fissa è 2^31\=268435455, come risulta dal prodotto della lunghezza per la dimensione dell'elemento.
