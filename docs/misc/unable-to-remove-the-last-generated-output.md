# Unable to remove the last generated output
L'ultimo output generato noto non è stato rimosso dal disco e\/o dal progetto.  
  
 Se uno strumento personalizzato restituisce un errore, l'ultimo output generato, se presente, verrà rimosso dal disco e dal progetto.  
  
 La situazione si verifica nel caso in cui l'output generato sia bloccato.  
  
 **Per correggere l'errore**  
  
-   Chiudere e riavviare [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].
