# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
Eccezione generata quando lo stato serializzato XML dell'oggetto nella cache non corrisponde alle informazioni sul tipo nel manifesto dei dati memorizzati nella cache.
