# Errore del compilatore CS0635
'attribute': System.Interop.UnmanagedType.CustomMarshaller richiede gli argomenti denominati ComType e Marshal  
  
 È necessario che gli argomenti **ComType** e **Marshal** siano specificati quando il formato di marshalling è **System.Interop.UnmanagedType.CustomMarshaller**.
