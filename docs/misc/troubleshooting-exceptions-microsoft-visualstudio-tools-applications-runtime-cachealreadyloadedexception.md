# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CacheAlreadyLoadedException
Eccezione generata quando viene effettuato un tentativo di avviare la memorizzazione nella cache dopo che la cache è già stata caricata.
