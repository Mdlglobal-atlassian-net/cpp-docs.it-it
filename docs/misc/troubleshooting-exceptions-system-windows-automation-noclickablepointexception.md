# Risoluzione dei problemi relativi alle eccezioni: System.Windows.Automation.NoClickablePointException
Un'eccezione <xref:System.Windows.Automation.NoClickablePointException> contiene informazioni sull'eccezione generata quando l'oggetto <xref:System.Windows.Automation.AutomationElement.GetClickablePoint%2A> viene chiamato in un oggetto <xref:System.Windows.Automation.AutomationElement> di interfaccia utente privo di un punto su cui fare clic.  
  
## Note  
 Questa eccezione può essere generata quando la finestra dell'applicazione è ridotta a icona o quando l'elemento è esterno allo schermo.
