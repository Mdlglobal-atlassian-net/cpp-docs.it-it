# Risoluzione dei problemi relativi alle eccezioni: System.Runtime.InteropServices.MarshalDirectiveException
Eccezione generata dal gestore di marshalling quando rileva un oggetto <xref:System.Runtime.InteropServices.MarshalAsAttribute> non supportato.
