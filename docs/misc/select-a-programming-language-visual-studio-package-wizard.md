# Selezionare un linguaggio di programmazione, Creazione guidata pacchetto di Visual Studio
Questo passaggio consente di:  
  
-   Selezionare il linguaggio di programmazione da usare per creare il pacchetto VSPackage.  
  
-   Generare o usare un file di chiave esistente per firmare l'assembly di un VSPackage.  
  
 **Visual C\#**  
 Crea il VSPackage in C\#.  
  
 **Visual Basic**  
 Crea il VSPackage in Visual Basic.  
  
 **Visual C\+\+**  
 Crea il VSPackage in Visual C\+\+.  
  
## Ottieni chiavi per la firma dell'assembly  
 **Genera un nuovo file di chiave per firmare l'assembly**  
 Determina la generazione di un file di chiave per l'utente con la procedura guidata. Il file di chiave generato viene chiamato key.snk.  
  
 **Usare il file di chiave seguente per firmare l'assembly**  
 Consente di digitare il percorso di un file di chiave esistente, oppure fare clic su **Sfoglia** per ottenere il file. La procedura guidata crea una copia del file denominandola key.snk.  
  
 **Annulla**  
 Annulla la creazione del pacchetto VSPackage.  
  
 **Precedente**  
 Passa alla pagina precedente della procedura guidata.  
  
 **Successivo**  
 Passa alla pagina successiva della procedura guidata.  
  
 **Fine**  
 Crea il pacchetto VSPackage.
