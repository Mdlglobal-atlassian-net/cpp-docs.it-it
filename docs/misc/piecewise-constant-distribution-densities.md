# piecewise_constant_distribution::densities
Restituisce le densità archiviate per la distribuzione.  
  
## Sintassi  
  
```  
std::vector<double> densities() const;  
```  
  
## Note  
 La funzione membro restituisce le densità archiviate, calcolate dai parametri di peso.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
