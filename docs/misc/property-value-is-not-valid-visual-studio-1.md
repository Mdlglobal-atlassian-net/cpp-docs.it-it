# Property value is not valid. (Visual Studio) (1)
Generalmente questo errore si verifica quando si immette un valore non valido per una proprietà,  ad esempio, quando la proprietà è di tipo boolean e i valori validi sono TRUE e FALSE, il valore MAYBE non risulta valido per la proprietà.  
  
### Per correggere l'errore  
  
1.  Immettere un valore valido per la proprietà.  
  
    > [!TIP]
    >  Cercare nella documentazione online il nome della proprietà per individuare i valori validi.
