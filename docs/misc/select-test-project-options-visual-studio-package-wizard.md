# Selezionare le opzioni dei progetti di test, Creazione guidata pacchetto di Visual Studio
Questo passaggio consente di:  
  
-   Generare un progetto test di integrazione per il pacchetto VSPackage.  
  
-   Generare un progetto unit test per il pacchetto VSPackage.  
  
 **Progetto test di integrazione**  
 Crea un progetto di test di integrazione. I test di integrazione vengono eseguiti in un'istanza sperimentale di Visual Studio e usano il modello di automazione per testare l'integrazione del pacchetto in Visual Studio.  
  
 **Progetto unit test**  
 Creare un progetto di unit test. Gli unit test sono piccole istanze di MSTest che Ã¨ possibile usare per testare singole classi e singoli membri nel pacchetto.  
  
 **Annulla**  
 Annulla e interrompe il comando per il nuovo progetto.  
  
 **Indietro**  
 Torna al passaggio precedente.  
  
 **Fine**  
 Crea il pacchetto VSPackage.
