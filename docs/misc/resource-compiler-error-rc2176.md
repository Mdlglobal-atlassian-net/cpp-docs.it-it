# Errore del compilatore di risorse RC2176
**DIB precedente nel nome file. Passarlo tramite SDKPAINT.**  
  
 Un vecchio formato bitmap indipendente dalla periferica \(DIB\) è stato trovato nel file specificato. Deve essere convertito nel formato corrente. A tale scopo, è possibile usare l'applicazione SDKPAINT fornita in Windows 3.0 SDK o un'applicazione equivalente.
