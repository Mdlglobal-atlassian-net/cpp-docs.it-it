# A failure occurred while attempting to start the compilation
È stato rilevato un errore sconosciuto durante l'invio della richiesta **Avvia compilazione** al compilatore [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] o [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)] .  
  
 **Per correggere l'errore**  
  
-   Riavviare [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     L'errore comporterà la mancata riuscita del processo di compilazione.
