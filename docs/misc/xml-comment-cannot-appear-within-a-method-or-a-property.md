# Il commento XML non pu&#242; essere specificato all&#39;interno di un metodo o di una propriet&#224;
Non è possibile specificare un commento XML all'interno di un metodo o di una proprietà. Il commento XML verrà ignorato. I blocchi di commento XML non possono essere inseriti in metodi o proprietà.  
  
 **ID errore:** BC42303  
  
### Per correggere l'errore  
  
-   Rimuovere il blocco di commento o spostarlo in una posizione più appropriata.
