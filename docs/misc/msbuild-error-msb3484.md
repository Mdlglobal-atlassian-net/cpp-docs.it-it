# MSBuild Error MSB3484
**MSB3484: impossibile trovare la destinazione di firma '\<file\>'.**  
  
 Questo errore viene generato quando il file che si è tentato di firmare non è stato trovato.  
  
### Per correggere l'errore  
  
-   Accertarsi di specificare un file valido per la firma.
