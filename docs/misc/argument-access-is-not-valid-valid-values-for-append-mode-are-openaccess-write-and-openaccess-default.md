# L&#39;argomento &#39;Access&#39; non &#232; valido. I valori validi per la modalit&#224; Append sono &#39;OpenAccess.Write&#39; e &#39;OpenAccess.Default&#39;
Un valore access non è valido in modalità Append.  
  
### Per correggere l'errore  
  
1.  Modificare il valore in `OpenAccess.Write` o `OpenAccess.Default`.
