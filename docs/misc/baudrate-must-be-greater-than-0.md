# BaudRate deve essere maggiore di 0
L'argomento `BaudRate` specificato per il metodo `My.Computer.Ports.OpenSerialPort` deve essere maggiore di zero.  
  
### Per correggere l'errore  
  
-   Modificare il valore dell'argomento `BaudRate` su un numero positivo.
