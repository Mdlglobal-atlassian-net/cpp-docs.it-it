# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
Il componente aggiuntivo si è arrestato in modo anomalo e potrebbe causare altri problemi nell'ambiente di sviluppo integrato.  
  
### Per correggere l'errore  
  
1.  Scegliere Sì per rimuovere il componente aggiuntivo dall'elenco dei componenti aggiuntivi disponibili.  
  
     In alternativa  
  
     Scegliere No.  
  
    > [!NOTE]
    >  Riportare il problema all'autore del componente aggiuntivo.
