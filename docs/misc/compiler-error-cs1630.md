# Errore del compilatore CS1630
Opzione 'option' non valida per \/errorreport; specificare prompt, send, queue o none  
  
 L'opzione della riga di comando [\/errorreport](../Topic/-errorreport%20\(C%23%20Compiler%20Options\).md) deve essere seguita da **prompt**, **send**, **queue** o **none**, specificando l'azione da intraprendere quando si verifica un errore interno del compilatore.
