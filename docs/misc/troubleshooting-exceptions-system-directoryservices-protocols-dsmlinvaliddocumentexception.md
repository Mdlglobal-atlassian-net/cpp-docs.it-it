# Risoluzione dei problemi relativi alle eccezioni: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
Eccezione generata quando il documento di una risposta o di una richiesta DSML \(Directory Services Markup Language\) non è un documento XML ben formato oppure non può essere convalidato rispetto allo schema DSMLV2.
