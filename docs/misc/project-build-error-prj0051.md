# Errore di compilazione progetto PRJ0051
Non è stato possibile copiare 'file' nella directory di destinazione.  Verificare che il file di destinazione non sia in uso.  
  
 Il sistema di compilazione non è riuscito a copiare un file.  
  
 Verificare che il file non sia aperto nel computer o usato da un altro programma.
