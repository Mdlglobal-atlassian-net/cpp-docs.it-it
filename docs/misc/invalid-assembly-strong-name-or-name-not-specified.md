# Invalid assembly strong name or name not specified
La proprietà del progetto nel modello a oggetti del sistema del progetto che consente di specificare un nome chiave o un contenitore chiave per i progetti che verranno compilati in un assembly con nome sicuro, o assembly firmato, non è stata specificata.  Se viene visualizzato questo messaggio, il processo di compilazione non sarà portato a termine.  
  
### Per correggere l'errore  
  
1.  Assicurarsi che nel codice sia presente un nome di contenitore chiave o un file di chiave.
