# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException
Quando un filtro o una tabella di filtri tenta di accedere al corpo di un messaggio privo di buffer, viene generata un'eccezione <xref:System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException>.
