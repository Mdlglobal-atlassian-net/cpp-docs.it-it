# PAVE OVER Risoluzione dei problemi relativi agli errori di configurazione e installazione per Visual Studio
Utilizzare le seguenti risorse per trovare assistenza nella risoluzione dei problemi d'installazione per [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
-   Forum relativo all'[installazione e configurazione di Visual Studio](http://go.microsoft.com/fwlink/?LinkID=151190).  Rivedere domande e risposte di altri componenti della community di Visual Studio.  Se non si trovato ciò che è necessario, porre le proprie domande.  
  
-   Sito Web [Centro di supporto di Visual Studio](http://go.microsoft.com/fwlink/?LinkID=251019).  Leggere gli articoli delle conoscenze di base \(KB\) e imparare come contattare il supporto di Microsoft per informazioni sui problemi di installazione di Visual Studio.
