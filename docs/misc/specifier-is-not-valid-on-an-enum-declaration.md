# &#39;&lt;identificatore&gt;&#39; non &#232; valido in una dichiarazione Enum
È stato usato un identificatore che non è valido in una dichiarazione `Enum`.  
  
 **ID errore:** BC30396  
  
### Per correggere l'errore  
  
1.  Rimuovere l'identificatore.
