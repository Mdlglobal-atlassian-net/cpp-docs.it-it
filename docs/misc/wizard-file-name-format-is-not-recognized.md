# Wizard file &lt;name&gt; format is not recognized.

### Per correggere l'errore  
  
1.  Verificare che il file della creazione guidata abbia lo stesso numero di versione della versione di Visual Studio elencata come valore di VSWIZARD, come `VSWIZARD 7.0`.  
  
     \-oppure\-  
  
     Reinstallare Visual Studio per ripristinare il file danneggiato.
