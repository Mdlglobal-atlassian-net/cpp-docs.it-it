# Top-level menu names cannot contain separators.
Generalmente questo errore si verifica quando si tenta di trascinare testo contenente un carattere separatore, ad esempio un trattino \(\-\), in una voce di menu.  I menu di primo livello non possono contenere questo carattere, che Ã¨ riservato alla separazione di gruppi di voci di menu.  
  
### Per correggere l'errore  
  
1.  Immettere il testo senza utilizzare il carattere separatore.
