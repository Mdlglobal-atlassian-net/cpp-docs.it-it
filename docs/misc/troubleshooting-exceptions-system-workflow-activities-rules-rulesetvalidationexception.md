# Risoluzione dei problemi relativi alle eccezioni: System.Workflow.Activities.Rules.RuleSetValidationException
Quando l'elaborazione non può continuare per l'impossibilità di convalidare un set di regole, viene generata un'eccezione <xref:System.Workflow.Activities.Rules.RuleSetValidationException>. L'eccezione viene generata dal costruttore <xref:System.Workflow.Activities.Rules.RuleEngine> se l'oggetto <xref:System.Workflow.Activities.Rules.RuleSet> utilizzato presenta errori di convalida.
