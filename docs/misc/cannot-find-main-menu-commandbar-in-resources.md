# Cannot find main menu commandbar in resources.
Questo errore viene generato quando l'ambiente di sviluppo integrato non trova la barra dei comandi MenuBar all'avvio.  Il file .dll è stato eliminato o spostato oppure è danneggiato.  
  
### Per correggere l'errore  
  
-   Riparare o reinstallare il prodotto tramite **Installazione applicazioni**.
