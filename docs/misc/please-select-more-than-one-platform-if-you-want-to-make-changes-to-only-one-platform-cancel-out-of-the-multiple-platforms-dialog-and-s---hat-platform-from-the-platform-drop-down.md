# Selezionare pi&#249; di una piattaforma. Se si desidera modificare solo una piattaforma, chiudere la finestra di dialogo Piattaforme multiple e selezionare la piattaforma desiderata dall&#39;elenco a discesa.
In genere questo errore si verifica quando viene selezionata un'unica piattaforma nella finestra di dialogo **Piattaforme multiple**.  
  
### Per correggere l'errore  
  
1.  Selezionare altre piattaforme.  
  
     \-oppure\-  
  
2.  Se si desidera aggiornare una singola piattaforma, scegliere Annulla e selezionare la piattaforma corretta dall'elenco Piattaforme.
