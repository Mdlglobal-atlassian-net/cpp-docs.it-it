# Risoluzione dei problemi relativi alle eccezioni: System.Runtime.InteropServices.SafeArrayTypeMismatchException
Eccezione generata quando il tipo della matrice `SAFEARRAY` in ingresso non corrisponde al tipo specificato nella firma gestita.
