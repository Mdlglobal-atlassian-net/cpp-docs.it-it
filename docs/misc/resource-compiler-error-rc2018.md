# Errore del compilatore di risorse RC2018
carattere 'hexnumber' sconosciuto  
  
 Il carattere ASCII corrispondente al numero esadecimale specificato è presente nel file di origine ma è un carattere non valido.  
  
 Questo errore può essere dovuto al danneggiamento del file di origine.
