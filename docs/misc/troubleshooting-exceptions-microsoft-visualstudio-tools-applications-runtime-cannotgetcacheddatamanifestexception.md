# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotGetCachedDataManifestException
Eccezione generata quando il manifesto nel quale sono elencati tutti gli oggetti dati memorizzati nella cache risulta danneggiato o mancante.
