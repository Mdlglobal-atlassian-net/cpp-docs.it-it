# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.ActionNotSupportedException
In genere un'eccezione <xref:System.ServiceModel.ActionNotSupportedException> viene generata sul client quando l'azione correlata all'operazione richiamata non corrisponde ad alcuna azione di operazioni nel server.
