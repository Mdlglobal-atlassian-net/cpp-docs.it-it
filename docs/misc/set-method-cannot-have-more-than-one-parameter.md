# Il metodo &#39;Set&#39; non pu&#242; avere pi&#249; di un parametro
Più parametri sono stati definiti per le routine di una proprietà `Set`.  
  
 **ID errore:** BC31063  
  
### Per correggere l'errore  
  
-   Rimuovere i parametri aggiuntivi dalle routine `Set` di una proprietà.
