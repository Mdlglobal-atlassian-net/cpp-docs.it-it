# MSBuild Error MSB3114
**MSB3114: impossibile trovare il file '\<file\>' a cui fa riferimento l'assembly '\<assembly\>'.**  
  
 Questo errore viene generato quando viene rilevato un riferimento non risolvibile in un manifesto esistente.  
  
### Per correggere l'errore  
  
-   Verificare che i manifesti non presentino conflitti nei riferimenti dei file.
