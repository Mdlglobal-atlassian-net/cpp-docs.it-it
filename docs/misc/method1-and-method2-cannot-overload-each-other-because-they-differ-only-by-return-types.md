# &#39;&lt;metodo1&gt;&#39; e &#39;&lt;metodo2&gt;&#39; non possono essere in rapporto di overload perch&#233; si differenziano solo per i tipi restituiti
Si è provato a eseguire l'overload di un metodo con un altro metodo che è diverso dal primo solo per il tipo restituito. Per l'overload, è necessario distinguere due versioni di un metodo tramite i relativi elenchi di argomenti. Per distinguere i metodi è possibile usare solo elenchi di argomenti.  
  
 **ID errore:** BC30301  
  
### Per correggere l'errore  
  
-   Verificare che i metodi si distinguano per i relativi elenchi di argomenti.
