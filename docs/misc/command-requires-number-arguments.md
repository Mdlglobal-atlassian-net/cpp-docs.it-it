# Command requires &lt;number&gt; arguments.
Generalmente questo errore si verifica quando si immette un numero eccessivo o insufficiente di argomenti per il comando.  
  
### Per correggere l'errore  
  
1.  Controllare la sintassi corretta del comando nella documentazione, quindi riprovare.
