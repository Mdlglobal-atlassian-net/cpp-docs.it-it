# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotLoadManifestException
Eccezione generata quando non è possibile caricare un manifesto perché non è presente nel documento, è danneggiato oppure non è valido.
