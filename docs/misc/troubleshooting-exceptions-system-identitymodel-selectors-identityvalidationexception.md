# Risoluzione dei problemi relativi alle eccezioni: System.IdentityModel.Selectors.IdentityValidationException
Quando il certificato del destinatario non è valido, viene generata un'eccezione <xref:System.IdentityModel.Selectors.IdentityValidationException>. Ciò può verificarsi se il certificato non presenta una concatenazione corretta fino al certificato radice.
