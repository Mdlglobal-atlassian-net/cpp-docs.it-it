# Risoluzione dei problemi relativi alle eccezioni: System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException
Quando si tenta di applicare un operatore incompatibile con gli operandi durante la valutazione di una regola, viene generata un'eccezione <xref:System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException>.
