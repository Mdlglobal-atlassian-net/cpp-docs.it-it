# &#39;Option&#39; deve essere seguito da &#39;Compare&#39;, &#39;Explicit&#39;, &#39;Infer&#39; o &#39;Strict&#39;
Le uniche opzioni che Ã¨ possibile impostare con l'istruzione `Option` sono `Compare`, `Explicit`, `Infer` e `Strict`.  
  
 **ID errore:** BC30206  
  
### Per correggere l'errore
