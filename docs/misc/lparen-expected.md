# Previsto &#39;(&#39;
Un elenco di argomenti in una chiamata di routine non inizia con un carattere di parentesi di apertura \(`(`\).  
  
 **ID errore:** BC30199  
  
### Per correggere l'errore  
  
-   Aggiungere un carattere di parentesi di apertura all'inizio dell'elenco.
