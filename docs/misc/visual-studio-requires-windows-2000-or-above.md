# Visual Studio requires Windows 2000 or above.
Visual Studio richiede Windows NT 4.0, build 1381 con Service Pack 3, Windows 95, build 708, o versioni successive.  
  
### Per correggere l'errore  
  
1.  Aggiornare la versione corrente di Windows NT o Windows alle versioni indicate.
