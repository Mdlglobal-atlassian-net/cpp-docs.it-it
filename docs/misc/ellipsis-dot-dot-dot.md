# Puntini di sospensione (...)
L'operatore ellipsis \(`...`\) viene utilizzato in vari modi in C e C\+\+, a seconda del contesto.  Originariamente utilizzato in C come dichiaratore astratto per elenchi di parametri di funzioni variabili, viene inoltre utilizzato nella programmazione macro del preprocessore sotto forma di macro variadic.  In C\+\+, `...` viene utilizzato in blocchi `catch` di gestione delle eccezioni e in C\+\+11, utilizzato per i modelli variadic.
