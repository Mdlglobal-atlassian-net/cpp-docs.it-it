# Avviso del compilatore (livello 1) CS2023
Opzione \/noconfig ignorata perché specificata in un file di risposta  
  
 L'opzione del compilatore [\/noconfig](../Topic/-noconfig%20\(C%23%20Compiler%20Options\).md) è stata specificata in un file di risposta. Questa operazione non è consentita.  
  
 Questo avviso non può essere eliminato con l'opzione [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md).
