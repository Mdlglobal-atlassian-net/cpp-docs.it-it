# Command &lt;name&gt; does not accept arguments or switches.
Generalmente questo errore si verifica quando si immette un'opzione o un argomento per un comando o un alias che non accetta opzioni o argomenti.  Se si immette un alias, nel messaggio di errore viene visualizzato il nome completo del comando al posto dell'alias.  Per un elenco di comandi validi che utilizzano opzioni e argomenti, vedere [Comandi di Visual Studio](../Topic/Visual%20Studio%20Commands.md).  
  
### Per correggere l'errore  
  
1.  Immettere di nuovo il comando e gli argomenti o immettere un nuovo comando.
