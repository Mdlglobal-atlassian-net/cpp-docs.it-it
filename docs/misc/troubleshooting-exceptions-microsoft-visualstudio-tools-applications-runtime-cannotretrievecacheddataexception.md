# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotRetrieveCachedDataException
Eccezione generata quando l'isola di dati memorizzati nella cache di un documento risulta danneggiata o mancante.
