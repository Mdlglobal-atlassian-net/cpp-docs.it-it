# Option is not currently valid.
Generalmente questo errore si verifica quando per un comando si specifica un argomento o un'opzione che al momento non è disponibile.  Se ad esempio si immette `>Edit.Find test /sel` e non è stata effettuata alcuna selezione nel file, l'opzione \/sel non è disponibile.  
  
### Per correggere l'errore  
  
1.  Verificare che gli argomenti e le opzioni immessi siano al momento disponibili.  Per ulteriori informazioni, vedere [Comandi con argomenti di Visual Studio](../Topic/Visual%20Studio%20Commands.md).
