# Il carattere &#39;character&#39; (0x&lt;esadecimale&gt;) non &#232; consentito in un nome XML
Un carattere non supportato è incluso nel nome di un valore letterale XML o di una proprietà axis XML.  
  
 **ID errore:** BC31170  
  
### Per correggere l'errore  
  
-   Rimuovere il carattere non supportato dal nome del valore letterale XML o della proprietà axis XML.
