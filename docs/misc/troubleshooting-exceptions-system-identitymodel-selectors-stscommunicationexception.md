# Risoluzione dei problemi relativi alle eccezioni: System.IdentityModel.Selectors.StsCommunicationException
Quando esiste un problema di comunicazione con il servizio token di sicurezza, viene generata un'eccezione <xref:System.IdentityModel.Selectors.StsCommunicationException>.  
  
 Il problema può essere causato dalle condizioni seguenti:  
  
-   Errore della rete.  
  
-   Errore nel raggiungere l'endpoint MEX.  
  
-   Il servizio token di sicurezza non ha un'identità valida dell'endpoint.
