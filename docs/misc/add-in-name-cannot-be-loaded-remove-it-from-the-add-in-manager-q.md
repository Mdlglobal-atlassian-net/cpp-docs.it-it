# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
Non è possibile trovare il componente aggiuntivo selezionato.  Generalmente questo errore si verifica quando non è disponibile un file DLL o un altro componente richiesto dal componente aggiuntivo selezionato.  
  
### Per correggere l'errore  
  
1.  Reinstallare il componente aggiuntivo.
