# &#39;MyClass&#39; non pu&#242; essere utilizzata all&#39;esterno di una classe
È stata usata la parola chiave `MyClass` all'esterno di una struttura di classe.  
  
 **ID errore:** BC30470  
  
### Per correggere l'errore  
  
-   Spostare la dichiarazione `MyClass` all'interno della struttura di classe.
