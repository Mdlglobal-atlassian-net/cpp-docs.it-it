# Risoluzione dei problemi relativi alle eccezioni: System.Data.SqlServerCe.SqlCeException
Quando il provider sottostante restituisce un avviso o un errore da un'origine dati SQL Server Compact, viene generata un'eccezione `System.Data.SqlServerCe.SqlCeException`. Per altre informazioni sulla causa dell'errore, vedere l'articolo sugli [errori di SQL Server Compact Edition](http://msdn2.microsoft.com/library/ms171849.aspx) nel sito Web di MSDN.

 Un'eccezione `System.Data.SqlServerCe.SqlCeException` contiene sempre almeno un'istanza dell'oggetto `System.Data.SqlServerCe.SqlCeError`. La classe non pu√≤ essere ereditata.
