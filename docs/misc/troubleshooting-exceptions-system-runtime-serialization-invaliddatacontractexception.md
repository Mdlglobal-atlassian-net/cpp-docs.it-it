# Risoluzione dei problemi relativi alle eccezioni: System.Runtime.Serialization.InvalidDataContractException
Quando l'oggetto<xref:System.Runtime.Serialization.InvalidDataContractException> o <xref:System.Runtime.Serialization.DataContractSerializer> incontra un contratto dati non valido durante la serializzazione e la deserializzazione, viene generata un'eccezione <xref:System.Runtime.Serialization.NetDataContractSerializer>.
