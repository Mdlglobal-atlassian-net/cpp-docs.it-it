# Risoluzione dei problemi relativi alle eccezioni: System.Web.Caching.TableNotEnabledForNotificationException
L'eccezione generata quando viene utilizzata una classe <xref:System.Web.Caching.SqlCacheDependency> viene utilizzata con una tabella di database non attivata per le notifiche delle modifiche.
