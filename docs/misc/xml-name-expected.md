# &#200; previsto il nome XML
Un valore letterale XML o una proprietà axis XML è stata specificata senza un nome.  
  
 **ID errore:** BC31146  
  
### Per correggere l'errore  
  
-   Aggiungere un nome per il valore letterale XML o per la proprietà axis XML.
