# There was an error trying to open or create the application definition file for this project. &#39;&lt;error&gt;&#39;
Questo errore si verifica quando si tenta di aprire il file di definizione dell'applicazione \(ADF\), ma non è disponible.  È possibile che sia stato spostato o eliminato.  
  
### Per correggere l'errore  
  
-   Ricreare il file di definizione dell'applicazione \(ADF\).
