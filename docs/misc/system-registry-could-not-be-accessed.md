# System registry could not be accessed.
Generalmente questo errore si verifica quando il sistema operativo non è in grado di trovare il file del Registro di sistema, il file è danneggiato oppure non è possibile leggere o scrivere nel file.  
  
### Per correggere l'errore  
  
1.  Reinstallare Visual Studio.
