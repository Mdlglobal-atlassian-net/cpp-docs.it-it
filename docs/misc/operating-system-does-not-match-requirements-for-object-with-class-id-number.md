# Operating system does not match requirements for object with class ID &lt;number&gt;.
Generalmente questo errore si verifica quando viene rilevato un sistema operativo incompatibile o una versione incompatibile del sistema operativo.  
  
### Per correggere l'errore  
  
1.  Verificare i requisiti di sistema per l'applicazione.  
  
2.  Installare uno dei sistemi operativi compatibili, quindi riprovare.
