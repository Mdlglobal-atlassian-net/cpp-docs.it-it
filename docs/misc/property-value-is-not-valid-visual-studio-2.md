# Property value is not valid. (Visual Studio) (2)
Generalmente questo errore si verifica quando si immette un valore non valido per una proprietà,  ad esempio quando si immette il formato errato per un Integer o un valore troppo grande per un valore short.  
  
### Per correggere l'errore  
  
1.  Immettere un valore valido per la proprietà.  
  
    > [!TIP]
    >  Cercare nella documentazione online quali sono i valori validi per la proprietà.
