# gamma_distribution::beta
Restituisce il parametro di distribuzione beta.  
  
## Sintassi  
  
```  
RealType beta() const;  
```  
  
## Note  
 La funzione membro restituisce il valore `beta` archiviato.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
