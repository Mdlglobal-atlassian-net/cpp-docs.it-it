# Unable to Open Linked File
È stato aggiunto al progetto un file come file collegato ma,  all'apertura del progetto non è stato possibile trovare il file.  
  
 **Per correggere l'errore**  
  
-   Eliminare il file collegato da Esplora soluzioni.  
  
     \- oppure \-  
  
-   Spostare il file nel percorso previsto dal sistema di progetto.
