# Disk full: &lt;name&gt;
L'unità contenente il programma non dispone di spazio sufficiente per salvare i file di programma, ad esempio toolbox.tbd, alla chiusura del programma.  
  
### Per correggere l'errore  
  
1.  Rimuovere alcuni file dall'unità, quindi riprovare a chiudere il programma.
