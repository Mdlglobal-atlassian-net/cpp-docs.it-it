# MSBuild Error MSB2007
**Impossibile trovare il file di progetto "{0}".**  
  
 Impossibile trovare un file di progetto specificato da parte di [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  
  
### Per correggere l'errore  
  
1.  Controllare che il nome del file di progetto sia stato digitato correttamente.
