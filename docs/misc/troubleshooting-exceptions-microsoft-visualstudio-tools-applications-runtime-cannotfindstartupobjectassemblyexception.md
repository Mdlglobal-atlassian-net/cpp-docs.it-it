# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotFindStartupObjectAssemblyException
Eccezione generata quando l'assembly principale della soluzione non contiene l'oggetto di avvio specificato nel manifesto dell'applicazione.
