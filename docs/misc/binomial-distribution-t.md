# binomial_distribution::t
Restituisce il parametro di distribuzione `t`.  
  
## Sintassi  
  
```  
IntType t() const;  
```  
  
## Valore restituito  
 Parametro `t` dall'oggetto `param_type` archiviato.  
  
## Note  
 Per un esempio di codice, vedere [Classe binomial\_distribution](../standard-library/binomial-distribution-class.md).  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
