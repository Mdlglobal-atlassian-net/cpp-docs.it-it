# poisson_distribution::mean
Restituisce il parametro di distribuzione Mean.  
  
## Sintassi  
  
```  
RealType mean() const;  
```  
  
## Note  
 La funzione membro restituisce il valore `mean` archiviato.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
