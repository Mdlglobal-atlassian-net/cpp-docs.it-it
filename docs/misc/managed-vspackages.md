# VSPackage gestiti
Negli argomenti riportati di seguito viene illustrato come creare un package VS.  Un VSPackage è un modulo di software che estende l'ambiente di sviluppo integrato di [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] \(IDE\) fornendo agli elementi dell'\(UI\) interfaccia utente, servizi web, i progetti, editor e finestre di progettazione.  Per ulteriori informazioni, vedere [Package VS](../Topic/VSPackages.md).
