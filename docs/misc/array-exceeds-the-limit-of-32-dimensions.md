# La matrice supera il limite di 32 dimensioni
Una matrice non può avere più di 32 dimensioni.  
  
 **ID errore:** BC30052  
  
### Per correggere l'errore  
  
-   Ridurre il numero di dimensioni nella dichiarazione della matrice.
