# Risoluzione dei problemi relativi alle eccezioni: System.Data.TypedDataSetGeneratorException
Eccezione generata quando si verifica un conflitto tra nomi durante la generazione di un oggetto <xref:System.Data.DataSet> fortemente tipizzato.
