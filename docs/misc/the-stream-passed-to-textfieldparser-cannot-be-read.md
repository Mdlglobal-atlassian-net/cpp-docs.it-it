# Impossibile leggere il flusso passato a TextFieldParser
L'[TextFieldParser Object](../Topic/TextFieldParser%20Object.md) non può leggere il flusso passato. Può trattarsi di un risultato del tentativo di lettura di un file che non è un file di testo.  
  
### Per correggere l'errore  
  
-   Verificare che il file sia un file di testo.
