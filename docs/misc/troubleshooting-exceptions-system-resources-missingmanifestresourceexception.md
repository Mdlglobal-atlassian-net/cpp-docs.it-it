# Risoluzione dei problemi relativi alle eccezioni: System.Resources.MissingManifestResourceException
Eccezione generata quando l'assembly principale non contiene le risorse per le impostazioni cultura non associate ad alcun paese e tali risorse sono necessarie perché l'assembly satellite appropriato non è disponibile.
