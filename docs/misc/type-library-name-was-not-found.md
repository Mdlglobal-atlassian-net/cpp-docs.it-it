# Type library &lt;name&gt; was not found.
Impossibile eseguire Visual Studio senza le librerie dei tipi DTE.olb e vsext.tlb.  Generalmente questo errore si verifica quando questi due file sono stati rimossi o eliminati.  
  
### Per correggere l'errore  
  
1.  Reinstallare Visual Studio.
