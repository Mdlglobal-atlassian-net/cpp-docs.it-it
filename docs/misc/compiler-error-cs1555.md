# Errore del compilatore CS1555
Impossibile trovare 'class' specificata per il metodo Main  
  
 Nell'opzione del compilatore [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) è stata specificata una classe, ma il nome della classe non è stato trovato nel codice sorgente.
