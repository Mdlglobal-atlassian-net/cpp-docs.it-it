# MSBuild Error MSB3176
**MSB3176: la versione minima richiesta specificata è successiva alla versione pubblicazione corrente.  Specificare una versione precedente o corrispondente alla versione pubblicazione corrente.**  
  
 Questo errore viene generato quando la versione dell'applicazione minima richiesta è successiva alla versione corrente del manifesto di distribuzione.  
  
### Per correggere l'errore  
  
-   Specificare una versione richiesta minima inferiore o uguale alla versione pubblicazione corrente.
