# MSBuild Error MSB3159
**MSB3159: errore di convalida XML nel file '\<file\>': '\<errore\>'**  
  
 Questo errore viene generato quando il file del prodotto o package.xml non è conforme allo schema XML.  La seconda parte dell'errore è uno specifico messaggio di errore.  
  
### Per correggere l'errore  
  
-   Aggiornare il file XML per correggere l'errore di convalida.
