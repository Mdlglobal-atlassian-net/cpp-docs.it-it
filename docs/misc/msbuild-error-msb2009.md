# MSBuild Error MSB2009
**Attributo '\<nome attributo\>' dell'elemento '\<nome elemento\>' non valido.**  
  
 Il nome dell'attributo non è stato digitato correttamente o non viene riconosciuto da [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  
  
### Per correggere l'errore  
  
-   Controllare che il nome dell'attributo sia stato digitato correttamente.  
  
-   Verificare se il file di progetto è stato modificato o danneggiato.  In tal caso, aprire il progetto nella versione di [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] in cui è stato creato, salvarlo, quindi tentare nuovamente la conversione.
