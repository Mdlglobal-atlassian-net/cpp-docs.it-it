# Risoluzione dei problemi relativi alle eccezioni: System.Windows.Automation.ElementNotAvailableException
Un'eccezione <xref:System.Windows.Automation.ElementNotAvailableException> contiene informazioni sull'eccezione generata quando si tenta di accedere a un elemento <xref:System.Windows.Automation.AutomationElement> dell'interfaccia utente che corrisponde a una parte dell'interfaccia utente non più disponibile.  
  
## Note  
 Questa eccezione può essere generata se l'elemento si trova in una finestra di dialogo che è stata chiusa o in un'applicazione che è stata chiusa o terminata.
