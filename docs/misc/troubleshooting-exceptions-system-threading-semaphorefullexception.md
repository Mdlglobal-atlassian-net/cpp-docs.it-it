# Risoluzione dei problemi relativi alle eccezioni: System.Threading.SemaphoreFullException
Eccezione generata quando l'overload del metodo <xref:System.Threading.Semaphore.Release%2A> viene chiamato su un semaforo per cui è già stato raggiunto il valore massimo.
