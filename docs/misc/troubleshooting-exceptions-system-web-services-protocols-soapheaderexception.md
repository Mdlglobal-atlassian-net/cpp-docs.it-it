# Risoluzione dei problemi relativi alle eccezioni: System.Web.Services.Protocols.SoapHeaderException
Eccezione generata quando viene chiamato un metodo di un servizio Web XML attraverso SOAP e si verifica un'eccezione durante l'elaborazione dell'intestazione SOAP.
