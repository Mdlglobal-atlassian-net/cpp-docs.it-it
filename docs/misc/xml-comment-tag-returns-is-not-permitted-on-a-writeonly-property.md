# Il tag &#39;returns&#39; del commento XML non &#232; consentito in una propriet&#224; &#39;WriteOnly&#39;
Il tag 'returns' del commento XML non è consentito in una proprietà 'WriteOnly'. Il commento XML verrà ignorato.  
  
 Un commento XML per una dichiarazione di proprietà di sola scrittura non può contenere un tag \<returns\>.  
  
 **ID errore:** BC42313  
  
### Per correggere l'errore  
  
-   Rimuovere il tag \<returns\> non supportato.
