# &#200; necessario il riferimento al modulo &#39;&lt;nomemodulo&gt;&#39; contenente il tipo &#39;&lt;nomemembro&gt;&#39;
È necessario il riferimento al modulo '\<nomemodulo\>' contenente il tipo '\<nomemembro\>'. Aggiungerne uno al progetto.  
  
 Il progetto contiene un tipo che dipende da un altro tipo di un assembly che non è incluso nei riferimenti a progetti.  
  
 **ID errore:** BC30653  
  
### Per correggere l'errore  
  
-   Aggiungere un riferimento a progetto per i tipi dipendenti nel progetto.
