# Errore dell&#39;analizzatore di espressioni CXX0003
errore interno nell'analizzatore di espressioni  
  
 L'analizzatore di espressioni ha rilevato un errore interno.  
  
 Prendere nota delle circostanze dell'errore e contattare Microsoft Corporation. È possibile segnalare l'errore online all'indirizzo [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) oppure chiamare il Servizio supporto tecnico Microsoft. Per altre informazioni, fare clic su **Supporto tecnico** nel menu **?** in Visual C\+\+.  
  
 Questo errore è identico all'errore CAN0003.
