# Utilit&#224; della riga di comando di VsRegEx
L'utilità VsRegEx è stata sostituita con l'[Utilità CreateExpInstance](../Topic/CreateExpInstance%20Utility.md) in Visual Studio 2010 e versioni successive.  
  
## Note  
 In Visual Studio 2008 le informazioni di configurazione relative all'istanza sperimentale dell'IDE sono state archiviate nel Registro di sistema. In Visual Studio 2010 e versioni successive sono archiviate nel file system.
