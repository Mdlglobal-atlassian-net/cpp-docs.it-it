# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
Il valore permanente di una proprietà `Boolean` non è uno dei seguenti valori:  
  
-   1 o 0  
  
-   On o Off  
  
-   true o false  
  
 L'errore è solitamente dovuto alla modifica manuale del file di progetto.  
  
 **Per correggere l'errore**  
  
-   Ripristinare una condizione soddisfacente del file di progetto.  
  
     Il valore della proprietà interessata verrà impostato su false.
