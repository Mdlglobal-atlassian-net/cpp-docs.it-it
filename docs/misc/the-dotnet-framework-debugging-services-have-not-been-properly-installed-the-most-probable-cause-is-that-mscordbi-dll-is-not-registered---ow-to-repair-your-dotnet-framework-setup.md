# I servizi di debug di .NET Framework non sono stati installati correttamente. La causa pi&#249; probabile &#232; una registrazione non corretta di mscordbi.dll. Fare clic su ? per altre informazioni su come riparare l&#39;installazione di .NET Framework.
Questo errore si verifica in genere quando Visual Studio non trova il file mscordbi.dll, installato con i servizi di debug di Common Language Runtime.  
  
### Per correggere l'errore  
  
-   Riparare l'installazione di .NET Framework nel computer.
