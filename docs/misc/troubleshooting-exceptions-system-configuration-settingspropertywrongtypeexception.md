# Risoluzione dei problemi relativi alle eccezioni: System.Configuration.SettingsPropertyWrongTypeException
Eccezione generata quando viene effettuato un tentativo di applicare una proprietà di configurazione che non può essere applicata a causa del relativo tipo.
