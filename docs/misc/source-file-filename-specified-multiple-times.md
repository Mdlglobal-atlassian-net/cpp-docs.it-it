# Il file di origine &lt;nomefile&gt; &#232; stato specificato pi&#249; volte
Un file è stato specificato più volte.  
  
 **ID errore:** BC2002  
  
### Per correggere l'errore  
  
-   Rimuovere gli identificatori di file ridondanti.
