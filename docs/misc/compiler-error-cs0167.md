# Errore del compilatore CS0167
Nel delegato 'delegate' manca il metodo Invoke  
  
 È stato importato e usato un programma \(che usa Common Language Runtime di .NET Framework\) creato con un altro compilatore. Il compilatore ha consentito un oggetto [delegate](../Topic/delegate%20\(C%23%20Reference\).md) con un formato non corretto. Pertanto, il metodo `Invoke` non è disponibile. Per altre informazioni, vedere [Delegati](../Topic/Delegates%20\(C%23%20Programming%20Guide\).md).
