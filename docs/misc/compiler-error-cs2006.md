# Errore del compilatore CS2006
Errore nella sintassi della riga di comando: manca 'text' per l'opzione 'option'  
  
 La sintassi per *option* richiede testo aggiuntivo. Per informazioni, vedere [Opzioni del compilatore](../Topic/C%23%20Compiler%20Options.md).
