# Internal error processing pattern.
Generalmente questo errore si verifica quando in una stringa di ricerca viene inclusa un'espressione regolare la cui sintassi contiene degli errori.  
  
### Per correggere l'errore  
  
1.  Immettere nuovamente la stringa di ricerca che utilizza la sintassi corretta per l'espressione regolare.  Per ulteriori informazioni, vedere [Espressioni regolari](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
