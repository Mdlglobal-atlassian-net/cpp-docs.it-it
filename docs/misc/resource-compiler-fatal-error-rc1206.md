# Errore irreversibile del compilatore di risorse RC1206
la tabella codici specificata nella riga di comando non esiste nel Registro di sistema  
  
 L'opzione di specifica della tabella codici \(\/c\) è stata seguita da una pagina di codice la cui posizione non è specificata nel Registro di sistema. Eseguire l'installazione per installare la tabella codici specificata.
