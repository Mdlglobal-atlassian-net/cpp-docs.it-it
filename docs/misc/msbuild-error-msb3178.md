# MSBuild Error MSB3178
**MSB3178: l'assembly '\<assembly\>' è stato specificato in modo errato come file.**  
  
 Questo avviso viene generato durante la generazione del manifesto di applicazione quando il processo di compilazione rileva che un riferimento di file è in effetti un assembly, gestito o nativo.  
  
### Per correggere l'errore  
  
-   Per ulteriori informazioni sulla causa e le possibili soluzioni di un errore di MSBuild, vedere [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
