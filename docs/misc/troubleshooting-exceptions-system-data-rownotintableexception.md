# Risoluzione dei problemi relativi alle eccezioni: System.Data.RowNotInTableException
Eccezione generata quando viene effettuato un tentativo di eseguire un'operazione su un oggetto <xref:System.Data.DataRow> che non si trova in un oggetto <xref:System.Data.DataTable>.
