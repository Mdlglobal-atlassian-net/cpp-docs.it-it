# Errore MSB4137 di MSBuild
**MSB4137: valore non valido specificato nel file di configurazione in "\<x\>".  Il nome della proprietà o il nome della versione degli strumenti non può essere una stringa vuota.**  
  
 Una definizione del set di strumenti contiene nomi o valori di proprietà non validi.  
  
### Per correggere l'errore  
  
-   Verificare che la definizione del set di strumenti non contenga nomi o valori della proprietà non validi o vuoti.
