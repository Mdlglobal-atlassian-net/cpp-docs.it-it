# Errore del compilatore CS2022
Le opzioni '\/out' e '\/target' devono trovarsi prima dei nomi dei file di origine  
  
 Le opzioni del compilatore [\/out \(impostazione del nome del file di output\)](../Topic/-out%20\(C%23%20Compiler%20Options\).md) e [\/target \(specifica del formato del file di output\)](../Topic/-target%20\(C%23%20Compiler%20Options\).md), quando sono specificate nella riga di comando, devono precedere i file del codice sorgente.
