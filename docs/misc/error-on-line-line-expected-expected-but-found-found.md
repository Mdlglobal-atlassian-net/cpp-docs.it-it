# Error on line &lt;line&gt;. Expected &#39;expected&#39; but found &#39;found&#39;
Si tratta di un errore di analisi XML generale che si verifica durante la lettura del file di progetto.  Questo errore può verificarsi per i file di progetto \(VBPROJ\/CSPROJ\) e per i file delle impostazioni utente \(VBPROJ.USER\/CSPROJ.USER\).  Ulteriori informazioni vengono fornite dal sistema del progetto nei parametri `expected` e `found`.  
  
 Questo errore è solitamente dovuto alla modifica manuale del file di progetto o utente.  
  
 **Per correggere l'errore**  
  
-   Ripristinare una condizione soddisfacente del file di progetto.  
  
     La possibilità di aprire il progetto è determinata dalla gravità dell'errore.
