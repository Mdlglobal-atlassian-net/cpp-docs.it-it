# Errore irreversibile del compilatore di risorse RC1049
argomento numerico 'stringa' non valido  
  
 Il compilatore di risorse ha ricevuto la stringa specificata invece dell'argomento numerico previsto.  
  
 Questo errore può verificarsi se si specifica un numero esadecimale senza il necessario prefisso \\x prefix oppure se è presente un numero a virgola mobile in formato non corretto.
