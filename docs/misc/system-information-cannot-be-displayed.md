# System information cannot be displayed.
Non è possibile visualizzare le informazioni sul sistema operativo, probabilmente perché manca un valore nel Registro di sistema o non è possibile eseguire msinfo32.exe.  
  
### Per correggere l'errore  
  
1.  Reinstallare Visual Studio.
