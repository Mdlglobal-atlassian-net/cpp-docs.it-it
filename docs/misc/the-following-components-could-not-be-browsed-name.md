# The following components could not be browsed: &lt;name&gt;
Generalmente questo errore si verifica quando si tenta di aggiungere un tipo di file che non supporta l'esplorazione di oggetti, ad esempio file HTM e TXT, nella finestra di dialogo Selettore componenti.  Il Visualizzatore oggetti supporta i seguenti tipi di file: BSC, OLB, TLB, DLL, EXE e OCX.  
  
### Per correggere l'errore  
  
1.  Rimuovere il tipo di file non supportato dall'elenco Componenti selezionati, quindi selezionare un file diverso.
