# Risoluzione dei problemi relativi alle eccezioni: System.RankException
Un'eccezione <xref:System.RankException> viene generata quando a un metodo viene passata una matrice con un numero di dimensioni errato.  
  
## Suggerimenti associati  
 **Verificare che la matrice disponga del numero di dimensioni necessario.**  
 Per altre informazioni, gli utenti di Visual Basic possono vedere [Array Dimensions in Visual Basic](../Topic/Array%20Dimensions%20in%20Visual%20Basic.md).  
  
 Per altre informazioni, gli utenti di C\# possono vedere [Matrici](../Topic/Arrays%20\(C%23%20Programming%20Guide\).md).

 Dato che gli indici di matrice partono da 0, l'indice minimo di qualsiasi dimensione Ã¨ sempre 0.
