# Previsto &#39;Lib&#39;
Un'istruzione`Declare` specifica un nome di libreria senza usare la parola chiave `Lib`.  
  
 **ID errore:** BC30218  
  
### Per correggere l'errore  
  
-   Aggiungere la parola chiave `Lib` prima del nome della libreria.
