# The application cannot shut down.
Generalmente questo errore si verifica quando un programma esterno, ad esempio Gestione attività, tenta di chiudere Visual Studio mentre in Visual Studio è ancora aperta una finestra di dialogo modale.  
  
### Per correggere l'errore  
  
1.  Chiudere la finestra di dialogo modale e riprovare.
