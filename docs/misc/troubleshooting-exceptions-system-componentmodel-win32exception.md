# Risoluzione dei problemi relativi alle eccezioni: System.ComponentModel.Win32Exception
Eccezione generata per un codice di errore Win32.  
  
## Note  
 Quando vengono visualizzati, i codici di errore Win32 vengono convertiti dalle relative rappresentazioni numeriche in un messaggio di sistema. Utilizzare la proprietà <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> per accedere alla rappresentazione numerica del codice di errore.
