# File &lt;filename&gt; cannot be created.
Generalmente questo errore si verifica durante la creazione di un report Web sui commenti del codice, nel caso in cui l'utente corrente non disponga di accesso in scrittura.  
  
### Per correggere l'errore  
  
1.  Verificare di disporre dell'accesso in scrittura.  
  
2.  Verificare di disporre dello spazio sufficiente per il salvataggio delle pagine.
