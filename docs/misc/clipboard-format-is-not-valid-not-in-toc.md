# Clipboard format is not valid.
Gli Appunti accettano solo dati in alcuni formati specifici.  Generalmente questo errore si verifica quando si cerca di inserire i dati di un componente negli Appunti, ma il formato dei dati non è compatibile con gli Appunti.  
  
### Per correggere l'errore  
  
1.  Consultare la documentazione relativa al componente per determinare se è possibile utilizzare gli Appunti per trasferire dati dal componente.  
  
     \-oppure\-  
  
2.  Selezionare un altro elemento e riprovare.
