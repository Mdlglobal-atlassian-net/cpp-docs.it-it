# Errore MSB8012 di MSBuild
*{0}\(\\{1\\}\)* non corrisponde al valore della proprietà OutputFile di *\\{2\\}* \(*\\{3\\}*\). È possibile che il progetto non venga compilato correttamente. Per risolvere il problema, accertarsi che i valori di proprietà $\(OutDir\), $\(TargetName\) e $\(TargetExt\) corrispondano al valore specificato in %\(\\{4\\}.OutputFile\).
