# &#39;&lt;spaziodeinomi&gt;&#39; non &#232; un nome valido e non pu&#242; essere usato come nome per lo spazio dei nomi radice
Uno spazio dei nomi di qualificazione non Ã¨ un nome di elemento valido. Un nome deve contenere solo caratteri alfabetici, cifre decimali e caratteri di sottolineatura e deve iniziare con un carattere alfabetico o un carattere di sottolineatura \(`_`\).  
  
 **ID errore:** BC30113  
  
### Per correggere l'errore  
  
-   Se lo spazio dei nomi fa parte di un set di spazi dei nomi annidati, verificare se uno dei nomi esterni contiene caratteri non validi.
