# Please specify valid search string.
Generalmente questo errore si verifica quando si tenta di eseguire una ricerca senza immettere una stringa da cercare nella casella **Trova** della finestra di dialogo **Trova simbolo**.  
  
### Per correggere l'errore  
  
1.  Digitare il testo corretto nella casella **Trova**, quindi scegliere di nuovo **Trova**.
