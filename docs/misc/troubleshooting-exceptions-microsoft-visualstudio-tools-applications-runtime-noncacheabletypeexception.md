# Risoluzione dei problemi relativi alle eccezioni: Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
Eccezione generata quando si tenta di memorizzare nella cache un tipo che non corrisponde a <xref:System.Data.DataSet>, <xref:System.Data.DataTable> o un tipo conforme ai requisiti specificati in [Caching Data](http://msdn.microsoft.com/it-it/e60dc731-2cb0-4de6-a861-0f0d131d780a).
