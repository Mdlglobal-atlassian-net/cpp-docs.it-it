# MSBuild Error MSB3166
**MSB3166: impossibile trovare il file obbligatorio '\<file\>' per l'elemento '\<package\>'.**  
  
 Questo errore viene generato quando manca un contratto di licenza con l'utente finale o un file di controllo esterno.  Assicurarsi che il file esista sul disco.  
  
### Per correggere l'errore  
  
-   Reinstallare Visual Studio SDK  
  
     \- oppure \-  
  
-   Copiare i file obbligatori nella posizione appropriata.
