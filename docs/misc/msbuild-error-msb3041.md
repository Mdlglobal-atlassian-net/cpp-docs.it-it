# MSBuild Error MSB3041
**Impossibile creare un nome di risorsa di manifesto per "\<file\>".**  
  
 L'attività `CreateManifestResourceName` non è in grado di creare un nome di risorsa di manifesto per il file specificato.  
  
### Per correggere l'errore  
  
-   Per ulteriori informazioni sulla causa e le possibili soluzioni di un errore di MSBuild, vedere [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
