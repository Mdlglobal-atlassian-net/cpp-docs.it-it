# Load of property &#39;property&#39; failed. &lt;reason&gt;
Un valore di una proprietà non è compreso nell'intervallo.  
  
 L'errore è solitamente dovuto alla modifica manuale del file di progetto.  
  
 **Per correggere l'errore**  
  
-   Correggere l'errore manualmente nel file di progetto.  
  
     Verranno ripristinati i valori predefiniti delle proprietà modificate, ovvero i valori delle proprietà alla creazione del progetto.
