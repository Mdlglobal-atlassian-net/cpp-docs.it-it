# Errore del compilatore CS2018
Non è possibile trovare il file di messaggi 'cscmsgs.dll'  
  
 Non è stato trovato il file con estensione dll che contiene i messaggi di errore e di avviso del compilatore. Questo file deve trovarsi nella stessa directory degli altri file di supporto del compilatore.
