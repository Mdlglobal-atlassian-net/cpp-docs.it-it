# Avviso del compilatore (livello 1) CS1592
Un codice XML con un formato non corretto è stato rilevato nel file dei commenti incluso \-\- 'reason'  
  
 Un problema, segnalato come ***reason***, è stato trovato nel file specificato dal tag [\<include\>](../Topic/%3Cinclude%3E%20\(C%23%20Programming%20Guide\).md).
