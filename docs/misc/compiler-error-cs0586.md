# Errore del compilatore CS0586
Errore interno del compilatore: fase 'stage'  
  
 Provare a determinare se il compilatore non è riuscito perché non è in grado di analizzare una sintassi imprevista. Se non è questo il caso, provare a [Comunicazioni con Microsoft](../Topic/Talk%20to%20Us.md).
