# Wizard file &lt;name&gt; is corrupted.
Il formato del file VSZ della creazione guidata non è corretto.  Questo errore si verifica quando un file VSZ è danneggiato o se il file appartiene a un prodotto diverso da Visual Studio.  
  
### Per correggere l'errore  
  
1.  Verificare che la creazione guidata appartenga a Visual Studio.  
  
     oppure  
  
     Reinstallare Visual Studio per sostituire il file danneggiato.
