# Risoluzione dei problemi relativi alle eccezioni: System.IO.Log.ReservationNotFoundException
Un'eccezione <xref:System.IO.Log.ReservationNotFoundException> viene generata quando non viene trovata una specifica prenotazione dello spazio in un oggetto <xref:System.IO.Log.LogRecordSequence>.
