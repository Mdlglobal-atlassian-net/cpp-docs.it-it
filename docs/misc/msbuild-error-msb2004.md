# MSBuild Error MSB2004
**L'elemento \<{0}\> non può contenere più di un nodo di lingua.**  
  
 Il file di progetto può contenere un solo nodo di lingua.  
  
### Per correggere l'errore  
  
-   Verificare se il file di progetto è stato modificato o danneggiato.  In tal caso, aprire il progetto nella versione di [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] in cui è stato creato, salvarlo, quindi tentare nuovamente la conversione.
