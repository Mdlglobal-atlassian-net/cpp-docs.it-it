# Switch &lt;name&gt; does not allow a value.
Generalmente questo errore si verifica quando si specifica un valore per un'opzione che non accetta valori.  
  
### Per correggere l'errore  
  
1.  Immettere di nuovo il comando, omettendo un valore per l'opzione.
