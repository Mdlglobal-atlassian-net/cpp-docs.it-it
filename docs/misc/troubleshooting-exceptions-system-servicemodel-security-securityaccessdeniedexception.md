# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.Security.SecurityAccessDeniedException
Un'eccezione <xref:System.ServiceModel.Security.SecurityAccessDeniedException> rappresenta l'eccezione di sicurezza generata quando una richiesta di autorizzazione di sicurezza non riesce.
