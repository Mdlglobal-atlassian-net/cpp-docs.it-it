# Errore irreversibile del compilatore di risorse RC1023
impossibile aprire il file di origine 'filename'  
  
 Il file specificato non esiste, non può essere aperto oppure non è stato trovato.  
  
 Verificare che le impostazioni dell'ambiente siano valide e che il percorso del file specificato sia corretto.  
  
 Se questo errore viene visualizzato senza messaggio di errore, il compilatore di risorse ha esaurito gli handle di file.
