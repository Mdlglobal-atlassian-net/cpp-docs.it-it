# Errore del compilatore di risorse RC2132
È prevista la parola chiave VALUE, BLOCK o END  
  
 Un blocco in una risorsa **VERSION** non termina in modo corretto o un nuovo blocco non inizia in modo corretto. Di conseguenza, l'istruzione **VERSION** non è un blocco valido.
