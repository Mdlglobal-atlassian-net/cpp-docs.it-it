# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
La sottosezione \(sottosezione\) è Includi e la sezione \(sezione\) è File.  
  
 L'errore è solitamente dovuto alla modifica manuale del file di progetto.  
  
 **Per correggere l'errore**  
  
-   Utilizzare la modalità **Mostra tutti i file**, accessibile scegliendo l'apposito pulsante dalla barra degli strumenti di Esplora soluzioni, e includere i file appropriati.  
  
     Se si verifica questo errore, sarà possibile aprire il progetto ma non aggiungervi i file.
