# Risoluzione dei problemi relativi alle eccezioni: System.IdentityModel.Selectors.UserCancellationException
Quando l'utente decide di annullare la creazione del token e di chiudere l'interfaccia utente, viene generata un'eccezione <xref:System.IdentityModel.Selectors.UserCancellationException>.
