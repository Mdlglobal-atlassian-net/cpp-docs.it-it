# Visual Studio has encountered an unexpected error.
Questo errore si verifica generalmente se Visual Studio riceve dati in un formato inatteso o riceve dati da un'origine inattesa.  
  
### Per correggere l'errore  
  
-   Chiudere e riavviare Visual Studio.
