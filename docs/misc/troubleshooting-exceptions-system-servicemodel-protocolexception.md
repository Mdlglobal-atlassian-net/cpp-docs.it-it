# Risoluzione dei problemi relativi alle eccezioni: System.ServiceModel.ProtocolException
Si verifica un'eccezione <xref:System.ServiceModel.ProtocolException> sul client quando la comunicazione con la controparte remota è impossibile a causa di protocolli di trasferimento dati non corrispondenti.  
  
## Note  
 L'eccezione <xref:System.ServiceModel.ProtocolException> può essere causata da associazioni incompatibili tra endpoint, dall'utilizzo di un indirizzo di endpoint remoto non corretto o da una configurazione simile errata.
