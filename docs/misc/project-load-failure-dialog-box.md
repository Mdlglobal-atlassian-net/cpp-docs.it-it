# Finestra di dialogo Errore di caricamento del progetto
Questa finestra di dialogo viene visualizzata quando non è possibile caricare un file di progetto.  
  
 **Correggere l'errore o gli errori specificati e ricaricare il progetto**  
 Fornisce informazioni sull'errore.  Utilizzare le informazioni riportate in questa finestra di dialogo per individuare e risolvere l'errore nel file di progetto.  Salvare il progetto, quindi ricaricarlo nell'ambiente di sviluppo integrato \(IDE\).  
  
 **Non visualizzare più questo messaggio nella sessione**  
 Quando è selezionata l'opzione, la finestra di dialogo **Errore di caricamento del progetto** non verrà più visualizzata fino alla prossima apertura dell'ambiente di sviluppo integrato \(IDE\).
