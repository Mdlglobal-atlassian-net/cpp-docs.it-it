# Risoluzione dei problemi relativi alle eccezioni: System.Reflection.CustomAttributeFormatException
L'eccezione generata quando il formato binario di un attributo personalizzato non è valido.  
  
## Note  
 È possibile che si sia verificato un bug in uno strumento utilizzato per creare il tipo di dati che genera l'errore.
