# Errore del compilatore CS0653
Impossibile applicare la classe Attribute 'class' perché è di tipo astratto  
  
 Una classe Attribute personalizzata [abstract](../Topic/abstract%20\(C%23%20Reference\).md) non può essere usata come attributo.  
  
 L'esempio seguente genera l'errore CS0653:  
  
```  
// CS0653.cs using System; public abstract class MyAttribute : Attribute { } public class My2Attribute : MyAttribute { } [My]   // CS0653 // try the following line instead // [My2] class MyClass { public static void Main() { } }  
```
