# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
Questo messaggio indica l'impossibilità di fare riferimento a uno specifico progetto da un altro progetto.  
  
 La causa più comune di questo errore è rappresentata dal riferimento a un progetto che compila un file EXE.  Questo errore indica inoltre che un progetto fa riferimento a un altro progetto il cui output non è un assembly, ad esempio un'applicazione C\+\+ nativa.
