# File &lt;name&gt; cannot be registered as an COM component.
Non è possibile aggiungere il file ai riferimenti del progetto.  Questo errore si verifica quando un file è stato danneggiato, non contiene librerie dei tipi, utilizza un tipo di file incompatibile o il sistema non dispone di risorse sufficienti per completare l'operazione.  
  
### Per correggere l'errore  
  
1.  Verificare che il file contenga una libreria dei tipi valida, quindi riprovare.  
  
     \-oppure\-  
  
     Reinstallare il prodotto che ha installato il file.  
  
    > [!NOTE]
    >  Il prodotto che ha installato il file potrebbe non essere Visual Studio.
