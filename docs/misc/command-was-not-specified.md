# Command was not specified.
Questo errore si verifica in genere quando si immette una stringa vuota nella finestra di comando  o nella casella **Trova\/Comando**.  
  
### Per correggere l'errore  
  
1.  Immettere un comando valido.  Per ulteriori informazioni, vedere [Comandi di Visual Studio](../Topic/Visual%20Studio%20Commands.md).
