# Command &lt;name&gt; is not available.
Generalmente questo errore si verifica quando nella finestra **Comando** o nella casella **Trova\/Comando** si immette un comando non valido rispetto allo stato corrente di Visual Studio.  Ad esempio, il comando `File.CloseSolution`  non è valido se in Visual Studio non è correntemente aperta una soluzione.  
  
### Per correggere l'errore  
  
1.  Scegliere un comando diverso.  
  
     \-oppure\-  
  
     Effettuare una selezione diversa su cui il comando possa agire.
