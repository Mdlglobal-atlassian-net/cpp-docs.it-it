# Component is not correctly registered.
Generalmente questo errore si verifica quando il Registro di sistema è danneggiato, il sistema non è in grado di accedere al Registro di sistema o nel Registro di sistema è registrato un elemento in un formato errato.  
  
### Per correggere l'errore  
  
1.  Reinstallare il componente.
