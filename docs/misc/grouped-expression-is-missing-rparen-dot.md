# Grouped expression is missing &#39;)&#39;.
Generalmente questo errore si verifica quando in una stringa di ricerca si specifica una parentesi tonda di apertura, `(`, ma si omette la parentesi tonda di chiusura, `)`.  
  
### Per correggere l'errore  
  
1.  Per cercare il valore letterale di carattere `(`, immettere `\(`.  
  
2.  Per raggruppare un'espressione, immettere la parentesi `)` mancante.
