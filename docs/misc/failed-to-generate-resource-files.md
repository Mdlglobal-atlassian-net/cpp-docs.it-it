# Failed to generate resource files
Si è verificato un errore generale nel sistema del progetto durante la lettura di un file RESX nell'ambito del processo di creazione dei file RESOURCES binari riconosciuti dal gestore di risorse .NET.  
  
 L'errore è provocato in genere da un file RESX danneggiato.  
  
 Questo avviso non comporterà la mancata riuscita del processo di compilazione.  Tuttavia, il file RESX a cui fa riferimento questo messaggio di errore non verrà incluso nel progetto come risorsa di manifesto assembly.
