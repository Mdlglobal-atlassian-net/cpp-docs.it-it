# Command does not accept switches.
Generalmente questo errore si verifica quando si specifica un'opzione per un comando che riconosce solo argomenti.  
  
### Per correggere l'errore  
  
1.  Verificare i requisiti della sintassi per il comando.  Per ulteriori informazioni, vedere [Comandi con argomenti di Visual Studio](../Topic/Visual%20Studio%20Commands.md).
