# Argument is missing in the pattern.
Generalmente questo errore si verifica quando nella stringa di ricerca viene immessa un'espressione regolare che presenta una sintassi incompleta.  
  
### Per correggere l'errore  
  
1.  Immettere nuovamente la stringa.  Per ulteriori informazioni sulla sintassi, vedere [Espressioni regolari](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
