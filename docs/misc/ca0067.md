# CA0067
Directory non trovata.  
  
 Impossibile trovare il valore dell'opzione **\/directory** nella riga di comando FxCopCmd.  
  
 Questo avviso può essere visualizzato in caso di utilizzo della proprietà **CodeAnalysisDependentAssemblyPaths** di [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] per specificare percorsi di ricerca dell'assembly di riferimento aggiuntivi e uno di tali percorsi non esiste.  
  
-   Se viene visualizzato l'avviso CA0067 con errori [CA0058](../misc/ca0058.md) o avvisi [CA0060](../misc/ca0060.md), la risoluzione degli altri errori in genere consente di risolvere CA0067.  
  
-   Se l'avviso CA0067 viene visualizzato senza altri errori o avvisi, è in genere possibile ignorare l'avviso poiché la directory non è necessaria.
