# File name or default path is too long.
Generalmente questo errore si verifica durante la creazione di un report Web sui commenti del codice, nel caso in cui il nome di file o il percorso specificato nella casella Salva pagine Web in della finestra di dialogo Compila pagine Web di commento sia troppo lungo.  
  
### Per correggere l'errore  
  
1.  Specificare un nome di file o un percorso con un numero di caratteri inferiore a 260.
