# piecewise_constant_distribution::intervals
Restituisce gli intervalli archiviati per la distribuzione.  
  
## Sintassi  
  
```  
std::vector<RealType> intervals() const;  
```  
  
## Note  
 La funzione membro restituisce gli intervalli archiviati.  
  
## Requisiti  
 **Intestazione:** \<random\>  
  
 **Spazio dei nomi:** std
