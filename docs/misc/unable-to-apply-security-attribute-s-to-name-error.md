# Impossibile applicare gli attributi di sicurezza a &#39;&lt;nome&gt;&#39;: &lt;errore&gt;
Si è verificato un errore durante l'applicazione degli attributi di sicurezza. Potrebbe trattarsi di un risultato di un uso errato degli attributi di sicurezza o potrebbe indicare un problema con il compilatore.  
  
 **ID errore:** BC31520  
  
### Per correggere l'errore  
  
1.  Verificare la sintassi e gli argomenti per l'attributo.  
  
2.  Ripetere la compilazione del programma, per controllare se l'errore si verifica di nuovo.  
  
3.  Se l'errore si ripresenta, salvare il lavoro e riavviare [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
4.  Se l'errore si ripete, reinstallare [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)].  
  
5.  Se l'errore persiste dopo la reinstallazione, inviare una notifica al Servizio Supporto Tecnico Clienti Microsoft.
