# Risoluzione dei problemi relativi alle eccezioni: System.Data.InvalidExpressionException
Un'eccezione <xref:System.Data.InvalidExpressionException> viene generata quando viene eseguito un tentativo di aggiungere un oggetto <xref:System.Data.DataColumn> contenente una proprietà <xref:System.Data.DataColumn.Expression%2A> non valida a un insieme <xref:System.Data.DataColumnCollection>.  
  
## Suggerimenti associati  
 **Controllare la sintassi dell'espressione.**  
 La proprietà <xref:System.Data.DataColumn.Expression%2A> viene utilizzata per calcolare il valore di una colonna o per creare un'aggregazione di colonne.
