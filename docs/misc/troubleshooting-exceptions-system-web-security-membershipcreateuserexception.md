# Risoluzione dei problemi relativi alle eccezioni: System.Web.Security.MembershipCreateUserException
Eccezione generata quando un utente non viene creato correttamente da un provider di appartenenze.  
  
## Note  
 La causa di questa eccezione è indicata dalla proprietà <xref:System.Web.Security.MembershipCreateUserException.StatusCode%2A>.
