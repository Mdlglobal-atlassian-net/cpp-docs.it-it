# Miscellaneous Files projects can contain &lt;number&gt; items.
In un progetto di file esterni possono essere salvati fino a 256 elementi.  Generalmente questo errore si verifica quando si immette un numero minore di 1 o maggiore di 256 nella casella **In caso di file esterni, salva gli ultimi \<numero\> elementi** nel riquadro Documenti dell'opzione Ambiente nella finestra di dialogo Opzioni.  
  
### Per correggere l'errore  
  
1.  Immettere un numero compreso tra 1 e 256 nella casella.
