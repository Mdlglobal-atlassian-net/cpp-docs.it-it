# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
Questo errore si verifica quando si seleziona **Crea nuove piattaforme progetto**, ma nessuno dei progetti supporta la nuova piattaforma soluzione.  
  
### Per correggere l'errore  
  
-   Deselezionare l'opzione **Crea nuove piattaforme progetto**, quindi scegliere **OK**.  
  
     \-oppure\-  
  
     Deselezionare l'opzione **Crea nuove piattaforme progetto**, scegliere il nome di un'altra piattaforma dall'elenco a discesa, quindi scegliere **OK**.
