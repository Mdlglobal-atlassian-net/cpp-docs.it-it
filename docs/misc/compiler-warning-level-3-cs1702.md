# Avviso del compilatore (livello 3) CS1702
Se il riferimento all'assembly "Nome assembly 1" corrisponde a "Nome assembly 2", potrebbe essere necessario fornire i criteri di runtime  
  
 I riferimenti ai due assembly hanno numeri di build e\/o di revisione diversi, quindi non si unificheranno automaticamente. Potrebbe essere necessario fornire i criteri di runtime per imporre l'unificazione usando direttive nel file config dell'applicazione.
