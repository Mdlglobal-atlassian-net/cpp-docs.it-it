# Error getting Codezone Community sites.
Questo errore viene generato quando non viene trovata la connessione al servizio Web online.  I servizi Web inseriscono le informazioni nella pagina **Online**, in **?**, **Ambiente**, della finestra di dialogo **Opzioni**.  
  
### Per correggere l'errore  
  
1.  Verificare che sia attiva una connessione Internet.  
  
     \-oppure\-  
  
     Se la connessione a Internet viene eseguita tramite un server proxy, selezionare le impostazioni relative al server proxy.  
  
2.  Provare nuovamente ad accedere alla pagina Online.
