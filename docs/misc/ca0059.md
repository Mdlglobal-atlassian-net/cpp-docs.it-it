# CA0059
Errore di opzione della riga di comando.  
  
 Questo errore si verifica se un'opzione della riga di comando obbligatoria è mancante, ad esempio  
  
 se non è stata specificata l'opzione \/out o \/console.  Questo errore si verifica anche quando esiste un'opzione non valida, come ad esempio un valore di timeout non corretto.
