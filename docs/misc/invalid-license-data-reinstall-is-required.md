# Invalid license data. Reinstall is required.
Generalmente questo errore si verifica quando le informazioni relative al codice di licenza presenti nel Registro di sistema sono state eliminate o sono state rese illeggibili.  
  
### Per correggere l'errore  
  
1.  Reinstallare il prodotto.
