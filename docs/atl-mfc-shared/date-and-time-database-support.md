# Date and Time: Database Support
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

A partire dalla versione 4,0, la programmazione di database MFC utilizza le classi [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) e [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) per rappresentare le informazioni di data e ora.  Queste classi, anche utilizzate in automazione, derivano dalla classe [COleVariant](../mfc/reference/colevariant-class.md).  Assicurarsi di un migliore supporto per gestire le informazioni su data e ora [CTime](../atl-mfc-shared/reference/ctime-class.md) e [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md).
