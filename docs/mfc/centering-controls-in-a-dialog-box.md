# Centering Controls in a Dialog Box
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Per centrare i controlli in una finestra di dialogo  
  
1.  Selezionare i controlli che si desidera riordinare.  
  
2.  Scegliere **Centra nella finestra** dal menu **Formato**, quindi scegliere una delle disposizioni riportate di seguito.  
  
    -   **Verticale**: consente di centrare verticalmente i controlli nella finestra di dialogo.  
  
    -   **Orizzontale**: consente di centrare orizzontalmente i controlli nella finestra di dialogo.
