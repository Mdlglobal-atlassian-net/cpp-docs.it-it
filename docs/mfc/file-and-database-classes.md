# Classi File e Database
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Queste classi consentono di archiviare le informazioni in un database o in un file su disco.  Esistono tre set di classi di database \- OLE DB, ODBC e DAO \- che forniscono una funzionalit√† simile.  Il gruppo di OLE DB viene implementato mediante OLE DB e di funziona con i modelli consumer OLE DB, il gruppo di DAO viene implementato mediante l'oggetto di accesso ai dati e il gruppo ODBC viene implementato mediante ODBC.  Esistono inoltre un set di classi per modificare file standard, i flussi attivi e flussi HTML.
