# Utilizzo dell&#39;editor finestre per aggiungere controlli
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Quando si crea la risorsa di dialogo\- modello con [editor finestre](../mfc/dialog-editor.md), trascinare i controlli da una tavolozza dei comandi e li rilascia nella finestra di dialogo.  Ciò aggiunge le specifiche per il tipo di controllo alla risorsa di dialogo\- modello.  Quando si costruisce un oggetto finestra di dialogo e chiama la relativa funzione membro di `DoModal` o di **Crea**, il framework crea un controllo Windows e posizionarlo nella finestra di dialogo sullo schermo.  
  
 È invece possibile [creare manualmente i controlli](../mfc/adding-controls-by-hand.md) se lo si desidera.  È più lavoro.
