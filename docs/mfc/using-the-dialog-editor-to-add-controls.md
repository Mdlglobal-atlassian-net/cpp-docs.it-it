# <a name="using-the-dialog-editor-to-add-controls"></a>Utilizzo dell'editor finestre per aggiungere controlli

Quando si crea la risorsa modello di finestra con il [editor finestre](../windows/dialog-editor.md), si trascina i controlli da una tavolozza dei controlli e rilasciarli nella finestra di dialogo. Aggiunge le specifiche per quel tipo di controllo per la risorsa modello di finestra di dialogo. Quando si costruisce un oggetto finestra di dialogo e chiamare relativi `Create` o `DoModal` funzione membro, il framework crea un controllo di Windows e lo inserisce nella finestra di dialogo sullo schermo.
