# <a name="debugging-support-classes"></a>Debug di classi di supporto
MFC fornisce le classi seguenti che consentono di eseguire il debug dei problemi di allocazione di memoria dinamica.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Fornisce una destinazione per i dump di diagnostica.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Struttura che fornisce gli snapshot dell'utilizzo della memoria. Inoltre utilizzato per confrontare gli snapshot di memoria precedenti e successivi.
