# Convalida e DDX (Dialog Data Exchange)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Lo scambio di dati della finestra di dialogo \(DDX\) è un modo semplice per inizializzare i controlli nella finestra di dialogo e di raccogliere l'immissione dei dati dall'utente.  La convalida di dati della finestra di dialogo \(DDV\) è un modo semplice per convalidare l'immissione di dati in una finestra di dialogo.  Per sfruttare DDX e di DDV nelle finestre di dialogo, utilizzare [Aggiunta guidata variabile membro](../ide/add-member-variable-wizard.md) per creare i membri dati e impostare i relativi tipi di dati e specificare le regole di convalida.
