# <a name="ready-to-use-map-classes"></a>Classi Map pronte all'uso
Le seguenti classi sono classi map pronte all'uso.  
  
 [CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)  
 Usa `void` puntatori come chiavi per la ricerca di altri `void` puntatori.  
  
 [CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)  
 Usa `void` puntatori come chiavi per la ricerca di dati di tipo **WORD**.  
  
 [CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)  
 Usa `CString` oggetti come chiavi per la ricerca `CObject` puntatori.  
  
 [CMapStringToPtr](../mfc/reference/cmapstringtoptr-class.md)  
 Usa `CString` oggetti come chiavi per la ricerca `void` puntatori.  
  
 [CMapStringToString](../mfc/reference/cmapstringtostring-class.md)  
 Usa `CString` oggetti come chiavi per la ricerca di altri `CString` oggetti.  
  
 [CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)  
 Utilizza i dati di tipo **WORD** trovare `CObject` puntatori.  
  
 [CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)  
 Utilizza i dati di tipo **WORD** trovare `void` puntatori.
