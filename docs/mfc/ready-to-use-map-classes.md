# Classi Map pronte all&#39;uso
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le classi seguenti classi sono pronte per l'uso della mappa.  
  
 [CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)  
 Utilizza puntatori di `void` come chiavi per trovare altri puntatori di `void`.  
  
 [CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)  
 Utilizza puntatori di `void` come chiavi per cercare dati di tipo **WORD**.  
  
 [CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)  
 Utilizza oggetti di `CString` come chiavi per trovare i puntatori di `CObject`.  
  
 [CMapStringToPtr](../mfc/reference/cmapstringtoptr-class.md)  
 Utilizza oggetti di `CString` come chiavi per trovare i puntatori di `void`.  
  
 [CMapStringToString](../mfc/reference/cmapstringtostring-class.md)  
 Utilizza oggetti di `CString` come chiavi per individuare altri oggetti di `CString`.  
  
 [CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)  
 Utilizza i dati di tipo **WORD** per trovare i puntatori di `CObject`.  
  
 [CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)  
 Utilizza i dati di tipo **WORD** per trovare i puntatori di `void`.
