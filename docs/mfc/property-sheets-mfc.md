# Finestre delle propriet&#224; (MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa famiglia di articoli spiega come implementare il supporto per le finestre delle proprietà nelle applicazioni MFC.  Una finestra delle proprietà, nota anche come finestra di dialogo scheda, fornisce un modo di gestire molti controlli in una finestra di dialogo.  La finestra delle proprietà contiene le pagine delle proprietà, ognuna basata su una risorsa di modello di dialogo separata.  È possibile dividere i comandi della finestra di dialogo in gruppi logici e inserire ogni gruppo nella relativa pagina delle proprietà.
