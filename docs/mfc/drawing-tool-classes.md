# Disegno di classi strumento
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Tali classi includono gli strumenti di disegno utilizzati per disegnare in un contesto di dispositivo.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 La classe base per gli strumenti di grafica GDI.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 Incapsula un pennello GDI che può essere selezionato come il pennello corrente in un contesto di dispositivo.  Pennelli utilizzati per riempire l'interno degli oggetti disegnati.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 Incapsula una penna GDI che può essere selezionata come la penna corrente in un contesto di dispositivo.  Penne vengono utilizzate per tracciare linee del bordo di oggetti.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 Incapsula un tipo di carattere GDI che può essere selezionata come il tipo corrente in un contesto di dispositivo.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 Incapsula una bitmap GDI, fornendo un'interfaccia per modificare le bitmap.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 Incapsula una tavolozza dei colori GDI per l'utilizzo come interfaccia tra l'applicazione e un dispositivo di output di colore come visualizzazione.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 Visualizzazioni e gestire l'interfaccia utente di ridimensionamento e di spostamento di oggetti rettangolari.
