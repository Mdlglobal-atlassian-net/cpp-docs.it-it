# Classi List pronte all&#39;uso
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le classi seguenti classi sono pronte per l'utilizzo dell'elenco.  
  
 [CObList](../mfc/reference/coblist-class.md)  
 Puntatori di archiviare gli oggetti di classe `CObject` o a oggetti delle classi derivate da `CObject` in un elenco collegato.  
  
 [CPtrList](../mfc/reference/cptrlist-class.md)  
 Archivia i puntatori a `void` \(puntatori generici\) in un elenco collegato.  
  
 [CStringList](../mfc/reference/cstringlist-class.md)  
 Archivia gli oggetti di `CString` in un elenco collegato.
