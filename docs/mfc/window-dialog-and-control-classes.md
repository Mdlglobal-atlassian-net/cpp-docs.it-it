# Classi Window, Dialog e Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Classificare `CWnd` e le relative classi derivate incapsulano `HWND`, un handle a una finestra di windows.  `CWnd` può essere utilizzato da solo o come base per derivare nuove classi.  Le classi derivate fornite dalla libreria di classi rappresentano diversi tipi di finestre.  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 La classe base per tutte le finestre.  È possibile utilizzare una delle classi derivate da `CWnd` o derivare classi personalizzate direttamente da.
