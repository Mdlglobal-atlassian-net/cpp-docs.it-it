# Utilizzo di CStatusBarCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile utilizzare il controllo barra di stato \([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)\) per creare una finestra di controllo che riflette i vari tipi di informazioni sullo stato dell'applicazione.  La finestra di stato può essere suddivisa in parti che visualizzano più tipi di informazione.
