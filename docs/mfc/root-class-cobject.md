# <a name="root-class-cobject"></a>Classe Root: CObject
La maggior parte delle classi nella libreria Microsoft Foundation classe (MFC) sono derivata da una singola classe di base nella radice della gerarchia di classi. `CObject`fornisce una serie di funzionalità utili per tutte le classi derivate, con un overhead molto basso. Per ulteriori informazioni su `CObject` e relative funzionalità, vedere [CObject tramite](../mfc/using-cobject.md).  
  
 [CObject](../mfc/reference/cobject-class.md)  
 La principale classe base della maggior parte delle classi MFC. Supporta la serializzazione dei dati e ottenere informazioni in fase di esecuzione su una classe.  
  
 [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)  
 Struttura utilizzata per determinare la classe esatta di un oggetto in fase di esecuzione.
