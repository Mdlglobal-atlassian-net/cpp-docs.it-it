# Classe Root: CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La maggior parte delle classi nella libreria Microsoft Foundation classes \(MFC\) sono derivate da una sola classe base alla radice della gerarchia di classi.  `CObject` fornisce numerose funzionalità utili a tutte le classi derivate da, con sovraccarico molto basso.  Per ulteriori informazioni su `CObject` e le relative funzionalità, vedere [Utilizzando CObject](../mfc/using-cobject.md).  
  
 [CObject](../mfc/reference/cobject-class.md)  
 La classe base finale di maggior parte delle classi MFC.  Supporto di serializzare i dati e che consentono di ottenere informazioni di runtime su classe.  
  
 [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)  
 Struttura utilizzata per determinare la classe esatta di un oggetto in fase di esecuzione.
