# <a name="root-class-cobject"></a>Classe Root: CObject

La maggior parte delle classi nella libreria di classi MFC (Microsoft Foundation) sono derivata da una singola classe di base nella radice della gerarchia di classi. `CObject` fornisce una serie di funzionalità utili per tutte le classi derivate da quest'ultimo, con un overhead molto basso. Per altre informazioni sulle `CObject` e le relative funzionalità, vedere [CObject tramite](../mfc/using-cobject.md).

[CObject](../mfc/reference/cobject-class.md)<br/>
La classe di base fondamentale della maggior parte delle classi MFC. Supporta la serializzazione dei dati e ottenere informazioni di run-time su una classe.

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
Struttura utilizzata per determinare la classe esatta di un oggetto in fase di esecuzione.
