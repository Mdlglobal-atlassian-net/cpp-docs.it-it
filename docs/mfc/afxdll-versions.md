# <a name="afxdll-versions"></a>Versioni AFXDLL
Invece di compilare l'applicazione il collegamento statico alle librerie di codice oggetto MFC, è possibile compilare l'applicazione per utilizzare una delle librerie di AFXDLL, che contengono MFC in una DLL che possono condividere più applicazioni in esecuzione. Per una tabella dei nomi AFXDLL, vedere [DLL: convenzioni di denominazione](../build/naming-conventions-for-mfc-dlls.md).  
  
> [!NOTE]
>  Per impostazione predefinita, la creazione guidata applicazione MFC crea un progetto AFXDLL. Per utilizzare invece il collegamento statico del codice MFC, impostare il **Usa MFC in una libreria statica** opzione nella creazione guidata applicazione MFC. Collegamento statico non è disponibile nell'edizione Standard di Visual C++.
