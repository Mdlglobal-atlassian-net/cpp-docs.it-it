# Framework applicazione
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Il centro della libreria MFC \(Microsoft Foundation Class \(MFC\) è un incapsulamento di grandi dimensioni parte dell'API Windows form in C\+\+.  Le librerie di classi rappresentano le finestre, finestre di dialogo, i contesti di periferica, gli oggetti GDI comuni quali pennelli e penne, controlli e altri elementi standard di windows.  Queste classi forniscono un'interfaccia funzionale di funzione membro C\+\+ alle strutture in windows che incapsulano.  Per ulteriori informazioni su queste classi, vedere [Argomenti dell'oggetto Window](../mfc/window-objects.md).  
  
 Ma la libreria MFC fornisce inoltre un livello di funzionalità dell'applicazione aggiuntiva compilato sull'incapsulamento di C\+\+ l'API Windows.  Questo livello è un framework di applicazione per windows che fornisce la maggior parte dell'interfaccia utente comune prevista dei programmi per windows, incluse le barre degli strumenti, barre di stato, la stampa, l'anteprima di stampa, il supporto del database e il supporto ActiveX.  [Utilizzo delle classi per la creazione di applicazioni per windows](../mfc/using-the-classes-to-write-applications-for-windows.md) viene illustrato in dettaglio il framework.
