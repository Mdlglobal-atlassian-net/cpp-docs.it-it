# <a name="walkthroughs-mfc"></a>Procedure dettagliate (MFC)
In questa sezione sono contenuti articoli che analizzano le diverse attività associate alle funzionalità della nuova libreria MFC.  
  
## <a name="in-this-section"></a>In questa sezione  
 [Procedura dettagliata: uso dei nuovi controlli Shell MFC](../mfc/walkthrough-using-the-new-mfc-shell-controls.md)  
 In questa procedura dettagliata verrà creata un'applicazione simile a Esplora risorse. Si creerà una finestra che contiene due riquadri. Riquadro a sinistra conterrà un [CMFCShellTreeCtrl](../mfc/reference/cmfcshelltreectrl-class.md) oggetto che verrà visualizzato il Desktop in una visualizzazione gerarchica. Il riquadro di destra contiene un [oggetto CMFCShellListCtrl](../mfc/reference/cmfcshelllistctrl-class.md) che visualizzerà i file nella cartella selezionata nel riquadro a sinistra.  
  
 [Procedura dettagliata: inserimento di controlli nelle barre degli strumenti](../mfc/walkthrough-putting-controls-on-toolbars.md)  
 Barre degli strumenti MFC moderne può ospitare i controlli non siano pulsanti semplici. In questo articolo viene illustrato come eseguire l'operazione.  
  
 [Procedura dettagliata: aggiunta di un oggetto D2D a un progetto MFC](../mfc/walkthrough-adding-a-d2d-object-to-an-mfc-project.md)  
 Viene illustrato come aggiungere un oggetto D2D a un progetto e viene illustrato come utilizzare D2D.  
  
 [Procedura dettagliata: aggiunta di animazione a un progetto MFC](../mfc/walkthrough-adding-animation-to-an-mfc-project.md)  
 Viene illustrato come aggiungere un oggetto grafico a un progetto e viene illustrato come l'animazione.
