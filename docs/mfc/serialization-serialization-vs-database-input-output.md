# Serializzazione: serializzazione e input/output del database
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In questo articolo viene illustrato quando utilizzare gli oggetti documento e la serializzazione per attività basato su file \(DEI\/O\) e quando altre tecniche dei\/O sono adatte \- perché l'applicazione legge e scrive i dati in base alle transazioni, come nelle applicazioni di database.  Se non si utilizza la serializzazione, non sarà necessario il file aperto, risparmiate e non risparmiate come controlli.  Vengono trattati i seguenti argomenti:  
  
-   [Requisiti per gestire input\/output](../mfc/recommendations-for-handling-input-output.md)  
  
-   [Gestire il menu File nelle applicazioni di database](../mfc/file-menu-in-an-mfc-database-application.md)
