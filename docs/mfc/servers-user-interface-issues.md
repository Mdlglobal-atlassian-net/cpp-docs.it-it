# Server: problemi dell&#39;interfaccia utente
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Un'applicazione server può numerose funzionalità che devono essere aggiunti all'interfaccia utente agli elementi OLE di fornire alle applicazioni contenitori.  Per ulteriori informazioni sui menu e delle risorse aggiuntive che devono essere aggiunti a un'applicazione server, vedere [Menu e risorse: Aggiunta di server](../mfc/menus-and-resources-server-additions.md).
