Classe [CAsyncSocket](../mfc/reference/casyncsocket-class.md) incapsula le API di Windows Sockets per uno, fornendo ai programmatori di rete avanzate maggiore potenza e flessibilit√†. Classe [CSocket](../mfc/reference/csocket-class.md) fornisce un'interfaccia semplificata per la serializzazione di dati da e verso un [CArchive](../mfc/reference/carchive-class.md) oggetto.
