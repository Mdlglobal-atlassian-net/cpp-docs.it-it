# TN049: livello di conversione da MFC/OLE MBCS a Unicode (MFCANS32)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa nota originariamente descritto come MFCANS32.DLL fornisce interfacce ANSI principalmente nel mondo Unicode OLE a 32 bit.  Questa DLL non è più utilizzato da MFC.
