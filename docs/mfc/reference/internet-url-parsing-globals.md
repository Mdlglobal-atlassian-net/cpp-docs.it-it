# Funzioni globali di analisi degli URL Internet
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Quando un client invia una query al server Internet, è possibile utilizzare uno dei globals di traccia URL per estrarre le informazioni sul client.  
  
### Internet URL che analizza Globals  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|Analizza una stringa URL e restituisce il tipo di servizio e dei relativi componenti.|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|Analizza una stringa URL e restituisce il tipo di servizio e dei relativi nonché fornisce il nome utente e la password.|
