# CCommonDialog Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La classe base per le classi che incapsulano funzionalità comuni di Windows dialogs.  
  
## Sintassi  
  
```  
class CCommonDialog : public CDialog  
```
