## <a name="remarks"></a>Note  
 La classe `CMFCTasksPane` implementa la funzionalità seguenti:  
  
-   Gli elementi possono essere raggruppati e ogni raggruppamento di elementi può avere una didascalia collegata.  
  
-   I raggruppamenti di elementi possono essere compressi o espansi.  
  
-   Un'icona può essere assegnata a ciascun elemento del riquadro attività.  
  
-   I singoli elementi possono essere associati a un ID di comando che viene eseguito quando un utente fa clic sull'elemento. Quando il clic si verifica, il messaggio `WM_COMMAND` viene inviato al proprietario del controllo del riquadro attività.  
  
 Per usare i controlli `CMFCTasksPane` nell'applicazione, attenersi alla procedura seguente:  
  
1.  Importare un oggetto `CMFCTasksPane` nella classe della finestra cornice principale.  
  
2.  Quando si effettua l'elaborazione del messaggio `WM_CREATE`, chiamare il metodo `Create`. È possibile utilizzare i normali [CControlBar](../../mfc/reference/ccontrolbar-class.md) stili. Per altre informazioni, vedere `CControlBar::Create`.  
  
3.  Chiamare il [CMFCTasksPane::AddGroup](#addgroup) metodo per aggiungere i vari gruppi.  
  
4.  Chiamare il [CMFCTasksPane::AddTask](#addtask), [CMFCTasksPane::AddLabel](#addlabel) o [CMFCTasksPane::AddMRUFilesList](#addmrufileslist) le funzioni membro per aggiungere nuovi elementi (attività) a ogni gruppo.  
  
5.  Chiamare [CMFCTasksPane::EnableGroupCollapse](#enablegroupcollapse) per specificare se è possono comprimere i gruppi di articoli.

##  <a name="addgroup"></a>CMFCTasksPane::AddGroup  
 Aggiunge un nuovo gruppo di attività al controllo del riquadro attività.  
  
```  
int AddGroup(
    int nPageIdx,  
    LPCTSTR lpszGroupName,  
    BOOL bBottomLocation = FALSE,  
    BOOL bSpecial = FALSE,  
    HICON hIcon = NULL);

 
int AddGroup(
    LPCTSTR lpszGroupName,  
    BOOL bBottomLocation = FALSE,  
    BOOL bSpecial = FALSE,  
    HICON hIcon = NULL);
```  
  
### <a name="parameters"></a>Parametri  
 [in] `nPageIdx`  
 Specifica l'indice in base zero della pagina.  
  
 [in] `lpszGroupName`  
 Specifica il nome del gruppo.  
  
 [in] `bBottomLocation`  
 `TRUE`Per creare il gruppo nella parte inferiore del controllo riquadro attività. in caso contrario, `FALSE`.  
  
 [in] `bSpecial`  
 `TRUE`Per contrassegnare questo gruppo è un *speciali* gruppo; in caso contrario, `FALSE`. Per ulteriori informazioni sui gruppi speciali, vedere la sezione Note della `CMFCTasksPane`.  
  
 [in] `hIcon`  
 Specifica l'icona da visualizzare nella didascalia del gruppo.  
  
### <a name="return-value"></a>Valore restituito  
 Indice in base zero del gruppo nell'elenco interno dei gruppi che gestisce la classe.  
  
### <a name="remarks"></a>Note  
 Chiamare questo metodo per creare un gruppo di attività e aggiungere tale gruppo per il controllo del riquadro attività.  
  
 Il framework consente di visualizzare i gruppi di attività nella parte superiore del controllo del riquadro attività o nella parte inferiore. Il framework consente di visualizzare un solo gruppo alla parte inferiore. Questo gruppo deve essere aggiunti per ultimi.

 Mostra o nasconde un'attività.  
  
```  
BOOL ShowTask(
    int nGroup,  
    int nTask,  
    BOOL bShow = TRUE,  
    BOOL bRedraw = TRUE);
```  
  
### <a name="parameters"></a>Parametri  
 [in] `nGroup`  
 Specifica l'indice in base zero del gruppo.  
  
 [in] `nTask`  
 Specifica l'indice in base zero dell'attività per mostrare o nascondere.  
  
 [in] `bShow`  
 `TRUE`Per visualizzare l'attività. `FALSE` per nascondere l'attività.  
  
 [in] `bRedraw`  
 `TRUE`Per ridisegnare il riquadro attività. in caso contrario, `FALSE`.  
  
### <a name="return-value"></a>Valore restituito  
 `TRUE`Se l'attività è stata correttamente mostrato o nascosto; `FALSE` se il gruppo specificato o l'attività non esiste.  
  
### <a name="remarks"></a>Note  
 Utilizzare [CMFCTasksPane::ShowTaskByCmdId](#showtaskbycmdid) per mostrare o nascondere un'attività in base al relativo ID di comando.  
  
##  <a name="showtaskbycmdid"></a>CMFCTasksPane::ShowTaskByCmdId  
 Mostra o nasconde un'attività in base all'ID di comando.  
  
```  
BOOL ShowTaskByCmdId(
    UINT uiCommandID,  
    BOOL bShow = TRUE,  
    BOOL bRedraw = TRUE);
```  
  
### <a name="parameters"></a>Parametri  
 [in] `uiCommandID`  
 Specifica l'ID di comando dell'attività per mostrare o nascondere.  
  
 [in] `bShow`  
 `TRUE`Per visualizzare l'attività. `FALSE` per nascondere l'attività.  
  
 [in] `bRedraw`  
 `TRUE`Per ridisegnare il riquadro attività. in caso contrario, `FALSE`.  
  
### <a name="return-value"></a>Valore restituito  
 `TRUE`Se l'attività è stata correttamente mostrato o nascosto; `FALSE` se un'attività con l'ID di comando specificato non esiste.  
  
### <a name="remarks"></a>Note  
 Utilizzare [CMFCTasksPane::ShowTask](#showtask) per mostrare o nascondere un'attività in base al relativo ID di comando.  
  
##  <a name="update"></a>CMFCTasksPane::Update  
 Consente di aggiornare tutti i controlli in un riquadro attività.  
  
```  
virtual void Update();
```  
  
### <a name="remarks"></a>Note  
 Questo metodo aggiorna la didascalia del riquadro attività, consente di regolare la barra di scorrimento, Riposiziona tutte le attività e ridisegna tutti i controlli del riquadro attività.  
  
 L'override del metodo in una classe derivata può eseguire codice personalizzato quando il framework aggiorna il riquadro attività.
