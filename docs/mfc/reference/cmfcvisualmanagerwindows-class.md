# <a name="cmfcvisualmanagerwindows-class"></a>Classe CMFCVisualManagerWindows

`CMFCVisualManagerWindows`simula l'aspetto di Microsoft Windows XP o Microsoft Vista quando l'utente seleziona un tema di Windows XP o Vista.

Per ulteriori dettagli, vedere il codice sorgente che si trova nella cartella **\\\\mfc di VC atlmfc\\** dell'installazione di Visual Studio.

## <a name="syntax"></a>Sintassi

```
class CMFCVisualManagerWindows : public CMFCVisualManagerOfficeXP
```

## <a name="remarks"></a>Osservazioni

Utilizzare `CMFCVisualManagerWindows` la classe per modificare l'aspetto dell'applicazione per simulare il tema corrente nel computer in cui viene eseguita l'applicazione.

Tuttavia, un tema di Windows potrebbe non essere disponibile se l'applicazione è in esecuzione su una versione di Windows precedente a Windows XP o se i temi sono disabilitati perché l'utente utilizza la visualizzazione **classica.** Se non è disponibile alcun tema, l'applicazione utilizza il gestore visualizzazione predefinito definito in [CMFCVisualManager](../../mfc/reference/cmfcvisualmanager-class.md).
