# Aggiunta di un consumer ODBC MFC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Il consumer ODBC MFC è costituito da una classe recordset ODBC e dalle associazioni dati necessarie per accedere a un'origine dati.  
  
### Per aggiungere un consumer ODBC MFC  
  
1.  In Visualizzazione classi fare clic con il pulsante destro del mouse sul progetto.  Scegliere **Aggiungi** dal menu di scelta rapida, quindi **Aggiungi classe**.  
  
2.  Nella cartella Visual C\+\+ fare doppio clic sull'icona **Consumer ODBC MFC** oppure selezionarla e scegliere **Apri**.  
  
     Verrà visualizzata la Creazione guidata consumer ODBC MFC.  
  
3.  Definire le impostazioni come descritto nella [Creazione guidata consumer ODBC MFC](../../mfc/reference/mfc-odbc-consumer-wizard.md).  
  
4.  Scegliere **Fine** per uscire dalla procedura guidata. Il nuovo codice del consumer ODBC verrà inserito nel progetto.
