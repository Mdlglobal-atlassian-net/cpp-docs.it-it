# <a name="styles-used-by-mfc"></a>Stili utilizzati da MFC
Utilizzare gli stili seguenti quando si crea l'oggetto MFC corrispondente. Nella maggior parte dei casi, questi stili sono impostati `dwStyle` parametro della classe **crea** (funzione).  
  
### <a name="mfc-styles"></a>Stili MFC  
  
|Stile|Descrizione|  
|-----------|-----------------|  
|[Stili dei pulsanti](../../mfc/reference/button-styles.md)|Si applica a [classe CButton](../../mfc/reference/cbutton-class.md) oggetti, ad esempio pulsanti, controllare finestre e pulsanti di comando. Specificare una combinazione di stili di `dwStyle` parametro di [CButton::Create](../../mfc/reference/cbutton-class.md#create).|  
|[Stili casella combinata](../../mfc/reference/combo-box-styles.md)|Si applica a [CComboBox (classe)](../../mfc/reference/ccombobox-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CComboBox::Create](../../mfc/reference/ccombobox-class.md#create).|  
|[Modifica stili](../../mfc/reference/edit-styles.md)|Si applica a [CEdit Class](../../mfc/reference/cedit-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CEdit::Create](../../mfc/reference/cedit-class.md#create).|  
|[Stili frame-window](../../mfc/reference/frame-window-styles-mfc.md)|Si applica a [CFrameWnd (classe)](../../mfc/reference/cframewnd-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CFrameWnd::Create](../../mfc/reference/cframewnd-class.md#create).|  
|[Stili casella di riepilogo](../../mfc/reference/list-box-styles.md)|Si applica a [CListBox (classe)](../../mfc/reference/clistbox-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CListBox::Create](../../mfc/reference/clistbox-class.md#create).|  
|[Stili finestra di messaggio](../../mfc/reference/message-box-styles.md)|Si applica a [AfxMessageBox](../../mfc/reference/cstring-formatting-and-message-box-display.md#afxmessagebox) elementi. Specificare una combinazione di stili di `nType` parametro di `AfxMessageBox`.|  
|[Stili barra di scorrimento](../../mfc/reference/scroll-bar-styles.md)|Si applica a [CScrollBar classe](../../mfc/reference/cscrollbar-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CScrollBar::Create](../../mfc/reference/cscrollbar-class.md#create).|  
|[Stili statici](../../mfc/reference/static-styles.md)|Si applica a [classe CStatic](../../mfc/reference/cstatic-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CStatic::Create](../../mfc/reference/cstatic-class.md#create).|  
|[Stili finestra](../../mfc/reference/window-styles.md)|Si applica a [classe CWnd](../../mfc/reference/cwnd-class.md) oggetti. Specificare una combinazione di stili di `dwStyle` parametro di [CWnd:: Create](../../mfc/reference/cwnd-class.md#create) o [CWnd::CreateEx](../../mfc/reference/cwnd-class.md#createex).|  
|[Stili finestra estesi](../../mfc/reference/extended-window-styles.md)|Si applica a [classe CWnd](../../mfc/reference/cwnd-class.md) oggetti. Specificare una combinazione di stili di `dwExStyle` parametro di [CWnd::CreateEx](../../mfc/reference/cwnd-class.md#createex).|
