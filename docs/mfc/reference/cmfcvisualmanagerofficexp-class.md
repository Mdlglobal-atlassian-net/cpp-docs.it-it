# <a name="cmfcvisualmanagerofficexp-class"></a>Classe CMFCVisualManagerOfficeXP
Il `CMFCVisualManagerOfficeXP` fornisce a un'applicazione un aspetto di Microsoft Office XP.  
  
## <a name="syntax"></a>Sintassi  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## <a name="members"></a>Membri  
 Tutti i membri di questa classe sono funzioni virtuali che sono derivate dal predecessore di questa classe, [CMFCVisualManager classe](../../mfc/reference/cmfcvisualmanager-class.md).  
  
## <a name="remarks"></a>Note  
 Utilizzare la `CMFCVisualManagerOfficeXP` classe per modificare l'aspetto visivo dell'applicazione sono simili a quello di Microsoft Office XP.
