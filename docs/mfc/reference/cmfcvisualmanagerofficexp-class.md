# CMFCVisualManagerOfficeXP Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`CMFCVisualManagerOfficeXP` fornisce a un'applicazione un aspetto di Microsoft Office XP.  
  
## Sintassi  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## Membri  
 Tutti i membri di questa classe sono funzioni virtuali che vengono derivate dal predecessore di questa classe, [CMFCVisualManager Class](../../mfc/reference/cmfcvisualmanager-class.md).  
  
## Note  
 Utilizza la classe `CMFCVisualManagerOfficeXP` per modificare l'aspetto visivo dell'applicazione simile a quella di Microsoft Office XP.
