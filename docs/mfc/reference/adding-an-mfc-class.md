# <a name="adding-an-mfc-class"></a>Aggiunta di una classe MFC
Per aggiungere le classi derivate dalle classi della libreria di classe MFC (Microsoft Foundation) al progetto, utilizzare il **Aggiungi classe** i comandi disponibili nel [Visualizzazione classi](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Specificare il nome della nuova classe, selezionare la classe di base e selezionare l'ID della finestra di dialogo a cui è associato (se presente). La procedura guidata codice crea un file di intestazione e un file di implementazione e li aggiunge al progetto.  
  
> [!NOTE]
>  È possibile aggiungere classi MFC a un'applicazione ATL COM se è inizialmente [l'applicazione creata con il supporto di MFC](../../atl/reference/mfc-support-in-atl-projects.md). È anche possibile aggiungere classi MFC per i progetti Win32 che dispongono del supporto MFC.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>Per aggiungere al progetto una classe MFC  
  
1.  Da Visualizzazione classi, fare clic sul nome del progetto. Fare clic su **Aggiungi** e quindi fare clic su **Aggiungi classe** per aprire la [Aggiungi classe](../../ide/add-class-dialog-box.md) la finestra di dialogo.  
  
2.  Nel riquadro Modelli selezionare **classe MFC** e premere il **Aggiungi** pulsante.  
  
3.  Definire le impostazioni per la nuova classe nel [Creazione guidata classe MFC](../../mfc/reference/mfc-add-class-wizard.md) la finestra di dialogo.  
  
4.  Fare clic su **fine** per chiudere la procedura guidata e visualizzare la nuova classe in visualizzazione classi. È inoltre possibile visualizzare i file creati dalla procedura guidata in **Esplora**.
