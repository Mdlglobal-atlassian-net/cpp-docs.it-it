## <a name="remarks"></a>Osservazioni

Un `CMFCToolBarMenuButton` può essere visualizzato come un menu, una voce di menu con un sottomenu, un pulsante che esegue un comando o visualizza un menu o un pulsante che visualizza solo un menu. È possibile determinare il comportamento e l'aspetto del pulsante di menu specificando parametri quali l'immagine, `CMFCToolbarMenuButton::CMFCToolbarMenuButton`il testo, la maniglia di menu e l'ID di comando associati al pulsante nel costruttore .

Una classe personalizzata `CMFCToolbarMenuButton` derivata dalla classe deve utilizzare la macro [DECLARE_SERIAL.](run-time-object-model-services.md#declare_serial) La macro [DECLARE_DYNCREATE](run-time-object-model-services.md#declare_dyncreate) genera un errore alla chiusura dell'applicazione.

## <a name="cmfctoolbarmenubuttoncmfctoolbarmenubutton"></a><a name="cmfctoolbarmenubutton"></a>CMFCToolBarMenuButton::CMFCToolBarMenuButton

Costruisce un oggetto `CMFCToolBarMenuButton`.

```
CMFCToolBarMenuButton();
CMFCToolBarMenuButton(const CMFCToolBarMenuButton& src);

CMFCToolBarMenuButton(
    UINT uiID,
    HMENU hMenu,
    int iImage,
    LPCTSTR lpszText=NULL,
    BOOL bUserButton=FALSE);
```

### <a name="parameters"></a>Parametri

*src*<br/>
[in] Oggetto `CMFCToolBarMenuButton` esistente da copiare `CMFCToolBarMenuButton` in questo oggetto.

*uiID*<br/>
[in] ID del comando da eseguire quando un utente fa clic sul pulsante; o (UINT)-1 per un pulsante di menu che non esegue direttamente un comando.

*Hmenu*<br/>
[in] Una maniglia per un menu; o NULL se il pulsante non dispone di un menu.

*iImage (immagine)*<br/>
[in] Indice dell'immagine per il pulsante; o -1 se questo pulsante non dispone di un'icona o utilizza l'icona per il comando specificato da *uiID*. L'indice è lo `CMFCToolBarImages` stesso per ogni oggetto nell'applicazione.

*lpszText*<br/>
[in] Testo del pulsante del menu della barra degli strumenti.

*bPulsante utente*<br/>
[in] TRUESe il pulsante visualizza un'immagine definita dall'utente. FALSE se il pulsante visualizza un'immagine predefinita associata al comando specificato da *uiID*.

### <a name="remarks"></a>Osservazioni

Se *uiID* è un ID di comando valido, il pulsante esegue tale comando quando l'utente fa clic su di esso. Se *hMenu* è una maniglia di menu valida, il pulsante fornisce un menu a discesa quando viene visualizzato in una barra degli strumenti o un sottomenu quando viene visualizzato in un menu. Se sia *uiID* che *hMenu* sono validi, il pulsante è un pulsante di divisione con una parte che eseguirà il comando quando l'utente fa clic su di esso e una parte con una freccia verso il basso che verrà a discesa un menu quando l'utente fa clic su di esso. Tuttavia, se *hMenu* è valido, un utente non sarà in grado di fare clic sul pulsante per eseguire un comando quando il pulsante viene inserito in un menu.
