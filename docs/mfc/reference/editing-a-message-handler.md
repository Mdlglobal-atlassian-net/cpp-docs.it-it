# Modifica di un gestore messaggi
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Dopo aver definito un gestore messaggi, è possibile passare alla definizione della funzione membro per aggiungere o modificare il codice.  
  
 Per passare alla definizione di una funzione membro dall'[editor finestre](../../mfc/dialog-editor.md), fare doppio clic su un controllo per il quale è già stato definito un gestore.  Ciò consente di passare al file in cui è stato definito il gestore messaggi del controllo selezionato.  
  
 Per passare alla definizione di una funzione membro da [Visualizzazione classi](http://msdn.microsoft.com/it-it/8d7430a9-3e33-454c-a9e1-a85e3d2db925), fare doppio clic sul nome della funzione.
