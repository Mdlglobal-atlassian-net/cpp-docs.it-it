# Tipi di dati (MFC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

In questo argomento sono elencati i tipi di dati più utilizzati nella libreria MFC.  La maggior parte dei tipi di dati sono uguali a quelli in software development kit \(SDK\) della piattaforma, mentre altre sono univoci a MFC.  
  
 Per informazioni sui tipi di dati utilizzati sia in [!INCLUDE[winSDK](../../atl/includes/winsdk_md.md)] che in MFC, vedere [Tipi di dati windows](http://msdn.microsoft.com/library/windows/desktop/aa383751).  
  
 I tipi di dati univoci alla libreria MFC includono:  
  
-   Valore di**POSIZIONE**A utilizzato per indicare la posizione di un elemento in una raccolta; utilizzato dalle classi collection MFC.  
  
-   Puntatore a 32 bit di**LPCRECT**A una struttura \(non modificabile\) costante di `RECT`.
