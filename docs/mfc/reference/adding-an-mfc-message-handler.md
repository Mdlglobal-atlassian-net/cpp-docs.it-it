# Aggiunta di un gestore messaggi MFC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

È possibile [utilizzare la finestra Proprietà per aggiungere un gestore messaggi](../../mfc/reference/mapping-messages-to-functions.md) \(una funzione membro che gestisce i messaggi di Windows\) a una classe e associare i messaggi di Windows al gestore.  È anche possibile aggiungere [un gestore eventi per ciascun controllo finestra di dialogo](../../mfc/adding-event-handlers-for-dialog-box-controls.md).  
  
 Utilizzando la finestra Proprietà per definire le funzioni di gestione di eventi e messaggi, è possibile aggiornare automaticamente la tabella di invio dei messaggi, o mappa messaggi, e il file di intestazione della classe.  
  
> [!NOTE]
>  È possibile aggiungere un gestore messaggi a una classe ATL nella finestra Proprietà, ma con risultati variabili.  Per ulteriori informazioni, vedere l'argomento [Aggiunta di un gestore messaggi ATL](../../atl/adding-an-atl-message-handler.md).
