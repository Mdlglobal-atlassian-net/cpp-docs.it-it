# <a name="adding-an-mfc-message-handler"></a>Aggiunta di un gestore messaggi MFC
È possibile [utilizzare la finestra proprietà per aggiungere un gestore di messaggi](../../mfc/reference/mapping-messages-to-functions.md) (una funzione membro che gestisce i messaggi di Windows) a una classe e associare i messaggi di Windows per il gestore di messaggi. È inoltre possibile aggiungere [un gestore eventi per qualsiasi controllo finestra di dialogo](../../windows/adding-event-handlers-for-dialog-box-controls.md).  
  
 Utilizzando la finestra proprietà per definire le funzioni di gestione di eventi e messaggi, è possibile aggiornare automaticamente la tabella di invio dei messaggi (o mappa messaggi) e file di intestazione della classe.  
  
> [!NOTE]
>  È possibile aggiungere un gestore di messaggi a una classe ATL utilizzando la finestra Proprietà. Tuttavia, alcuni risultati possono variare. Per ulteriori informazioni, vedere l'argomento [aggiunta di un gestore messaggi ATL](../../atl/adding-an-atl-message-handler.md).
