# Gestione dei menu durante l&#39;esecuzione del programma
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Utilizzare la classe `CMenu` per modificare i menu e di voci di menu immediatamente.  `CMenu` incapsula gli handle di `HMENU` di windows e fornisce le funzioni membro per l'utilizzo dei menu.  
  
 Vedere i cenni preliminari sulla classe [CMenu](../mfc/reference/cmenu-class.md) per i dettagli.
