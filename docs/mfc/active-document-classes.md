# Classi documento attivo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

I documenti attivi possono essere visualizzati nell'intera finestra client di un browser, ad esempio Internet Explorer 5.5, oppure in un contenitore attivo, ad esempio il raccoglitore Microsoft Office, tale documenti attivi databinding.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 Esegue il mapping delle interfacce del documento attivo e inizializza e attiva un oggetto documento attivo.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 Verbi del server OLE implements in particolare per i server di documenti attivi.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 Implementa Active Document Containment.
