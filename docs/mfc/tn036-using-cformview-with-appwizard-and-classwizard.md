# TN036: utilizzo di CFormView con AppWizard e ClassWizard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Questa nota tecnica descritta come modificare AppWizard ha generato l'applicazione in modo che conteneva `CFormView` anziché `CView` predefinita della classe visualizzazione principale.  Questo comportamento è supportato direttamente con questa versione di Visual C\+\+.
