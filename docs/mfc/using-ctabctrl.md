# Utilizzo di CTabCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

"Un controllo Struttura a schede" è analogo ai divisori di un blocco appunti o nelle cartelle contrassegnate in un file di installazione.  Utilizzare il controllo Struttura a schede, rappresentato dalla classe [CTabCtrl](../mfc/reference/ctabctrl-class.md), per illustrare più pagine di informazioni o controlli a un utente, uno alla volta, in un formato che suggerisce una relazione peer o logica tra ciascuna pagina.  
  
 Per ulteriori informazioni sui controlli scheda, vedere [Controllo Struttura a schede](http://msdn.microsoft.com/library/windows/desktop/bb760550) in [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)].
