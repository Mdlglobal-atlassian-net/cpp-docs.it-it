# Classi Array pronte all&#39;uso
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le classi seguenti classi sono pronte per l'uso di matrice.  
  
 [CByteArray](../mfc/reference/cbytearray-class.md)  
 Contiene gli elementi di tipo **BYTE** in una matrice.  
  
 [CDWordArray](../mfc/reference/cdwordarray-class.md)  
 Contiene gli elementi di tipo `DWORD` in una matrice.  
  
 [CObArray](../mfc/reference/cobarray-class.md)  
 Puntatori di archiviare gli oggetti di classe `CObject` o a oggetti delle classi derivate da `CObject` in una matrice.  
  
 [CPtrArray](../mfc/reference/cptrarray-class.md)  
 Archivia i puntatori a `void` \(puntatori generici\) in una matrice.  
  
 [CUIntArray](../mfc/reference/cuintarray-class.md)  
 Contiene gli elementi di tipo **UINT** in una matrice.  
  
 [CWordArray](../mfc/reference/cwordarray-class.md)  
 Contiene gli elementi di tipo **WORD** in una matrice.  
  
 [CStringArray](../mfc/reference/cstringarray-class.md)  
 Archivia gli oggetti di `CString` in una matrice.
