# <a name="using-cricheditctrl"></a>Utilizzo di CRichEditCtrl

Un controllo rich edit è una finestra in cui l'utente può immettere e modificare il testo. Il testo può essere assegnato a carattere e formattazione di paragrafo e può includere oggetti OLE incorporati. Controllo rich edit è rappresentato in MFC per la [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) classe.
