# <a name="using-cricheditctrl"></a>Utilizzo di CRichEditCtrl

Un controllo Rich Edit è una finestra in cui l'utente può immettere e modificare il testo. Al testo può essere assegnata la formattazione di caratteri e paragrafi e possono includere oggetti OLE incorporati. Il controllo Rich Edit è rappresentato in MFC dalla classe [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) .
