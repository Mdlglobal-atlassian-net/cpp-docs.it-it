# <a name="document-template-classes"></a>Classi Document-Template
Gli oggetti modello di documento coordinano la creazione di documenti, visualizzazione e oggetti della finestra cornice quando un nuovo documento o la vista viene creata.  
  
 [CDocTemplate](../mfc/reference/cdoctemplate-class.md)  
 La classe base per i modelli di documento. Questa classe non verrà mai utilizzato direttamente. In alternativa, utilizzare una delle altre classi di modello di documento derivate da questa classe.  
  
 [CMultiDocTemplate](../mfc/reference/cmultidoctemplate-class.md)  
 Un modello per i documenti di interfaccia a documenti multipli (MDI). Le applicazioni MDI possono avere più documenti aprire contemporaneamente.  
  
 [CSingleDocTemplate](../mfc/reference/csingledoctemplate-class.md)  
 Un modello per i documenti di interfaccia a documento singolo (SDI). SDI (applicazioni) che un solo documento aperto in una fase.  
  
## <a name="related-class"></a>Classe correlata  
 [CCreateContext](../mfc/reference/ccreatecontext-structure.md)  
 Una struttura passati da un modello di documento alle funzioni di creazione della finestra per coordinare la creazione di oggetti di documenti, visualizzazione e finestra cornice.
