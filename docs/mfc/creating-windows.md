# <a name="creating-windows"></a>Creazione di finestre
Il framework crea automaticamente la maggior parte delle finestre che è necessario in un programma di framework. [Creazione documento/visualizzazione](../mfc/document-view-creation.md) viene illustrato come il framework crea le finestre cornice associate a documenti e visualizzazioni. Ma per utilizzi specifici è possibile creare proprie finestre, incluse le proprie finestre figlio di finestre cornice o viste: oltre alle finestre fornite dal framework.
