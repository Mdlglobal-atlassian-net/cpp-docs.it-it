# Utilizzo di un modello di finestra di dialogo in memoria
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Anziché utilizzare i metodi forniti in [Tabella di creazione della finestra di dialogo](../mfc/creating-a-dialog-class-with-code-wizards.md), è possibile creare indirettamente qualsiasi tipo di finestra di dialogo da un modello di finestra di dialogo in memoria.  Per ulteriori informazioni, vedere la classe [CDialog](../mfc/reference/cdialog-class.md) in *Riferimento MFC*.
