# Funzioni membro comunemente aggiunte
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Se la finestra di dialogo contenente i pulsanti diverso da quello OK o cancel, è necessario scrivere i funzione membro per la gestione dei messaggi nella classe della finestra di dialogo per rispondere ai messaggi di notifica di controllo genera.  Per un esempio, vedere l'esempio di programma di [Scribble](../top/visual-cpp-samples.md).  È inoltre possibile gestire i messaggi di notifica del controllo da altri controlli nella finestra di dialogo.
