# Dialog Editor States (Guides and Grids)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile disporre i controlli nelle finestre di dialogo con l'editor finestre in uno dei tre stati seguenti:  
  
-   Con le guide e i margini attivati \(impostazione predefinita\)  
  
-   Con la griglia di layout attivata  
  
-   Senza funzionalità di blocco o allineamento  
  
 La [barra degli strumenti dell'editor finestre](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md) contiene pulsanti che controllano lo stato.  Per modificare lo stato, fare clic sull'icona appropriata.  È possibile modificare lo stato anche scegliendo il comando **Impostazioni guide** dal menu **Formato**.  
  
-   [Creazione e impostazione di guide e margini](../mfc/creating-and-setting-guides-and-margins.md)  
  
-   [Allineamento di controlli su una guida](../mfc/aligning-controls-on-a-guide.md)  
  
-   [Disattivazione delle guide](../mfc/disabling-guides.md)  
  
-   [Modifica della griglia di layout](../mfc/modifying-the-layout-grid.md)  
  
 Per informazioni sull'aggiunta di risorse a progetti gestiti, vedere [Risorse nelle applicazioni](../Topic/Resources%20in%20Desktop%20Apps.md) nella *Guida per gli sviluppatori di .NET Framework.* Per informazioni sull'aggiunta manuale di file di risorse a progetti gestiti, sull'accesso alle risorse, sulla visualizzazione di risorse statiche e sull'assegnazione di stringhe di risorse alle proprietà, vedere [Procedura dettagliata: localizzazione di Windows Form](http://msdn.microsoft.com/it-it/9a96220d-a19b-4de0-9f48-01e5d82679e5) e [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
