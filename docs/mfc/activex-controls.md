# <a name="activex-controls"></a>Controlli ActiveX

In Visual C++ è possibile creare controlli ActiveX con MFC o ATL.

>[!IMPORTANT]
> ActiveX è una tecnologia legacy che non deve essere utilizzata per nuove attività di sviluppo. Molte funzionalità dei controlli ActiveX possono essere eseguite in modo più semplice e molto più sicuro con le tecnologie moderni come HTML5 e JavaScript, le estensioni del browser moderni o WebAssembly moduli. Per altre informazioni, vedere [un'interruzione passata, parte 2: browser attachEvent ActiveX, VBScript,...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) e [messaggistica Native](/microsoft-edge/extensions/guides/native-messaging) e [estensioni Microsoft Edge](/microsoft-edge/extensions) e [WebAssembly](https://webassembly.org/).
