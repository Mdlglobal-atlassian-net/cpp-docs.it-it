# Come vengono cercate le mappe messaggi nel framework
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Il framework esegue la ricerca della tabella della mappa messaggi le corrispondenze con i messaggi in arrivo.  Quando si scrive una voce della mappa messaggi per ogni messaggio desiderato una classe per gestire e scrivere i gestori corrispondente, il framework chiama i gestori automaticamente.  Negli argomenti seguenti vengono illustrati la ricerca della mappa messaggi:  
  
-   [Dove trovare le mappe messaggi](../mfc/where-to-find-message-maps.md)  
  
-   [Mappe messaggi derivate](../mfc/derived-message-maps.md)  
  
-   [Intervalli di mapping dei messaggi, degli ID di comando, o control ID a un gestore](../mfc/handlers-for-message-map-ranges.md)
