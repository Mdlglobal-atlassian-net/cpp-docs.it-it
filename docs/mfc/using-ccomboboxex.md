# Utilizzo di CComboBoxEx
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Il controllo casella combinata estesa è un'estensione del controllo casella combinata standard che fornisce il supporto nativo di immagini dell'elemento.  Queste immagini possono essere utilizzate per indicare lo stato di singoli elementi della casella combinata, ad esempio gli elementi attualmente selezionati e deselezionati.  Per rendere le immagini dell'elemento facilmente accessibili, il controllo fornisce il supporto di elenco.  
  
 Utilizzare questo controllo per fornire la funzionalità di una casella combinata senza dover manualmente disegnare grafica dell'elemento.
