# <a name="using-ccomboboxex"></a>Utilizzo di CComboBoxEx

Il controllo casella combinata estesa è un'estensione del controllo casella combinata standard che fornisce il supporto nativo per le immagini di elementi. Queste immagini possono essere utilizzate per indicare lo stato dei singoli elementi nella casella combinata, ad esempio gli elementi attualmente selezionati e deselezionati. Per rendere facilmente accessibili le immagini degli elementi, il controllo fornisce supporto per l'elenco immagini.

Utilizzare questo controllo per fornire la funzionalità di una casella combinata senza dover tracciare manualmente la grafica dell'elemento.
