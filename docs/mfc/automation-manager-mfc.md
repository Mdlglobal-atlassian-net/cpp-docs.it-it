# <a name="automation-manager-mfc"></a>Gestione automazione (MFC)
AUTMGR32. File EXE deve essere copiati nella directory di sistema di Windows di ogni computer che non sia progettata per fornire oggetti di automazione remota. Per Windows 95 e Windows 98, questa directory è in genere C:\WINDOWS\SYSTEM. Per Windows NT e Windows 2000, è in genere C:\WINNT\SYSTEM32.  
  
 Se si desidera abilitare callback dal server al client, il file eseguibile dovrebbe essere copiato anche nella directory di sistema di ciascun computer client.  
  
 Quando è in esecuzione il gestore di automazione, visualizza una piccola finestra sul computer server che contiene informazioni sullo stato breve. Se si desidera nascondere, consultare l'articolo Q138067 della Microsoft Knowledge Base.
