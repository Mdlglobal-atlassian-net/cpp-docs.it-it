# Creazione di una classe di finestre di dialogo con creazioni guidate codice
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le seguenti attività correlate dialogo\- della tabella elenchi che le creazioni guidate codice consentono di gestire.  
  
### Attività correlate Dialogo\-  
  
|Attività|Applica a. . .|  
|--------------|--------------------|  
|[Creare una nuova classe derivata da cdialog](../mfc/creating-your-dialog-class.md) per gestire la finestra di dialogo.|Ogni finestra di dialogo.|  
|[Messaggi di windows della mappa](../mfc/handling-windows-messages-in-your-dialog-box.md) alla classe della finestra di dialogo.|Ogni messaggio desiderato gestito.|  
|Dichiarare le variabili membro di una classe per rappresentare i controlli nella finestra di dialogo.|Ogni controllo che rende un testo o un valore numerico si desidera accedere dal programma.|  
|Specificare [come i dati vengano scambiati](../mfc/dialog-data-exchange-and-validation.md) tra i controlli e le variabili membro.|Ogni controllo che si desidera accedere dal programma.|  
|Specificare [regole di convalida](../mfc/dialog-data-exchange-and-validation.md) per le variabili membro.|Ogni controllo che rende un testo o un valore numerico, se necessario.|
