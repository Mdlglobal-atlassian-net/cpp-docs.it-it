# Utilizzo di CToolBarCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile utilizzare il controllo toolbar \([CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md)\) per creare una finestra di controllo contenente pulsanti e spazi facoltativi.  Ogni pulsante nella finestra di controllo toolbar invia un messaggio di comando alla finestra padre quando l'utente la sceglie.  In genere, i pulsanti nella barra degli strumenti corrispondono agli elementi nel menu dell'applicazione, fornendo all'utente una modalità aggiuntiva e più diretta di accedere ai controlli di un'applicazione.
