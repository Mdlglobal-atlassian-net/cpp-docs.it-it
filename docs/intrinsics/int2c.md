# <a name="int2c"></a>__int2c
**Sezione specifica Microsoft**  
  
 Genera il `int 2c` (istruzione), che attiva il `2c` interrupt.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void __int2c(void);  
```
