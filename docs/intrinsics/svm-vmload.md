# <a name="svmvmload"></a>__svm_vmload

**Sezione specifica Microsoft**

Carica un subset di stato del processore dal blocco di controllo (VMCB) della macchina virtuale specificata.

## <a name="syntax"></a>Sintassi

```
void __svm_vmload(
   size_t VmcbPhysicalAddress
);
```

#### <a name="parameters"></a>Parametri

|Parametro|Descrizione|
|---------------|-----------------|
|*VmcbPhysicalAddress*|[in] L'indirizzo fisico del VMCB.|

## <a name="remarks"></a>Note

Il `__svm_vmload` Ã¨ equivalente alla funzione il `VMLOAD` istruzioni in linguaggio macchina. Questa funzione supporta l'interazione di monitoraggio della macchina virtuale di un host con un sistema operativo guest e le relative applicazioni. Per altre informazioni, cercare il documento, "Volume manuale per programmatori dell'architettura AMD64 2: programmazione di sistema," documento numero 24593, revisione 3.11, la [corporation AMD](https://developer.amd.com/resources/developer-guides-manuals/) sito.
