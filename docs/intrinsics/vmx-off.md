# <a name="vmxoff"></a>__vmx_off
**Sezione specifica Microsoft**  
  
 Disattiva l'operazione della macchina virtuale (VMX) le estensioni del processore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void __vmx_off();  
```  
  
## <a name="remarks"></a>Note

Questa funzione supporta l'interazione di monitoraggio della macchina virtuale di un host con un sistema operativo guest e le relative applicazioni.
