# <a name="cipow"></a>_CIpow
Calcola *x* elevato alla potenza *y* in base ai primi valori dello stack.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void __cdecl _CIpow();  
```  
  
## <a name="remarks"></a>Note
