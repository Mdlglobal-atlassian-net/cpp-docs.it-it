# Modifica di stringhe (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Queste routine agiscono su un carattere a byte singolo con terminazione null, a caratteri estesi e stringhe di caratteri multibyte.  Utilizzare le routine di modifica del buffer, descritte in [Modifica del buffer](../c-runtime-library/buffer-manipulation.md), per utilizzare gli array di caratteri che non terminano con un carattere null.
