# <a name="ciatan"></a>_CIatan
Calcola l'arcotangente del primo valore dello stack.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void __cdecl _CIatan();  
```  
  
## <a name="remarks"></a>Note
