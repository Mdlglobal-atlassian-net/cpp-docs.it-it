# Funzioni CRT non supportate da Windows Runtime
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Molte API di runtime del linguaggio C \(CRT\) non possono essere usate nelle app [!INCLUDE[win8_appname_long](../build/includes/win8_appname_long_md.md)] in esecuzione nel [!INCLUDE[wrt](../atl/reference/includes/wrt_md.md)].  Queste app vengono compilate usando il flag del compilatore \/ZW.  Per un elenco di funzioni CRT non supportate, vedere [Funzioni CRT non supportate da \/ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).  
  
 Tutte le API CRT sono descritte nella sezione [Riferimento alfabetico alle funzioni](../c-runtime-library/reference/crt-alphabetical-function-reference.md) della documentazione.
