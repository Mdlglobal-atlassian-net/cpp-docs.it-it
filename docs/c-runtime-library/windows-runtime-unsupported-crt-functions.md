# <a name="windows-runtime-unsupported-crt-functions"></a>Funzioni CRT non supportate da Windows Runtime
Molte API CRT non possono essere usate nelle app UWP (Universal Windows Platform) in esecuzione in Windows Runtime. Queste app vengono compilate usando il flag del compilatore /ZW. Per un elenco di funzioni CRT non supportate, vedere [CRT functions not supported by /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx) (Funzioni CRT non supportate da /ZW).  
  
 Tutte le API CRT sono descritte nella sezione [Riferimento alfabetico alle funzioni](../c-runtime-library/reference/crt-alphabetical-function-reference.md) della documentazione.
