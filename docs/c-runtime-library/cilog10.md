# <a name="cilog10"></a>_CIlog10

Esegue un'operazione `log10` sul primo valore dello stack.

## <a name="syntax"></a>Sintassi

```
void __cdecl _CIlog10();
```

## <a name="remarks"></a>Osservazioni
