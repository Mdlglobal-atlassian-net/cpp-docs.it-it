# <a name="cilog"></a>_CIlog
Calcola il logaritmo naturale del primo valore dello stack.  
  
## <a name="syntax"></a>Sintassi  
  
```  
void __cdecl _CIlog();  
```  
  
## <a name="remarks"></a>Note
