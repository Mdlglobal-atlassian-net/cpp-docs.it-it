# _DEBUG
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Il compilatore definisce `_DEBUG` quando si specifica l'opzione \/MDd o \/MTd.  Queste opzioni specificano le versioni di debug della libreria di runtime del linguaggio C.  
  
 Per ulteriori informazioni, consultare [Tecniche di debug di CRT](../Topic/CRT%20Debugging%20Techniques.md).
