# Mapping testo generico
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Per semplificare la scrittura di codice per i mercati internazionali, in TCHAR.H sono definite le mappature di testo generico:  
  
-   [Tipi di dati](../c-runtime-library/data-type-mappings.md)  
  
-   [Costanti e variabili globali](../c-runtime-library/constant-and-global-variable-mappings.md)  
  
-   [Mapping di routine](../c-runtime-library/routine-mappings.md)  
  
 Per ulteriori informazioni, vedere [Utilizzare le mappature di testo generico](../c-runtime-library/using-generic-text-mappings.md).  Le mappature di testo generico sono estensioni Microsoft non compatibili con lo standard ANSI.
