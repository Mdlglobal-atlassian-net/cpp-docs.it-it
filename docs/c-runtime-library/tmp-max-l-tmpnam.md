# TMP_MAX, L_tmpnam
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Sintassi  
  
```  
#include <stdio.h>  
```  
  
## Note  
 `TMP_MAX` è il numero massimo dei nomi dei file univoci che la funzione `tmpnam` può generare.  `L_tmpnam` è la lunghezza dei nomi di file temporanei generati da `tmpnam`.
