# Routine di runtime per categoria
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In questa sezione vengono elencate e descritte le routine di libreria di runtime di Microsoft per categoria.  Per praticità di riferimento, alcune procedure sono elencate in più categorie.  Le routine di caratteri multibyte e le routine a caratteri estesi vengono raggruppati con byte singoli \- controparti di carattere, in cui si trovano.  
  
 In questa sezione vengono inoltre forniti metodi o classi .NET Framework equivalenti dove disponibile.
