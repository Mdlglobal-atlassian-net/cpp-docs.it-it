# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>Sintassi

```

#include <stdlib.h>

```

## <a name="remarks"></a>Note

La costante `RAND_MAX` è il valore massimo che può essere restituito dalla funzione `rand`. `RAND_MAX` viene definito come valore 0x7fff.
