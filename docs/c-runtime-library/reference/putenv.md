# <a name="putenv"></a>putenv
Questa funzione POSIX è deprecata. In alternativa, usare [_putenv](../../c-runtime-library/reference/putenv-wputenv.md) conforme a ISO C++ o [_putenv_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md) con sicurezza avanzata.  
  
> [!IMPORTANT]
>  Non è possibile usare questa API nelle applicazioni eseguite in Windows Runtime. Per altre informazioni, vedere l'articolo relativo alle [funzioni CRT non supportate con /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
