# <a name="cgets"></a>cgets
Questa funzione POSIX è deprecata. In alternativa usare [_cgets](../../c-runtime-library/cgets-cgetws.md) conforme a ISO C++ o [_cgets_s](../../c-runtime-library/reference/cgets-s-cgetws-s.md) con sicurezza avanzata.  
  
> [!IMPORTANT]
>  Non è possibile usare questa API nelle applicazioni eseguite in Windows Runtime. Per altre informazioni, vedere [HYPERLINK "http://msdn.microsoft.com/en-us/library/windows/apps/jj606124.aspx" \o "CRT functions not supported with /ZW" Funzioni CRT non supportate con /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
