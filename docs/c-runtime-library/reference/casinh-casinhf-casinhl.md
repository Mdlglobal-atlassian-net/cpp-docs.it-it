# <a name="casinh-casinhf-casinhl"></a>casinh, casinhf, casinhl
Recupera il seno iperbolico inverso di un numero complesso, con tagli branch all'esterno dell'intervallo [-i, + i] lungo l'asse immaginaria.  
  
## <a name="syntax"></a>Sintassi  
  
```  
_Dcomplex casinh(   
   _Dcomplex z   
);  
_Fcomplex casinh(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex casinh(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex casinhf(   
   _Fcomplex z   
);  
_Lcomplex casinhl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parametri  
 `z`  
 Numero complesso che rappresenta un angolo, espresso in radianti.  
  
## <a name="return-value"></a>Valore restituito  
 Seno iperbolico inverso di `z`, espresso in radianti. Il risultato viene disassociato lungo l'asse reale e nell'intervallo [-iπ/2 + iπ/2] lungo l'asse immaginaria.  
  
## <a name="remarks"></a>Note  
 Dato che C++ consente l'overload, è possibile chiamare degli overload di `casinh` che accettino e restituiscano valori `_Fcomplex` e `_Lcomplex`. In un programma C `casinh` accetta e restituisce sempre un valore `_Dcomplex` .
