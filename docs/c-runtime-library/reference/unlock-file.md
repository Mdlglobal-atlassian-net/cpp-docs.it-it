# <a name="_unlock_file"></a>_unlock_file

Sblocca un file, consentendo ad altri processi di accedere al file.

## <a name="syntax"></a>Sintassi

```C
void _unlock_file(
   FILE* file
);
```

### <a name="parameters"></a>Parametri

*ﬁle*<br/>
Handle di file.

## <a name="remarks"></a>Osservazioni

La funzione **_unlock_file** sblocca il file specificato dal *file*. Lo sblocco di un file consente l'accesso al file da altri processi. Questa funzione non deve essere chiamata a meno che **_lock_file** sia stata chiamata in precedenza sul puntatore del *file.* La chiamata **_unlock_file** su un file che non è bloccato può causare un deadlock. Per un esempio, vedere [_lock_file](lock-file.md).

Per impostazione predefinita, lo stato globale di questa funzione ha come ambito l'applicazione. Per modificare questa impostazione, vedere [Stato globale in CRT](../global-state.md).
