# Piattaforme Windows (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le librerie di runtime C per [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] supportano [!INCLUDE[win8](../build/includes/win8_md.md)], [!INCLUDE[winserver8](../build/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 \(SP3\) per x86, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 \(SP2\) per x64 e [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Service Pack 2 \(SP2\) sia per x86 che per x64.  Tutti questi sistemi operativi supportano l'interfaccia di programmazione \(API\) di applicazione Windows e forniscono supporto Unicode.  Inoltre, qualsiasi applicazione Win32 può utilizzare un set di caratteri multibyte \(MBCS\).  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] non supporta [!INCLUDE[winxp](../build/includes/winxp_md.md)] e lo sviluppo di [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].  È necessario installare [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] per aggiungere il supporto per [!INCLUDE[winxp](../build/includes/winxp_md.md)] e [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
