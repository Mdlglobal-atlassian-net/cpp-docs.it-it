# Variabili globali e tipi standard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La libreria di run\-time di Microsoft contiene le definizioni per [variabili globali](../c-runtime-library/global-variables.md), [flag di controllo](../c-runtime-library/control-flags.md) e [tipi standard](../c-runtime-library/standard-types.md) utilizzate dalle routine della libreria.  Accedere a tali variabili, flags e tipi dichiarandoli nel programma o includendo i file di intestazione appropriati.
