# Caratteri multibyte
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.1.2** Sposta gli stati per i caratteri multibyte  
  
 I caratteri multibyte vengono utilizzati da alcune implementazioni, ad esempio Microsoft C, per rappresentare i caratteri in lingua straniera non rappresentati nel set di caratteri di base.  Tuttavia, Microsoft C non supporta determinate codifiche dipendenti dallo stato.  Non sono pertanto presenti stati di spostamento.  Per ulteriori informazioni, vedere [Caratteri multibyte e "wide"](../c-language/multibyte-and-wide-characters.md).
