# Funzione clock (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** L'era per la funzione `clock`  
  
 L'era della funzione `clock` inizia \(con un valore 0\) quando il programma C inizia l'esecuzione.  Restituisce i tempi misurati in 1\/**CLOCKS\_PER\_SEC** \(uguale a 1\/1000 per Microsoft C\).
