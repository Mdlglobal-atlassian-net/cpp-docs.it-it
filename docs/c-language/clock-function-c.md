# <a name="clock-function-c"></a>Funzione clock (C)
**ANSI 4.12.2.1** L'era della funzione `clock`  
  
 L'era della funzione `clock` inizia (con un valore 0) quando il programma C inizia l'esecuzione. Restituisce i tempi misurati in 1/**CLOCKS_PER_SEC** (uguale a 1/1000 per Microsoft C).
