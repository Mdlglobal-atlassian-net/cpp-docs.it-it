# Registri: disponibilit&#224; di registri
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.1** La misura in cui gli oggetti possono essere effettivamente inseriti nei registri dall'utilizzo dell'identificatore classe di archiviazione del registro  
  
 Il compilatore non soddisfa le richieste degli utenti relative alle variabili di registro.  Invece, fa le proprie scelte durante l'ottimizzazione.
