# Segno dei campi di bit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.2.1** Se un campo `int` "normale" viene considerato come campo di bit **signed int** o come campo di bit Unsigned Integer  
  
 I campi di bit possono essere con o senza segno.  I campi di bit normali vengono considerati come con segno.
