# <a name="initialization"></a>Inizializzazione

"Inizializzatore" è un valore o una sequenza di valori da assegnare alla variabile dichiarata. È possibile impostare una variabile su un valore iniziale applicando un inizializzatore al dichiaratore nella dichiarazione di variabile. Il valore o i valori dell'inizializzazione sono assegnati alla variabile.

Nelle sezioni seguenti viene descritto come inizializzare le variabili dei tipi [scalare](../c-language/initializing-scalar-types.md), [aggregato](../c-language/initializing-aggregate-types.md) e [stringa](../c-language/initializing-strings.md). I "tipi scalari" includono tutti i tipi aritmetici, più i puntatori. I "tipi aggregati" includono matrici, strutture e unioni.
