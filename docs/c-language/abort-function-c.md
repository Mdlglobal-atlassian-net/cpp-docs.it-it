# Funzione abort (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.1** Il comportamento della funzione **abort** relativamente a file aperti o temporanei  
  
 La funzione **abort** non chiude i file che sono aperti o temporanei.  Non svuota i buffer del flusso.
