# Funzione fmod
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** Se un errore del dominio si verifica o viene restituito zero quando la funzione `fmod` dispone di un secondo argomento di zero  
  
 Quando la funzione `fmod` dispone di un secondo argomento di zero, la funzione restituisce zero.
