# Funzione atexit (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** Lo stato restituito dalla funzione `atexit` se il valore dell'argomento è diverso da zero, da EXIT\_SUCCESS o da EXIT\_FAILURE  
  
 La funzione `atexit` restituisce zero se eseguita correttamente, mentre restituisce un valore diverso da zero se non è stata eseguita correttamente.
