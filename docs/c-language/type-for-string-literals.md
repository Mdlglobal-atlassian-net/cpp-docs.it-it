# <a name="type-for-string-literals"></a>Tipo per valori letterali stringa

I valori letterali stringa hanno una matrice di tipo `char` (ovvero, **char[ ]**). (Le stringhe di caratteri wide hanno una matrice `wchar_t` di tipo (ovvero, **wchar_t []**). Ciò significa che una stringa è una matrice con elementi di tipo `char`. Il numero di elementi della matrice è uguale al numero di caratteri della stringa più uno per il carattere di terminazione null.
