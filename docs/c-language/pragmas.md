|||||
|-|-|-|-|
|**alloc_text**|**data_seg**|**include_alias**|`setlocale`|
|**auto_inline**|**function**|**intrinsic**|**warning**|
|**check_stack**|**hdrstop**|**message**||
|**code_seg**|**inline_depth**|**optimize**||
|**comment**|**inline_recursion**|`pack`||
