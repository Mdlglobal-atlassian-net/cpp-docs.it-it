# Riepilogo di valori letterali stringa
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*string\-literal*:  
 **'** *s\-char\-sequence*  opt **'**  
  
 **L'** *s\-char\-sequence*  opt             **'**  
  
 *s\-char\-sequence*:  
 *s\-char*  
  
 *s\-char\-sequence s\-char*  
  
 *s\-char*:  
 qualsiasi membro del set di caratteri di origine eccetto le virgolette doppie \("\), la barra rovesciata \(\\\) o il carattere di nuova riga *escape\-sequence*
