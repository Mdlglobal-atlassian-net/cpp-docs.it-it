# <a name="remainders"></a>Resti
**ANSI 3.3.5** Segno del resto nella divisione di interi  
  
 Il segno del resto è uguale al segno del dividendo. Di seguito è riportato un esempio:  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
