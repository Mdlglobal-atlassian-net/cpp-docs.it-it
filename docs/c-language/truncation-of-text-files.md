# Troncamento di file di testo
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Indica se una scrittura in un flusso di testo provoca il troncamento del file associato oltre tale punto  
  
 La scrittura in un flusso di testo non tronca il file oltre tale punto.
