# Intervallo di valori char
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.2.1.1** Indica se un tipo **char** normale ha lo stesso intervallo di valori di un tipo **signed char** o `unsigned char`  
  
 Tutti i valori di caratteri con segno si trovano in un intervallo compreso tra \-128 e 127.  Tutti i valori di caratteri senza segno si trovano in un intervallo compreso tra 0 e 255.  
  
 L'opzione del compilatore \/J modifica l'impostazione predefinita da **signed** a `unsigned`.
