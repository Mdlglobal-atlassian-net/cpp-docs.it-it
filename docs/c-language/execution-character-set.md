# <a name="execution-character-set"></a>Set di caratteri di esecuzione
Questo contenuto fa spesso riferimento al "set di caratteri di esecuzione". Il set di caratteri di esecuzione non Ã¨ necessariamente lo stesso del set di caratteri di origine utilizzato per la scrittura di programmi C. Il set di caratteri di esecuzione include tutti i caratteri nel set di caratteri di origine e il carattere null, il carattere di nuova riga, il backspace, la tabulazione orizzontale, la tabulazione verticale, il ritorno a capo e le sequenze di escape. I set di caratteri di esecuzione e di origine possono variare in altre implementazioni.
