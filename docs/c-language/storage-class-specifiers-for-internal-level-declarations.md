# <a name="storage-class-specifiers-for-internal-level-declarations"></a>Identificatori di classi di archiviazione per dichiarazioni di livello interno

È possibile usare uno dei quattro terminali *storage-class-specifier* per le dichiarazioni delle variabili a livello interno. Quando si omette *Storage-Class-specifier* da tale dichiarazione, la classe di archiviazione predefinita è **auto**. Pertanto, la parola chiave **auto** viene raramente visualizzata in un programma C.
