# Identificatori di classi di archiviazione per dichiarazioni di livello interno
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile utilizzare uno dei quattro terminali *storage\-class\-specifier* per le dichiarazioni delle variabili a livello interno.  Quando si omette *storage\-class\-specifier* da tale dichiarazione, la classe di archiviazione predefinita è **auto**.  Di conseguenza, la parola chiave **auto** viene vista raramente in un programma C.
