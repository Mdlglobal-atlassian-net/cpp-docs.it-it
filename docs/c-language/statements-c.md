# Istruzioni (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le istruzioni di un programma C controllano il flusso di esecuzione del programma.  In C, come negli altri linguaggi di programmazione, sono disponibili diversi tipi di istruzioni per eseguire cicli, per selezionare altre istruzioni da eseguire e per trasferire il controllo.  A seguito di una breve [panoramica sulla sintassi delle istruzioni](../c-language/overview-of-c-statements.md), in questa sezione vengono descritte le istruzioni di C in ordine alfabetico:
