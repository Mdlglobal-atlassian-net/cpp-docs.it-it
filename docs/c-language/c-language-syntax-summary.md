# Riepilogo di sintassi del linguaggio C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In questa sezione viene fornita la descrizione completa del linguaggio C e delle funzionalità specifiche Microsoft del linguaggio C.  In questa sezione è possibile utilizzare una notazione sintattica per determinare la sintassi corretta per ogni componente del linguaggio.  La spiegazione della sintassi compare nella sezione di questo manuale in cui verrà illustrato l'argomento.  
  
> [!NOTE]
>  Questo riepilogo della sintassi non fa parte dello standard ANSI C, ma viene incluso solo per uno scopo informativo.  La sintassi specifica Microsoft viene indicata nei commenti a seguito della sintassi.
