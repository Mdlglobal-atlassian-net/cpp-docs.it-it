# <a name="compiler-error-c2149"></a>Errore del compilatore C2149

I campi di bit possono avere larghezza zero solo se non sono denominati.

L'esempio seguente genera l'errore C2149:

```cpp
// C2149.cpp
// compile with: /c
struct C {
   int i : 0;   // C2149
   int j : 2;   // OK
};
```
