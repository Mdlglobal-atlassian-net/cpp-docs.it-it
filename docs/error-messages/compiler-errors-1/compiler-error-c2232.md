 L'operando a sinistra dell'operatore `->` non Ã¨ un puntatore. Utilizzare l'operatore punto (.) per una classe, struttura o unione.  
  
 L'esempio seguente genera l'errore C2232:  
  
```  
// C2232.c  
struct X {  
    int member;  
} x, *px;
```
