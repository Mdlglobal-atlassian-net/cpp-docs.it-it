# Errore del compilatore C2007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sintassi \#define  
  
 Non Ã¨ presente alcun identificatore dopo `#define`.  Per correggere l'errore, utilizzare un identificatore.  
  
 Il seguente codice di esempio genera l'errore C2007:  
  
```  
// C2007.cpp  
#define   // C2007  
```  
  
 Possibile soluzione:  
  
```  
// C2007b.cpp  
// compile with: /c  
#define true 1  
```
