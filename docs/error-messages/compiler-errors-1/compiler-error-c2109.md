# Errore del compilatore C2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'indice richiede un tipo puntatore o matrice  
  
 L'indice è stato utilizzato su una variabile che non è una matrice.  
  
 Il seguente codice di esempio genera l'errore C2109:  
  
```  
// C2109.cpp  
int main() {  
   int a, b[10] = {0};  
   a[0] = 1;   // C2109  
   b[0] = 1;   // OK  
}  
```
