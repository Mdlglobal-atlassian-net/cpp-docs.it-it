# <a name="fatal-error-c1046"></a>Errore irreversibile C1046

limite del compilatore: struttura eccessivamente annidate

La struttura, unione o classe ha superato il limite di nidificazione, 15 livelli. Riscrivere la definizione per ridurre il livello di nidificazione. Suddividere la struttura, unione o classe in due o più parti usando `typedef` per definire uno o più strutture annidate.
