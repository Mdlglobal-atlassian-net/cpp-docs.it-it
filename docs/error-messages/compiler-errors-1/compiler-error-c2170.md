# <a name="compiler-error-c2170"></a>Errore del compilatore C2170
'identifier': funzione intrinseca non dichiarata come funzione, non può essere  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili  
  
1.  Pragma `intrinsic` viene utilizzato per un elemento diverso da una funzione.  
  
2.  Pragma `intrinsic` viene utilizzato per una funzione senza formato intrinseco.
