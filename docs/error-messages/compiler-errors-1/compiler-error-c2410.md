# Errore del compilatore C2410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': nome membro ambiguo in 'contesto'  
  
 L'identificatore è un membro di più strutture o unioni in questo contesto.  
  
 Utilizzare sull'operando che provoca l'errore un identificatore di struttura o di unione,  Un identificatore di struttura o di unione è un identificatore di tipo `struct` o `union` \(un nome `typedef` o una variabile dello stesso tipo della struttura o dell'unione a cui viene fatto riferimento\).  L'identificatore deve essere l'operando sinistro del primo operatore di selezione dei membri \(.\) a utilizzare l'operando.
