# Errore del compilatore C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' provider di attributi ATL  
  
 L'attributo fa riferimento a un identificatore non dichiarato. Questo errore potrebbe essere associato all'[errore del provider ATL ATL2010](http://msdn.microsoft.com/it-it/27a25924-b52b-4971-80d3-c3b3e9ffbe41).
