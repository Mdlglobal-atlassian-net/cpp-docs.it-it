Le specifiche di eccezione non sono consentite nelle funzioni membro delle classi gestite.

L'esempio seguente genera l'errore C2353:

```
// C2353.cpp
// compile with: /clr /c
ref class X {
   void f() throw(int);   // C2353

};
```
