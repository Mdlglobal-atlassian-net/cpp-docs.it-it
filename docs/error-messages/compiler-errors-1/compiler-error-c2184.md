# <a name="compiler-error-c2184"></a>Errore del compilatore C2184
'type': tipo non valido per espressione __except. Deve essere un tipo integrale  
  
 È stato utilizzato un tipo in un [except](../../c-language/try-except-statement-c.md) istruzione, ma il tipo non è consentito.  
  
 L'esempio seguente genera l'errore C2184:  
  
```  
// C2184.cpp  
void f() {  
   int * p;  
   __try{}  
   __except(p){};   // C2184  
}  
```  
  
 Possibile soluzione:  
  
```  
// C2184b.cpp  
// compile with: /c  
void f() {  
   int i = 0;  
   __try{}  
   __except(i){};  
}  
```
