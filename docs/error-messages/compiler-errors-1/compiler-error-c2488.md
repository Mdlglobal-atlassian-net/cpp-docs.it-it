# Errore del compilatore C2488
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': 'naked' può essere applicato solo a definizioni di funzioni non membro  
  
 L'attributo [naked](../../cpp/naked-cpp.md) è stato applicato a una dichiarazione di funzione.  
  
 Il seguente codice di esempio genera l'errore C2488:  
  
```  
// C2488.cpp  
// compile with: /c  
// processor: x86  
__declspec( naked ) void func();   // C2488  declaration, not definition  
__declspec( naked ) void i;   // C2488  i is not a function  
  
__declspec( naked ) void func() {}   // OK  
```
