# Errore irreversibile C1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generics non supportati dalla versione del runtime installata  
  
 La versione di Common Language Runtime usata dal compilatore non è una versione del runtime supportato dal compilatore corrente.  La funzionalità generics, ad esempio, richiede un runtime che corrisponda al compilatore.  
  
 Potrebbe essere necessario modificare la specifica del percorso.
