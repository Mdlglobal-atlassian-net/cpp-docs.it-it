# Errore del compilatore C2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

errore di sintassi: '\*' deve essere seguito da un qualificatore di tipo  
  
 È presente un qualificatore di tipo \(`const` o `volatile`\) in una posizione in cui non è consentito.
