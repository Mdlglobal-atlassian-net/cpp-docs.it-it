# <a name="compiler-error-c2104"></a>Errore del compilatore C2104
'&' su campo di bit ignorato  
  
 Ãˆ possibile accettare l'indirizzo di un campo di bit.  
  
 L'esempio seguente genera l'errore C2104:  
  
```  
// C2104.cpp  
struct X {  
   int sb : 1;  
};  
  
int main() {  
   X x;  
   &x.sb;   // C2104   
   x.sb;   // OK  
}  
```
