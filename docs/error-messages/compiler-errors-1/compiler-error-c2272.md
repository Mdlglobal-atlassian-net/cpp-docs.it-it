# Errore del compilatore C2272
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': modificatori non consentiti per funzioni membro statiche  
  
 Una funzione membro `static` Ã¨ dichiarata con un identificatore di modello di memoria, ad esempio [const](../../cpp/const-cpp.md) o [volatile](../../cpp/volatile-cpp.md), ma questi modificatori non sono consentiti in funzioni membro `static`.  
  
 Il seguente codice di esempio genera l'errore C2272:  
  
```  
// C2272.cpp  
// compile with: /c  
class CMyClass {  
public:  
   static void func1() const volatile;   // C2272  func1 is static  
   void func2() const volatile;   // OK  
};  
```
