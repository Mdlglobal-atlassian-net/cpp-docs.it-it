# Errore del compilatore C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': membro di struct\/union non valido in 'contesto'  
  
### Possibili cause  
  
1.  L'identificatore non è un membro di una struttura o di un'unione visibile in questo contesto.  
  
2.  L'identificatore non è un membro della struttura o dell'unione specificata con l'operatore di selezione dei membri \(.\).
