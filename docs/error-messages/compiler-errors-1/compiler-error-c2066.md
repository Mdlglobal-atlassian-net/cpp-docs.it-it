# Errore del compilatore C2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

cast a tipo funzione non valido  
  
 In ANSI C i cast tra un puntatore a una funzione e un puntatore ai dati non sono validi.
