# Errore del compilatore C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

un'enumerazione senza membri non è valida  
  
 Quando è selezionata l'opzione \/Za \(Disabilita le estensioni Microsoft\), un'enumerazione deve avere almeno un membro.
