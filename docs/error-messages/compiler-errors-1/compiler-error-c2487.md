# Errore del compilatore C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': un membro di una classe di interfaccia di una dll non può essere dichiarato con un'interfaccia dll  
  
 È possibile dichiarare una classe completa o alcuni membri di una classe di interfaccia non DLL con interfaccia DLL.  Non è possibile dichiarare una classe con interfaccia DLL, quindi dichiarare un membro di tale classe con interfaccia DLL.
