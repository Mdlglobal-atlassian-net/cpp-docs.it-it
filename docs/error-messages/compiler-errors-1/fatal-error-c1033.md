# <a name="fatal-error-c1033"></a>Errore irreversibile C1033
Impossibile aprire il pdb di database di programma  
  
 Questo errore può dipendere da un errore di disco.  
  
 In Visual C++ .NET 2002, impostazioni locali dell'utente devono essere impostata correttamente quando il nome del file o percorso della directory per il nome del file contiene caratteri MBCS. La selezione delle impostazioni locali del sistema non è sufficiente, perché è necessario che tali impostazioni supportino l'elaborazione dei caratteri MBCS.  
  
 Per ulteriori informazioni, vedere [http://support.microsoft.com/default.aspx?scid=kb;en-us;246007](http://support.microsoft.com/default.aspx?scid=kb;en-us;246007).
