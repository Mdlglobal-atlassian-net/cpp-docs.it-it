# <a name="compiler-error-c2095"></a>Errore del compilatore C2095

' Function ': il parametro effettivo è di tipo ' void ': parametro ' Number '

Il parametro passato alla funzione è di tipo `void`, che non è consentito. In alternativa, usare un puntatore a void (`void *`).

Il `number` indica quale parametro è `void`.
