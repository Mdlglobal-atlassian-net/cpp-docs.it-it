# Errore del compilatore C2270
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': modificatori non consentiti per funzioni non membro  
  
 Una funzione non membro Ã¨ dichiarata con [const](../../cpp/const-cpp.md), [volatile](../../cpp/volatile-cpp.md) o un altro modificatore del modello di memoria.  
  
 Il seguente codice di esempio genera l'errore C2270:  
  
```  
// C2270.cpp  
// compile with: /c  
void func1(void) const;   // C2270, nonmember function  
  
void func2(void);  
  
class CMyClass {  
public:  
   void func2(void) const;  
};  
```
