# Errore del compilatore C2279
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

non Ã¨ possibile specificare eccezioni nelle dichiarazioni typedef  
  
 In **\/Za** non sono consentite [specifiche delle eccezioni](../../cpp/exception-specifications-throw-cpp.md) nelle dichiarazioni typedef.  
  
 Il seguente codice di esempio genera l'errore C2279:  
  
```  
// C2279.cpp  
// compile with: /Za /c  
typedef int (*xy)() throw(...);   // C2279  
typedef int (*xyz)();   // OK  
```
