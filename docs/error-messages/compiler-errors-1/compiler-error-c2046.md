# <a name="compiler-error-c2046"></a>Errore del compilatore C2046
case non valido  
  
 La parola chiave `case` pu√≤ trovarsi solo all'interno di un'istruzione `switch` .  
  
 L'esempio seguente genera l'errore C2046:  
  
```  
// C2046.cpp  
int main() {  
   case 0:   // C2046  
}  
```  
  
 Possibile soluzione:  
  
```  
// C2046b.cpp  
int main() {  
   int i = 0;  
  
   switch(i) {  
      case 0:  
      break;  
  
      default:  
      break;  
   }  
}  
```
