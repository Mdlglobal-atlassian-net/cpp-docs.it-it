# Errore irreversibile C1207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Modelli gestiti non supportati dalla versione del runtime installata  
  
 L'errore C1207 si verifica quando è presente un compilatore per la versione corrente, ma un Common Language Runtime di una versione precedente.  
  
 Alcune funzionalità del compilatore potrebbero non funzionare in una versione precedente del runtime.  
  
 Per risolvere l'errore C1207, installare la versione di Common Language Runtime che deve essere usata con il compilatore.
