# <a name="compiler-error-c2224"></a>Errore del compilatore C2224

a sinistra di 'Identifier' deve avere un tipo struct/union

L'operando a sinistra del punto (.) non è una classe, struttura o unione.

Questo errore può essere causato da un operando di sinistra è una variabile non definita (pertanto digitare `int`).
