# <a name="compiler-error-c2087"></a>Errore del compilatore C2087
'identifier': indice mancante  
  
 La definizione di una matrice con pi√π indici manca un valore di indice per una dimensione supera a uno.  
  
 L'esempio seguente genera l'errore C2087:  
  
```  
// C2087.cpp  
int main() {  
   char a[10][];   // C2087  
}  
```  
  
 Possibile soluzione:  
  
```  
// C2087b.cpp  
int main() {  
   char b[4][5];  
}  
```
