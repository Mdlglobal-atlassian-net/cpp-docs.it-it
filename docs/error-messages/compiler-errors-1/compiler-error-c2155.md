# Errore del compilatore C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' : operando sinistro non valido. Previsto tipo puntatore o aritmetico  
  
 Non è possibile confrontare un'espressione sul lato sinistro di `?` con zero.  È necessario utilizzare un'espressione aritmetica o di puntatore che può essere confrontata con zero.
