# <a name="compiler-error-c2155"></a>Errore del compilatore C2155

'?': operando sinistro non valido, tipo aritmetico o puntatore previsto

Un'espressione a sinistra del `?` non può essere confrontata con zero. È necessario utilizzare un'espressione aritmetica o puntatore che può essere confrontata con zero.
