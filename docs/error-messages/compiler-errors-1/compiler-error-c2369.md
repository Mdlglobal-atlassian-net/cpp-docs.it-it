# <a name="compiler-error-c2369"></a>Errore del compilatore C2369
'array': ridefinizione. Indici differenti  
  
 La matrice è già dichiarata con un indice diverso.  
  
 L'esempio seguente genera l'errore C2369:  
  
```  
// C2369.cpp  
// compile with: /c  
int a[10];  
int a[20];   // C2369  
int b[20];   // OK  
```
