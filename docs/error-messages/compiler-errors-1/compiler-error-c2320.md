# Errore del compilatore C2320
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

previsto ':' dopo l'identificatore di accesso 'specifier'  
  
 La parola chiave `public`, `protected` o `private` deve essere seguita dai due punti.
