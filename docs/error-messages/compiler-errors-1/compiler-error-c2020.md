# Errore del compilatore C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'membro': ridefinizione membro 'classe'  
  
 Un membro ereditato da una struttura o da una classe base è stato ridefinito.  Non è possibile ridefinire i membri ereditati, a meno che non siano dichiarati come `virtual` nella classe base.
