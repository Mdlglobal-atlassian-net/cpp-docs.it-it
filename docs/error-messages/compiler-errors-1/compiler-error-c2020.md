# <a name="compiler-error-c2020"></a>Errore del compilatore C2020
'member': ridefinizione membro di 'class'  
  
 Ridefinizione di un membro ereditato da una classe di base o una struttura. I membri ereditati non possono essere ridefiniti a meno che non dichiarati come `virtual` nella classe base.
