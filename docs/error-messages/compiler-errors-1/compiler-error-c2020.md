# <a name="compiler-error-c2020"></a>Errore del compilatore C2020

'member': ridefinizione membro 'class'

Un membro ereditato da una classe di base o una struttura viene ridefinito. I membri ereditati non possono essere ridefiniti a meno che non dichiarati come `virtual` nella classe di base.
