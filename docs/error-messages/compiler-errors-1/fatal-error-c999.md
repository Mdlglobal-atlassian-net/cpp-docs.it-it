# Fatal Error C999
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ERRORE SCONOSCIUTO    Per altre informazioni, scegliere Supporto tecnico dal menu ? di Visual C\+\+ o aprire il file della Guida relativo al supporto tecnico  
  
 Questo errore indica in genere che si dispone di file diversi da diverse versioni del compilatore oppure l'installazione del compilatore è danneggiata.  Usare l'applet **Programmi e funzionalità** nel Pannello di controllo per ripristinare o reinstallare il prodotto.
