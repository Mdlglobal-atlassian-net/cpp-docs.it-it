# Errore del compilatore C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token': dimensione di allineamento non valida  
  
 La dimensione utilizzata con la direttiva `ALIGN` è mancante o non è compresa nell'intervallo valido.
