# <a name="compiler-error-c2412"></a>Errore del compilatore C2412
'label': etichetta senza maiuscole/minuscole ridefinita  
  
 L'etichetta è definita più volte nella funzione corrente. Modificare l'ortografia dell'etichetta e i relativi riferimenti.
