# <a name="compiler-error-c2412"></a>Errore del compilatore C2412

'label': etichetta senza maiuscole/minuscole ridefinita

L'etichetta è definito più volte nella funzione corrente. Modificare l'ortografia di etichetta e i relativi riferimenti.
