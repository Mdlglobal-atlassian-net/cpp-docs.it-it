# Errore del compilatore C2154
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type': è consentito solo un tipo di enumerazione come argomento per compilare il tratto di tipo intrinseco '\_\_underlying\_type'  
  
 È possibile ottenere solo il tipo sottostante di un tipo di enumerazione.  
  
 Per altre informazioni, vedere la pagina relativa al [Compiler Support for Type Traits](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
