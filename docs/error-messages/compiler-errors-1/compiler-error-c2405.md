# <a name="compiler-error-c2405"></a>Errore del compilatore C2405
non valido riferimento short in avanti con offset  
  
 I riferimenti in avanti breve devono fare riferimento solo a un'etichetta. Impossibile utilizzare un offset aggiuntivo.
