# Errore del compilatore C2147
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

errore di sintassi: 'identificatore' è una nuova parola chiave  
  
 È stato utilizzato un identificatore che è una nuova parola chiave riservata nel linguaggio.  
  
 Il seguente codice di esempio genera l'errore C2147:  
  
```  
// C2147.cpp  
// compile with: /clr  
int main() {  
   int gcnew = 0;   // C2147  
   int i = 0;   // OK  
}  
```
