# <a name="compiler-error-c2309"></a>Errore del compilatore C2309

 Un gestore catch non dispone di alcun tipo tra parentesi.  
  
 L'esempio seguente genera l'errore C2309:  
  
```  
// C2309.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch C {}   // C2309  
   // try the following line instead  
   // catch( C ) {}  
}  
```
