# Errore del compilatore C2223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'elemento a sinistra di '\-\>identifier' deve puntare a struct\/union  
  
 L'operando a sinistra di `->` non è un puntatore a una classe, una struttura o un'unione.  
  
 Questo errore può essere causato da un operando di sinistra che rappresenta una variabile non definita, vale a dire un tipo `int`.
