# Errore del compilatore C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': nome non definito in 'context'  
  
 Un identificatore non definito viene usato con `SIZE`, `LENGTH` o member\-selection \(.\) operatore.
