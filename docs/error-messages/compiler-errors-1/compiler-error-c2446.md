# Errore del compilatore C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operatore': nessuna conversione da 'tipo1' a 'tipo2'  
  
 Non è possibile convertire `type1` in `type2`.  È possibile che la conversione non rispetti la semantica di C\/C\+\+.
