# <a name="compiler-error-c2446"></a>Errore del compilatore C2446

' operator ': nessuna conversione da' tipo1' a' tipo2'

Il compilatore non è in grado di convertire `type1` in `type2`. La conversione potrebbe non essere utile perché viola la semantica CC++ /.
