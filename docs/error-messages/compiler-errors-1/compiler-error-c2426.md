# Errore del compilatore C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token':operatore non valido in 'contesto'  
  
 Il token non può essere utilizzato come operatore in questo contesto.  Ad esempio, non è possibile annidare gli operatori di indice.
