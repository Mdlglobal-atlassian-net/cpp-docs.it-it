# <a name="compiler-error-c2175"></a>Errore del compilatore C2175

'delle impostazioni locali': impostazioni locali non valido

Le impostazioni locali specificate non sono valida. Vedere [stringhe di paese/area geografica e lingua](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) nel *Run-Time Library Reference* per impostazioni locali supportate.
