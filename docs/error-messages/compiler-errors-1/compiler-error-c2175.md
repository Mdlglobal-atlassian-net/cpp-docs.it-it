# Errore del compilatore C2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'impostazioni locali': impostazioni locali non valide  
  
 Le impostazioni locali specificate non sono valide.  Per conoscere le impostazioni locali supportate, vedere [Stringhe relative a lingue e paesi](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) in *Riferimenti alla libreria di runtime*.
