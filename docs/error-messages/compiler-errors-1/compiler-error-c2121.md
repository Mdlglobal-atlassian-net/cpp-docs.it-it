# Errore del compilatore C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#': carattere non valido. Probabile risultato dell'espansione di una macro  
  
 È possibile che un carattere \# non valido sia stato inserito da una macro errata che utilizza l'operatore di concatenamento dei token \(\#\#\) anziché l'operatore per la creazione di stringhe \(\#\).
