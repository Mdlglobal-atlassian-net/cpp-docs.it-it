# <a name="compiler-error-c2163"></a>Errore del compilatore C2163
'function': non è disponibile come funzione intrinseca  
  
 Un `intrinsic` o `function` pragma elenca una funzione non è disponibile in formato intrinseco. Ad esempio, alcune funzioni intrinseche non sono disponibili quando si compila un programma che utilizza la programmazione /clr.
