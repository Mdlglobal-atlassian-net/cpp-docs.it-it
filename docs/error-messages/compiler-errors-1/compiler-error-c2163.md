# <a name="compiler-error-c2163"></a>Errore del compilatore C2163

' Function ': non disponibile come funzione intrinseca

Un pragma `intrinsic` o `function` elenca una funzione non disponibile nel formato intrinseco. Alcune funzioni intrinseche, ad esempio, non sono disponibili quando si compila un programma che utilizza la programmazione/CLR.
