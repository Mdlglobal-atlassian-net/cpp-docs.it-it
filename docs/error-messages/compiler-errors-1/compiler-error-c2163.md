# Errore del compilatore C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': non disponibile come funzione intrinseca  
  
 Un pragma `intrinsic` o `function` elenca una funzione non disponibile in formato intrinseco.  Alcune funzioni intrinseche non sono ad esempio disponibili quando si compila un programma che utilizza \/clr.
