# Errore del compilatore C2193
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': già esistente in un segmento  
  
 Una funzione è stata inserita in due segmenti differenti utilizzando i pragma `alloc_text` e `code_seg`.  
  
 Il seguente codice di esempio genera l'errore C2193:  
  
```  
// C2193.cpp  
// compile with: /c  
extern "C" void MYFUNCTION();  
#pragma alloc_text(MYCODE, MYFUNCTION)  
#pragma code_seg("MYCODE2")  
extern "C" void MYFUNCTION() {}   // C2193  
extern "C" void MYFUNCTION2() {}  
```
