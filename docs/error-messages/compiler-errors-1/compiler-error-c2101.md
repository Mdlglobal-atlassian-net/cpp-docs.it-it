# <a name="compiler-error-c2101"></a>Errore del compilatore C2101
'&' in costante  
  
 L'operatore address-of ( `&` ) deve avere un valore l-value come operando.  
  
 L'esempio seguente genera l'errore C2101:  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
