# <a name="compiler-error-c2181"></a>Errore del compilatore C2181
else non valido senza if corrispondente  
  
 Ogni `else` deve avere un oggetto `if`corrispondente.  
  
 L'esempio seguente genera l'errore C2181:  
  
```  
// C2181.cpp  
int main() {  
   int i = 0;  
   else   // C2181  
      i = 1;  
}  
```  
  
 Possibile soluzione:  
  
```  
// C2181b.cpp  
int main() {  
   int i = 0;  
   if(i)  
      i = 0;  
   else  
      i = 1;  
}  
```
