# Errore del compilatore C2450
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

le espressioni switch di tipo 'tipo' non sono valide  
  
 L'espressione `switch` restituisce un tipo non valido.  Il risultato deve essere un tipo intero o un tipo classe con conversione non ambigua in un tipo intero.  Se il risultato Ã¨ un tipo definito dall'utente, occorre fornire un operatore di conversione.  
  
 Il seguente codice di esempio genera l'errore C2450:  
  
```  
// C2450.cpp  
class X {  
public:  
   int i;  
} x;  
  
class Y {  
public:  
   int i;  
   operator int() { return i; }   // conversion operator  
} y;  
  
int main() {  
   int j = 1;  
   switch ( x ) {   // C2450, x is not type int  
   // try the following line instead  
   // switch ( y ) {  
      default:  ;  
   }  
}  
```
