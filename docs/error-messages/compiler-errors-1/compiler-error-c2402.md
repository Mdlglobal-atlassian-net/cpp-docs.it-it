# Errore del compilatore C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': il registro deve essere di tipo indice in 'context'  
  
 Il registro usato in un operando di memoria indiretto deve essere un registro di indice in questo contesto.
