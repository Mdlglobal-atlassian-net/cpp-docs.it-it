# Errore del compilatore C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

corrispondenza errata per il delimitatore sinistro trovato in 'percorso'  
  
 La corrispondenza di un delimitatore è errata.  
  
 Questo errore può essere causato dall'errata corrispondenza delle parentesi in una dichiarazione di modello di funzione.
