# <a name="compiler-error-c2873"></a>Errore del compilatore C2873
'symbol': simbolo non può essere utilizzato in una dichiarazione using  
  
 Oggetto `using` direttiva non è presente un [dello spazio dei nomi](../../cpp/namespaces-cpp.md) (parola chiave). In questo modo il compilatore erroneamente il codice come un [dichiarazione using](../../cpp/using-declaration.md) anziché [direttiva using](../../cpp/namespaces-cpp.md#using_directives).
