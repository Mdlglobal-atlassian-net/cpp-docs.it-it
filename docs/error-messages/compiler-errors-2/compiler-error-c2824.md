# Errore del compilatore C2824
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il tipo restituito per 'operator new' deve essere 'void \*'  
  
 In caso di puntatori non based, gli overload dell'operatore `new` devono restituire `void *`.  
  
 Il seguente codice di esempio genera l'errore C2824:  
  
```  
// C2824.cpp  
// compile with: /c  
class   A {  
   A* operator new(size_t i, char *m);   // C2824  
   // try the following line instead  
   // void* operator new(size_t i, char *m);  
};  
```
