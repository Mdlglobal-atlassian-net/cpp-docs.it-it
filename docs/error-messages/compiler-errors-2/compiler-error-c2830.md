# Errore del compilatore C2830
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

solo i parametri di posizione di 'operator new' possono avere valori predefiniti  
  
 I parametri formali standard per [operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md) non possono avere valori predefiniti.  Solo i parametri di posizione definiti dall'utente possono specificare valori predefiniti.
