# <a name="compiler-error-c2830"></a>Errore del compilatore C2830
solo i parametri di posizione per 'operator new' possono avere valori predefiniti  
  
I parametri formali standard per [operatore new](../../standard-library/new-operators.md#op_new) non possono avere valori predefiniti. Solo i parametri di posizione definita dall'utente possono specificare valori predefiniti.
