# Errore del compilatore C2777
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

per ciascuna proprietà è possibile specificare un solo metodo 'get'  
  
 Un modificatore declspec di [property](../../cpp/property-cpp.md) ha più di una proprietà `put`.  
  
 Il seguente codice di esempio genera l'errore C2777:  
  
```  
// C2777.cpp  
struct A {  
   __declspec(property(put=PutProp,put=PutPropToo))   // C2777  
   // try the following line instead  
   // __declspec(property(put=PutProp))  
      int prop;  
   int PutProp(void);  
   int PutPropToo(void);  
};  
```
