# <a name="compiler-error-c2871"></a>Errore del compilatore C2871

'name': uno spazio dei nomi con il nome specificato non esiste

Questo errore si verifica quando si passa un identificatore che non sia uno spazio dei nomi per un [usando](../../cpp/namespaces-cpp.md#using_directives) direttiva.

## <a name="example"></a>Esempio

L'esempio seguente genera l'errore C2871:

```cpp
// C2871.cpp
// compile with: /c
namespace a {
   int fn(int i) { return i; }
}
namespace b {
   using namespace d;   // C2871 because d is not a namespace
   using namespace a;   // OK
}
```
