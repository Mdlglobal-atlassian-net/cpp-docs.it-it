# <a name="compiler-error-c2871"></a>Errore del compilatore C2871
'nome': uno spazio dei nomi con questo nome non esiste  
  
Questo errore si verifica quando viene passato un identificatore che non Ã¨ uno spazio dei nomi a un [utilizzando](../../cpp/namespaces-cpp.md#using_directives) (direttiva).  
  
## <a name="example"></a>Esempio  
Nell'esempio seguente viene generato l'errore C2871:  
  
```cpp  
// C2871.cpp  
// compile with: /c
namespace a {
   int fn(int i) { return i; }
} 
namespace b { 
   using namespace d;   // C2871 because d is not a namespace  
   using namespace a;   // OK
}  
```
