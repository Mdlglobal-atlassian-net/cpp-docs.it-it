# Errore del compilatore C3902
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione di accesso': il tipo dell'ultimo parametro deve essere 'tipo'  
  
 Il tipo dell'ultimo parametro di almeno un metodo set deve corrispondere al tipo della propriet√†.  Per ulteriori informazioni, vedere [property](../../windows/property-cpp-component-extensions.md).  
  
 Il seguente codice di esempio genera l'errore C3902:  
  
```  
// C3902.cpp  
// compile with: /clr /c  
using namespace System;  
ref class X {  
   property String ^Name {  
      void set(int);   // C3902  
      // try the following line instead  
      // void set(String^){}  
   }  
  
   property double values[int,int] {  
      void set(int, int, float);   // C3902  
      // try the following line instead  
      // void set(int, int, double){}  
   }  
};  
```
