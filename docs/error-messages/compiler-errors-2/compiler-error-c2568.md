# <a name="compiler-error-c2568"></a>Errore del compilatore C2568
'identifier1': Impossibile risolvere l'overload della funzione  
  
 Il compilatore non è possibile determinare quale overload di funzione da chiamare. I parametri effettivi passati alla funzione devono essere impostati per corrispondere ai parametri formali per una delle funzioni in overload, ma nessuna corrispondenza è migliore in modo non ambiguo di tutti gli altri utenti.
