# Errore del compilatore C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore1': impossibile risolvere l'overload della funzione  
  
 Non è possibile stabilire quale funzione in overload chiamare.  È necessario eseguire il cast dei parametri effettivi passati alla funzione affinché corrispondano ai parametri formali per una delle funzioni in overload, ma nessuna corrispondenza è meno ambigua delle altre.
