# Errore del compilatore C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo restituito 'tipo' non valido per 'operator \-\>' di overload  
  
 L'operatore `->` deve restituire una classe, una struttura, un'unione oppure un riferimento a una di queste.
