# Errore del compilatore C2713
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

in ciascuna funzione è permessa una sola forma di gestione delle eccezioni  
  
 Non è possibile utilizzare la gestione delle eccezioni strutturata \(`__try`\/`__except`\) e la gestione delle eccezioni C\+\+ \(`try`\/`catch`\) nella stessa funzione.
