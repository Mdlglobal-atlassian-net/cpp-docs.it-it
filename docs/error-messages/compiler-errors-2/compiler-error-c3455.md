# <a name="compiler-error-c3455"></a>Errore del compilatore C3455
'attribute': nessuno dei costruttori di attributo corrispondente agli argomenti  
  
 Ãˆ stato usato un valore non valido per dichiarare un attributo.  Vedere [attributo](../../windows/attribute.md) per ulteriori informazioni.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore C3455.  
  
```  
// C3455.cpp  
// compile with: /clr /c  
using namespace System;  
  
[attribute("MyAt")]   // C3455  
// try the following line instead  
// [attribute(All)]  
ref struct MyAttr {  
   MyAttr() {}  
};  
```
