# <a name="compiler-error-c3157"></a>Errore del compilatore C3157

Attributo ParamArray può essere applicato solo all'ultimo parametro

Il [ParamArray](https://msdn.microsoft.com/library/system.paramarrayattribute.aspx) attributo è stato applicato al parametro errato.
