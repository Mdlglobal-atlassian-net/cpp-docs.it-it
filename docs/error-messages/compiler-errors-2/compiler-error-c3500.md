# <a name="compiler-error-c3500"></a>Errore del compilatore C3500

ProgID non valido 'progid'

Un progid non valido specificato con il `#import` istruzione. Controllare il Registro di sistema di Windows per assicurarsi di avere specificato un valore valido progid.
