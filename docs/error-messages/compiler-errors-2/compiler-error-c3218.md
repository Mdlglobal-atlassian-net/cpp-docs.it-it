# Errore del compilatore C3218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'tipo': tipo non consentito come vincolo  
  
 Affinch√© il tipo sia un vincolo, deve essere un tipo di valore o di riferimento per una classe o interfaccia gestita.  
  
## Esempio  
 Nell'esempio seguente viene generato l'errore C3218:  
  
```  
// C3218.cpp  
// compile with: /clr /c  
class A {};  
ref class B {};  
  
// Delete the following 3 lines to resolve.  
generic <class T>  
where T : A   // C3218  
ref class C {};  
  
// OK  
generic <class T>  
where  T : B  
ref class D {};  
```
