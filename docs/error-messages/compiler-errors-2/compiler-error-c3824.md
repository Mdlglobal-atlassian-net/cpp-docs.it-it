 Puntatori di blocco non possono essere parametri di funzione, tipi restituiti o dichiarate `static`.  
  
## <a name="example"></a>Esempio  
 Nell'esempio seguente viene generato l'errore C3824:  
  
```  
// C3824a.cpp  
// compile with: /clr /c  
void func() {  
   static pin_ptr<int> a; // C3824  
   pin_ptr<int> b; // OK  
}  
```
