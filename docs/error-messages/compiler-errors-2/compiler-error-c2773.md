# Errore del compilatore C2773
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import e \#using disponibili solo nel compilatore C\+\+  
  
 Il compilatore C non riconosce la direttiva `#import` per il preprocessore .  Compilare il codice sorgente come C\+\+.  Se necessario, utilizzare [\/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md).
