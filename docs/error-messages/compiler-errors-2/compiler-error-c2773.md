# <a name="compiler-error-c2773"></a>Errore del compilatore C2773
\#importare e #using disponibili solo nel compilatore C++  
  
 Il compilatore C non riconosce il `#import` direttiva del preprocessore. Compilare il codice sorgente come C++. Utilizzare [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) se necessario.
