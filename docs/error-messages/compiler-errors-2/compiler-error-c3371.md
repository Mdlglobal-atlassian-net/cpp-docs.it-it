# <a name="compiler-error-c3371"></a>Errore del compilatore C3371
'idl_module': solo la proprietà 'name' è ammessa in questo contesto  
  
 [idl_module](../../windows/idl-module.md) utilizzo direttamente in una dichiarazione di funzione non può avere altri parametri oltre al nome.  
  
 L'esempio seguente genera l'errore C3371:  
  
```  
// C3371.cpp  
[idl_module(name="Name", dllname="Some.dll")];  
[idl_module(name="Name", helpstring="Some help")]   // C3371  
int f1();  
// try  
// [idl_module(name="Name")]  
// int f1();  
  
int main()  
{  
}  
```
