# Errore del compilatore C3618
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': impossibile definire un metodo contrassegnato DllImport  
  
 Un metodo contrassegnato con <xref:System.Runtime.InteropServices.DllImportAttribute> Ã¨ definito nella DLL specificata.  
  
## Esempio  
 Nell'esempio seguente viene generato l'errore C3618:  
  
```  
// C3618.cpp  
// compile with: /clr /c  
using namespace System;  
using namespace System::Runtime::InteropServices;  
  
[ DllImport("user32.dll", EntryPoint="MessageBox", CharSet=CharSet::Ansi) ]  // CHANGED   
void Func();   
  
void Func() {}   // C3618, remove this function definition to resolve  
```
