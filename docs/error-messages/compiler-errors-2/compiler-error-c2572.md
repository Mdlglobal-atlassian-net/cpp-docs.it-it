# <a name="compiler-error-c2572"></a>Errore del compilatore C2572

'member': ridefinizione del parametro predefinito: parametro param

I parametri predefiniti non possono essere ridefiniti. Se Ã¨ necessario un altro valore per il parametro, il parametro predefinito deve essere lasciato non definito.

L'esempio seguente genera l'errore C2572:

```
// C2572.cpp
// compile with: /c
void f(int i = 1);   // function declaration

// function definition
void f(int i = 1) {}   // C2572

// try the following line instead
// void f(int i) {}
```
