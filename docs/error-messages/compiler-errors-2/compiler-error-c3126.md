 Un'unione non possa essere definita all'interno di un tipo gestito.  
  
 Nell'esempio seguente viene generato l'errore C3126:  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
