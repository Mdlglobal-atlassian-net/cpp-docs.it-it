# <a name="compiler-error-c2531"></a>Errore del compilatore C2531
'identifier': riferimento a un bit campo non valido  
  
 Non sono consentiti riferimenti a campi di bit.  
  
 L'esempio seguente genera l'errore C2531:  
  
```  
// C2531.cpp  
// compile with: /c  
class P {  
   int &b1 : 10;   // C2531  
   int b2 : 10;   // OK  
};  
```
