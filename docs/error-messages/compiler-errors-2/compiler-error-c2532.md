# Errore del compilatore C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': modificatore non valido per riferimento  
  
 Il riferimento è stato cambiato.  Non è possibile modificare i riferimenti in modo che facciano riferimento a un altro oggetto.  Utilizzare un puntatore.
