# Errore del compilatore C2752
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'modello': pi√π specializzazioni parziali corrispondenti all'elenco degli argomenti di template  
  
 Una creazione di un'istanza era ambigua.  
  
 Il seguente codice di esempio genera l'errore C2752:  
  
```  
// C2752.cpp  
template<class T, class U>   
struct A {};  
  
template<class T, class U>   
struct A<T*, U> {};  
  
template<class T, class U>   
struct A<T,U*> {};  
  
// try the following line instead  
// template<class T, class U> struct A<T*,U*> {};  
  
int main() {  
   A<char*,int*> a;   // C2752 an instantiation  
  
   // OK  
   A<char*,int> a1;  
   A<char,int*> a2;  
   A<char,int> a3;  
}  
```
