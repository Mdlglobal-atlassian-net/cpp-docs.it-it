# Errore del compilatore C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile elaborare l'attributo 'attributo'  
  
 Un attributo definito dall'utente che eredita da `System.Security.Permissions.SecurityAttribute` è utilizzato per definire un attributo di sicurezza.  Non è possibile utilizzare tale attributo nello stesso assembly in cui è definito.
