# <a name="compiler-error-c3603"></a>Errore del compilatore C3603
'Symbol': tipo 'Type' non è ancora supportato  
  
 Si è tentato di utilizzare un tipo di dati che non è ancora supportato dal runtime .NET in un oggetto gestito.
