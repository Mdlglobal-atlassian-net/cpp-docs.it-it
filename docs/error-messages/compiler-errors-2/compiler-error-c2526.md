# Errore del compilatore C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore1': una funzione con collegamento C non può restituire la classe C\+\+ 'identificatore2'  
  
 Una funzione definita con il collegamento C non può restituire un tipo definito dall'utente.
