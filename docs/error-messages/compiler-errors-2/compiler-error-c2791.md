# Errore del compilatore C2791
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

utilizzo non valido di 'super': 'classe' non ha una classe base  
  
 La parola chiave [super](../../cpp/super.md) Ã¨ stata utilizzata nel contesto di una funzione membro di una classe priva di classi base.  
  
 Il seguente codice di esempio genera l'errore C2791:  
  
```  
// C2791.cpp  
struct D {  
   void mf() {  
      __super::mf();   // C2791  
   }
```
