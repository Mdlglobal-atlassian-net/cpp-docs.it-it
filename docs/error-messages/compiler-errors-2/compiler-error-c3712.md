# Errore del compilatore C3712
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'metodo': un metodo per la gestione eventi deve restituire lo stesso tipo dell'origine 'metodo'  
  
 Ãˆ stato definito un metodo per la gestione degli eventi che non ha restituito lo stesso tipo del metodo dell'origine eventi.  Per correggere questo errore, assegnare al metodo per la gestione degli eventi lo stesso tipo restituito del metodo dell'origine eventi.  
  
 Il seguente codice di esempio genera l'errore C3712:  
  
```  
// C3712.cpp  
// compile with: /c  
[event_source(native)]  
class CEventSrc {  
public:  
   __event void event1();  
};  
  
[event_receiver(native)]  
class CEventRec {  
public:  
   int handler1() { return 0; }  
   // try the following line instead  
   // void handler1() {}  
  
   void HookEvents(CEventSrc* pSrc) {  
      __hook(&CEventSrc::event1, pSrc, &CEventRec::handler1);   // C3712  
   }  
   void UnhookEvents(CEventSrc* pSrc) {  
      __unhook(&CEventSrc::event1, pSrc, &CEventRec::handler1);   // C3712  
   }  
};  
```
