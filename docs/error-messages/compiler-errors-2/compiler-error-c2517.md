# <a name="compiler-error-c2517"></a>Errore del compilatore C2517
'identifier': a destra di ':: ' non è definito  
  
 L'identificatore a destra dell'operatore di risoluzione dell'ambito (`::`) deve essere un membro definito della classe, struttura o unione a sinistra. Se non è denominata alcuna classe, struttura o unione, l'identificatore a destra deve essere dichiarato con ambito globale.
