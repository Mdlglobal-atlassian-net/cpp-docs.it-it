# <a name="compiler-error-c2886"></a>Errore del compilatore C2886

' Class:: Identifier ': Impossibile utilizzare il simbolo in una dichiarazione using di membro

Una dichiarazione `using` usa un simbolo, ad esempio un nome di spazio dei nomi. Una dichiarazione di `using` Ã¨ per dichiarare i membri della classe base.

L'esempio seguente genera l'C2886:

```cpp
// C2886.cpp
// compile with: /c
namespace Z {
    int i;
}

class B {
protected:
    int i;
};

class D : public B {
    // Error: Z is a namespace
    using Z::i;   // C2886

    // OK: B is a base class
    using B::i;
};
```
