# Errore del compilatore C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nome tipo': il nome eccede il limite di 'limite' caratteri stabilito per i metadati  
  
 È stato troncato il nome per un tipo gestito nei metadati.  A causa del troncamento, il tipo non potrà essere utilizzato con la direttiva `#using` o la direttiva equivalente in un altro linguaggio.  
  
 Il limite del nome dei tipi include qualsiasi qualificazione dello spazio dei nomi.
