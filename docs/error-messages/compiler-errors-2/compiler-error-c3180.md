# <a name="compiler-error-c3180"></a>Errore del compilatore C3180
'type name': nome supera il limite di metadati di caratteri 'limite'  
  
 È stato troncato il nome di un tipo gestito nei metadati. Il troncamento verrà rendere inutilizzabile con il tipo di `#using` (direttiva) (o l'equivalente in un altro linguaggio).  
  
 Il limite di nome di tipo include qualsiasi qualificazione dello spazio dei nomi.
