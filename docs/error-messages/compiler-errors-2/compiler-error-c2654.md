# Errore del compilatore C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': tentativo di accedere a un membro all'esterno di una funzione membro  
  
 Si accede a un membro in una dichiarazione.  È possibile accedere ai dati dei membri solo nelle funzioni membro.  
  
 Questo errore può verificarsi quando si tenta di inizializzare le variabili in una dichiarazione.  Utilizzare un costruttore a questo scopo.
