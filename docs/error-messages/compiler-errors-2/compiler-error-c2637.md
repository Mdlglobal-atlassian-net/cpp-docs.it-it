# Errore del compilatore C2637
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': impossibile modificare i puntatori ai membri dati  
  
 Un puntatore a un membro dati non pu√≤ avere una convenzione di chiamata.  Per correggere l'errore, rimuovere la convenzione di chiamata o dichiarare un puntatore a una funzione membro.  
  
 Il seguente codice di esempio genera l'errore C2637:  
  
```  
// C2637.cpp  
// compile with: /c  
struct S {};  
int __stdcall S::*pms1;   // C2637  
  
// OK  
int S::*pms2;  
int (__stdcall S::*pms3)(...);  
```
