# Errore del compilatore C3887
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'variabile': l'inizializzatore per un membro dati literal deve essere un'espressione costante  
  
 Un membro dati [literal](../../windows/literal-cpp-component-extensions.md) pu√≤ essere inizializzato solo con un'espressione costante.  
  
 Il seguente codice di esempio genera l'errore C3887:  
  
```  
// C3887.cpp  
// compile with: /clr  
ref struct Y1 {  
   static int i = 9;  
   literal  
   int staticConst = i;   // C3887  
};  
```  
  
 Possibile soluzione:  
  
```  
// C3887b.cpp  
// compile with: /clr /c  
ref struct Y1 {  
   literal  
   int staticConst = 9;  
};  
```
