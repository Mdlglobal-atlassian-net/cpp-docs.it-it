# Errore del compilatore C3701
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': origine evento senza eventi  
  
 Si è tentato di utilizzare [event\_source](../../windows/event-source.md) su una classe che non ha metodi di evento.  Per correggere questo errore, aggiungere uno o più eventi alla classe.  
  
 Il seguente codice di esempio genera l'errore C3701:  
  
```  
// C3701.cpp  
[ event_source(native) ]  
class CEventSrc {  
public:  
   // uncomment the following line to resolve this C3701  
   // __event void fireEvent(int i);  
};   // C3701  
  
int main() {  
}  
```
