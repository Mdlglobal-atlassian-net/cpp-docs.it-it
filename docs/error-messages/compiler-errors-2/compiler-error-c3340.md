# <a name="compiler-error-c3340"></a>Errore del compilatore C3340
'interface': l'interfaccia non pu√≤ essere sia 'restricted' che 'default' nella coclasse 'class'  
  
 Il [limitato](../../windows/restricted.md) attributo e il [predefinito](../../windows/default-cpp.md) attributo si escludono a vicenda.  
  
 L'esempio seguente genera l'errore C3340:  
  
```  
// C3340.cpp  
#include <windows.h>  
[module(name="MyModule")];  
  
[ object, uuid(373a1a4c-469b-11d3-a6b0-00c04f79ae8f) ]  
__interface IMyIface  
{  
   HRESULT f1();  
};  
  
[ coclass, uuid(373a1a4d-469b-11d3-a6b0-00c04f79ae8f),  
default(IMyIface),  
source(IMyIface),restricted(IMyIface) ]  
class CmyClass // C3340  
{  
};  
  
int main()  
{  
}  
```
