# Errore del compilatore C3749
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'attributo': impossibile utilizzare un attributo personalizzato all'interno di una funzione  
  
 Non Ã¨ possibile utilizzare un attributo personalizzato all'interno di una funzione. Per ulteriori informazioni sugli attributi personalizzati, vedere l'argomento relativo agli [attributi](../../windows/attribute.md).  
  
 Il seguente codice di esempio genera l'errore C3749:  
  
```  
// C3749a.cpp  
// compile with: /clr /c  
using namespace System;  
  
[AttributeUsage(AttributeTargets::All)]  
public ref struct ABC : public Attribute {  
   ABC() {}  
};  
  
void f1() { [ABC]; };  // C3749  
```  
  
 Il seguente codice di esempio genera l'errore C3749:  
  
```  
// C3749b.cpp  
// compile with: /clr:oldSyntax /c  
using namespace System;  
  
[attribute(All)]  
public __gc struct ABC {  
   ABC() {}  
};  
  
void f1() { [ABC]; };  // C3749  
```
