# <a name="compiler-error-c2890"></a>Errore del compilatore C2890
'classe': una classe di riferimento può avere solo una classe di base non di interfaccia  
  
 Una classe di riferimento può avere solo una classe di base.  
  
 Nell'esempio seguente viene generato l'errore C2890:  
  
```  
// C2890.cpp  
// compile with: /clr /c  
ref class A {};  
ref class B {};  
ref class C : public A, public B {};   // C2890  
ref class D : public A {};   // OK  
```
