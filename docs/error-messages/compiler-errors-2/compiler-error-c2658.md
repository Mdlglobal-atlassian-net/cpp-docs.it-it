# <a name="compiler-error-c2658"></a>Errore del compilatore C2658
'member': ridefinizione in struct/union anonima  
  
 Due strutture o unioni anonime contenevano le dichiarazioni dei membri con lo stesso identificatore ma con tipi diversi. In [/Za](../../build/reference/za-ze-disable-language-extensions.md), si otterr√† l'errore per i membri con lo stesso identificatore e tipo.  
  
 L'esempio seguente genera l'errore C2658:  
  
```  
// C2658.cpp  
// compile with: /c  
struct X {  
   union { // can be struct too  
      int i;  
   };  
   union {  
      int i;   // Under /Za, C2658  
      // int i not needed here because it is defined in the first union  
   };  
};  
  
struct Z {  
   union {  
      char *i;  
   };  
  
   union {  
      void *i;   // C2658 redefinition of 'i'  
      // try the following line instead  
      // void *ii;  
   };  
};  
```
