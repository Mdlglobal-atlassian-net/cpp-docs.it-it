# Errore del compilatore C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': il modello di funzione non può convertire il parametro numero dal tipo 'tipo'  
  
 Non è stato possibile convertire un parametro nel tipo richiesto.  
  
 È possibile correggere questo errore creando una conversione esplicita.
