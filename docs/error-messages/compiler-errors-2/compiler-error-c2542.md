# <a name="compiler-error-c2542"></a>Errore del compilatore C2542
'identifier': oggetto di classe non ha un costruttore per l'inizializzazione  
  
 È presente alcun costruttore con un elenco di parametri corrispondente all'inizializzazione.  
  
 Questo errore può essere causato da parametri errati nell'inizializzazione.
