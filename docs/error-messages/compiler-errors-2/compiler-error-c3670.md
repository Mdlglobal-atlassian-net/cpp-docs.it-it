# Errore del compilatore C3670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'override': impossibile eseguire l'override del metodo della classe base inaccessibile 'metodo'  
  
 Ãˆ possibile eseguire l'override solo in una funzione il cui livello di accesso la rende disponibile in un tipo derivato.  Per ulteriori informazioni, vedere [Override espliciti](../../windows/explicit-overrides-cpp-component-extensions.md).  
  
 Il seguente codice di esempio genera l'errore C3670:  
  
```  
// C3670.cpp  
// compile with: /clr /c  
public ref class C {  
// Uncomment the following line to resolve.  
// public:  
   virtual void g() { }  
};  
  
public ref class D : public C {  
public:  
   virtual void f() new sealed = C::g {};   // C3670  
};  
```
