# <a name="compiler-error-c3135"></a>Errore del compilatore C3135
'property': una proprietà non può avere un 'const' o 'volatile' di tipo  
  
 Il [const](../../cpp/const-cpp.md) e [volatile](../../cpp/volatile-cpp.md) parole chiave non sono consentite per le proprietà.
