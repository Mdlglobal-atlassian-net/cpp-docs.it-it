# Errore del compilatore C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'proprietà': una proprietà non può avere un tipo 'const' o 'volatile'  
  
 Le parole chiave [const](../../cpp/const-cpp.md) e [volatile](../../cpp/volatile-cpp.md) non sono consentite per le proprietà.
