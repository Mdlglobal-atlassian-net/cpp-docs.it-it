# <a name="compiler-error-c3018"></a>Errore del compilatore C3018

'variabile1': 'For' test o l'incremento OpenMP deve utilizzare la variabile dell'indice 'variabile2'

Per il test e l'incremento di un ciclo `for` in un'istruzione OpenMP Ã¨ necessario usare la stessa variabile usata per l'indice.

L'esempio seguente genera l'errore C3018:

```
// C3018.cpp
// compile with: /openmp
int main()
{
   int i = 0, j = 5;

   #pragma omp parallel
   {
      #pragma omp for
      for (i = 0; j < 10; ++i)   // C3018
      // try the following line instead
      // for (i = 0; i < 10; ++i)
         j *= 2;

      #pragma omp for
      for (i = 0; i < 10; j = j + i)   // C3018
      // try the following line instead
      // for (i = 0; i < 10; i = j + i)
         j *= 2;
   }
}
```
