# <a name="compiler-error-c2948"></a>Errore del compilatore C2948

creazione di un'istanza esplicita. Identificatore di classe di archiviazione 'specifier' non valido in una specializzazione

Non è possibile usare gli identificatori di classi di archiviazione (ad esempio `extern`) in una specializzazione della classe modello che è stata creata in precedenza un'istanza in modo esplicito.
