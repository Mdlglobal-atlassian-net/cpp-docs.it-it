# <a name="compiler-error-c2948"></a>Errore del compilatore C2948
creazione di un'istanza esplicita. Identificatore di classe di archiviazione 'specifier' non valido in una specializzazione  
  
 È possibile utilizzare gli identificatori di classe di archiviazione (ad esempio `extern`) in una specializzazione di una classe modello che è stata creata in precedenza un'istanza in modo esplicito.
