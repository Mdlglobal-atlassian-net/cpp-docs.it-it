# Errore del compilatore C3369
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'module name': idl\_module già definito  
  
 L'oggetto [idl\_module](../../windows/idl-module.md) in cui è stata definita la DLL può essere usato una sola volta in un programma.  
  
 L'esempio seguente genera l'errore C3369:  
  
```  
// C3369.cpp // compile with: /c [idl_module(name="name1", dllname="x.dll")]; // C3369 [idl_module(name="name1", dllname="x.dll")];  
```
