# <a name="compiler-error-c3369"></a>Errore del compilatore C3369
'module name': idl_module già definito  
  
 Il [idl_module](../../windows/idl-module.md) utilizzo in cui è definita la DLL può verificarsi solo una volta in un programma.  
  
 L'esempio seguente genera l'errore C3369:  
  
```  
// C3369.cpp  
// compile with: /c  
[idl_module(name="name1", dllname="x.dll")]; // C3369  
[idl_module(name="name1", dllname="x.dll")];  
```
