# Errore del compilatore C2819
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il tipo 'tipo' non ha un membro 'operator \-\>' di overload  
  
 Ãˆ necessario definire `operator->()` per utilizzare questa operazione del puntatore.  
  
 Il seguente codice di esempio genera l'errore C2819:  
  
```  
// C2819.cpp  
// compile with: /c  
class A {  
   public:  
      int i;  
};  
  
class B {};  
  
void C(B j) {  
   j->i;   // C2819  
}  
  
class D {  
   A* pA;  
  
   public:  
      A* operator->() {  
         return pA;  
      }  
};  
  
void F(D j) {  
   j->i;  
}  
```
