# Errore del compilatore C2616
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversione': impossibile convertire in modo implicito un valore non l\-value 'tipo1' in un tipo 'tipo2' che non sia const  
  
 Non è possibile inizializzare un riferimento da un valore non l\-value.  
  
 Questo è un errore in compatibilità ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) e un avviso con le estensioni Microsoft \(**\/Ze**\).
