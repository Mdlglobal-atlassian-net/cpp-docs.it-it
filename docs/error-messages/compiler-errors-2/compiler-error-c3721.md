# <a name="compiler-error-c3721"></a>Errore del compilatore C3721

'signature': firma non compatibile per l'evento

Un evento è stato dichiarato in modo non corretto. Per altre informazioni, vedere [event](../../cpp/event.md).

È raggiungibile tramite l'opzione del compilatore obsolete solo C3721 **/CLR: oldSyntax**.
