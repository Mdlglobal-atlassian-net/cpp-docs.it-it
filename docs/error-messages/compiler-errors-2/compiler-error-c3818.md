# <a name="compiler-error-c3818"></a>Errore del compilatore C3818

dichiarazione di proprietà di matrice 'property1' non comporterà l'overload di una proprietà indicizzata 'property2'

Un overload non è possibile che le proprietà quando uno è un indicizzatore e l'altro è una proprietà di matrice.

È raggiungibile tramite l'opzione del compilatore obsolete solo C3818 **/CLR: oldSyntax**.
