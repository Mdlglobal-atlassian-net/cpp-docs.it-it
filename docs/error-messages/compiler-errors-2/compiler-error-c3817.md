# <a name="compiler-error-c3817"></a>Errore del compilatore C3817
'dichiarazione': la proprietà può essere applicata solo a una funzione  
  
 Il `property` parola chiave può essere applicata solo a una definizione di funzione.  
  
 C3817 è raggiungibile tramite l'opzione del compilatore obsoleta **oldSyntax**.
