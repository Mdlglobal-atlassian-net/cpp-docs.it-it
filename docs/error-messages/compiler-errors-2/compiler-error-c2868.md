# Errore del compilatore C2868
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': sintassi non valida per la dichiarazione using. Previsto nome completo  
  
 Per una [dichiarazione using](../../cpp/using-declaration.md) Ã¨ necessario un [nome completo](http://msdn.microsoft.com/it-it/3fefb16d-8120-4627-8b3f-3d90fbdcd1df).  
  
 Il seguente codice di esempio genera l'errore C2868:  
  
```  
// C2868.cpp  
class X {  
public:  
   int i;  
};  
  
class Y : X {  
public:  
   using X::i;   // OK  
};  
  
int main() {  
   using X;   // C2868  
}  
```
