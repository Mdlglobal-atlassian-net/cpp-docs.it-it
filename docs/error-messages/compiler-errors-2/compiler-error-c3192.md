# Errore del compilatore C3192
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

errore di sintassi: '^' non è la forma prefissa di un operatore \(si intendeva '\*'?\)  
  
 Non è possibile utilizzare un handle come operatore di dereferenziazione.  
  
 Il seguente codice di esempio genera l'errore C3192:  
  
```  
// C3192.cpp  
// compile with: /clr  
using namespace System;  
  
ref class MyClass {  
public:  
   MyClass () {}  
   MyClass(MyClass%) {}  
};  
  
int main() {  
   MyClass ^ s = gcnew MyClass;   
   MyClass b = ^s;   // C3192  
  
   // OK  
   MyClass b2 = *s;  
}  
```
