# Errore del compilatore C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore interno di compilazione: impossibile scrivere nel PDB il blocco di codice inserito  
  
 Questo errore si verifica se il compilatore non è riuscito a scrivere un blocco di codice inserito nel file pdb  ed è generalmente provocato da spazio insufficiente su disco.
