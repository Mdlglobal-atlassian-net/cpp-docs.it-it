# Errore del compilatore C2524
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'distruttore': un distruttore\/finalizzatore deve avere un elenco di parametri 'void'  
  
 L'elenco di parametri del distruttore o del finalizzatore non Ã¨ [void](../../cpp/void-cpp.md).  Non sono consentiti altri tipi di parametri.  
  
## Esempio  
 Nel codice seguente viene riprodotto l'errore C2524.  
  
```  
// C2524.cpp  
// compile with: /c  
class A {  
   A() {}  
   ~A(int i) {}    // C2524  
   // try the following line instead  
   // ~A() {}  
};  
```
