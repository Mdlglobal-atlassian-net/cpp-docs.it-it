# <a name="compiler-error-c3903"></a>Errore del compilatore C3903

'property': viene non ha un set o get (metodo)

Una propriet√† deve avere almeno una `get` o un `set` (metodo). Per altre informazioni, vedere [property](../../windows/property-cpp-component-extensions.md).

L'esempio seguente genera l'errore C3903:

```
// C3903.cpp
// compile with: /clr
ref class X {
   property int P {
      void f(int){}
      // Add one or both of the following lines.
      // void set(int){}
      // int get(){return 0;}
   };   // C3903

   property double Q[,,,,] {
      void f(){}
      // Add one or both of the following lines.
      // void set(int, char, int, char, double, double){}
      // double get(int, char, int, char, double){return 1.1;}
   }   // C3903
};
```
