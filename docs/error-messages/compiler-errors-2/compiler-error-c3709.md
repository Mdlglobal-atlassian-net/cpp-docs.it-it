# <a name="compiler-error-c3709"></a>Errore del compilatore C3709

'function': sintassi non corretta per la specifica di eventi in hook /\__unhook

Quando si specifica un'origine eventi contenente [hook](../../cpp/hook.md) oppure [unhook](../../cpp/unhook.md), il primo parametro deve essere un metodo di evento valido e il secondo parametro deve essere un oggetto di origine evento valido (non un metodo).

L'esempio seguente genera l'errore C3709:

```
// C3709.cpp
// compile with: /LD
[event_source(native)]
class CEventSrc
{
public:
   __event void event1();
};

[event_receiver(native)]
class CEventRec
{
public:
   void handler1()
   {
   }

   void HookEvents(CEventSrc* pSrc)
   {
      __hook(bad, pSrc, CEventRec::handler1);   // C3709
      // Try the following line instead:
      // __hook(&CEventSrc::event1, pSrc, CEventRec::handler1);
   }

   void UnhookEvents(CEventSrc* pSrc)
   {
      __unhook(&CEventSrc::event1, pSrc, CEventRec::handler1);
   }
};
```
