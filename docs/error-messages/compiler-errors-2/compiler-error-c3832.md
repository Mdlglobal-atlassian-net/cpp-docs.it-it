# Errore del compilatore C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'libreria tipi': la libreria dei tipi sembra essere stata compilata per puntatori a 32 bit. Modificare il qualificatore 'ptrsize'  
  
 Le informazioni esplicite fornite con l'attributo `ptrsize` della direttiva [\#import](../../preprocessor/hash-import-directive-cpp.md) non corrispondono alle informazioni rilevate dal compilatore nella libreria dei tipi.
