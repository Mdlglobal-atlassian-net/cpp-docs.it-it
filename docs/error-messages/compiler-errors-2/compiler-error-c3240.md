# Errore del compilatore C3240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': deve essere una funzione membro astratta di 'tipo' non in overload  
  
 In un tipo base Ã¨ contenuta una funzione definita.  La funzione deve essere di tipo virtuale.  
  
## Esempio  
 Nell'esempio seguente viene generato l'errore C3240:  
  
```  
// C3240.cpp  
// compile with: /c  
__interface I {  
   void f();  
};  
  
struct A1 : I {   
   void f() {}  
};  
  
struct A2 : I {   
   void f() = 0;  
};  
  
template <class T>   
struct A3 : T {  
   void T::f() {}  
};  
  
template <class T>   
struct A4 : T {  
   void T::f() {}  
};  
  
A3<A1> x;   // C3240  
A3<I> x2;  
A4<A2> x3;  
```
