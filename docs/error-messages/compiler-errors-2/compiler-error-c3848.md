# Errore del compilatore C3848
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

espressione con tipo 'tipo' deve perdere alcuni qualificatori const\-volatile per poter chiamare 'funzione'  
  
 Una variabile con un tipo const\-volatile specificato pu√≤ chiamare solo funzioni membro definite con qualificazioni const\-volatile uguali o maggiori.  
  
 Negli esempi seguenti viene generato l'errore C3848:  
  
```  
// C3848.cpp  
void glbFunc1()  
{  
}  
  
typedef void (* pFunc1)();  
  
struct S3  
{  
   operator pFunc1() // const  
   {  
      return &glbFunc1;  
   }  
};  
  
int main()  
{  
   const S3 s3;  
   s3();   // C3848, uncomment const qualifier  
}  
```
