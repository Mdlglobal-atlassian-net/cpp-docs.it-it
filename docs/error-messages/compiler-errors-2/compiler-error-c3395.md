 `__declspec(dllexport)`e [clrcall](../../cpp/clrcall.md) non sono compatibili.  Per ulteriori informazioni, vedere [dllexport, dllimport](../../cpp/dllexport-dllimport.md).

```  
// C3395.cpp

__declspec(dllexport) void Test3(){}   // OK  
```
