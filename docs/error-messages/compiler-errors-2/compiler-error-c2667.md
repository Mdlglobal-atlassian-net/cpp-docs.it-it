# Errore del compilatore C2667
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': nessuno dei numero overload presenta una conversione ottimale  
  
 Una chiamata di funzione in overload è ambigua e non può essere risolta.  
  
 La conversione necessaria per far corrispondere i parametri effettivi nella chiamata di funzione a una delle funzioni in overload deve essere migliore delle conversioni richieste da tutte le altre funzioni in overload.  
  
 Per ulteriori informazioni sull'ordine parziale dei template di funzione, vedere l’articolo della Knowledge Base Q240869 \(informazioni in lingua inglese\).
