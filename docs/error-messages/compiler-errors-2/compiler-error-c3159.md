# <a name="compiler-error-c3159"></a>Errore del compilatore C3159
'puntatore': Impossibile dichiarare matrici di puntatori al tipo di valore  
  
 Non è possibile dichiarare una matrice di puntatori a un tipo di valore.  
  
 C3159 è raggiungibile tramite l'opzione del compilatore obsoleta **oldSyntax**.
