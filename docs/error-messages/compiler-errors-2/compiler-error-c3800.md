# <a name="compiler-error-c3800"></a>Errore del compilatore C3800
'dichiarazione': Impossibile combinare proprietà ed eventi  
  
 È possibile dichiarare un costrutto di una proprietà e un evento.  
  
 C3800 è raggiungibile tramite l'opzione del compilatore obsoleta **oldSyntax**.
