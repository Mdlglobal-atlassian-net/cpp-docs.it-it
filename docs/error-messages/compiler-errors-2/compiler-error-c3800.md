# <a name="compiler-error-c3800"></a>Errore del compilatore C3800
'declaration': Impossibile combinare proprietà ed eventi  
  
 È possibile dichiarare un costrutto di una proprietà e un evento.  
  
 C3800 solo è raggiungibile anche tramite l'opzione del compilatore obsoleta **oldSyntax**.
