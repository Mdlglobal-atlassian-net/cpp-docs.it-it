# Errore del compilatore C3173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

conflitto di versioni nell'unione idl  
  
 Questo errore viene generato quando un file oggetto contiene IDL incorporato generato con una versione precedente del compilatore,  il quale codifica un numero di versione per accertarsi che lo stesso compilatore venga utilizzato per generare il contenuto IDL incorporato nei file obj e per unire l'IDL incorporato.
