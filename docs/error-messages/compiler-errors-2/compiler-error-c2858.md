# <a name="compiler-error-c2858"></a>Errore del compilatore C2858
opzione della riga di comando ' /Yc (/ /Fdnomefile)' non è coerente con l'intestazione precompilata, utilizzato ' / /Fdnomefile '  
  
 Il database di programma specificato dall'opzione Usa intestazione precompilata ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) opzione non è quello specificato per il precedente Crea intestazione precompilata ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) opzione.
