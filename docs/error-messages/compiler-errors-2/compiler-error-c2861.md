# Errore del compilatore C2861
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'nome funzione': impossibile definire una funzione membro di interfaccia  
  
 È stata rilevata la parola chiave interface o è stata dedotta una struct come interfaccia, ma in seguito è stata individuata una definizione di funzione membro.  Un'interfaccia non può contenere una definizione per una funzione membro.  
  
## Esempio  
 Il seguente codice di esempio genera l'errore C2861:  
  
```  
// C2861.cpp  
// compile with: /c  
#include <objbase.h>   // required for IUnknown definition  
[ object, uuid("00000000-0000-0000-0000-000000000001") ]  
__interface IMyInterface : IUnknown {  
   HRESULT mf(int a);  
};  
  
HRESULT IMyInterface::mf(int a) {}   // C2861  
  
```
