# Errore del compilatore C3656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'override': impossibile ripetere l'indicatore di override  
  
 La parola chiave di un override esplicito pu√≤ essere specificata una sola volta.  Per ulteriori informazioni, vedere [Override espliciti](../../windows/explicit-overrides-cpp-component-extensions.md).  
  
 Il seguente codice di esempio genera l'errore C3656:  
  
```  
// C3656.cpp  
// compile with: /clr /c  
public interface struct O {  
   int f();  
};  
  
public ref struct V : O {  
   int f() override override { return 0; }   // C3656  
   // try the following line instead  
   // int f() override { return 0; }  
};  
```
