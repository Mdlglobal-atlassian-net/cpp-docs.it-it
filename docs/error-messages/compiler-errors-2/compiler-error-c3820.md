# <a name="compiler-error-c3820"></a>Errore del compilatore C3820

'name': gli inizializzatori devono essere gestiti

Anche l'inizializzatore *name* per un tipo gestito deve essere gestito.

### <a name="to-correct-this-error"></a>Per correggere l'errore

1. Specificare un metodo di inizializzazione gestito.
