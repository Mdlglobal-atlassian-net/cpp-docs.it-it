# Errore del compilatore C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operatore': impossibile trovare l'operatore di overload  
  
 L'operatore non può essere utilizzato con gli operandi forniti.  Occorre fornire un operatore di overload con gli operandi richiesti.  
  
 Questo errore può verificarsi se il tipo degli operandi non è corretto.  
  
 È possibile correggerlo definendo un costruttore o un operatore di conversione che accetta un singolo parametro.
