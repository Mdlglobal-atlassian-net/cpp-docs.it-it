# <a name="compiler-error-c2545"></a>Errore del compilatore C2545

'operator': Impossibile trovare sottoposti a overload l'operatore

L'operatore non è utilizzabile con gli operandi forniti. È necessario specificare un operatore di overload con gli operandi necessari.

Questo errore può essere causato se gli operandi hanno un tipo non corretto.

Questo errore può essere risolto se si definisce un operatore di conversione o costruttore che accetta un singolo parametro.
