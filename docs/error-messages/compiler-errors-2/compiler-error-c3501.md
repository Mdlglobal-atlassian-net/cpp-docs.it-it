# <a name="compiler-error-c3501"></a>Errore del compilatore C3501

Nessun TypeLib registrato per il ProgID ' ProgID '

All'ID di classe per un ProgID specificato non è associata una libreria dei tipi. Non è quindi possibile passare questo ProgID all'istruzione `#import`.
