# <a name="compiler-error-c3501"></a>Errore del compilatore C3501

Nessuna libreria dei tipi registrata per il ProgID 'progid'

L'ID della classe per un dato progid è privo di una libreria dei tipi associati. Pertanto, è possibile passare questo progid per il `#import` istruzione.
