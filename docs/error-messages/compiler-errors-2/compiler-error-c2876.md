# Errore del compilatore C2876
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe::simbolo': non tutti gli overload sono accessibili  
  
 Tutti gli overload di una funzione in una classe base devono essere accessibili alla classe derivata.  
  
 Il seguente codice di esempio genera l'errore C2876:  
  
```  
// C2876.cpp  
// compile with: /c  
class A {  
public:     
   double a(double);  
private:  
   int a(int);  
};  
  
class B : public A {  
   using A::a;   // C2876 one overload is private in base class  
};  
```
