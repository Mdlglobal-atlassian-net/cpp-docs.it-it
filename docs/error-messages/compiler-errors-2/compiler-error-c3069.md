# <a name="compiler-error-c3069"></a>Errore del compilatore C3069

'operator': non consentito per il tipo di enumerazione

Un operatore non Ã¨ supportato per le enumerazioni CLR.  Per altre informazioni, vedere [Procedura: Definire e usare enumerazioni in C++/CLI](../../dotnet/how-to-define-and-consume-enums-in-cpp-cli.md).

## <a name="example"></a>Esempio

L'esempio seguente genera l'errore C3069:

```
// C3069.cpp
// compile with: /clr
enum struct E { e1 };
enum F { f1 };

int main() {
   E e = E::e1;
   bool tf;
   tf = !e;   // C3069

   // supported for native enums
   F f = f1;
   tf = !f;
}
```
