# Errore del compilatore C3055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'symbol': impossibile fare riferimento al simbolo prima di utilizzarlo nella direttiva 'threadprivate'  
  
 Un simbolo è stato indicato e usato in una clausola [threadprivate](../../parallel/openmp/reference/threadprivate.md). Questa operazione non è consentita.  
  
 L'esempio seguente genera l'errore C3055:  
  
```  
// C3055.cpp // compile with: /openmp int x, y; int z = x; #pragma omp threadprivate(x, y)   // C3055 void test() { #pragma omp parallel copyin(x, y) { x = y; } }  
```  
  
 Possibile soluzione:  
  
```  
// C3055b.cpp // compile with: /openmp /LD int x, y, z; #pragma omp threadprivate(x, y) void test() { #pragma omp parallel copyin(x, y) { x = y; } }  
```
