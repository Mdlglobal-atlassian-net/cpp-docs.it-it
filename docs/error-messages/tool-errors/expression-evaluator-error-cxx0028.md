# Errore dell‘analizzatore di espressioni CXX0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

elemento non di tipo struttura\/unione  
  
 È stata specificata un'espressione del tipo 'Struct.Member' o 'pStruct\-\>Member', ma \<member\> non è un elemento della struttura.  
  
 È possibile che l'espressione non sia stata inclusa tra parentesi in modo corretto.  
  
 Questo errore è identico all'errore CAN0028.
