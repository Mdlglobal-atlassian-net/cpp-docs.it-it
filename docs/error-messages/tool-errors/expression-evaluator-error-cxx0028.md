# <a name="expression-evaluator-error-cxx0028"></a>Errore dell‘analizzatore di espressioni CXX0028
elemento non struct/union  
  
 È stata specificata un'espressione del form 'Tipo' o 'pStruct -> Member', ma \<membro > non è un elemento della struttura.  
  
 L'espressione potrebbe non essere correttamente tra parentesi.  
  
 Questo errore è identico all'errore CAN0028.
