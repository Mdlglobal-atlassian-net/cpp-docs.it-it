# Errore BK1507 di BSCMAKE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile aprire il file temporaneo "nomefile" \[: motivo\]  
  
 Non è possibile aprire un file temporaneo con BSCMAKE.  
  
### Possibili cause  
  
1.  La variabile **TMP** non è impostata su un'unità e una directory valide.  
  
2.  Disco pieno.
