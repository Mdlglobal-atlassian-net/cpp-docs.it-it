# <a name="linker-tools-error-lnk2023"></a>Errore degli strumenti del linker LNK2023
dll non valida o punto di ingresso \<dll o punto di ingresso >  
  
 Il linker sta caricando una versione non corretta di msobj90. Verificare che link.exe e msobj90 nel proprio percorso abbiano la stessa versione.  
  
 Una dipendenza di msobj90 non può essere presente. L'elenco di dipendenze per msobj90 è:  
  
-   Msvcr90.dll  
  
-   Kernel32.dll  
  
 Controllare il computer per eventuali altre copie di msobj90 che possono essere aggiornate.
