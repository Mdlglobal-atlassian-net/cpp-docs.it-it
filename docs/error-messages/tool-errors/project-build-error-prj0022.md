# Errore di compilazione progetto PRJ0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Strumento sconosciuto. La proprietà "proprietà" contiene il nome di file non valido "file".  
  
 Il nome di file ***file***, specificato nella proprietà ***proprietà***, non è valido.  
  
 È possibile che sia stata utilizzata una macro sconosciuta o non valida.  
  
 Per informazioni sulle modalità di accesso alle proprietà del progetto, vedere [Impostazione delle proprietà dei progetti Visual C\+\+](../../ide/working-with-project-properties.md).
