# Errore del compilatore di risorse RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sequenza di escape non valida  
  
 È stata rilevata una sequenza di escape in una posizione non consentita.  
  
 Una sequenza di escape, ovvero una barra rovesciata \( **\\** \) seguita da un numero o da una lettera, può trovarsi solo in una costante di stringa o di carattere.
