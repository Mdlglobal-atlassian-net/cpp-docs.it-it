# Errore dell‘analizzatore di espressioni CXX0063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'operatore di overload \-\> non è supportato  
  
 L'espressione utilizza un operatore di overload di accesso a membri di classe \(**\-\>**\).  
  
 Questo errore è identico all'errore CAN0063.
