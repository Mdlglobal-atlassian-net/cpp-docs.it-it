# Errore della riga di comando D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\/opzione" non consentita con più file di origine.  
  
 Le seguenti opzioni di compilazione non possono essere utilizzate con più file di origine:  
  
-   \/Fa: un file di listato dell'assembly  
  
-   \/Fo: consente di rinominare un file oggetto  
  
-   \/Fr: consente di creare informazioni di visualizzazione di origine senza variabili locali  
  
-   \/FR: consente di creare informazioni di visualizzazione di origine nelle quali sono incluse variabili locali
