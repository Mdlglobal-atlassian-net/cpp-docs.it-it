# Errore degli strumenti del linker LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo di computer tipo non valido  
  
 Il tipo di computer specificato nell'intestazione dell'oggetto non è valido.  
  
 Questo errore può verificarsi se il file è danneggiato.  Ricompilare l'oggetto.
