# Errore del compilatore di risorse RC2103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

fine file imprevista in stringa letterale  
  
 La fine di file è stata raggiunta prima della fine della stringa.  È possibile che nella stringa manchi un segno di virgolette doppie di chiusura \(**"**\).
