# <a name="profile-guided-optimization-warning-pg1039"></a>Avviso di ottimizzazione PGO PG1039

Non è possibile attendere per effettuare lo sweep.

L'applicazione non consentiva [pgosweep](../../build/pgosweep.md) per l'esecuzione.  Se possibile, eseguire pgosweep quando l'applicazione è in uno stato di inattività.
