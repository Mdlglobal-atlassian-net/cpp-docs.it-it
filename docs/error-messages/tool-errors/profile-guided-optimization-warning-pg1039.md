# Avviso di ottimizzazione PGO PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile attendere per effettuare lo sweep.  
  
 L'applicazione non ha consentito l'esecuzione di [pgosweep](../../build/reference/pgosweep.md).  Se possibile, eseguire pgosweep quando l'applicazione è in uno stato di inattività.
