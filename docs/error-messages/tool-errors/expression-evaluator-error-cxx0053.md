# Errore dell‘analizzatore di espressioni CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

rilevato simbolo che non è una funzione durante l'associazione dei punti di interruzione  
  
 Un simbolo utilizzato come punto di interruzione non è una funzione.  L'errore può essere causato dall'aver specificato un membro dati come punto di interruzione.  
  
 Questo errore è identico all'errore CAN0053.
