# Errore dell‘analizzatore di espressioni CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile trovare l'operatore di overload  
  
 Un tipo di classe è stato specificato come operando di sinistra in un'espressione, senza che per la classe sia stato definito un operatore di overload.  
  
 Questo errore è identico all'errore CAN0058.
