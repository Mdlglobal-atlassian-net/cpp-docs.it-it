# <a name="expression-evaluator-error-cxx0058"></a>Errore dell‘analizzatore di espressioni CXX0058
operatore di overload non trovata  
  
 È stato specificato un tipo di classe dell'operando di sinistra in un'espressione, ma non è stato definito un operatore di overload per la classe.  
  
 Questo errore è identico all'errore CAN0058.
