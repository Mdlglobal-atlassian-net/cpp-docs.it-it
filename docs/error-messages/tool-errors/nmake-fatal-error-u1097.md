# Errore irreversibile U1097 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La sintassi relativa ai nomi di file richiede un dipendente.  
  
 Nella dipendenza corrente non è disponibile né un dipendente esplicito né uno implicito.  La sintassi relativa ai nomi di file, nella quale simbolo di percentuale \(`%`\) viene utilizzato come identificatore, rappresenta i componenti del primo dipendente della destinazione corrente.
