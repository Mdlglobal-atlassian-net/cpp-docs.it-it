# <a name="expression-evaluator-error-cxx0072"></a>Errore dell‘analizzatore di espressioni CXX0072

Errore: tipo di informazioni mancanti o sconosciuto

Il file con estensione PCH non è stato ottenere collegato o il codice contiene un riferimento a un tipo che si trova in un modulo non è stato compilato con /Zi.

Questo errore è identico all'errore CAN0067.
