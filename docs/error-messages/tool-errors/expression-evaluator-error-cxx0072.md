# Errore dell‘analizzatore di espressioni CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore: informazioni sul tipo mancanti o sconosciute  
  
 Il file PCH non è stato collegato o il codice fa riferimento a un tipo contenuto in un modulo non compilato con \/Zi.  
  
 Questo errore è identico all'errore CAN0067.
