# <a name="expression-evaluator-error-cxx0072"></a>Errore dell‘analizzatore di espressioni CXX0072
Errore: tipo di informazioni mancanti o sconosciute  
  
 Il file PCH non collegato, o il codice contiene un riferimento a un tipo in un modulo non è stato compilato con /Zi.  
  
 Questo errore è identico all'errore CAN0067.
