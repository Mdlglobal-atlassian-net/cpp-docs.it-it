# <a name="expression-evaluator-error-cxx0022"></a>Errore dell‘analizzatore di espressioni CXX0022
chiamata di funzione prima di Main  
  
 L'analizzatore di espressioni C non è possibile valutare una funzione prima del debugger la funzione **Main**. Il programma non è inizializzato correttamente finché **Main** è stato chiamato.  
  
 Questo errore è identico all'errore CAN0022.
