# Errore degli strumenti del linker LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

rilevati \<count\> tipi Windows Runtime importati in un modulo e definiti in un altro modulo  
  
 Quando ha prodotto la destinazione corrente, il linker rileva \<`count`\> tipi di Runtime di Windows, ognuno dei quali è incluso in un modulo ed è anche definito in un altro modulo.  
  
### Per correggere l'errore  
  
-   Correggere tutti gli errori LNK2039 nella compilazione a seconda del suggerimento nel messaggio di errore.
