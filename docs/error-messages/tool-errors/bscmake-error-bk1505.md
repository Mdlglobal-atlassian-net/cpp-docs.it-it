# Errore BK1505 di BSCMAKE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile leggere dal file "nomefile" \[: motivo\]  
  
 Impossibile leggere dal file con BSCMAKE.  
  
### Possibili cause  
  
1.  File danneggiato.  
  
2.  Troncamento del file dovuto all'insufficienza di spazio su disco o all'interruzione del compilatore durante la creazione del file SBR.
