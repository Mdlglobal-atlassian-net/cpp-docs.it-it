# <a name="bscmake-error-bk1505"></a>Errore BK1505 di BSCMAKE
Impossibile leggere dal file 'nomefile' [: motivo]  
  
 BSCMAKE non è possibile leggere il file.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili  
  
1.  Danneggiamento del file.  
  
2.  Il compilatore sta esaurendo lo spazio su disco o viene interrotta durante la creazione del file SBR troncamento del file.
