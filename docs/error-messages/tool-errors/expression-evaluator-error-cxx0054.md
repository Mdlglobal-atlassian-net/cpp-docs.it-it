# Errore dell‘analizzatore di espressioni CXX0054
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

registro nell'espressione del punto di interruzione non valido  
  
 Un registro non può essere utilizzato nell'espressione di un punto di interruzione.  
  
 Questo errore è identico all'errore CAN0054.
