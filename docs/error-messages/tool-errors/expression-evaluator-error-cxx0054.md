# <a name="expression-evaluator-error-cxx0054"></a>Errore dell‘analizzatore di espressioni CXX0054
registrare nell'espressione di punto di interruzione non valido  
  
 Un registro non può essere utilizzato in un'espressione con punto di interruzione.  
  
 Questo errore è identico all'errore CAN0054.
