# Errore di compilazione progetto PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile creare un file temporaneo nella directory "directory".  
  
 La chiamata per la creazione di un file temporaneo non è riuscita.  Di seguito sono indicate alcune delle possibili cause dell'errore:  
  
-   Non vi sono più nomi di file temporanei disponibili.  
  
-   La directory temp è di sola lettura.  
  
-   Non è presente alcuna directory temp o variabile d'ambiente TMP.  
  
-   Lo spazio disponibile su disco è insufficiente.
