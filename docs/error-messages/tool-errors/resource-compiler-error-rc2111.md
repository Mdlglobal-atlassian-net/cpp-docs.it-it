# Errore del compilatore di risorse RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Tipo di controllo non valido.  
  
 Le istruzioni di controllo ammesse all'interno di un'istruzione **DIALOG** sono: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICON, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, SCROLLBAR, USERBUTTON.  
  
 Verificare che i nomi di queste istruzioni di controllo siano stati digitati correttamente.
