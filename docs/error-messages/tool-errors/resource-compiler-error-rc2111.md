# <a name="resource-compiler-error-rc2111"></a>Errore del compilatore di risorse RC2111

tipo di controllo non valido

Ogni istruzione di controllo in una **dialogo** istruzione deve essere uno dei seguenti: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BMODIFICA, CASELLA DI CONTROLLO, CASELLA COMBINATA, CONTROLLO, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICONA, IEDIT, CASELLA DI RIEPILOGO, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, DELLA BARRA DI SCORRIMENTO, USERBUTTON.

Assicurarsi che queste istruzioni di controllo siano state digitate correttamente.
