# Errore di compilazione progetto PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile creare la directory di output "directory".  
  
 Non è stato possibile creare una directory di output.  Le ragioni possibili sono:  
  
-   La directory in cui doveva essere creata la directory di output è di sola lettura.  
  
-   Lo spazio disponibile su disco è insufficiente.  
  
-   È stato specificato un percorso di directory non valido.  
  
-   Le autorizzazioni dell'utente non sono sufficienti per creare la directory.
