# Errore dell‘analizzatore di espressioni CXX0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

elenco di argomenti richiesto per la funzione membro  
  
 Una funzione membro è stata chiamata da un'espressione senza che siano stati specificati parametri effettivi.  
  
 Questo errore è identico all'errore CAN0046.
