# Errore dell‘analizzatore di espressioni CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

non è un puntatore a una struttura  
  
 L'operatore di selezione membro \(**\-\>**\) è stato applicato a un'espressione che non è un puntatore a una struttura.  
  
 Verificare che l'intera espressione sia inclusa tra parentesi in modo corretto oppure eseguire il cast di tipo sull'espressione di indirizzo nell'appropriato tipo di puntatore a struttura  
  
 Questo errore è identico all'errore CAN0029.
