# Errore R6024 del linguaggio C in fase di esecuzione 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Spazio per la tabella \_onexit\/atexit insufficiente.  
  
 Non vi è memoria sufficiente per la funzione `_onexit` o `atexit`.  L'errore è provocato da una condizione di memoria insufficiente.
