# Errore dell‘analizzatore di espressioni CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operatore non supportato  
  
 È stato specificato in un'espressione un operatore C non supportato.  Scrivere un'espressione equivalente utilizzando operatori C supportati.  
  
 Questo errore è identico all'errore CAN0005.
