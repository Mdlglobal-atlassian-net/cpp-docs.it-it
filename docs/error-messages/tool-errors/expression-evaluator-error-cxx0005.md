# <a name="expression-evaluator-error-cxx0005"></a>Errore dell‘analizzatore di espressioni CXX0005
operatore non è supportato  
  
 È stato specificato un operatore C non supportato in un'espressione. Scrivere un'espressione equivalente utilizzando gli operatori C supportati.  
  
 Questo errore è identico all'errore CAN0005.
