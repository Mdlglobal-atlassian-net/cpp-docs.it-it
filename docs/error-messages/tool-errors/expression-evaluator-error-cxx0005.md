# <a name="expression-evaluator-error-cxx0005"></a>Errore dell‘analizzatore di espressioni CXX0005

operatore non supportato

È stato specificato un operatore di C non supportato in un'espressione. Scrivere un'espressione equivalente che usa gli operatori C supportati.

Questo errore è identico all'errore CAN0005.
