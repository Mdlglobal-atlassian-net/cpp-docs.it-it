# Errore matematico M6110
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

overflow dello stack  
  
 Un'espressione a virgola mobile ha causato un overflow dello stack a virgola mobile.  
  
 Le eccezioni di overflow dello stack a virgola mobile vengono intercettate sino al limite di sette livelli in aggiunta agli otto generalmente supportati dal coprocessore 8087\/287\/387.  
  
 Il programma termina con codice di uscita 138.
