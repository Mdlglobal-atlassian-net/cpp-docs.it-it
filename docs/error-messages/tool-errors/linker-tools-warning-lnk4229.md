# Avviso degli strumenti del linker LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la direttiva '\/directive' rilevata non è valida e verrà ignorata  
  
 Una direttiva passata con il pragma [comment](../../preprocessor/comment-c-cpp.md) non è valida.  La direttiva `/directive` verrà ignorata.
