# <a name="nmake-fatal-error-u1071"></a>Errore irreversibile U1071 di NMAKE 
ciclo nella struttura ad albero di dipendenza per la destinazione 'targetname'  
  
 Nell'albero della dipendenza per la destinazione è presente una dipendenza circolare. La destinazione specificata è un dipendente di uno dei dipendenti di destinazione specificato. Dipendenze circolari non sono validi.
