# Errore irreversibile U1071 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ciclo nella struttura ad albero delle dipendenze per la destinazione "nomedestinazione"  
  
 Nella struttura ad albero delle dipendenze della destinazione specificata è presente una dipendenza circolare.  La destinazione specificata è a sua volta dipendente di uno dei suoi dipendenti.  Le dipendenze circolari non sono valide.
