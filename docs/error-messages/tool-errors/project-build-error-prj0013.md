# <a name="project-build-error-prj0013"></a>Errore di compilazione progetto PRJ0013
Le risorse di sistema potrebbero non essere sufficienti. Impossibile creare una pipe necessaria per avviare la compilazione.  
  
 Questo errore indica che le risorse di sistema non sono sufficienti. Per risolverlo, ridurre l'uso delle risorse di sistema da parte di altri processi o applicazioni.  
  
 Questo errore può verificarsi anche se il livello di protezione non è sufficiente per la creazione (vedere [pipe](http://msdn.microsoft.com/library/windows/desktop/aa365152.aspx)).
