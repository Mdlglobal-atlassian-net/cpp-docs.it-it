# Errore di compilazione progetto PRJ0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le risorse di sistema potrebbero non essere sufficienti.Impossibile creare una pipe necessaria per avviare la compilazione.  
  
 Questo errore indica che le risorse di sistema non sono sufficienti.  Per risolverlo, ridurre l'uso delle risorse di sistema da parte di altri processi o applicazioni.  
  
 Questo errore può anche verificarsi se il livello di sicurezza disponibile non è sufficiente per la creazione di pipe \(vedere [Funzione CreatePipe](http://msdn.microsoft.com/it-it/library/windows/desktop/aa365152.aspx)\).
