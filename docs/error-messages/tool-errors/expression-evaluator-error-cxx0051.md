# Errore dell‘analizzatore di espressioni CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

più di un simbolo di overload specificato nel punto di interruzione  
  
 L'analizzatore di espressioni non è stato in grado di determinare quale simbolo di overload utilizzare come punto di interruzione.  
  
 Questo errore è identico all'errore CAN0051.
