# <a name="linker-tools-error-lnk1120"></a>Errore degli strumenti del linker LNK1120
*numero* riferimenti esterni non risolti  
  
Errore LNK1120 riporta il conteggio (*numero*) di errori di simbolo esterno non risolto per questa operazione di collegamento. La maggior parte non risolti gli errori di simbolo esterno vengono segnalati singolarmente [Errore degli strumenti del Linker LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) e [Errore degli strumenti del Linker LNK2019](../../error-messages/tool-errors/linker-tools-error-lnk2019.md), che precedono questo messaggio di errore, una volta per ciascun esterno non risolto Errore di simbolo.  
  
Per correggere l'errore, correggere tutti gli altri errori esterni non risolti o altri errori del linker che lo precedono nell'output di compilazione. Questo errore non viene segnalato quando non rimangano alcun errore esterno non risolto.
