# Errore degli strumenti del linker LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

numero esterni non risolti  
  
 Questo errore fornisce un conteggio \(`number`\) di esterni non risolti per il collegamento corrente.  Le condizioni che provocano gli esterni non risolti sono descritte con l'errore [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md), che precede questo messaggio di errore, una volta per ciascun esterno non risolto.  
  
 Per ulteriori informazioni, vedere [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;it\-it;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650).
