# <a name="project-build-error-prj0002"></a>Errore di compilazione progetto PRJ0002
Errore restituito da 'riga di comando'.  
  
 Il comando ***riga di comando***, che è stato creato dall'input utente nel **pagine delle proprietà** , ha restituito un codice di errore, ma nessuna informazione viene visualizzata nella finestra di Output.  
  
 La risoluzione dell'errore dipende strumento che ha generato l'errore. Per MIDL, si otterrà un'idea di cosa è verificato un errore se viene definito /o (reindirizza Output).  
  
 Un file batch, ad esempio un passaggio di compilazione personalizzata o un evento di compilazione, che non fornisce informazioni sulle condizioni di errore potrebbe essere il motivo dell'errore.
