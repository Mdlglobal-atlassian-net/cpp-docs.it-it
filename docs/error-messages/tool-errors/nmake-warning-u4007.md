# Avviso U4007 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nome di file "nomefile" troppo lungo; verrà troncato in base al formato 8.3  
  
 Nel nome di base del file specificato sono presenti più di otto caratteri o più di tre nell'estensione.  Il nome o l'estensione è stato troncato da NMAKE in modo da ottenere un nome file in formato 8.3.  
  
 Se nel sistema operativo in uso sono supportati i nomi file lunghi, racchiudere il nome file tra virgolette doppie \(**"**\).
