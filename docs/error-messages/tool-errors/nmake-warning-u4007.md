# <a name="nmake-warning-u4007"></a>Avviso U4007 di NMAKE 

nome del file 'nomefile' troppo lungo. Troncamento a 8,3

Il nome di base del file specificato contiene più di otto caratteri, o l'estensione dispone di più di tre caratteri. NMAKE troncato il nome di una base di otto caratteri e un'estensione di tre caratteri.

Se i nomi file lunghi sono supportati dal file system, racchiuderlo tra virgolette (**"**).
