# Errore dell‘analizzatore di espressioni CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

registro non disponibile  
  
 Un registro non utilizzabile è stato specificato da un'espressione.  L'errore può essere causato dal tentativo di accesso a un registro che non esiste nel computer in uso.  
  
 Questo errore è identico all'errore CAN0060.
