# <a name="expression-evaluator-error-cxx0060"></a>Errore dell‘analizzatore di espressioni CXX0060

registrazione non disponibile

Un registro non può essere usati è specificata un'espressione. Questo errore può essere causato dal tentativo di accesso a un registro che non esiste nel computer in esecuzione.

Questo errore è identico all'errore CAN0060.
