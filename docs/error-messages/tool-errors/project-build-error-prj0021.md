# Errore di compilazione progetto PRJ0021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Strumento "strumento". La proprietà "proprietà" contiene un nome di file non valido.  
  
 Il nome di file specificato nella proprietà ***proprietà*** per lo strumento ***strumento*** non è valido.  
  
 È possibile che sia stata utilizzata una macro sconosciuta o non valida.  
  
 Per informazioni sulle modalità di accesso alle proprietà del progetto, vedere [Impostazione delle proprietà dei progetti Visual C\+\+](../../ide/working-with-project-properties.md).
