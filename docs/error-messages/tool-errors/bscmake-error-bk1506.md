# <a name="bscmake-error-bk1506"></a>Errore BK1506 di BSCMAKE
Impossibile aprire il file 'nomefile' [: motivo]  
  
 BSCMAKE non è possibile aprire il file.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili  
  
1.  File bloccato da un altro processo. Se `reason` afferma **autorizzazione negata**, il browser può utilizzare il file. Chiudere la finestra di ricerca, quindi ripetere la compilazione.  
  
2.  Un disco pieno.  
  
3.  Un errore hardware.  
  
4.  Il file di output specificato ha lo stesso nome di una sottodirectory esistente.
