# <a name="bscmake-error-bk1506"></a>Errore BK1506 di BSCMAKE

Impossibile aprire il file 'nomefile' [: motivo]

BSCMAKE non è possibile aprire il file.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili

1. File è bloccato da un altro processo. Se `reason` afferma **autorizzazione negata**, il browser sta utilizzando il file. Chiudere la finestra di esplorazione e ripetere la compilazione.

1. Un disco pieno.

1. Un errore hardware.

1. Il file di output specificata ha lo stesso nome di una sottodirectory esistente.
