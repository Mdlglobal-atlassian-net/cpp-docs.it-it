# <a name="expression-evaluator-error-cxx0020"></a>Errore dell‘analizzatore di espressioni CXX0020
tipi di operando non valido per questa operazione  
  
 È stato applicato un operatore a un'espressione con un tipo non valido per questo operatore.  
  
 Non è ad esempio, valido per accettare l'indirizzo di un registro o di un indice di matrice con un'espressione a virgola mobile.  
  
 Questo errore è identico all'errore CAN0020.
