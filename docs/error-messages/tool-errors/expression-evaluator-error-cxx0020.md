# Errore dell‘analizzatore di espressioni CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipi di operandi non validi per l'operazione  
  
 Si è applicato un operatore a un'espressione con un tipo non valido per tale operatore.  
  
 Non è, ad esempio, consentito accettare l'indirizzo di un registro né utilizzare come indice di una matrice un'espressione in virgola mobile.  
  
 Questo errore è identico all'errore CAN0020.
