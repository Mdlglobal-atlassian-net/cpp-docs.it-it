# <a name="linker-tools-error-lnk1166"></a>Errore degli strumenti del linker LNK1166
non è possibile modificare il codice all'offset = offset, va = valore  
  
 COLLEGAMENTO non è in grado di adattare il codice in base alle esigenze.  
  
 Alcune istruzioni non è consentiti attraversare i limiti di pagina su alcuni processori. COLLEGAMENTO tenta di aggiungere riempimenti per risolvere questo problema. In questo caso, collegamento potrebbe non risolvere il problema.
