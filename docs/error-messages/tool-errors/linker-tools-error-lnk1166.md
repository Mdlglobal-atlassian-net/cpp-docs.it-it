# <a name="linker-tools-error-lnk1166"></a>Errore degli strumenti del linker LNK1166

Impossibile modificare il codice in corrispondenza dell'offset = offset, valutazione della vulnerabilità = valore

COLLEGAMENTO non è riuscito riempire il codice in base alle esigenze.

Alcune istruzioni non sono consentiti a oltrepassare i limiti di pagina su alcuni processori. COLLEGAMENTO tenta di aggiungere riquadri per correggere il problema. In questo caso, collegamento potrebbe non risolvere il problema.
