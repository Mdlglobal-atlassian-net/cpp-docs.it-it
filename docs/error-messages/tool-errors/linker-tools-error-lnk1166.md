# Errore degli strumenti del linker LNK1166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile adattare il codice in offset\=offset, va\=value  
  
 Non è stato possibile adattare il codice nel modo richiesto.  
  
 Determinate istruzioni non possono superare i limiti di pagina su alcuni processori.  LINK tenta di aggiungere riempimenti per correggere la situazione.  In questo caso, non è stato possibile risolvere il problema.
