# Errore irreversibile U1088 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

separatore "::" non valido per la regola di inferenza  
  
 Una regola di inferenza deve essere seguita da un segno di due punti singoli \(**:**\).
