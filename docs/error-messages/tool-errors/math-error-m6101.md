# Errore matematico M6101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Non valido.  
  
 Operazione non valida.  
  
 Questo errore può verificarsi quando un operando non è un numero oppure è infinito.  
  
 Il programma termina con codice di uscita 129.
