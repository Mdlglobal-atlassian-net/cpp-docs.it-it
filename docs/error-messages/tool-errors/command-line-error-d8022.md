# Errore della riga di comando D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile aprire 'filemessaggio'  
  
 Il file indicato non si trova nella directory corrente o in una directory specificata nella variabile di ambiente PATH.  Nel file di messaggio è contenuto un breve riepilogo della sintassi e delle opzioni di riga di comando del compilatore.  
  
 Spostare il file nella directory corrente o in una directory all'interno del percorso corrente.  Se non è possibile trovare il file, eseguire il programma di installazione per copiarlo dai dischi originali.
