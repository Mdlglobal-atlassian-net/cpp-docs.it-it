# Errore degli strumenti del linker LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la dimensione dell'immagine \('dimensione'\) supera la dimensione massima consentita \(80000000\)  
  
 È stato rilevato che la dimensione del file di output supera la dimensione massima consentita per un'immagine di programma.  È possibile creare il programma in più DLL.  Per ulteriori informazioni sull'eseguibile di programma \(PE\), vedere l'articolo "Peering Inside the PE: A Tour of the Win32 Portable Executable File Format" in [MSDN Library](http://go.microsoft.com/fwlink/?linkid=556).
