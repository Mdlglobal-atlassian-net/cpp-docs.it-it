# <a name="linker-tools-error-lnk1248"></a>Errore degli strumenti del linker LNK1248

> dimensioni dell'immagine ('*dimensioni*') supera la dimensione massima consentita (80000000)

È stato rilevato che le dimensioni del file di output supera la dimensione massima consentita per un'immagine del programma a 32 bit. È possibile creare il programma in più DLL.
