# <a name="resource-compiler-error-rc2152"></a>Errore del compilatore di risorse RC2152
carattere di controllo non valido  
  
 Un carattere di controllo nell'istruzione **ACCELERATORS** non è valido. Un carattere di controllo valido è costituito da una lettera (solo) in seguito a un punto di inserimento (**^**).
