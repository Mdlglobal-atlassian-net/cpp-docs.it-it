# Errore BK1512 di BSCMAKE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nomefile: limite di capacità superato  
  
 Non è possibile utilizzare BSCMAKE per compilare un file di informazioni di visualizzazione poiché il numero di definizioni, riferimenti, moduli o altre informazioni supera il limite consentito.  
  
### Per correggere valutando le seguenti possibili soluzioni  
  
1.  Escludere alcune informazioni utilizzando \/Em, \/Es o \/Ei.  
  
2.  Omettere l'opzione \/Iu.
