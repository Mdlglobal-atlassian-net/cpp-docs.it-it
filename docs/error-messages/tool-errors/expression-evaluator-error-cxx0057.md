# <a name="expression-evaluator-error-cxx0057"></a>Errore dell‘analizzatore di espressioni CXX0057
Nessun codice nel numero di riga  
  
 È stato generato alcun codice per il numero di riga specificato. Non può essere utilizzato come un punto di interruzione.  
  
 Questo errore è identico all'errore CAN0057.
