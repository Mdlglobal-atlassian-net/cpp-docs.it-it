# Errore dell‘analizzatore di espressioni CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nessun codice nel numero di riga  
  
 Non è stato generato codice per il numero di riga specificato,  che non può quindi essere utilizzato come punto di interruzione.  
  
 Questo errore è identico all'errore CAN0057.
