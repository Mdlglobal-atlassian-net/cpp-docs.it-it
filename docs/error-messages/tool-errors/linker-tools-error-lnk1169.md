# Errore degli strumenti del linker LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

rilevato uno o più simboli definiti più volte  
  
 La compilazione non è riuscita a causa di più definizioni di uno o più simboli.  Questo errore è preceduto dall'errore [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 L'opzione [\/FORCE](../../build/reference/force-force-file-output.md) o **\/FORCE:MULTIPLE** ignora questo errore.
