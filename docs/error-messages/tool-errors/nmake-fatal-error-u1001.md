# Errore irreversibile U1001 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

errore di sintassi: carattere "carattere" non valido nella macro  
  
 Il carattere presente nella macro non è una lettera, un numero o un segno di sottolineatura.  
  
 Questo errore può essere causato dall'assenza di un carattere di due punti in un'espansione macro:  
  
```  
syntax error : illegal character '=' in macro  
```
