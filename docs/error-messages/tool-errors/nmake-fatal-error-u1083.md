# Errore irreversibile U1083 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

la macro di destinazione "destinazione" non è definita  
  
 La macro di destinazione specificata è una chiamata a una macro non definita o con valore Null.  Non è possibile elaborare una destinazione non specificata.
