# <a name="expression-evaluator-error-cxx0047"></a>Errore dell‘analizzatore di espressioni CXX0047
elenco di argomenti non corrisponde a una funzione  
  
 Un'espressione chiamata una funzione con un elenco di parametri effettivi non corrispondente elenco di parametri formali di qualsiasi funzione con lo stesso nome definito nel programma.  
  
 Le funzioni in overload possono essere chiamate solo se è presente una corrispondenza esatta di parametro o una corrispondenza che non richiede la costruzione di un oggetto.  
  
 Questo errore è identico all'errore CAN0047.
