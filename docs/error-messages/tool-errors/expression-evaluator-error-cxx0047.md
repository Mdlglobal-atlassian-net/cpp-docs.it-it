# Errore dell‘analizzatore di espressioni CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'elenco di argomenti non corrisponde a una funzione  
  
 Un'espressione ha chiamato una funzione con un elenco di parametri effettivi che non corrisponde all'elenco di parametri formali di nessuna funzione con lo stesso nome definita nel programma.  
  
 Le funzioni in overload possono essere chiamate solo se esiste un'esatta corrispondenza di parametri o una corrispondenza che non richiede la costruzione di un oggetto.  
  
 Questo errore è identico all'errore CAN0047.
