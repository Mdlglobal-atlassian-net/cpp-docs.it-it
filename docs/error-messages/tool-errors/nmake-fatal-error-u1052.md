# Errore irreversibile U1052 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile trovare il file "nomefile"  
  
 NMAKE non trova il file specificato con uno degli elementi seguenti:  
  
-   Un'opzione **\/F**  
  
-   Una direttiva per il preprocessore **\!INCLUDE**  
  
-   Un segno di chiocciola \(**@**\) quale identificatore di un file di risposta  
  
 Verificare che il file esista e che il nome file sia stato digitato correttamente.
