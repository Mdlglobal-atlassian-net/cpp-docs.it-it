# Errore degli strumenti del linker LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile risolvere uno o più simboli non decorati  
  
 Questo errore è preceduto da un avviso [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) per ciascun simbolo non decorato che non è possibile risolvere e da almeno due avvisi [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) per i simboli duplicati trovati per il simbolo non decorato.
