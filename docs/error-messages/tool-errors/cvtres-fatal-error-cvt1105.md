# Errore irreversibile CVT1105 di CVTRES
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile spostarsi nel file  
  
 CVTRES non è in grado di raggiungere una posizione nel file.  
  
 Questo errore può essere dovuto al danneggiamento del file.
