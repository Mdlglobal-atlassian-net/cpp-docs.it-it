# <a name="linker-tools-error-lnk1282"></a>Errore degli strumenti del linker LNK1282
Impossibile eseguire il file /REBASE; è stato firmato  
  
 Si è tentato di modificare l'indirizzo di base di un assembly firmato con l'opzione /REBASE per [editbin](../../build/reference/editbin-reference.md). A tale scopo, modificare innanzitutto l'indirizzo di base e quindi firmare l'assembly.
