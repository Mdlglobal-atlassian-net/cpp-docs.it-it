# Errore irreversibile del compilatore di risorse RC1047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Troppe opzioni opzione, "stringa".  
  
 L'opzione indicata è stata specificata troppe volte.  La stringa indicata è l'argomento dell'opzione che ha causato l'errore.
