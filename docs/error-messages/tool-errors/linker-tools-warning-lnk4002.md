# Avviso degli strumenti del linker LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

simbolo definito in oggetto  
  
 Il simbolo, visualizzato nel formato decorato, è stato specificato nel formato non decorato in `object`, ma non è stata rilevata una corrispondenza univoca con un simbolo decorato.  Questo avviso è sempre preceduto dall'avviso [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) e seguito dall'errore irreversibile [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
