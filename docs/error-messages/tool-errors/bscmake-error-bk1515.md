# <a name="bscmake-error-bk1515"></a>Errore BK1515 di BSCMAKE

FileBSC: versione incompatibile: non è possibile aggiornare in modo incrementale

File BSC è stato creato con un'altra versione di BSCMAKE. Ricreare il file BSC.
