# <a name="project-build-error-prj0026"></a>Errore di compilazione progetto PRJ0026
File di risposta 'file' contenuto Unicode che non può essere tradotto in una tabella codici ANSI dell'utente.  
  
 ***Contenuto UNICODE del file***  
  
 Il sistema del progetto trovata Unicode contenuto in un file di risposta che non è possibile convertire correttamente nella tabella codici ANSI corrente dell'utente.  
  
 La risoluzione di questo errore consiste nell'aggiornare il contenuto del file di risposta per utilizzare ANSI o installare la tabella codici nel computer in uso e impostarlo come predefinito del sistema.
