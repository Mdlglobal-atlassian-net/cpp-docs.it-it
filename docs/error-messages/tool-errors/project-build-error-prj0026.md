# Errore di compilazione progetto PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile convertire il contenuto Unicode del file di risposta "file" nella tabella codici ANSI dell'utente.  
  
 ***Contenuto UNICODE del file***  
  
 Il sistema del progetto ha trovato in un file di risposta contenuto Unicode che non Ã¨ possibile convertire correttamente nella tabella codici ANSI corrente dell'utente.  
  
 Per correggere l'errore, aggiornare il contenuto del file di risposta in modo da utilizzare ANSI oppure installare la tabella codici nel computer in uso e impostarla come predefinita per il sistema.
