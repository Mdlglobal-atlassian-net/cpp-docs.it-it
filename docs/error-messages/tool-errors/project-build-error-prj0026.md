# <a name="project-build-error-prj0026"></a>Errore di compilazione progetto PRJ0026

> File di risposta '*file*' il contenuto Unicode che non può essere tradotto in tabella codici ANSI dell'utente.
>
> *Contenuto UNICODE del file*

Il sistema di progetto trovato contenuto Unicode in un file di risposta che non può essere convertito correttamente alla tabella codici ANSI corrente dell'utente.

La risoluzione di questo errore consiste nell'aggiornare il contenuto del file di risposta per usare ANSI o per installare la tabella codici nel computer e impostarlo come impostazione predefinita del sistema.
