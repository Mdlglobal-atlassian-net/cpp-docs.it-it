# <a name="linker-tools-warning-lnk4065"></a>Avviso degli strumenti del linker LNK4065
'function' non può essere ordinato; ignorato  
  
 La funzione specificata non è stata compilata come una funzione di package. Ricompilare con [/Gy](../../build/reference/gy-enable-function-level-linking.md).
