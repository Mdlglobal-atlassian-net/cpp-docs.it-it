# <a name="project-build-error-prj0019"></a>Errore di compilazione progetto PRJ0019
Uno strumento ha restituito un codice di errore da  
  
 Un livello di errore è diverso da zero per un passaggio di compilazione personalizzata o un evento di compilazione.  
  
 PRJ0019 si verifica anche quando uno strumento ha restituito un codice di errore, ma nessun messaggio di errore. Può accadere, ad esempio, se si reindirizza l'output di MIDL NUL.  
  
 Vedere [risoluzione dei problemi di istruzioni di compilazione personalizzate e sugli eventi di compilazione](../../ide/troubleshooting-build-customizations.md) per ulteriori informazioni.  
  
 Questo errore può verificarsi anche quando si eseguono come membro del gruppo Users ed è necessario l'accesso amministrativo. Per ulteriori informazioni, vedere [in esecuzione come membro del gruppo di utenti](../../security/running-as-a-member-of-the-users-group.md).
