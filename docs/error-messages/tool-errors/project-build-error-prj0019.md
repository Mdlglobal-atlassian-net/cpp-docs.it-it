# Errore di compilazione progetto PRJ0019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Uno strumento ha restituito un codice di errore da  
  
 Un livello di errore è diverso da zero per un'istruzione di compilazione personalizzata o un evento di compilazione.  
  
 Il messaggio di errore PRJ0019 viene anche visualizzato quando uno strumento restituisce un codice di errore senza messaggio di errore.  Questa eventualità può verificarsi, ad esempio, se si reindirizza l'output di MIDL a NUL.  
  
 Per ulteriori informazioni, vedere [Risoluzione dei problemi relativi alle istruzioni di compilazione personalizzate e agli eventi di compilazione](../../ide/troubleshooting-build-customizations.md).  
  
 Questo errore può verificarsi anche quando l'esecuzione avviene come membro del gruppo Users ed è necessario l'accesso amministrativo.  Per ulteriori informazioni, vedere [Esecuzione come membro del gruppo Users](../../top/running-as-a-member-of-the-users-group.md).
