# Errore dell‘analizzatore di espressioni CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Espressione non espandibile.  
  
 È stato rilevato un errore interno durante il calcolo dell'espressione.  
  
 Si consiglia di scrivere un'espressione equivalente che possa essere valutata in modo corretto.  
  
 Questo errore è identico all'errore CAN0031.
