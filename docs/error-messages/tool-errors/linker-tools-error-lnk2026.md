# Errore degli strumenti del linker LNK2026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

modulo unsafe per immagine SAFESEH  
  
 È stato specificato [\/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) ma un modulo non è compatibile con la funzionalità di gestione sicura delle eccezioni.  Se si desidera utilizzare questo modulo con **\/SAFESEH**, è necessario ricompilare il modulo utilizzando il compilatore di Visual C\+\+ .NET 2003 o versione successiva.
