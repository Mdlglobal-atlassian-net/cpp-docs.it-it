# <a name="expression-evaluator-error-cxx0008"></a>Errore dell‘analizzatore di espressioni CXX0008
**mancante "alla fine della stringa**  
  
 Le virgolette doppie prevista alla fine della stringa letterale è mancante. Digitare di nuovo l'espressione, racchiudendo la stringa tra virgolette doppie.  
  
 Questo errore è identico a CAN0008.
