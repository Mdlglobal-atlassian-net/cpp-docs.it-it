# Errore dell‘analizzatore di espressioni CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**virgolette doppie \("\) mancanti alla fine della stringa**  
  
 Un segno di virgolette doppie previsto alla fine della stringa letterale è mancante.  Digitare nuovamente l'espressione, racchiudendo la stringa letterale tra una coppia di virgolette doppie.  
  
 Questo errore è identico all'errore CAN0001.
