# <a name="bscmake-error-bk1508"></a>Errore BK1508 di BSCMAKE
Impossibile eliminare il file temporaneo 'filename' [: motivo]  
  
 BSCMAKE non è possibile eliminare uno dei file temporanei.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili  
  
1.  Il file di un altro processo è aperto.  
  
2.  Un errore hardware.
