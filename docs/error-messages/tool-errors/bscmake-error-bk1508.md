# Errore BK1508 di BSCMAKE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile eliminare il file temporaneo "nomefile" \[: motivo\]  
  
 Uno dei file temporanei di BSCMAKE non può essere eliminato.  
  
### Possibili cause  
  
1.  Il file è già aperto da un altro processo.  
  
2.  Errore hardware.
