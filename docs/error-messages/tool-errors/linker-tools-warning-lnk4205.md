# <a name="linker-tools-warning-lnk4205"></a>Avviso degli strumenti del linker LNK4205

'filename' mancano informazioni di debug correnti per fare riferimento a tale modulo. oggetto collegato senza informazioni di debug

Il file con estensione PDB contiene informazioni non aggiornate. Il linker continuerà collegare oggetti senza informazioni di debug. È possibile ricompilare il file oggetto usando il [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) opzione.
