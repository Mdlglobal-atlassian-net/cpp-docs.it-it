# Avviso degli strumenti del linker LNK4205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

in 'nomefile' mancano informazioni di debug correnti per il modulo di riferimento; oggetto collegato senza informazioni di debug  
  
 Il file PDB contiene informazioni non aggiornate.  Il collegamento dell'oggetto continuerà senza informazioni di debug.  Può essere utile ricompilare il file oggetto con l'opzione [\/Zi](../../build/reference/z7-zi-zi-debug-information-format.md).
