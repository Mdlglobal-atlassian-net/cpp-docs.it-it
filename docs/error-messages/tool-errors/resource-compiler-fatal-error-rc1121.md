# Errore irreversibile del compilatore di risorse RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore di I\/O durante la lettura del file.  
  
 Non è possibile leggere un file con il compilatore di risorse.  Verificare che l'unità contenente il file sia disponibile e che il file sia valido.
