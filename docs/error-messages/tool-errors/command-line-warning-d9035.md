# Avviso della riga di comando D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'opzione 'opzione' è deprecata e verrà rimossa in una futura versione  
  
 È stata specificata un'opzione del compilatore che verrà rimossa in una versione successiva del compilatore Visual C\+\+.  Questo avviso è seguito dall'avviso [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) se è disponibile una sostituzione suggerita per *option*.  
  
 Benché l'opzione specificata sia tuttora funzionante, è consigliabile aggiornare la configurazione della build.  Di conseguenza, è più probabile che la compilazione del progetto continui quando si aggiorna il compilatore.
