# Errore degli strumenti del linker LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

file danneggiato o non valido  
  
 Il file di input ha un'intestazione danneggiata oppure ha dimensioni pari a zero o eccessivamente piccole.  
  
 L'errore LNK1136 può inoltre verificarsi se il file non è stato specificato in modo corretto come dipendenza aggiuntiva.
