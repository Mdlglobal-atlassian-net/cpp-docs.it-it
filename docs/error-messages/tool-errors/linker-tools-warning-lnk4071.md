# Avviso degli strumenti del linker LNK4071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile eseguire un collegamento incrementale su collegamenti successivi  
  
 Sono state rilevate più definizioni per uno o più simboli, ma è stato utilizzato [\/FORCE](../../build/reference/force-force-file-output.md) o **\/FORCE:MULTIPLE** per creare un file di output indipendentemente dagli errori.  Il file di stato incrementale \(ilk\) è stato eliminato da LINK.
