# Errore di compilazione progetto PRJ0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile convertire il contenuto Unicode del file temporaneo "file" nella tabella codici ANSI dell'utente.  
  
 Con l'opzione del linker [\/MIDL \(Specifica opzioni della riga di comando MIDL\)](../../build/reference/midl-specify-midl-command-line-options.md) è stato specificato un valore che non è possibile risolvere mediante la tabella codici del sistema.  
  
 La tabella codici utilizzata quando è stato specificato il comando MIDL \(tabella codici di input\) deve corrispondere alla tabella codici del sistema.
