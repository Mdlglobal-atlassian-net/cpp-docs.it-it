# <a name="nmake-fatal-error-u1023"></a>Errore irreversibile U1023 di NMAKE 
Errore di sintassi nell'espressione  
  
 Un'espressione non è valida.  
  
 Verificare la validità degli operatori e precedenza degli operatori.
