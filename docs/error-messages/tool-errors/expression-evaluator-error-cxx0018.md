# Errore dell‘analizzatore di espressioni CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

nome di registro non valido  
  
 Un registro specificato non esiste o non può essere visualizzato.  
  
 Nella finestra Espressioni di controllo possono essere visualizzati i seguenti registri:  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 Questo errore è identico all'errore CAN0018.
