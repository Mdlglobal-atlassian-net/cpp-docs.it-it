# <a name="expression-evaluator-error-cxx0018"></a>Errore dell‘analizzatore di espressioni CXX0018
nome di registro non valido  
  
 Un registro specificato non esiste o non può essere visualizzato.  
  
 Finestra Espressioni di controllo è possibile visualizzare i seguenti registri:  
  
 IP DI AX SP DI DOMINIO ACTIVE DIRECTORY  
  
 BX BP ES FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP ES EFL  
  
 ECX ESI ADFS  
  
 EDX EDI CS  
  
 Questo errore è identico all'errore CAN0018.
