# <a name="expression-evaluator-error-cxx0016"></a>Errore dell‘analizzatore di espressioni CXX0016

costante troppo grande

L'analizzatore di espressioni C non può accettare una costante integer senza segno maggiore 4.294.967.295 (0FFFFFFFF esadecimale) o una costante a virgola mobile a cui ordine di grandezza è superiore a circa 1.8 e + 308.

Questo errore è identico all'errore CAN0016.
