# Errore dell‘analizzatore di espressioni CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

costante troppo grande  
  
 L'analizzatore di espressioni di C non accetta costanti integer senza segno maggiori di 4.294.967.295 \(0FFFFFFFF esadecimale\) o costanti a virgola mobile maggiori approssimativamente di 1,8E\+308.  
  
 Questo errore è identico all'errore CAN0016.
