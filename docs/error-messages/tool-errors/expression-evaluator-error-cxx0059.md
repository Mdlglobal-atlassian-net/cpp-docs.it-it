# <a name="expression-evaluator-error-cxx0059"></a>Errore dell‘analizzatore di espressioni CXX0059
operando sinistro è una classe non è un nome di funzione  
  
 L'operando sinistro di una chiamata di funzione è un nome di classe e non può essere risolto in una chiamata di funzione. Omettere il nome di una funzione membro in un'espressione può causare questo errore.  
  
 Questo errore è identico all'errore CAN0059.
