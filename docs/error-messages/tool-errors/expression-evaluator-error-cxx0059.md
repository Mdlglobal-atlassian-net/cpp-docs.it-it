# Errore dell‘analizzatore di espressioni CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'operando di sinistra è una classe e non un nome di funzione  
  
 L'operando di sinistra di una chiamata di funzione è un nome di classe e non può essere risolto in una chiamata di funzione.  L'errore può essere causato dall'omissione di una funzione membro in un'espressione.  
  
 Questo errore è identico all'errore CAN0059.
