# Errore dell‘analizzatore di espressioni CXX0025
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'operatore richiede struttura\/unione.  
  
 A un'espressione non di tipo `struct` o **unione** è stato applicato un operatore che accetta un'espressione di tipo `struct` o **unione**.  
  
 I componenti di variabili di classe, struttura o unione devono avere un nome completo.  Tali componenti possono essere immessi solo se specificati in modo completo.  
  
 Questo errore è identico all'errore CAN0025.
