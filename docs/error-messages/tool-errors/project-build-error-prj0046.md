# Errore di compilazione progetto PRJ0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Riga di comando specificata vuota. Impossibile generare.  
  
 È stata specificata una riga di comando vuota per una configurazione di makefile; la riga di comando è obbligatoria.
