# Errore dell‘analizzatore di espressioni CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

membro statico non presente  
  
 Un membro statico di una classe non è stato trovato o non è stato definito.  L'errore può essere dovuto a un membro di classe statico che è stato dichiarato ma non definito o che è stato definito e utilizzato in moduli che non contengono informazioni di debug.  
  
 Questo errore è identico all'errore CAN0066.
