# Errore irreversibile del compilatore di risorse RW1023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore di I\/O durante la scrittura del file. Unità piena.  
  
 La quantità di spazio disponibile deve essere almeno doppia rispetto a quella del file eseguibile che si intende creare.
