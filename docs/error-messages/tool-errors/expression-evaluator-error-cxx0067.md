# Errore dell‘analizzatore di espressioni CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

valutazione della funzione non supportata  
  
 Nell'espressione è contenuta una chiamata di funzione.  In alcuni analizzatori di funzioni non sono supportate le chiamate di funzione.  
  
 Questo errore è identico all'errore CAN0067.
