# Errore R6019 del linguaggio C in fase di esecuzione 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile aprire la periferica di console.  
  
 Il programma ha chiamato una funzione di console dichiarata in CONIO.h, ma il sistema operativo non consente l'accesso alla console.
