# <a name="nmake-fatal-error-u1086"></a>Errore irreversibile U1086 di NMAKE 

**regola di inferenza non può avere dipendenti**

I due punti (**:**) in un'inferenza regola deve essere seguita da uno dei seguenti:

- Carattere di nuova riga

- Punto e virgola (**;**), che può essere seguito da un comando

- Simbolo di cancelletto (**#**), che può essere seguito da un commento
