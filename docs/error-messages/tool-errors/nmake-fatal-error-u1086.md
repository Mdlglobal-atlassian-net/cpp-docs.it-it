# Errore irreversibile U1086 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**la regola di inferenza non può avere dipendenti**  
  
 Il segno di due punti \(**:**\) all'interno di una regola di inferenza può essere seguito solo da:  
  
-   Un carattere di nuova riga  
  
-   Un segno di punto e virgola \(**;**\), che può essere seguito da un comando  
  
-   Un segno di cancelletto \(**\#**\), che può essere seguito da un commento
