# <a name="resource-compiler-fatal-error-rw1030"></a>Errore irreversibile del compilatore di risorse RW1030
Errore di output  
  
 Questo errore può essere causato se il compilatore di risorse non è in grado di creare il file di destinazione. Assicurarsi che vi sia spazio su disco sufficiente e di disporre delle autorizzazioni di scrittura nel volume.
