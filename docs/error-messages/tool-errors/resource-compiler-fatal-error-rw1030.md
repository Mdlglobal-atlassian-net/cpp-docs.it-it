# Errore irreversibile del compilatore di risorse RW1030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore di output.  
  
 Questo errore può verificarsi se non è possibile creare il file di destinazione con il compilatore di risorse.  Verificare che lo spazio su disco sia sufficiente e che si disponga dell'autorizzazione di scrittura sul volume.
