# <a name="command-line-warning-d9024"></a>Avviso della riga di comando D9024
origine non riconosciuto file di tipo 'nomefile', utilizzato file oggetto  
  
 L'estensione del file specificato non è stato riconosciuto. Il file è stato considerato come un file oggetto e passato al linker.  
  
 Vengono riconosciute le seguenti estensioni:  
  
-   c (file di origine C)  
  
-   cxx (file di origine C++)  
  
-   . cpp (file di origine C++)  
  
-   obj (file oggetto)  
  
-   LIB (file di libreria)  
  
-   DEF (file di definizione moduli)  
  
-   EXP (linker Esporta file creato da LINK /LIB)
