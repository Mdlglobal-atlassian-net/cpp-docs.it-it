# Avviso della riga di comando D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

tipo di file di origine 'nomefile', non riconosciuto, utilizzato file oggetto  
  
 L'estensione del file specificato non è stata riconosciuta.  Il file è stato considerato come un file oggetto e passato al linker.  
  
 Vengono riconosciute le opzioni seguenti:  
  
-   c \(file di origine C\)  
  
-   cxx \(file di origine C\+\+\)  
  
-   cpp \(file di origine C\+\+\)  
  
-   obj \(file oggetto\)  
  
-   lib \(file di libreria\)  
  
-   def \(file di definizione moduli\)  
  
-   exp \(file di esportazione del linker creato da LINK \/LIB\)
