# <a name="project-build-error-prj0044"></a>Errore di compilazione progetto PRJ0044
La proprietà "Dipendenze aggiuntive" per la regola di compilazione personalizzata 'rule' assegnata al file 'file' non è valida. La proprietà contiene 'string' che restituisce 'value'.  
  
 Il **dipendenze aggiuntive** proprietà valutato in una stringa vuota o su una stringa che contiene caratteri non validi (caratteri che non può essere un nome file o directory). L'output dell'azione di compilazione è necessaria per le regole di compilazione personalizzate.  
  
 Per ulteriori informazioni, vedere [strumenti di compilazione personalizzato specificando](../../ide/specifying-custom-build-tools.md).
