# Errore di compilazione progetto PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La proprietà 'Dipendenze aggiuntive' della regola di compilazione personalizzata 'regola' assegnata al file 'file' non è valida. La proprietà contiene 'stringa' che restituisce 'valore'.  
  
 La proprietà **Dipendenze aggiuntive** ha restituito una stringa vuota oppure una stringa contenente caratteri non validi \(caratteri non ammessi in un nome di file o di directory\).  Per le regole di compilazione personalizzate è necessario l'output dell'operazione di compilazione.  
  
 Per ulteriori informazioni, vedere [Specifica di strumenti di compilazione personalizzata](../../ide/specifying-custom-build-tools.md).
