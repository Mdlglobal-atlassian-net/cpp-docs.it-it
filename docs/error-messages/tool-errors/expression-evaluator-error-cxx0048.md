# Errore dell‘analizzatore di espressioni CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sequenza di chiamata non supportata  
  
 In una funzione specificata nell'espressione viene utilizzata una sequenza di chiamata non supportata dall'analizzatore di espressioni di C.  Non è possibile chiamare la funzione in un'espressione della finestra Espressioni di controllo.  
  
 Questo errore è identico all'errore CAN0048.
