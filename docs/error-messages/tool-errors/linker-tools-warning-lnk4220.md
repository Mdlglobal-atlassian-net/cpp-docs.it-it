# Avviso degli strumenti del linker LNK4220
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

valore di 'opzione linker' non valido; verrà utilizzato il valore predefinito  
  
 È stato specificato un valore esterno all'intervallo con l'opzione [\/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md).  Il valore predefinito di **\/TLBID** è 1.
