# <a name="linker-tools-warning-lnk4220"></a>Avviso degli strumenti del linker LNK4220
valido 'opzione del linker' valore 'value'; impostazione predefinita  
  
 È stato specificato un valore di out-of-range con il [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) opzione. Il valore predefinito per **/TLBID** è 1.
