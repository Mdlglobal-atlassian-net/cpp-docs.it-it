# Errore matematico M6205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"funzione": errore \_TLOSS  
  
 Si è verificata una perdita totale di significato \(precisione\).  
  
 È possibile che l'errore sia stato causato dall'aver specificato un numero molto grande come operando di seno, coseno o tangente dato che l'operando deve essere ridotto a un numero compreso tra 0 e 2\*pi greco.
