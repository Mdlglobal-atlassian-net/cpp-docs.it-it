# Errore dell‘analizzatore di espressioni CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**tipi incompatibili con l'operatore**  
  
 I tipi di operando specificati non sono validi per l'operazione.  
  
 Un puntatore, ad esempio, non può essere moltiplicato per qualsiasi valore.  
  
 Si consiglia di eseguire il cast di tipo sugli operandi nel tipo compatibile con l'operatore.
