# Errore dell‘analizzatore di espressioni CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

divisione per 0  
  
 L'espressione contiene un divisore per zero, non consentito.  Il divisore può essere il numero zero letterale oppure un'espressione che dà come risultato zero.  
  
 Questo errore è identico all'errore CAN0032.
