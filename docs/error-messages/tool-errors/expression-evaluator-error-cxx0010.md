# Errore dell‘analizzatore di espressioni CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

parentesi quadra di sinistra mancante  
  
 L'espressione contiene parentesi quadre non corrette.  Digitare nuovamente l'espressione con l'esatta corrispondenza di parentesi quadre di apertura e di chiusura.  
  
 Questo errore è identico all'errore CAN0010.
