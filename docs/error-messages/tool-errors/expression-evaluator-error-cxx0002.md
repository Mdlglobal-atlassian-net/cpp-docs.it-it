# <a name="expression-evaluator-error-cxx0002"></a>Errore dell‘analizzatore di espressioni CXX0002

accesso alla memoria di utente errori

L'espressione tenta di fare riferimento a memoria non allocata per il programma sottoposto a debug.

Questo errore è identico all'errore CAN0002.
