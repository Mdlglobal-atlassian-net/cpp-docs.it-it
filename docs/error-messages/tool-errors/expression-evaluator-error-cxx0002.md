# Errore dell‘analizzatore di espressioni CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

errore nell'accesso alla memoria dell'utente  
  
 L'espressione tenta di fare riferimento a memoria non allocata al programma in corso di debug.  
  
 Questo errore è identico all'errore CAN0002.
