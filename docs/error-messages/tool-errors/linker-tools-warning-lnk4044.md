# Avviso degli strumenti del linker LNK4044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opzione 'opzione' non riconosciuta e ignorata  
  
 L'opzione specificata non è valida per questo strumento,  pertanto verrà ignorata.  
  
 Per visualizzare un elenco delle opzioni valide, eseguire lo strumento senza argomenti né opzioni.
