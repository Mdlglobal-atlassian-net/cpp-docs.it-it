# Errore irreversibile del compilatore di risorse RC1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Errore nella creazione del nome di risorsa.  
  
 Impossibile creare il file RES specificato.  Assicurarsi che non si stia tentando di creare il file su un'unità in sola lettura.  Utilizzare l'opzione \/V per verificare se è in corso la creazione del file.
