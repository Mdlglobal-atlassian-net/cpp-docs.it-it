# Avviso degli strumenti del linker LNK4010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il numero di versione numero del sottosistema non è valido. Verrà utilizzata la versione predefinita del sottosistema  
  
 È possibile specificare una versione per il sottosistema dell'immagine \([\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)\).  Ciascun sottosistema ha un requisito di versione minima.  Se la versione specificata è inferiore alla minima, verrà visualizzato questo avviso e utilizzato il sottosistema predefinito.
