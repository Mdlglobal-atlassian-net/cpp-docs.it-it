# <a name="c-runtime-errors-rxxxx"></a>Errori di runtime di C (Rxxxx)

La libreria di Runtime C (CRT) può segnalare un errore di runtime quando l'app viene caricata o è in esecuzione. Anche se ogni messaggio fa riferimento a Microsoft Visual C++ libreria di runtime, non significa che c'è un bug nella libreria. Questi errori indicano un bug nel codice dell'app o una condizione che non può gestire la libreria di runtime, ad esempio memoria insufficiente. Gli utenti finali dell'app possono incontrare questi errori, a meno che la scrittura di app per evitarli, o per acquisire gli errori e presentare un errore descrittivo del messaggio agli utenti invece.
