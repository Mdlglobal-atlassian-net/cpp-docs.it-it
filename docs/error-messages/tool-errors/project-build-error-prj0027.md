# Errore di compilazione progetto PRJ0027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Impossibile convertire il contenuto del messaggio di registro Unicode "contenuto" nella tabella codici ANSI dell'utente.  
  
 Questo avviso viene in genere visualizzato solo congiuntamente a errori relativi alla creazione di file batch e\/o di risposta.  
  
 Per correggere l'errore, aggiornare il contenuto del log di compilazione in modo da utilizzare ANSI oppure installare la tabella codici nel computer in uso e impostarla come predefinita per il sistema.
