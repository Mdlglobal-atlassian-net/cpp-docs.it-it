# Errore irreversibile CVT1103 di CVTRES
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile leggere il file "nomefile"  
  
 Si è verificato un errore irreversibile durante il tentativo di lettura del file specificato con CVTRES.  
  
 Questo errore può essere dovuto al danneggiamento del file.
