# <a name="cvtres-fatal-error-cvt1103"></a>Errore irreversibile CVT1103 di CVTRES
Impossibile leggere il nome di file  
  
 Si è verificato un errore irreversibile durante il tentativo di leggere il file specificato con CVTRES.  
  
 Questo errore può essere causato dal danneggiamento dei file.
