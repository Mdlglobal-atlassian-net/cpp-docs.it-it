# Errore irreversibile del compilatore di risorse RC1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

espressione costante integer non valida  
  
 L'espressione all'interno di una direttiva `#if` non esiste o non identifica una costante.
