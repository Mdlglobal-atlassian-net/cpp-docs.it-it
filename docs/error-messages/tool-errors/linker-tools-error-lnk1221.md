# Errore degli strumenti del linker LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile inferire un sottosistema: deve essere definito  
  
 Le informazioni disponibili non sono sufficienti per dedurre il sottosistema di destinazione.  
  
 Per correggere questo errore, utilizzare [\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md).
