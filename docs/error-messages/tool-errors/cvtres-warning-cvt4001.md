# <a name="cvtres-warning-cvt4001"></a>Avviso CVT4001 di CVTRES
**tipo di computer non è specificato. si presuppone che**   
 ***type***  
  
 CVTRES non è riuscito a trovare una specifica del computer. Si presuppone il tipo di computer specificato. Se il valore predefinito è corretto, eseguire nuovamente CVTRES utilizzando questa opzione.
