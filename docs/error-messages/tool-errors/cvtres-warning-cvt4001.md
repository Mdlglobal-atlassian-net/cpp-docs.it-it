# Avviso CVT4001 di CVTRES
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**tipo di computer non specificato; verrà utilizzato il**   
 ***tipo***  
  
 Non è stata rilevata alcuna specifica del tipo di computer da CVTRES.  Verrà quindi utilizzato il tipo di computer specificato.  Se il tipo predefinito non è corretto, eseguire nuovamente CVTRES utilizzando l'opzione \/MACHINE.
