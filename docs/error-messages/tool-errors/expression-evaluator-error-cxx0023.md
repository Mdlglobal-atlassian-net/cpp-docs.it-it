# Errore dell‘analizzatore di espressioni CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

radice non valida  
  
 L'analizzatore di espressioni di C non riconosce la radice specificata.  Sono ammesse solo radici decimali, esadecimali e ottali.  
  
 Questo errore è identico all'errore CAN0023.
