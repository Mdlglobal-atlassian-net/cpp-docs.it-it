# <a name="expression-evaluator-error-cxx0023"></a>Errore dell‘analizzatore di espressioni CXX0023
radice non valida  
  
 L'analizzatore di espressioni C non riconosce la radice specificata. Solo radici decimali, esadecimali e ottali sono validi.  
  
 Questo errore è identico all'errore CAN0023.
