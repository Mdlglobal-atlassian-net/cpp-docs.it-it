# <a name="expression-evaluator-error-cxx0023"></a>Errore dell‘analizzatore di espressioni CXX0023

radice non valida

L'analizzatore di espressioni C non riconosce la radice specificata. Solo le radici decimali, ottali ed esadecimali sono valide.

Questo errore è identico all'errore CAN0023.
