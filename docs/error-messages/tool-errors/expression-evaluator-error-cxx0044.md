# <a name="expression-evaluator-error-cxx0044"></a>Errore dell‘analizzatore di espressioni CXX0044
l'utilizzo del puntatore _based(void) richiede l'operatore :>  
  
 Un puntatore basato su `void` non può essere utilizzata direttamente. È necessario creare un puntatore completo utilizzando il **: >** operatore.  
  
 Questo errore è identico all'errore CAN0044.
