# Errore dell‘analizzatore di espressioni CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'utilizzo del puntatore \_based\(void\) richiede l'operatore :\>  
  
 Un puntatore basato su `void` non può essere utilizzato direttamente.  È necessario creare un puntatore completo utilizzando l'operatore **:\>**.  
  
 Questo errore è identico all'errore CAN0044.
