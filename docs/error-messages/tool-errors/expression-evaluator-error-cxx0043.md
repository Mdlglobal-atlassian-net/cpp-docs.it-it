# Errore dell‘analizzatore di espressioni CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Puntatore utilizzato all'esterno della funzione membro.  
  
 Il puntatore **this** può essere utilizzato solo per funzioni membro non statiche.  
  
 Questo errore è identico all'errore CAN0043.
