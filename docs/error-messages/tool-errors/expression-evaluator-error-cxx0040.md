# <a name="expression-evaluator-error-cxx0040"></a>Errore dell‘analizzatore di espressioni CXX0040
funzione richiede una conversione implicita  
  
 L'analizzatore di espressioni C non consente conversioni implicite con chiamate al costruttore.  
  
 Questo errore è identico all'errore CAN0040.
