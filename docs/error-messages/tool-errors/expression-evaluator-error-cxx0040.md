# Errore dell‘analizzatore di espressioni CXX0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La funzione richiede una conversione implicita.  
  
 Nell'analizzatore di espressioni di C non sono supportate conversioni implicite che implicano chiamate ai costruttori.  
  
 Questo errore è identico all'errore CAN0040.
