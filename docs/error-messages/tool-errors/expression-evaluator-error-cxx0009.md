# Errore dell‘analizzatore di espressioni CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

virgolette singole \('\) mancanti dopo la costante carattere  
  
 Il segno di virgolette singole previsto alla fine della costante carattere è mancante.  Digitare nuovamente l'espressione, racchiudendo la costante carattere tra una coppia di virgolette singole.  
  
 Questo errore è identico all'errore CAN0009.
