# <a name="nmake-warning-u4001"></a>Avviso U4001 di NMAKE

> il file di comando può essere richiamato solo dalla riga di comando

Un file di comando, che viene richiamato dall'identificatore di chiocciola ( **\@** ), non può contenere una specifica per un altro file di comando. Tale annidamento non è consentito. La specifica è stata ignorata.
