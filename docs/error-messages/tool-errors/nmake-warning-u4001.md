# Avviso U4001 di NMAKE 
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il file di comando può essere chiamato solo dalla riga di comando  
  
 In un file di comando, richiamato utilizzando come identificatore il simbolo di chiocciola \(**@**\), non possono essere contenute specifiche di altri file di comando.  Tale annidamento non è consentito.  La specifica è stata ignorata.
