# <a name="nmake-warning-u4001"></a>Avviso U4001 di NMAKE 
file di comando può essere richiamato solo dalla riga di comando  
  
 Un file di comando richiamato utilizzando il simbolo di chiocciola (**@**) identificatore, non può contenere una specifica di un altro file di comando. Non è consentita la nidificazione. La specifica è stata ignorata.
