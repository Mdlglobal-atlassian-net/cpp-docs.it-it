# Avviso del compilatore di risorse RW4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Opzioni di elaborazione EXE \(\/L \/M \/P \/T \/K \/E \/31 o \/30\).  
  
 Sono state specificate opzioni di elaborazione EXE senza che vi siano file eseguibili da elaborare.  Non utilizzare queste opzioni con file RES.
