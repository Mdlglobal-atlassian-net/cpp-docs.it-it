# <a name="resource-compiler-fatal-error-rc1208"></a>Errore irreversibile del compilatore di risorse RC1208
file di input. Estensione RES  
  
 Il file con estensione Estensione RES viene utilizzato per l'output del compilatore di risorse. Il file con estensione Estensione RC deve essere utilizzato per gli script del compilatore di risorse.
