# Errore dell‘analizzatore di espressioni CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

chiamate ai costruttori non supportate  
  
 È stata eseguita una chiamata a un costruttore da un'espressione.  Non è consentito eseguire chiamate esplicite ai costruttori o conversioni che richiedono una chiamata a un costruttore dalle espressioni.  
  
 Questo errore è identico all'errore CAN0062.
