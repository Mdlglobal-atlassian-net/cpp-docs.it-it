# <a name="linker-tools-warning-lnk4073"></a>Avviso degli strumenti del linker LNK4073
non è possibile creare mapping per il file ilk; il collegamento in modo non incrementale  
  
 Non è un grande spazio sufficiente contiguo in memoria condivisa per il collegamento creare il file di stato incrementale (ilk). È stata eseguita una compilazione non incrementale.
