# Avviso degli strumenti del linker LNK4073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

impossibile creare la mappa per il file ILK; collegamento eseguito in modo non incrementale  
  
 Lo spazio contiguo nella memoria condivisa non è sufficiente per creare il file di stato incrementale \(ilk\) con LINK.  È stata eseguita una compilazione non incrementale.
