# Errori e avvisi relativi alla compilazione di progetti (PRJxxxx)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

In questa sezione viene fornito un elenco di riferimento degli errori generati dagli strumenti di compilazione.  Per visualizzare le informazioni della Guida relative a un determinato messaggio di errore, fare clic sul corrispondente numero di errore nella finestra di output e premere F1 oppure digitare il numero dell'errore nella casella **Cerca** dell'Indice.
