# Errore irreversibile del compilatore di risorse RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

limite del compilatore: macro eccessivamente annidate "macro"  
  
 Il file supera il limite del compilatore di risorse per l'annidamento di macro.  Modificare il file di origine specificato in modo da diminuire il livello di annidamento delle relative macro.
