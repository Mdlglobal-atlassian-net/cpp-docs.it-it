# <a name="expression-evaluator-error-cxx0041"></a>Errore dell‘analizzatore di espressioni CXX0041
elemento della classe deve essere un membro statico o una funzione  
  
 Un membro non statico di una classe o struttura o unione, è stato utilizzato senza specificare quali la creazione di istanze della classe da utilizzare.  
  
 Solo i membri dati statici o funzioni membro possono essere utilizzate senza specificare un'istanza.  
  
 Questo errore è identico all'errore CAN0041.
