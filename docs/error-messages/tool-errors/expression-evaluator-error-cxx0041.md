# Errore dell‘analizzatore di espressioni CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'elemento della classe deve essere un membro di tipo statico o una funzione  
  
 È stato utilizzato un membro non statico di una classe \(o di una struttura o unione\) senza che sia stata specificata la creazione di istanza da utilizzare.  
  
 Solo le funzioni membro o i membri dati statici possono essere utilizzati senza specificare una creazione di istanza.  
  
 Questo errore è identico all'errore CAN0041.
