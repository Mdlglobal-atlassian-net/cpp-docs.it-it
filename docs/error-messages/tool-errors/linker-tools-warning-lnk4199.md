# <a name="linker-tools-warning-lnk4199"></a>Avviso degli strumenti del linker LNK4199
/DELAYLOAD: nomedll ignorata; Nessuna importazione trovata da nomedll  
  
 Ignorato dal linker `dllname` perché non è necessaria una delle funzioni che `dllname` Esporta.
