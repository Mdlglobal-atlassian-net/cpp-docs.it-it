# Avviso della riga di comando D9040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

l'opzione '\/analyze' verrà ignorata; gli avvisi dell'analisi codice non sono disponibili in questa edizione del compilatore  
  
 L'opzione della riga di comando **\/analyze** non è disponibile in tutte le edizioni di Visual Studio.  Per porre rimedio a questo avviso, passare ad una versione supportata di Visual Studio o rimuovere l'opzione della riga di comando.
