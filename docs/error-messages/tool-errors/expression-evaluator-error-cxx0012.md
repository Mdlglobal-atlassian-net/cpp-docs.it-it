# Errore dell‘analizzatore di espressioni CXX0012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

parentesi graffa di sinistra mancante  
  
 L'espressione contiene una parentesi graffa non corretta.  Digitare nuovamente l'espressione con l'esatta corrispondenza di parentesi graffe di apertura e di chiusura.  
  
 Questo errore è identico all'errore CAN0012.
