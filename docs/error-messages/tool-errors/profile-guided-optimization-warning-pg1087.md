# Avviso di ottimizzazione PGO PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

File .PGC 'file' non valido.  
  
 Un file pgc è danneggiato, probabilmente perché la sessione di profilatura è terminata in modo anomalo.  Eseguire di nuovo lo scenario di profilatura e terminare normalmente.
