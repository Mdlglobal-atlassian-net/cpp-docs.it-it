# <a name="profile-guided-optimization-warning-pg1087"></a>Avviso di ottimizzazione PGO PG1087

Il file PGC 'file' non è valido.

Un file pgc è danneggiato, probabilmente a causa di una sessione di profilatura terminata in modo anomalo.  Eseguire di nuovo lo scenario di profilatura e termina normalmente.
