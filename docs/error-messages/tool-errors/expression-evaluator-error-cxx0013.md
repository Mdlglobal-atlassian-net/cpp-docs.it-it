# Errore dell‘analizzatore di espressioni CXX0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operatore mancante  
  
 Un operatore previsto nell'espressione non è stato trovato.  Controllare la sintassi dell'espressione.  
  
 Questo errore è identico all'errore CAN0013.
