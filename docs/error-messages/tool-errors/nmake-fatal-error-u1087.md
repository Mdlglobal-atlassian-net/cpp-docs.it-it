# <a name="nmake-fatal-error-u1087"></a>Errore irreversibile U1087 di NMAKE 
non può avere: e:: dipendenti per la stessa destinazione  
  
 Una destinazione non può essere specificata in entrambi un singolo nome senza due punti (**:**) e un doppio carattere due punti (`::`) dipendenza.  
  
 Per specificare una destinazione in più blocchi di descrizione, utilizzare `::` in ogni riga della dipendenza.
