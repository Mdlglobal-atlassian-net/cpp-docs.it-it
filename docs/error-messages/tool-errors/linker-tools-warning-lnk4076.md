# <a name="linker-tools-warning-lnk4076"></a>Avviso degli strumenti del linker LNK4076

il file di stato incrementale ' filename ' non è valido; collegamento non incrementale

Il collegamento non è in grado di scrivere nel file di stato incrementale (. ilk). Il `filename` è danneggiato o non è un database di collegamento incrementale. Rimuovere il file e ricollegarlo.
