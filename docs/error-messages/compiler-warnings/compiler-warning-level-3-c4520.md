# Avviso del compilatore (livello 3) C4520
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe': specificato più di un costruttore predefinito  
  
 Se la classe dispone di più costruttori predefiniti,  viene utilizzato il primo.
