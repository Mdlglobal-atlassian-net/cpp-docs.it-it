# Avviso del compilatore (livello 1) C4329
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_\_declspec\(align\(\)\) ignorato su enum  
  
 Non Ã¨ possibile utilizzare la parola chiave [align](../../cpp/align-cpp.md) del modificatore [\_\_declspec](../../cpp/declspec.md) su un `enum`.  Il seguente codice di esempio genera l'errore C4329:  
  
```  
// C4329.cpp  
// compile with: /W1 /LD  
enum __declspec(align(256)) TestEnum {   // C4329  
   TESTVAL1,  
   TESTVAL2,  
   TESTVAL3  
};  
__declspec(align(256)) enum TestEnum1;  
```
