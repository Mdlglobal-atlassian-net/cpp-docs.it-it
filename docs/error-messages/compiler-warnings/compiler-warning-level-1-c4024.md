# <a name="compiler-warning-level-1-c4024"></a>Compilatore Warning (level 1) C4024

'function': diversi tipi di parametro formali ed effettivi 'number'

I parametri formali ed effettivi corrispondenti hanno tipi diversi. Il compilatore passa il parametro effettivo senza alcuna modifica. La funzione di ricezione converte il tipo di parametro nel tipo previsto.
