# Avviso del compilatore (livello 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': tipi diversi tra il parametro 'numero' formale e quello effettivo  
  
 I parametri formali ed effettivi corrispondenti hanno tipi diversi.  Il compilatore passa il parametro effettivo senza modificarlo.  La funzione ricevente converte il tipo di parametro nel tipo previsto.
