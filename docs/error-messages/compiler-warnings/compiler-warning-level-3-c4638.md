commento al documento XML di destinazione: riferimento al simbolo sconosciuto 'symbol'

 L'esempio seguente genera l'errore C4638:  
  
```  
// C4638.cpp  
// compile with: /clr /doc /LD /W3

public:  
   /// <summary> Text </summary>
```
