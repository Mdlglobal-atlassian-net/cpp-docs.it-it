# Avviso del compilatore (livello 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': tipi differenti per il parametro reale 'numero', parametro formale 'numero'  
  
 I parametri formali ed effettivi corrispondenti hanno tipi diversi.  Il parametro effettivo viene passato senza modifiche.  La funzione effettua il cast del parametro effettivo nel tipo specificato nella definizione della funzione.
