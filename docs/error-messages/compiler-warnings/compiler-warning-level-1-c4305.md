# Avviso del compilatore (livello 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': troncamento da 'tipo1' a 'tipo2'  
  
 L'identificatore viene convertito in un tipo più piccolo, con conseguente perdita di informazioni.  
  
 Se è stata utilizzata la Creazione guidata punto di connessione ATL, vedere l'articolo della Knowledge Base indicato di seguito \(informazioni in lingua inglese\):  
  
-   Q250847: ATL Connection Point Wizard Generated Code for Event with VARIANT Argument Gives C4305 Warning
