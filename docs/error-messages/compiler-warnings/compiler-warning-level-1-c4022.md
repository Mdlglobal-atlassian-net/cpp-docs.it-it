# Avviso del compilatore (livello 1) C4022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': errore di corrispondenza del puntatore per il parametro effettivo 'numero'  
  
 Il tipo di puntatore del parametro effettivo si differenzia dal tipo di puntatore del parametro formale corrispondente.  Il parametro effettivo viene passato senza modifiche.
