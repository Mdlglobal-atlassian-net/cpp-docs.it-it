# Avviso del compilatore (livello 3) C4013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione' non definita. Verrà considerata come esterna che restituisce un int  
  
 È stata rilevata una chiamata a una funzione non definita.  
  
### Possibili cause  
  
1.  Ortografia errata del nome della funzione.  
  
2.  Funzioni esterne senza prototipo `extern`
