# <a name="compiler-warning-level-3-c4013"></a>Compilatore avviso (livello 3) C4013

'function' non definita. verrà considerata come esterna che restituisce un int

Il compilatore ha rilevato una chiamata a una funzione non definita.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Per risolverlo è possibile verificare le seguenti cause possibili

1. Errori di ortografia del nome di funzione

1. Funzioni esterne non prototipo `extern`
