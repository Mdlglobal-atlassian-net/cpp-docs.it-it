# Avviso del compilatore (livello 1) C4553
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operatore': l'operatore non ha effetto. Si intendeva 'operatore'?  
  
 Se nella parte iniziale di un'istruzione di espressione Ã¨ presente un operatore senza effetti collaterali, si tratta probabilmente di un errore.  
  
 Il seguente codice di esempio genera l'errore C4553:  
  
```  
// C4553.cpp  
// compile with: /W1  
int func()  
{  
   return 0;  
}  
  
int main()  
{  
   int i;  
   i == func();   // C4553  
   // try the following line instead  
   // i = func();  
}  
```
