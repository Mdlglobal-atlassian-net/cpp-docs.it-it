# <a name="compiler-warning-level-1-c4144"></a>Compilatore Warning (level 1) C4144

'expression': espressione relazionale come espressione switch

L'espressione relazionale specificata Ã¨ stata utilizzata come espressione di controllo di un [commutatore](../../cpp/switch-statement-cpp.md) istruzione. Le istruzioni case associate verranno offerto valori booleani. L'esempio seguente genera l'errore C4144:

```
// C4144.cpp
// compile with: /W1
int main()
{
   int i = 0;
   switch(!i) {   // C4144, remove the ! to resolve
      case 1:
         break;
      default:
         break;
   }
}
```
