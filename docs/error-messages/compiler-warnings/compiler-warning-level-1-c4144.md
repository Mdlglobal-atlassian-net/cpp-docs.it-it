# Avviso del compilatore (livello 1) C4144
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'espressione': espressione relazionale come espressione switch  
  
 L'espressione relazionale specificata Ã¨ stata utilizzata come espressione di controllo di un'istruzione [switch](../../cpp/switch-statement-cpp.md).  Alle istruzioni case associate verranno forniti valori booleani.  Il seguente codice di esempio genera l'errore C4144:  
  
```  
// C4144.cpp  
// compile with: /W1  
int main()  
{  
   int i = 0;  
   switch(!i) {   // C4144, remove the ! to resolve  
      case 1:  
         break;  
      default:  
         break;  
   }  
}  
```
