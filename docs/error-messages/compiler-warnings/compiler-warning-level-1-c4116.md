# <a name="compiler-warning-level-1-c4116"></a>Compilatore Warning (level 1) C4116

definizione di tipo senza nome tra parentesi

Una struttura, unione o un tipo enumerato senza nome è definito in un'espressione tra parentesi. La definizione del tipo non è significativo.

In una chiamata di funzione C, la definizione ha ambito globale. In una chiamata di funzione C++, la definizione ha lo stesso ambito della funzione chiamata.
