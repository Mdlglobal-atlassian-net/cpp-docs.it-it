# <a name="compiler-warning-level-1-c4027"></a>Avviso del compilatore (livello 1) C4027
funzione dichiarata senza elenco di parametri formali  
  
 La dichiarazione di funzione non presenta parametri formali, ma esistono parametri formali nella definizione di funzione o parametri effettivi in una chiamata. Le chiamate successive a questa funzione presuppongono che la funzione accetti parametri effettivi dei tipi rilevati nella definizione di funzione o nella chiamata.
