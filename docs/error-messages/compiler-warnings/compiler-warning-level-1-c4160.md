# Avviso del compilatore (livello 1) C4160
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**\#pragma**   
 ***pragma* \(pop,...\): non ha trovato l'identificatore '**   
 ***identifier* ' precedentemente inserito**  
  
 Un'istruzione pragma nel codice sorgente tenta di estrarre un identificatore che non Ã¨ stato inserito. Per evitare questo avviso, assicurarsi che l'identificatore estratto sia stato inserito correttamente.  
  
 L'esempio seguente genera l'errore C4160:  
  
```  
// C4160.cpp // compile with: /W1 #pragma pack(push) #pragma pack(pop, id)   // C4160 // use identifier when pushing to resolve the warning // #pragma pack(push, id) int main() { }  
```
