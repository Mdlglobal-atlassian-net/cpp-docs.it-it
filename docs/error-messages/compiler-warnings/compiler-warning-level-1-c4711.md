# <a name="compiler-warning-level-1-c4711"></a>Avviso del compilatore (livello 1) C4711
funzione "funzione" selezionata per l'espansione inline  
  
 Il compilatore ha eseguito l'incorporamento della funzione specificata, anche se non è stato contrassegnato per l'incorporamento.  
  
 C4711 viene attivato se [/Ob2](../../build/reference/ob-inline-function-expansion.md) specificato.  
  
 Inline viene eseguita a discrezione del compilatore. Si tratta di un avviso informativo.  
  
 Per impostazione predefinita, questo avviso non è attivo. Per abilitare un avviso, utilizzare [#pragma avviso](../../preprocessor/warning.md). Per altre informazioni, vedere [Avvisi del compilatore disattivati per impostazione predefinita](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
