# Avviso del compilatore (livello 1) C4711
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

funzione "funzione" selezionata per l'espansione inline  
  
 La funzione specificata è stata resa inline, anche se non era contrassegnata per tale operazione.  
  
 L'avviso C4711 viene attivato se si specifica [\/Ob2](../../build/reference/ob-inline-function-expansion.md).  
  
 L'operazione di inline viene eseguita a discrezione del compilatore.  L'avviso è informativo.  
  
 Per impostazione predefinita, questo avviso non è attivo.  Per attivarlo, utilizzare [\#pragma warning](../../preprocessor/warning.md).  Per ulteriori informazioni, vedere [Avvisi del compilatore disattivati per impostazione predefinita](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
