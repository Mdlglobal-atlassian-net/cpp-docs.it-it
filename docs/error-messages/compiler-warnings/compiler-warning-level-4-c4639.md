# <a name="compiler-warning-level-4-c4639"></a>Avviso del compilatore (livello 4) C4639
Errore MSXML, documento XML non verranno elaborati i commenti. motivo  
  
 Questo avviso può verificarsi per diversi motivi.  
  
 Per risolvere il problema:  
  
-   Ricompilare.  
  
-   Reinstallare MSXML reinstallando common language runtime.  
  
-   Modificare o rimuovere il commento di documentazione che ha causato l'avviso e ricompilare.  
  
 Quando viene rilasciato C4639, elaborazione di tutte le ulteriori commenti XML è disabilitato e non viene generato il file xdc.
