# <a name="compiler-warning-level-4-c4639"></a>Avviso del compilatore (livello 4) C4639

Errore MSXML, documento XML non verranno elaborati i commenti. motivo

Questo avviso può verificarsi per diversi motivi.

Per risolvere il problema:

- RECOMPILE.

- Reinstallare MSXML, provare a reinstallare common language runtime.

- Modificare o rimuovere il commento relativo alla documentazione che ha causato l'avviso e recompile.

Quando viene emessa C4639, elaborazione di tutte le ulteriori commenti XML è disabilitato e non viene generato il file con estensione xdc.
