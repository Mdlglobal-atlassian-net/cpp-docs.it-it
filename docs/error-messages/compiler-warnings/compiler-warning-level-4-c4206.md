# Avviso del compilatore (livello 4) C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**utilizzata estensione non standard: unità di conversione vuota**  
  
 Dopo la pre\-elaborazione, il file è vuoto.  
  
 È possibile che l'estensione impedisca la portabilità del codice in altri compilatori.  Tale estensione, inoltre, genera un errore in compatibilità ANSI \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) e può essere applicata solo al codice sorgente C.
