# <a name="compiler-warning-level-1-c4098"></a>Compilatore (livello 1) Avviso C4098
'function': funzione void restituisce un valore  
  
 Una funzione dichiarata con tipo restituito [void](../../cpp/void-cpp.md) ha un `return` istruzione che restituisce un valore. Il compilatore presuppone che la funzione restituisce un valore di tipo `int`.
