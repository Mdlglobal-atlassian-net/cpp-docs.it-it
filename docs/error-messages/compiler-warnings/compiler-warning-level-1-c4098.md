# Avviso del compilatore (livello 1) C4098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': la funzione 'void' restituisce un valore  
  
 Una funzione dichiarata con tipo restituito [void](../../cpp/void-cpp.md) ha un'istruzione `return` che restituisce un valore.  Si presume che la funzione restituisca un valore di tipo `int`.
