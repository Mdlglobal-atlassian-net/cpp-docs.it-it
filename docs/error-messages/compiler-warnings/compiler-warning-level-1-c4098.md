# <a name="compiler-warning-level-1-c4098"></a>Compilatore Warning (level 1) C4098

'function': funzione void restituisce un valore

Una funzione dichiarata con tipo restituito [void](../../cpp/void-cpp.md) ha una `return` istruzione che restituisce un valore. Il compilatore presuppone che la funzione restituisce un valore di tipo `int`.
