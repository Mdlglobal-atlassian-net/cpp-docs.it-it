# <a name="compiler-warning-level-1-c4674"></a>Avviso del compilatore (livello 1) C4674

La firma di un operatore di conversione non era corretta. Il metodo non Ã¨ considerato una conversione definita dall'utente. Per altre informazioni sulla definizione degli operatori, vedere [operatori definiti dall'utente (C++/CLI)](../../dotnet/user-defined-operators-cpp-cli.md) e [conversioni definite dall'utente (C++/CLI)](../../dotnet/user-defined-conversions-cpp-cli.md).
