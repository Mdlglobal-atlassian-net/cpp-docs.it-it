# <a name="compiler-warning-level-1-c4029"></a>Compilatore Warning (level 1) C4029

elenco di parametri formali dichiarati diversa dalla definizione

I tipi di parametro formale nella dichiarazione di funzione non concordano con quelli nella definizione di funzione. Il compilatore Usa l'elenco dei parametri dalla definizione.
