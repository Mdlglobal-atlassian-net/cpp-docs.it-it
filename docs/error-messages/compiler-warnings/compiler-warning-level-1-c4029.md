# <a name="compiler-warning-level-1-c4029"></a>Compilatore (livello 1) Avviso C4029
elenco di parametri formali dichiarati diversa dalla definizione  
  
 Tipi di parametro formale nella dichiarazione di funzione non concordano con quelli nella definizione di funzione. Il compilatore utilizza l'elenco di parametri dalla definizione.
