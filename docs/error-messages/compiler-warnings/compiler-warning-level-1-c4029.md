# Avviso del compilatore (livello 1) C4029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

dichiarato un elenco di parametri formali diverso dalla definizione  
  
 I tipi di parametro formale presenti nella dichiarazione di funzione non concordano con quelli presenti nella definizione di funzione.  Viene utilizzato l'elenco di parametri della definizione.
