# Avviso del compilatore (livello 1) C4652
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opzione del compilatore 'opzione' non coerente con l'intestazione precompilata. L'opzione della riga di comando corrente avrà priorità su quanto definito nell'intestazione precompilata  
  
 L'opzione della riga di comando specificata è diversa da quella definita al momento della creazione dell'intestazione precompilata \(PCH\).  Viene utilizzata l'opzione specificata nella riga di comando corrente.  
  
 Per evitare la visualizzazione dell'avviso, rigenerare l'intestazione precompilata con l'opzione della riga di comando specificata.
