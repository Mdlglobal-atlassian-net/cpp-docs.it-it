# Avviso del compilatore (livello 1) C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione': errata corrispondenza del puntatore nel parametro reale 'numero', parametro formale 'numero'  
  
 I parametri formali ed effettivi corrispondenti hanno un livello diverso di riferimento indiretto.  Il parametro effettivo viene passato senza modifiche.  La funzione chiamata interpreta il valore come puntatore.
