# <a name="compiler-warning-level-1-c4088"></a>Compilatore Warning (level 1) C4088

'function': mancata corrispondenza del puntatore nel parametro effettivo 'number', parametro formale 'number'

I parametri formali ed effettivi corrispondenti hanno un livello diverso di riferimento indiretto. Il parametro effettivo viene passato senza alcuna modifica. La funzione chiamata interpreta il valore come un puntatore.
