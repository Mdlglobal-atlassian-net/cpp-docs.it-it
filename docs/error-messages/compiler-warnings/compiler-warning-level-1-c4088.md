# <a name="compiler-warning-level-1-c4088"></a>Compilatore (livello 1) Avviso C4088
'function': errata corrispondenza del puntatore nel parametro effettivo 'number', parametro formale 'number'  
  
 I parametri formali ed effettivi corrispondenti hanno un diverso livello di riferimento indiretto. Il parametro effettivo viene passato senza modifiche. La funzione chiamata interpreta il valore come un puntatore.
