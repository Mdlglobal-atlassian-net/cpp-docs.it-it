# Avviso del compilatore (livello 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**convenzione di chiamata non valida per un costruttore\/distruttore**  
  
 I costruttori e i distruttori non possono avere convenzioni di chiamata diverse da quelle predefinite per la piattaforma \(tranne quando si specifica esplicitamente **\_\_clrcall**\).
