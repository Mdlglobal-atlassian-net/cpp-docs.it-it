# <a name="compiler-warning-level-3-c4511"></a>Avviso del compilatore (livello 3) C4511
'class': Impossibile generare un costruttore di copia  
  
 Il compilatore non è stato possibile generare un costruttore di copia predefinito per una classe. una classe base può avere un costruttore di copia privato.
