# Avviso del compilatore (livello 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe': impossibile generare un costruttore di copia  
  
 Non è stato possibile generare un costruttore di copia predefinito per una classe. È possibile che una classe base presenti un costruttore di copia privato.
