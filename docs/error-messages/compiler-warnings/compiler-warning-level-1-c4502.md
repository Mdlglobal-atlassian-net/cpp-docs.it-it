# <a name="compiler-warning-level-1-c4502"></a>Avviso del compilatore (livello 1) C4502

'le specifiche di collegamento' richiede l'uso della parola chiave 'extern' e devono precedere tutti gli altri identificatori

È stato specificato un collegamento senza il `extern` (parola chiave). Un collegamento non è rilevante per i tipi non extern.

Il compilatore presuppone che il `extern` (parola chiave).
