# <a name="compiler-warning-level-3-c4554"></a>Avviso del compilatore (livello 3) C4554
'operator': controllare la precedenza tra operatori eventuali errori; Utilizzare le parentesi per chiarire la precedenza  
  
 L'esempio seguente genera l'errore C4554:  
  
```  
// C4554.cpp  
// compile with: /W3 /WX  
int main() {  
   int a = 0, b = 0, c = 0;  
   a = a << b + c;   // C4554  
   // probably intended (a << b) + c,  
   // but will get a << (b + c)  
}  
```
