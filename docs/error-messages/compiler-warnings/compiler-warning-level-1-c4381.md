# Avviso del compilatore (livello 1) C4381
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'funzione1': il metodo di interfaccia non sarà implementato dal metodo non pubblico 'funzione2'  
  
 Le classi devono implementare tutte le funzioni in un'interfaccia.  Una classe è in grado di soddisfare questa condizione se una delle relative classi base implementa la funzione.  È tuttavia necessario che venga implementata come funzione pubblica.
