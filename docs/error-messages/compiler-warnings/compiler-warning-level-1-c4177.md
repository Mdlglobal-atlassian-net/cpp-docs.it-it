# <a name="compiler-warning-level-1-c4177"></a>Avviso del compilatore (livello 1) C4177
\#il pragma pragma deve essere in ambito globale  
  
 Il [pragma](../../preprocessor/pragma-directives-and-the-pragma-keyword.md) pragma non deve essere utilizzato in ambito locale. Il pragma **pragma** non sar√† valido fino a quando non viene rilevato l'ambito globale dopo l'ambito corrente.  
  
 L'esempio seguente genera l'errore C4177:  
  
