# Avviso del compilatore (livello 4) C4513
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'classe': impossibile generare un distruttore  
  
 Non è possibile generare un distruttore predefinito per la classe specificata. Non è stato creato alcun distruttore.  Il distruttore si trova in una classe base non accessibile alla classe derivata.  Se la classe base ha un distruttore privato, renderlo pubblico o protetto.
