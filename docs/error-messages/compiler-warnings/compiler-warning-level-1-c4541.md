# Avviso del compilatore (livello 1) C4541
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore' utilizzato sul tipo polimorfico 'tipo' con \/GR\-. Possibili risultati imprevisti  
  
 Si è tentato di utilizzare una funzionalità che richiede informazioni sul tipo in fase di esecuzione senza attivare tali informazioni.  Effettuare nuovamente la compilazione con [\/GR](../../build/reference/gr-enable-run-time-type-information.md).
