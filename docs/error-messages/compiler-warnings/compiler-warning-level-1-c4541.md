# <a name="compiler-warning-level-1-c4541"></a>Avviso del compilatore (livello 1) C4541
'identifier' usato sul tipo polimorfico 'type' con /GR-. potrebbe verificarsi un comportamento imprevedibile  
  
 Si è tentato di utilizzare una funzionalità che richiede informazioni sul tipo in fase di esecuzione senza abilitare le informazioni sul tipo in fase di esecuzione. Ricompilare con [/GR](../../build/reference/gr-enable-run-time-type-information.md).
