# <a name="compiler-warning-level-1-c4340"></a>Avviso del compilatore (livello 1) C4340
'value': valore trasformato da positivo a negativo  
  
 Il `enum` valore è maggiore del più grande `enum` valore positivo trasformato in un valore negativo.
