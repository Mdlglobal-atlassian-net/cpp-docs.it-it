# Avviso del compilatore (livello 1) C4340
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'valore': valore trasformato da positivo a negativo  
  
 Il valore `enum` è maggiore del valore `enum` positivo più elevato trasformato in valore negativo.
