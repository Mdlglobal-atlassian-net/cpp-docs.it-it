# Avviso del compilatore (livello 1) C4403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

operatore PTR non valido  
  
 Un operatore PTR viene utilizzato in modo errato nel codice assembler inline.  
  
 L'operatore PTR viene ignorato.
