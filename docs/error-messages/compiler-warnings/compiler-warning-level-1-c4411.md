# Avviso del compilatore (livello 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificatore': il simbolo punta al registro di spostamento  
  
 L'identificatore è un simbolo locale che punta al registro di spostamento, pertanto può essere utilizzato su un operando con un altro simbolo.
