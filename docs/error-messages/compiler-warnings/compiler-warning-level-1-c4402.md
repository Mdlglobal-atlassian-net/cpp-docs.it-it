# Avviso del compilatore (livello 1) C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

utilizzare l'operatore PTR  
  
 Un tipo viene utilizzato su un operando senza un operatore PTR per il riferimento o il cast a un tipo nel codice assembly inline.  
  
 Viene utilizzato l'operatore PTR.
