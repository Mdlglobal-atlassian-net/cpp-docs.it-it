# Avviso del compilatore (livello 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

La versione 'versione' di Common Language Runtime non è supportata da questo compilatore.L'utilizzo di questa versione potrebbe provocare risultati imprevisti.  
  
 Nel computer è installata una versione precedente di Common Language Runtime e non la versione corrente.  Per risolvere il problema, installare la versione di Common Language Runtime fornita con il compilatore.
