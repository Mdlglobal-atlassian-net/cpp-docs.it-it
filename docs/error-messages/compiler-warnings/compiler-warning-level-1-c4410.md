# Avviso del compilatore (livello 1) C4410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

dimensione operando non valida  
  
 Uno degli operandi nell'istruzione ha una dimensione errata.  È stata utilizzata la più piccola dimensione valida per l'operando.
