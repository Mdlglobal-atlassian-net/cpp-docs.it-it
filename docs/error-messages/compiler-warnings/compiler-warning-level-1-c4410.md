# <a name="compiler-warning-level-1-c4410"></a>Avviso del compilatore (livello 1) C4410
dimensione operando non valida  
  
 Uno degli operandi sull'istruzione ha una dimensione non corretta. La dimensione più piccola valida per l'operando è stata usata.
