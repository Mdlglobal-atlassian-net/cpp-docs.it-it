# <a name="compiler-warning-level-1-c4679"></a>Avviso del compilatore (livello 1) C4679

> «*membro*': Impossibile importare il membro

Il compilatore ha rilevato un costrutto supportato, che non può essere importato dai metadati.

Non provare a utilizzare il costrutto.
