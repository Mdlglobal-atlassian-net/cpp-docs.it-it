# <a name="compiler-warning-level-4-c4536"></a>Avviso del compilatore (livello 4) C4536
'type name': nome del tipo supera il limite di metadati di '% d' caratteri  
  
 Se si trattasse di un tipo gestito, un nome di tipo verrà troncato nei metadati. Vedere [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) per ulteriori informazioni.  
  
 Per impostazione predefinita, questo avviso non è attivo. Per altre informazioni, vedere [Avvisi del compilatore disattivati per impostazione predefinita](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
