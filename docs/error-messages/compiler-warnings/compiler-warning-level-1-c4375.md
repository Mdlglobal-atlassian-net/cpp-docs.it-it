# Avviso del compilatore (livello 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

il metodo non pubblico 'metodo2' non esegue l'override di 'metodo1'  
  
 Un tipo che implementa un altro tipo ha definito un metodo di override, ma l'override non è pubblico.  Il metodo non eseguirà quindi l'override del metodo del tipo di base.
