# Avviso del compilatore (livello 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

modifica del comportamento: l'utilizzo di argomenti di template espliciti provoca la chiamata di 'funzione'  
  
 Una chiamata a una funzione che utilizza argomenti di template espliciti chiama una funzione diversa da quella chiamata se non fossero specificati argomenti espliciti.
