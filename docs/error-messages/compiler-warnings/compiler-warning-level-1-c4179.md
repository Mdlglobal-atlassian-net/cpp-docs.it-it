# <a name="compiler-warning-level-1-c4179"></a>Avviso del compilatore (livello 1) C4179

> '//*': analizzato come '/' è/\*': confusione con i commenti '//' standard

__//\*__ è un delimitatore di commento non corretto. Usare __//__ o __/\*__ .
