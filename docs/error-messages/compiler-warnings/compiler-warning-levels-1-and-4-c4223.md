# <a name="compiler-warning-levels-1-and-4-c4223"></a>Avviso del compilatore (livelli 1 e 4) C4223

utilizzata estensione non standard: matrice non lvalue convertita in puntatore

Nel linguaggio C standard, è possibile convertire una matrice non lvalue a un puntatore. Con le estensioni Microsoft predefinito ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), è possibile.
