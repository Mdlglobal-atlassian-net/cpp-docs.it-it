# Avviso del compilatore (livelli 1 e 4) C4223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

utilizzata estensione non standard: matrice non lvalue convertita in puntatore  
  
 Nel linguaggio C standard non è possibile convertire una matrice non lvalue in un puntatore.  Questa operazione è invece possibile con le estensioni Microsoft predefinite \([\/Ze](../../build/reference/za-ze-disable-language-extensions.md)\).
