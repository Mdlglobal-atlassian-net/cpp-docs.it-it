# <a name="compiler-warning-level-4-c4709"></a>Avviso del compilatore (livello 4) C4709
operatore virgola all'interno di espressione di indice di matrice  
  
 Quando Ã¨ presente una virgola in un'espressione di indice di matrice, il compilatore utilizza il valore dopo l'ultima virgola.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore C4709:  
  
```  
// C4709.cpp  
// compile with: /W4  
#include <stdio.h>  
  
int main()   
{  
    int arr[2][2];  
    arr[0][0] = 10;  
    arr[0][1] = 11;  
  
    // Prints 10, not 11  
    printf_s("\n%d",arr[0][1,0]);   // C4709  
}  
```
