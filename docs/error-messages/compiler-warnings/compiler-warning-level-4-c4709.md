operatore virgola all'interno di un'espressione che definisce l'indice di una matrice  
  
 Quando in un'espressione che definisce l'indice di una matrice Ã¨ presente una virgola, viene utilizzato il valore successivo all'ultima virgola.  
  
## Esempio  
 Il seguente codice di esempio genera l'errore C4709:  
  
```  
// C4709.cpp  
// compile with: /W4  
#include <stdio.h>  
  
int main()   
{  
    int arr[2][2];  
    arr[0][0] = 10;  
    arr[0][1] = 11;  
  
    // Prints 10, not 11  
    printf_s("\n%d",arr[0][1,0]);   // C4709  
}  
```
