# <a name="compiler-warning-level-3-c4622"></a>Avviso del compilatore (livello 3) C4622
Sovrascrittura delle informazioni di debug generate durante la creazione dell'intestazione precompilata nel file oggetto: 'file'  
  
 Informazioni di CodeView nel file specificato sono andate perse durante la compilazione con il [/Yu](../../build/reference/yu-use-precompiled-header-file.md) (Usa intestazione precompilata).  
  
 Rinominare il file oggetto (usando [/Fo](../../build/reference/fo-object-file-name.md)) durante la creazione o Usa intestazione precompilata file e collegare usando il nuovo file oggetto.
