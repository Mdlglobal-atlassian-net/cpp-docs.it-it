# <a name="compiler-warning-level-4-c4932"></a>Avviso del compilatore (livello 4) C4932
__identifier(Identifier) e \__identifier(identifier) distinguibili.  
  
 Il compilatore non Ã¨ in grado di distinguere tra **finally** e `__finally` o `__try` e **try** come un parametro passato a [Identifier](../../windows/identifier-cpp-cli.md). Non tentare di utilizzare entrambi come identificatori nello stesso programma, come risultato un [C2374](../../error-messages/compiler-errors-1/compiler-error-c2374.md) errore.  
  
 L'esempio seguente genera l'errore C4932:  
  
```  
// C4932.cpp  
// compile with: /clr /W4 /WX  
int main() {  
   int __identifier(_finally) = 245;   // C4932  
   int __identifier(__finally) = 25;   // C4932  
}  
```
