# Avviso del compilatore (livello 4) C4245
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'conversione': conversione da 'tipo1' a 'tipo2'. Errore di corrispondenza tra signed e unsigned  
  
 Si Ã¨ tentato di convertire una variabile **const** signed con valore negativo in `unsigned`.  
  
 Il seguente codice di esempio genera l'errore C4245:  
  
```  
// C4245.cpp  
// compile with: /W4 /c  
const int i = -1;  
unsigned int j = i; // C4245  
  
const int k = 1;  
unsigned int l = k; // okay  
  
int m = -1;  
unsigned int n = m; // okay  
  
void Test(size_t i) {}  
  
int main() {  
   Test( -19 );   // C4245  
}  
```
