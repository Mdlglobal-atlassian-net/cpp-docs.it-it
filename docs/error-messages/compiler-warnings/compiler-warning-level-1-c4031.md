# Avviso del compilatore (livello 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

secondo elenco di parametri formali più lungo del primo elenco  
  
 Una funzione è ridichiarata con parametri formali diversi.  Vengono utilizzati i parametri formali forniti nella prima dichiarazione.
