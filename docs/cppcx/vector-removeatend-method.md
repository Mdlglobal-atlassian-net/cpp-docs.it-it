# Metodo Vector::RemoveAtEnd
Elimina l'elemento alla fine dell'oggetto Vector corrente.  
  
## Sintassi  
  
```  
  
virtual void RemoveAtEnd();  
```  
  
## Requisiti  
 **Intestazione:** collection.h  
  
 **Spazio dei nomi:** Platform::Collections
