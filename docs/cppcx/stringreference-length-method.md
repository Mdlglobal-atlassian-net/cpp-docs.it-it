# Metodo StringReference::Length
Restituisce il numero di caratteri della stringa.  
  
## Sintassi  
  
```cpp  
unsigned int Length() const  
```  
  
## Valore restituito  
 Intero senza segno che specifica il numero di caratteri nella stringa..
