# Metodo StringReference::Data
Restituisce i contenuti di questo `StringReference` come matrice di valori char16.  
  
## Sintassi  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## Valore restituito  
 Matrice di caratteri di testo UNICODE char16.
