# Metodo Vector::Clear
Elimina tutti gli elementi nell'oggetto Vector corrente.  
  
## Sintassi  
  
```  
  
virtual void Clear();  
```  
  
## Requisiti  
 **Intestazione:** collection.h  
  
 **Spazio dei nomi:** Platform::Collections
