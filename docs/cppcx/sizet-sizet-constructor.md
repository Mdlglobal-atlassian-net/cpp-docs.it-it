# SizeT::SizeT (costruttore)
Inizializza una nuova istanza di SizeT con il valore specificato.  
  
## Sintassi  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## Parametri  
 value1  
 Valore a 32 bit Unsigned.  
  
 value2  
 Puntatore a un valore a 32 bit Unsigned.
