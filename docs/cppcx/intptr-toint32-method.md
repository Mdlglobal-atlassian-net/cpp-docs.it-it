# IntPtr::ToInt32 (metodo)
Converte il valore dell'elemento IntPtr corrente in un Integer a 32 bit.  
  
## Sintassi  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## Valore restituito  
 Integer a 32 bit.
