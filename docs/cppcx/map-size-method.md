# Metodo Map::Size
Restituisce il numero di elementi [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) in Map.  
  
## Sintassi  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Valore restituito  
 Numero di elementi in Map.  
  
## Requisiti  
 **Intestazione:** collection.h  
  
 **Spazio dei nomi:** Platform::Collections
