# Metodo StringReference::GetHSTRING
Restituisce il contenuto della stringa come `__abi_HSTRING`.  
  
## Sintassi  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## Valore restituito  
 Oggetto `__abi_HSTRING` contenente i dati in formato stringa.
