# <a name="platformtypecode-enumeration"></a>Platform::TypeCode (enumerazione)
Specifica una categoria numerica che rappresenta un tipo incorporato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
enum class TypeCode {};  
```

|UInt8|Tipo default::uint8.|
