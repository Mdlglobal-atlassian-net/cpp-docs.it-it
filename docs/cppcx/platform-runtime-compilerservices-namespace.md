# Platform::Runtime::CompilerServices (spazio dei nomi)
Questo spazio dei nomi è solo per uso interno e non deve essere utilizzato per lo sviluppo.  
  
## Sintassi  
  
```cpp  
  
namespace Platform {  
   namespace CompilerServices{  
}}  
```
