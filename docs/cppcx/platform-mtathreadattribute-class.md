# Platform::MTAThreadAttribute (classe)
Indica che il modello di threading per un'applicazione è un apartment a thread multipli \(MTA\).  
  
## Sintassi  
  
```  
public ref class MTAThreadAttribute sealed : Attribute  
```  
  
## Membri  
  
### Costruttori pubblici  
  
|Nome|Descrizione|  
|----------|-----------------|  
|Costruttore [Costruttore MTAThreadAttribute](../cppcx/mtathreadattribute-constructor-1.md)|Inizializza una nuova istanza della classe.|  
  
### Metodi pubblici  
 L'attributo MTAThreadAttribute eredita da [Classe Platform::Object](../cppcx/platform-object-class.md). MTAThreadAttribute esegue anche l'overload o contiene i membri seguenti:  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[MTAThreadAttribute::Equals](../cppcx/mtathreadattribute-equals.md)|Determina se l'oggetto specificato è uguale all'oggetto corrente.|  
|[MTAThreadAttribute::GetHashCode](../cppcx/mtathreadattribute-gethashcode.md)|Restituisce il codice hash per l'istanza.|  
|[MTAThreadAttribute::ToString](../cppcx/mtathreadattribute-tostring.md)|Restituisce una stringa che rappresenta l'oggetto corrente.|
