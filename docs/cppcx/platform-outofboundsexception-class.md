# Classe Platform::OutOfBoundsException
Generata quando un'operazione tenta di accedere a dati memorizzati al di fuori dell'intervallo valido.  
  
## Sintassi  
  
```cpp  
public ref class OutOfBoundsException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## Note  
 Per ulteriori informazioni, vedi la classe [COMException](../cppcx/platform-comexception-class.md).
