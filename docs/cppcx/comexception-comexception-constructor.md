# COMException::COMException (costruttore)
Inizializza una nuova istanza della classe COMException.  
  
## Sintassi  
  
```cpp  
COMException( int hresult )  
```  
  
#### Parametri  
 HRESULT  
 Errore HRESULT rappresentato dall'eccezione.
