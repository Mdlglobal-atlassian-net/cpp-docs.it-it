# STAThreadAttribute::ToString
Restituisce una stringa che rappresenta l'oggetto corrente.  
  
## Sintassi  
  
```cpp  
public:String^ ToString()  
```  
  
## Valore restituito  
 Stringa che rappresenta l'oggetto corrente.
