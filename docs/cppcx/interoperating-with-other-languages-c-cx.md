# <a name="interoperating-with-other-languages-ccx"></a>Interoperabilità con altri linguaggi (C++/CX)

Questa parte della documentazione descrive come usare C + + c++ /CX per creare componenti di Windows Runtime che possono essere utilizzati da altri linguaggi di programmazione e librerie. È anche possibile modificare i componenti che non possono essere utilizzati da tutti i linguaggi. In questa sezione descrive i diversi aspetti da considerare durante di C + + / applicazione CX interagisce con i componenti scritti con JavaScript, un linguaggio gestito .NET Framework o libreria di modelli di Windows Runtime C++.
