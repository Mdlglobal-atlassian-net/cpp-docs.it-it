# <a name="interoperating-with-other-languages-ccx"></a>Interoperabilità con altri linguaggi (C++/CX)

Questa parte della documentazione descrive come usare C++/CX per creare componenti di Windows Runtime che possono essere utilizzati da altri linguaggi di programmazione e librerie. È anche possibile modificare i componenti che non possono essere utilizzati da tutti i linguaggi. In questa sezione descrive i diversi aspetti da considerare durante la C++applicazione /CX interagisce con i componenti scritti con JavaScript, un linguaggio gestito .NET Framework o il Runtime di Windows C++ libreria di modelli.
