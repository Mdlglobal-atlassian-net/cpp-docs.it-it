# Distruttore Agile::~Agile
Eliminata l'istanza corrente della classe Agile.  
  
## Sintassi  
  
```cpp  
  
~Agile();  
```  
  
## Note  
 Questo distruttore rilascia inoltre l'oggetto rappresentato dall'oggetto Agile corrente.
