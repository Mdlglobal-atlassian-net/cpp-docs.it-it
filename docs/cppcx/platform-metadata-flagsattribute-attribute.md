# Platform::Metadata::FlagsAttribute (attributo)
Indica che un'enumerazione può essere gestita come un campo di bit, ovvero un set di flag.  
  
## Sintassi  
  
```cpp  
  
public ref class Flags abstract : Attribute  
```
