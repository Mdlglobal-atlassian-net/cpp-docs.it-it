# Metodo UnorderedMap::Size
Restituisce il numero di elementi [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) in UnorderedMap.  
  
## Sintassi  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## Valore restituito  
 Numero di elementi nell'oggetto Mappa non ordinata.  
  
## Requisiti  
 **Intestazione:** collection.h  
  
 **Spazio dei nomi:** Platform::Collections
