# Metodo String::Begin
Restituisce un puntatore che punta all'inizio della stringa corrente.  
  
## Sintassi  
  
```cpp  
  
char16* Begin()  
```  
  
## Valore restituito  
 Puntatore all'inizio della stringa corrente.
