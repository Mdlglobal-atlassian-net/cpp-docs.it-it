# Metodo String::IsEmpty
Indica se l'oggetto String corrente è vuoto.  
  
## Sintassi  
  
```cpp  
  
bool IsEmpty()  
```  
  
## Valore restituito  
 `true` se l'oggetto String corrente è `null` o la stringa vuota \(L""\); in caso contrario, `false`.
