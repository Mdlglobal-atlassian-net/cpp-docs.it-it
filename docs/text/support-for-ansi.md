# Supporto per ANSI
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

La maggior parte delle classi e dei metodi MFC supporta il set di caratteri ANSI, anche se il framework MFC nel suo insieme sta gradualmente evolvendo verso il supporto del solo set di caratteri Unicode.  Grazie ai continui miglioramenti di [!INCLUDE[windowsver](../build/reference/includes/windowsver_md.md)] e della versione 6.1 dei controlli comuni di Windows, il supporto di molte classi e metodi ANSI è deprecato.  Per ulteriori informazioni, vedere [API ANSI deprecate](../mfc/deprecated-ansi-apis.md) e [Supporto per Unicode](../text/support-for-unicode.md).
