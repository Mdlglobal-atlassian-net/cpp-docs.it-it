# <a name="fill-stlclr"></a>fill (STL/CLR)
Assegna lo stesso nuovo valore a ogni elemento di un intervallo specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
template<class _FwdIt, class _Ty> inline  
    void fill(_FwdIt _First, _FwdIt _Last, const _Ty% _Val);  
```  
  
## <a name="remarks"></a>Note  
 Questa funzione si comporta come la funzione della libreria Standard C++ `fill`. Per ulteriori informazioni, vedere [riempimento](../standard-library/algorithm-functions.md#fill).  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** \<cliext/algoritmo >  
  
 **Namespace:** cliext
