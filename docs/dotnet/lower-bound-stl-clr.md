# <a name="lowerbound-stlclr"></a>lower_bound (STL/CLR)
Trova la posizione del primo elemento in un intervallo ordinato con un valore inferiore o equivalente a un valore specificato, in cui il criterio di ordinamento pu√≤ essere specificato da un predicato binario.  
  
## <a name="syntax"></a>Sintassi  
  
```  
template<class _FwdIt, class _Ty> inline  
    _FwdIt lower_bound(_FwdIt _First, _FwdIt _Last, const _Ty% _Val);  
template<class _FwdIt, class _Ty, class _Pr> inline  
    _FwdIt lower_bound(_FwdIt _First, _FwdIt _Last,
```  
