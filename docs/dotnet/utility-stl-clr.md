# <a name="utility-stlclr"></a>utility (STL/CLR)
Includere l'intestazione STL/CLR `<cliext/utility>` per definire la classe modello `pair` e diverse funzioni di modello di supporto.  
  
## <a name="syntax"></a>Sintassi  
  
```  
#include <utility>  
```  
  
## <a name="declarations"></a>Dichiarazioni  
  
|Classe|Descrizione|  
|-----------|-----------------|  
|[pair (STL/CLR)](../dotnet/pair-stl-clr.md)|Eseguire il wrapping di una coppia di elementi.|  
  
|Operatore|Descrizione|  
|--------------|-----------------|  
|[operator== (pair) (STL/CLR)](../dotnet/operator-equality-pair-stl-clr.md)|Coppia uguale a confronto.|  
|[operator!= (pair) (STL/CLR)](../dotnet/operator-inequality-pair-stl-clr.md)|Coppia di confronto non è uguale.|  
|[operator< (pair) (STL/CLR)](../dotnet/operator-less-than-pair-stl-clr.md)|Coppia di minore di confronto.|  
|[operatore\<= (coppia) (STL/CLR)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|Coppia minore o uguale confronto.|  
|[operator> (pair) (STL/CLR)](../dotnet/operator-greater-than-pair-stl-clr.md)|Coppia di maggiore di confronto.|  
|[operator>= (pair) (STL/CLR)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|Coppia maggiore o uguale confronto.|  
  
|Funzione|Descrizione|  
|--------------|-----------------|  
|[make_pair (STL/CLR)](../dotnet/make-pair-stl-clr.md)|Rendere una coppia di una coppia di valori.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** \<cliext/utilità >  
  
 **Namespace:** cliext
