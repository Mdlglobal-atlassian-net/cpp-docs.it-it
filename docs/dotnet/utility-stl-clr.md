# utility (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Includere l'intestazione di `<cliext/utility>` STL\/CLR per definire la classe modello `pair` e alcune funzioni di supporto del modello.  
  
## Sintassi  
  
```  
#include <utility>  
```  
  
## Dichiarazioni  
  
|Classe|Descrizione|  
|------------|-----------------|  
|[pair](../dotnet/pair-stl-clr.md)|Eseguire il wrapping di una coppia di elementi.|  
  
|Operatore|Descrizione|  
|---------------|-----------------|  
|[operator\=\= \(pair\)](../dotnet/operator-equality-pair-stl-clr.md)|Confronto uguale di coppie.|  
|[operator\!\= \(pair\)](../dotnet/operator-inequality-pair-stl-clr.md)|Abbini il confronto non uguale.|  
|[operator\< \(pair\)](../dotnet/operator-less-than-pair-stl-clr.md)|Abbini minore del confronto.|  
|[operator\<\= \(pair\)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|Coppia minore o uguale confronto.|  
|[operator\> \(pair\)](../dotnet/operator-greater-than-pair-stl-clr.md)|Coppia maggiore del confronto.|  
|[operator\>\= \(pair\)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|Coppia maggiore o uguale confronto.|  
  
|Funzione|Descrizione|  
|--------------|-----------------|  
|[make\_pair](../dotnet/make-pair-stl-clr.md)|Fare una coppia da una coppia di valori.|  
  
## Requisiti  
 **Intestazione:**\<cliext\/utility\>  
  
 **Spazio dei nomi:** cliext
