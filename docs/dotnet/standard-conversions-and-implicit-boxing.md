# <a name="standard-conversions-and-implicit-boxing"></a>Conversioni standard e conversione boxing implicita

## <a name="example"></a>Esempio

```
// clr_implicit_boxing_Std_conversion.cpp
// compile with: /clr
int f3(int ^ i) {   // requires boxing
   return 1;
```
