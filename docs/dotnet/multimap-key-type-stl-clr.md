# <a name="multimapkeytype-stlclr"></a>multimap::key_type (STL/CLR)
Tipo di una chiave di ordinamento.  
  
## <a name="syntax"></a>Sintassi  
  
```  
typedef Key key_type;  
```  
  
## <a name="remarks"></a>Note  
 Il tipo Ã¨ un sinonimo del parametro di modello `Key`.  
  
## <a name="example"></a>Esempio  
  
```  
// cliext_multimap_key_type.cpp   
// compile with: /clr   
#include <cliext/map>   
  
typedef cliext::multimap<wchar_t, int> Mymultimap;   
int main()   
    {   
    Mymultimap c1;   
    c1.insert(Mymultimap::make_value(L'a', 1));   
    c1.insert(Mymultimap::make_value(L'b', 2));   
    c1.insert(Mymultimap::make_value(L'c', 3));   
  
// display contents " [a 1] [b 2] [c 3]" using key_type   
    for (Mymultimap::iterator it = c1.begin(); it != c1.end(); ++it)   
        {   // store element in key_type object   
        Mymultimap::key_type val = it->first;   
  
        System::Console::Write(" {0}", val);   
        }   
    System::Console::WriteLine();   
    return (0);   
    }  
  
```  
  
```Output  
a b c  
```
