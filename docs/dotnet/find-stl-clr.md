# <a name="find-stlclr"></a>find (STL/CLR)
Individua la posizione della prima occorrenza di un elemento in un intervallo con un valore specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## <a name="remarks"></a>Note  
 Questa funzione si comporta come la funzione della libreria Standard C++ `find`. Per ulteriori informazioni, vedere [trovare](../standard-library/algorithm-functions.md#find).  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** \<cliext/algoritmo >  
  
 **Namespace:** cliext
