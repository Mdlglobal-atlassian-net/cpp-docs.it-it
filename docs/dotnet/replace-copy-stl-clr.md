# replace_copy (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Esamina ogni elemento in un intervallo di origine e sostituirla se corrisponde a un valore specificato come copia il risultato in un nuovo intervallo di destinazione.  
  
## Sintassi  
  
```  
template<class _InIt, class _OutIt, class _Ty> inline  
    _OutIt replace_copy(_InIt _First, _InIt _Last, _OutIt _Dest,  
        const _Ty% _Oldval, const _Ty% _Newval);  
```  
  
## Note  
 Questa funzione si comporta come se fosse la funzione `replace_copy`STL.  Per ulteriori informazioni, vedere [replace\_copy](../Topic/replace_copy.md).  
  
## Requisiti  
 **Intestazione:**\<cliext\/algorithm\>  
  
 **Spazio dei nomi:** cliext
