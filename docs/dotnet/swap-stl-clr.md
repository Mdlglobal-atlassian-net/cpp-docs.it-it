# swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Scambia i valori degli elementi tra due tipi di oggetti, assegnanti il contenuto del primo oggetto dal secondo oggetto e il contenuto del secondo al primo.  
  
## Sintassi  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## Note  
 Questa funzione si comporta come se fosse la funzione `swap`STL.  Per ulteriori informazioni, vedere [swap](../Topic/swap.md).  
  
## Requisiti  
 **Intestazione:**\<cliext\/algorithm\>  
  
 **Spazio dei nomi:** cliext
