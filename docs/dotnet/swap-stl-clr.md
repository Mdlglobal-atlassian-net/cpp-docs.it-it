# <a name="swap-stlclr"></a>swap (STL/CLR)
Scambia i valori degli elementi tra due tipi di oggetti, assegnando il contenuto del primo oggetto al secondo oggetto e il contenuto del secondo al primo.  
  
## <a name="syntax"></a>Sintassi  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## <a name="remarks"></a>Note  
 Questa funzione si comporta come la funzione della libreria Standard C++ `swap`. Per ulteriori informazioni, vedere [scambio](http://msdn.microsoft.com/Library/b471a2de-035e-4aff-b1c7-345d85d93972).  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** \<cliext/algoritmo >  
  
 **Namespace:** cliext
