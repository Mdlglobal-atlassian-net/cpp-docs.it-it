# <a name="how-to-explicitly-request-boxing"></a>Procedura: richiedere la conversione boxing in modo esplicito
Ãˆ possibile richiedere in modo esplicito il boxing assegnando una variabile a una variabile di tipo `Object`.  
  
## <a name="example"></a>Esempio  
  
```  
// vcmcppv2_explicit_boxing3.cpp  
// compile with: /clr  
using namespace System;  
  
void f(int i) {  
   Console::WriteLine("f(int i)");  
}  
  
void f(Object ^o) {  
   Console::WriteLine("f(Object^ o)");  
}  
  
int main() {  
   int i = 5;  
   Object ^ O = i;   // forces i to be boxed  
   f(i);  
   f(O);  
   f( (Object^)i );  // boxes i  
}  
```  
  
```Output  
f(int i)  
f(Object^ o)  
f(Object^ o)  
```
