# <a name="noimplementation"></a>no_implementation
**Sezione specifica C++**

Elimina la generazione dell'intestazione con estensione tli, contenente implementazioni delle funzioni membro wrapper.

## <a name="syntax"></a>Sintassi

```
no_implementation
```
