# pop_macro
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Imposta il valore della macro *macro\_name* sul valore nella parte superiore dello stack per questa macro.  
  
## Sintassi  
  
```  
  
#pragma pop_macro("  
macro_name  
")  
  
```  
  
## Note  
 Ãˆ necessario generare una [push\_macro](../preprocessor/push-macro.md) per *macro\_name* prima di poter eseguire una **pop\_macro**.  
  
## Esempio  
  
```  
// pragma_directives_pop_macro.cpp  
// compile with: /W1  
#include <stdio.h>  
#define X 1  
#define Y 2  
  
int main() {  
   printf("%d",X);  
   printf("\n%d",Y);  
   #define Y 3   // C4005  
   #pragma push_macro("Y")  
   #pragma push_macro("X")  
   printf("\n%d",X);  
   #define X 2   // C4005  
   printf("\n%d",X);  
   #pragma pop_macro("X")  
   printf("\n%d",X);  
   #pragma pop_macro("Y")  
   printf("\n%d",Y);  
}  
```
