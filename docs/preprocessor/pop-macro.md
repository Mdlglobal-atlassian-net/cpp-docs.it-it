# <a name="popmacro"></a>pop_macro
Imposta il valore della *macro_name* macro per il valore all'inizio dello stack per questa macro.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
#pragma pop_macro("  
macro_name  
")  
  
```  
  
## <a name="remarks"></a>Note  
 Ãˆ necessario generare un [push_macro](../preprocessor/push-macro.md) per *macro_name* prima di effettuare un **pop_macro**.  
  
## <a name="example"></a>Esempio  
  
```  
// pragma_directives_pop_macro.cpp  
// compile with: /W1  
#include <stdio.h>  
#define X 1  
#define Y 2  
  
int main() {  
   printf("%d",X);  
   printf("\n%d",Y);  
   #define Y 3   // C4005  
   #pragma push_macro("Y")  
   #pragma push_macro("X")  
   printf("\n%d",X);  
   #define X 2   // C4005  
   printf("\n%d",X);  
   #pragma pop_macro("X")  
   printf("\n%d",X);  
   #pragma pop_macro("Y")  
   printf("\n%d",Y);  
}  
```  
  
```Output  
1  
2  
1  
2  
1  
3  
```
