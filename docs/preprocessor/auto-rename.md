# auto_rename
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Sezione specifica C\+\+**  
  
 Rinomina parole riservate C\+\+ aggiungendo due caratteri di sottolineatura \(\_\_\) al nome della variabile per risolvere potenziali conflitti di nomi.  
  
## Sintassi  
  
```  
auto_rename  
```  
  
## Note  
 Questo attributo viene utilizzato quando si importa una libreria dei tipi che utilizza una o pi√π parole riservate C\+\+ \(parole chiave o macro\) come nomi di variabile.
