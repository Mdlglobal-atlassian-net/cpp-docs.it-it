# <a name="autorename"></a>auto_rename
**Sezione specifica C++**  
  
 Rinomina parole riservate C++ aggiungendo due caratteri di sottolineatura (__) al nome della variabile per risolvere potenziali conflitti di nomi.  
  
## <a name="syntax"></a>Sintassi  
  
```  
auto_rename  
```
