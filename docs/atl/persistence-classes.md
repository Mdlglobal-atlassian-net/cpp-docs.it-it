# <a name="persistence-classes"></a>Classi di persistenza
Le seguenti classi implementano la persistenza degli oggetti:  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) consente a un client caricare e salvare le proprietà dell'oggetto a un elenco di proprietà.  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) consente a un client caricare e salvare i dati persistenti di un oggetto in un flusso.  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) consente a un client caricare e salvare i dati persistenti di un oggetto al servizio di archiviazione.
