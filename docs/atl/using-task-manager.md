# Using Task Manager
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Uno dei modi più semplici eseguire il debug di un servizio è l'utilizzo di Gestione attività di Windows NT 4.0 o Windows 2000.  Quando il servizio è in esecuzione, avviare Gestione attività e fare clic sulla scheda **Processes**.  Fare clic con il pulsante destro del mouse sul nome del file EXE e scegliere **Debug**.  Verrà avviato Visual C\+\+ associato a tale processo in esecuzione.  Ora, scegliere **Interrompi** il menu **Debug** per permettervi impostare i punti di interruzione nel codice.  Fare clic **Esegui** per lavorare ai punti di interruzione selezionati.
