# <a name="using-task-manager"></a>Tramite Gestione attività
Uno dei modi più semplici per eseguire il debug di un servizio è tramite Task Manager Windows NT 4.0 o Windows 2000. Mentre il servizio è in esecuzione, avviare Gestione attività e fare clic su di **processi** scheda. Fare doppio clic il nome del file EXE e quindi fare clic su **Debug**. Verrà avviato Visual C++ associato al processo in esecuzione. A questo punto, fare clic su **interrompere** sul **Debug** menu consente di impostare punti di interruzione nel codice. Fare clic su **eseguire** per eseguire i punti di interruzione selezionati.
