# <a name="using-task-manager"></a>Tramite Gestione attività

Uno dei modi più semplici per il debug di un servizio è tramite l'uso di Gestione attività. Mentre il servizio è in esecuzione, avviare Gestione attività e scegliere il **processi** scheda. Fare doppio clic il nome del file EXE e quindi fare clic su **Debug**. Verrà avviato Visual C++ collegato al processo in esecuzione. A questo punto, fare clic su **interrompere** nel **Debug** menu consente di impostare punti di interruzione nel codice. Fare clic su **eseguiti** per l'esecuzione per i punti di interruzione selezionati.
