# Modifying the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In questa fase, il progetto del controllo composito implementa gli oggetti necessari per il controllo composito.  È necessario aggiungere tutti i controlli che il controllo composito conterrà e gestirà gli eventi necessari.  
  
 Per aggiungere ActiveX aggiuntivo o controlli Windows, aggiungere un nuovo script di risorsa quindi utilizzare l'editor finestre.  Per ulteriori informazioni sull'aggiunta dei controlli e le attività correlate, vedere [Editor finestre](../mfc/dialog-editor.md).  
  
 Per gestire gli eventi necessari da controlli ActiveX, vedere [Aggiunta di funzionalità al controllo composito](../atl/adding-functionality-to-the-composite-control.md).
