# Interfacce, Creazione guidata controllo ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

È possibile utilizzare questa schermata della procedura guidata per identificare le interfacce supportate dal controllo.  Per impostazione predefinita, le interfacce supportate sono quelle generalmente utilizzate dalla maggior parte dei contenitori.  
  
> [!NOTE]
>  Se è stata selezionata l'opzione **Controllo minimo** nella scheda [Opzioni](../../atl/reference/options-atl-control-wizard.md), non viene visualizzata alcuna interfaccia predefinita nella casella di riepilogo **Supportate**.  
  
 **Non supportato**  
 Consente di verificare quali sono le interfacce disponibili non attualmente supportate per il controllo.  
  
 **Supportato**  
 Consente di verificare quali sono le interfacce attualmente supportate per il controllo.  
  
|Pulsante di trasferimento|Descrizione|  
|-------------------------------|-----------------|  
|**\>**|Utilizzare questa opzione per aggiungere all'elenco **Supportate** il nome dell'interfaccia attualmente selezionata nell'elenco **Non supportate**.|  
|**\>\>**|Utilizzare questa opzione per aggiungere all'elenco **Supportate** tutti i nomi delle interfacce disponibili nell'elenco **Non supportate**.|  
|**\<**|Utilizzare questa opzione per rimuovere il nome dell'interfaccia attualmente selezionata nell'elenco **Supportate**.|  
|**\<\<**|Utilizzare questa opzione per rimuovere tutti i nomi delle interfacce attualmente elencate nell'elenco **Supportate**.|
