# Service Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste macro consentono di definire i mapping e le voci di servizio.  
  
|||  
|-|-|  
|[BEGIN\_SERVICE\_MAP](../Topic/BEGIN_SERVICE_MAP.md)|Contrassegna l'inizio di un mapping di servizio ATL.|  
|[END\_SERVICE\_MAP](../Topic/END_SERVICE_MAP.md)|Contrassegna la fine di un mapping di servizio ATL.|  
|[SERVICE\_ENTRY](../Topic/SERVICE_ENTRY.md)|Indica che l'oggetto supporta un ID servizio specifica|  
|[SERVICE\_ENTRY\_CHAIN](../Topic/SERVICE_ENTRY_CHAIN.md)|Indica a [IServiceProviderImpl::QueryService](../Topic/IServiceProviderImpl::QueryService.md) concatenare all'oggetto specificato.|
