# Window Class Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste macro definiscono le utilità della classe della finestra.  
  
|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|Consente di specificare il nome della nuova classe della finestra.|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|Consente di specificare il nome di una classe finestra esistente in cui una nuova classe della finestra verrà basata su.|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|Consente di specificare i parametri di una classe.|
