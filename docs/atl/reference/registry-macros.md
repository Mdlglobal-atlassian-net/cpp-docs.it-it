# Registry Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste macro consentono la libreria dei tipi utili e le funzionalità del Registro di sistema.  
  
|||  
|-|-|  
|[\_ATL\_STATIC\_REGISTRY](../Topic/_ATL_STATIC_REGISTRY.md)|Indica che il codice di registrazione dell'oggetto dall'oggetto per evitare una dipendenza da ATL.DLL.|  
|[DECLARE\_LIBID](../Topic/DECLARE_LIBID.md)|Fornisce una modalità per ATL ottengano *LIBID* della libreria dei tipi.|  
|[DECLARE\_NO\_REGISTRY](../Topic/DECLARE_NO_REGISTRY.md)|Evita la registrazione predefinita ATL.|  
|[DECLARE\_REGISTRY](../Topic/DECLARE_REGISTRY.md)|Fornisce o rimuove la voce dell'obiettivo principale nel Registro di sistema.|  
|[DECLARE\_REGISTRY\_APPID\_RESOURCEID](../Topic/DECLARE_REGISTRY_APPID_RESOURCEID.md)|Specifica le informazioni richieste automaticamente per registrare *l'appid*.|  
|[DECLARE\_REGISTRY\_RESOURCE](../Topic/DECLARE_REGISTRY_RESOURCE.md)|Trova la risorsa denominata ed esegue lo script del Registro di sistema.|  
|[DECLARE\_REGISTRY\_RESOURCEID](../Topic/DECLARE_REGISTRY_RESOURCEID.md)|Trova la risorsa identificata da un numero ID ed esegue lo script del Registro di sistema.|
