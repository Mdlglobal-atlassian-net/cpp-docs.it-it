# Category Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste macro consentono di definire i mapping delle categorie.  
  
|||  
|-|-|  
|[BEGIN\_CATEGORY\_MAP](../Topic/BEGIN_CATEGORY_MAP.md)|Contrassegna l'inizio del mapping delle categorie.|  
|[END\_CATEGORY\_MAP](../Topic/END_CATEGORY_MAP.md)|Contrassegna la fine del mapping delle categorie.|  
|[IMPLEMENTED\_CATEGORY](../Topic/IMPLEMENTED_CATEGORY.md)|Indica le categorie distribuite dall'oggetto COM.|  
|[REQUIRED\_CATEGORY](../Topic/REQUIRED_CATEGORY.md)|Indica le categorie necessarie contenitore dall'oggetto COM.|
