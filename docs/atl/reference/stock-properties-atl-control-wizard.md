# Propriet&#224; predefinite, Creazione guidata controllo ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Utilizzando questa schermata della procedura guidata è possibile identificare le proprietà predefinite supportate dal controllo.  Per impostazione predefinita, non viene identificata alcuna proprietà.  
  
 **Non supportato**  
 Consente di verificare quali sono le proprietà disponibili non attualmente supportate per il controllo.  
  
 **Supportato**  
 Consente di verificare quali sono le proprietà attualmente supportate per il controllo.  
  
|Pulsante di trasferimento|Descrizione|  
|-------------------------------|-----------------|  
|**\>**|Utilizzare questa opzione per aggiungere all'elenco **Supportate** il nome della proprietà attualmente selezionata nell'elenco **Non supportate**.|  
|**\>\>**|Utilizzare questa opzione per aggiungere all'elenco **Supportate** tutti i nomi delle proprietà disponibili nell'elenco **Non supportate**.|  
|**\<**|Utilizzare questa opzione per rimuovere il nome della proprietà attualmente selezionata nell'elenco **Supportate**.|  
|**\<\<**|Utilizzare questa opzione per rimuovere tutti i nomi delle proprietà attualmente elencate nell'elenco **Supportate**.|
