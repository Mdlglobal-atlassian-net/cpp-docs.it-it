# <a name="adding-an-atl-com-10-component"></a>Aggiunta di un componente ATL COM+ 1.0
Questa procedura guidata aggiunge un oggetto al progetto che supporta i servizi di COM+ 1.0, incluse le transazioni.  
  
### <a name="to-add-an-atl-com-10-component-to-your-project"></a>Per aggiungere un componente ATL COM+ 1.0 al progetto  
  
1.  In entrambi **Esplora** o [Visualizzazione classi](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), fare doppio clic sul nome del progetto a cui si desidera aggiungere il componente ATL COM+ 1.0.  
  
2.  Menu di scelta rapida, fare clic su **Aggiungi**, quindi fare clic su **Aggiungi classe**.  
  
3.  Nel [Aggiungi classe](../../ide/add-class-dialog-box.md) la finestra di dialogo, nel riquadro Modelli fare clic su **componente ATL COM+ 1.0**, quindi fare clic su **aprire** per visualizzare il [ATL COM+ 1.0 aggiunta guidata componenti di](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
