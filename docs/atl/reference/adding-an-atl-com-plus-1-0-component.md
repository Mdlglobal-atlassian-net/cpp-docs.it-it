# <a name="adding-an-atl-com-10-component"></a>Aggiunta di un componente ATL COM+ 1.0

Questa procedura guidata aggiunge un oggetto per il progetto che supporta i servizi di COM+ 1.0, incluse le transazioni.

### <a name="to-add-an-atl-com-10-component-to-your-project"></a>Per aggiungere un componente ATL COM+ 1.0 al progetto

1. In uno **Esplora soluzioni** oppure [Visualizzazione classi](/visualstudio/ide/viewing-the-structure-of-code), fare doppio clic il nome del progetto a cui si desidera aggiungere il componente ATL COM+ 1.0.

2. Nel menu di scelta rapida, fare clic su **Add**, quindi fare clic su **Aggiungi classe**.

3. Nel [Aggiungi classe](../../ide/add-class-dialog-box.md) finestra di dialogo, nel riquadro dei modelli, fare clic su **componente ATL COM+ 1.0**e quindi fare clic su **Open** per visualizzare il [COM+ 1.0 Creazione guidata componente ATL ](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
