# Aggiunta di un componente ATL COM+ 1.0
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ãˆ possibile utilizzare questa procedura guidata per aggiungere al progetto un oggetto che supporta i servizi COM\+ 1.0, incluse le transazioni.  
  
### Per aggiungere un componente ATL COM\+ 1.0 al progetto  
  
1.  In **Esplora soluzioni** o [Visualizzazione classi](http://msdn.microsoft.com/it-it/8d7430a9-3e33-454c-a9e1-a85e3d2db925) fare clic con il pulsante destro del mouse sul nome del progetto a cui aggiungere il componente ATL COM\+ 1.0.  
  
2.  Scegliere **Aggiungi** dal menu di scelta rapida, quindi **Aggiungi classe**.  
  
3.  Nel riquadro Modelli della finestra di dialogo [Aggiungi classe](../../ide/add-class-dialog-box.md) fare clic su **Componente ATL COM\+ 1.0**, quindi scegliere **Apri** per visualizzare la [Creazione guidata componente ATL COM\+ 1.0](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
