# Connection Point Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste macro definiscono le mappe dei punti di connessione e le voci.  
  
|||  
|-|-|  
|[BEGIN\_CONNECTION\_POINT\_MAP](../Topic/BEGIN_CONNECTION_POINT_MAP.md)|Contrassegna l'inizio delle voci della mappa dei punti di connessione.|  
|[CONNECTION\_POINT\_ENTRY](../Topic/CONNECTION_POINT_ENTRY.md)|Immette i punti di connessione nella mappa.|  
|[END\_CONNECTION\_POINT\_MAP](../Topic/END_CONNECTION_POINT_MAP.md)|Contrassegna la fine delle voci della mappa dei punti di connessione.|
