# ATL Operators Alphabetical Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Questa sezione contiene argomenti di riferimento per gli operatori globali ATL.  
  
|Operatore|Descrizione|  
|---------------|-----------------|  
|[\=\= dell'operatore](../Topic/operator%20==%20\(ATL\).md)|Confronta due oggetti `CSid` o strutture `SID` l'uguaglianza.|  
|[operatore\! \=](../Topic/operator%20!=%20\(ATL\).md)|Confronta due oggetti `CSid` o strutture `SID` disuguaglianza.|  
|[operatore \<](../Topic/operator%20%3C%20\(ATL\).md)|Test se l'oggetto `CSid` o la struttura `SID` a sinistra dell'operatore è inferiore all'oggetto `CSid` o la struttura `SID` lato destro \(per la compatibilità STL\).|  
|[operatore \>](../Topic/operator%20%3E%20\(ATL\).md)|Test se l'oggetto `CSid` o la struttura `SID` a sinistra dell'operatore è maggiore dell'oggetto `CSid` o della struttura `SID` lato destro \(per la compatibilità STL\).|  
|[\<\= dell'operatore](../Topic/operator%20%3C=%20\(ATL\).md)|Test se l'oggetto `CSid` o la struttura `SID` a sinistra dell'operatore è minore o uguale all'oggetto `CSid` o la struttura `SID` lato destro \(per la compatibilità STL\).|  
|[\>\= dell'operatore](../Topic/operator%20%3E=%20\(ATL\).md)|Test se l'oggetto `CSid` o la struttura `SID` a sinistra dell'operatore è maggiore o uguale all'oggetto `CSid` o la struttura `SID` lato destro \(per la compatibilità STL\).|  
  
 Tutti questi operatori sono definiti nel file atlsecurity.h.
