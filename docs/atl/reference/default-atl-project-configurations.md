# <a name="default-atl-project-configurations"></a>Configurazioni progetto ATL predefinite

Per impostazione predefinita, la creazione guidata progetto ATL crea due configurazioni di progetto:

|Configurazione|Set di caratteri|Uso di ATL|
|-------------------|-------------------|----------------|
|Rilascio|MBCS|DLL|
|Debug|MBCS|DLL|

**Set di caratteri**, **uso di ATL** e può essere modificato nel **impostazioni del progetto** nella finestra di dialogo il **generali** scheda. È anche possibile aggiungere configurazioni personalizzate utilizzando Gestione configurazione. Per informazioni dettagliate, vedere [configurazioni della Build](/visualstudio/ide/understanding-build-configurations).
