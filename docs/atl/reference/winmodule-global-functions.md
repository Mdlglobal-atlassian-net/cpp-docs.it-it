# WinModule Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste funzioni consentono il supporto per le operazioni della struttura `_AtlCreateWndData`.  
  
> [!IMPORTANT]
>  Le funzioni elencate nella tabella seguente non possono essere utilizzate nelle applicazioni eseguite in [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlWinModuleAddCreateWndData](../Topic/AtlWinModuleAddCreateWndData.md)|Questa funzione viene utilizzata per inizializzare e aggiungere una struttura `_AtlCreateWndData`.|  
|[AtlWinModuleExtractCreateWndData](../Topic/AtlWinModuleExtractCreateWndData.md)|Chiamare la funzione per disegnare una struttura esistente `_AtlCreateWndData`.|
