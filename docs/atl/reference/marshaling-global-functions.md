# Marshaling Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste funzioni consentono il supporto per il marshalling e convertire i dati di marshalling dei puntatori a interfaccia.  
  
> [!IMPORTANT]
>  Le funzioni elencate nella tabella seguente non possono essere utilizzate nelle applicazioni eseguite in [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlFreeMarshalStream](../Topic/AtlFreeMarshalStream.md)|Inserisce i dati del marshalling e il puntatore `IStream`.|  
|[AtlMarshalPtrInProc](../Topic/AtlMarshalPtrInProc.md)|Crea un nuovo oggetto flusso ed effettua il marshalling di un puntatore a interfaccia specificato.|  
|[AtlUnmarshalPtr](../Topic/AtlUnmarshalPtr.md)|Converte i dati del marshalling di un flusso in un puntatore a interfaccia.|
