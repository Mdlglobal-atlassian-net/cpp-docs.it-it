# <a name="atl-structures"></a>Strutture ATL


Active Template Library include le seguenti strutture.  
  
## <a name="in-this-section"></a>In questa sezione  
 [ATL_DRAWINFO](../../atl/reference/atl-drawinfo-structure.md)  
 Contiene informazioni utilizzate per il rendering in varie destinazioni, ad esempio una stampante, metafile o controllo ActiveX.  
  
 [_AtlCreateWndData](../../atl/reference/atlcreatewnddata-structure.md)  
 Contiene i dati di istanza di classe nel codice di windowing in ATL.  
  
 [_ATL_BASE_MODULE70](../../atl/reference/atl-base-module70-structure.md)  
 Usato da qualsiasi progetto che utilizza ATL.  
  
 [_ATL_COM_MODULE70](../../atl/reference/atl-com-module70-structure.md)  
 Utilizzato da codice COM correlato in ATL.  
  
 [_ATL_FUNC_INFO](../../atl/reference/atl-func-info-structure.md)  
 Contiene informazioni sul tipo utilizzati per descrivere una propriet√† o metodo in un'interfaccia dispatch.  
  
 [_ATL_MODULE70](../../atl/reference/atl-module70-structure.md)  
 Contiene i dati utilizzati da ogni modulo ATL.  
  
 [_ATL_WIN_MODULE70](../../atl/reference/atl-win-module70-structure.md)  
 Utilizzato dal codice di windowing in ATL.
