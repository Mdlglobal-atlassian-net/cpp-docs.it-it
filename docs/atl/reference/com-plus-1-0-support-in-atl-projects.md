# <a name="com-10-support-in-atl-projects"></a>COM+ 1.0 supporta ATL (progetti)
È possibile utilizzare il [Creazione guidata progetto ATL](../../atl/reference/creating-an-atl-project.md) per creare un progetto con il supporto di base per i componenti COM+ 1.0.  
  
 COM+ 1.0 è progettato per lo sviluppo di applicazioni distribuite basate su componenti. Fornisce inoltre un'infrastruttura di runtime per la distribuzione e la gestione di queste applicazioni.  
  
 Se si seleziona il **supporto COM+ 1.0** casella di controllo, la procedura guidata consente di modificare lo script di compilazione in fase di collegamento. In particolare, di COM+ 1.0 progetto collegamenti a librerie seguenti:  
  
-   Comsvcs.lib  
  
-   Mtxguid.lib  
  
 Se si seleziona il **supporto COM+ 1.0** casella di controllo, è anche possibile selezionare **supporto registrazione componenti**. La registrazione del componente consente all'oggetto di COM+ 1.0 ottenere un elenco di componenti, componenti di registrare o annullare la registrazione di componenti (singolarmente o in una sola volta).
