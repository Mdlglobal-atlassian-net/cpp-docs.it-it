# Connection Point Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Queste funzioni consentono il supporto per i punti di connessione e effettuano il sink i mapping.  
  
> [!IMPORTANT]
>  Le funzioni elencate nella tabella seguente non possono essere utilizzate nelle applicazioni eseguite in [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlAdvise](../Topic/AtlAdvise.md)|Creazione di una connessione tra il punto di connessione di un oggetto e il sink di un client.|  
|[AtlUnadvise](../Topic/AtlUnadvise.md)|Termina la connessione stabilita con `AtlAdvise`.|  
|[AtlAdviseSinkMap](../Topic/AtlAdviseSinkMap.md)|Consigliabile o voci di unadvises in una mappa del sink di eventi.|
