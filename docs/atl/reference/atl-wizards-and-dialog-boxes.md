# Finestre di dialogo e procedure guidate ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Le procedure guidate di Active Template Library \(ATL\) generano il codice di boilerplate per diversi tipi di oggetti COM.  È possibile eseguire le procedure guidate aprendo il menu di scelta rapida per un progetto in **Esplora soluzioni** e scegliendo **Aggiungi**, **Classe**.
