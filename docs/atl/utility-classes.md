# <a name="utility-classes"></a>Classi di utilità
Vengono fornite le classi di utilità indipendente da MFC seguenti:  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) fornisce migliorato il supporto delle bitmap, inclusa la possibilità di caricare e salvare le immagini JPEG, GIF, BMP, e formati di grafica PNG (Portable Network).  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) fornisce un'implementazione per l'archiviazione delle coppie di coordinate (x, y).  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) fornisce un'implementazione per archiviare le coordinate delle aree rettangolari.  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) fornisce un'implementazione per l'archiviazione di distanza, posizioni relative o valori accoppiati.  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md) fornisce un'implementazione per l'archiviazione delle stringhe di caratteri.  
  
-   [CAdapt](../atl/reference/cadapt-class.md) un modello semplice utilizzato per eseguire il wrapping delle classi che consentono di ridefinire l'operatore address-of.  
  
-   [_U_RECT](../atl/reference/u-rect-class.md) una classe di adattatori di argomento che consente di utilizzare `RECT` puntatori o riferimenti deve essere passato a una funzione che viene implementata in termini di puntatori.
