# Scope of ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ALT consente di creare in modo semplice oggetti COM, i server di automazione e di controlli ActiveX.  ATL fornisce il supporto incorporato per molte interfacce COM sottostanti.  
  
 ATL viene fornito come codice sorgente di includere nell'applicazione.  ATL inoltre rende disponibile una DLL \(atl90.dll\), che contiene il codice che può essere condiviso tra i componenti.  Tuttavia, questa DLL non è necessario.
