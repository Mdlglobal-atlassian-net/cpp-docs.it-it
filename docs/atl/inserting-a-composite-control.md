# <a name="inserting-a-composite-control"></a>Inserimento di un controllo composito
Il **Aggiungi classe** la finestra di dialogo consente di inserire un oggetto ATL in un progetto. Accedere a questa finestra di dialogo facendo clic con il nome del progetto in Esplora soluzioni, scegliendo **Aggiungi**e quindi fare clic su **Aggiungi classe**.  
  
 Nel **Aggiungi classe** finestra di dialogo scegliere **controllo ATL**. Verrà avviata la [Creazione guidata controllo ATL](../atl/reference/atl-control-wizard.md). Per creare un controllo composito, selezionare il **opzioni** scheda, quindi scegliere il **controllo composito** casella di controllo.  
  
 Una pagina HTML predefinita verrà creata per la visualizzazione del controllo.
