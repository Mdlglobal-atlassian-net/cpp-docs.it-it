# <a name="inserting-a-composite-control"></a>Inserimento di un controllo composito

Il **Aggiungi classe** nella finestra di dialogo consente di inserire un oggetto ATL in un progetto. Accedere a questa finestra di dialogo facendo clic con il nome del progetto in Esplora soluzioni, scegliendo **Add**e quindi scegliendo **Aggiungi classe**.

Nel **Aggiungi classe** finestra di dialogo, scegliere **del controllo ATL**. Verrà avviata il [Creazione guidata controllo ATL](../atl/reference/atl-control-wizard.md). Per creare un controllo composito, selezionare la **le opzioni** e scegliere il **controllo composito** casella di controllo.

Una pagina HTML predefinita verrà creata per il controllo di visualizzazione.
