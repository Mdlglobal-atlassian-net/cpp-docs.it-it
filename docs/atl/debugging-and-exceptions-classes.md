# <a name="debugging-and-exceptions-classes"></a>Classi di eccezioni e debug

Queste classi forniscono supporto per la gestione e debug delle eccezioni.

- [CAtlDebugInterfacesModule](../atl/reference/catldebuginterfacesmodule-class.md) questa classe offre supporto per interfacce di debug.

- [CAtlException](../atl/reference/catlexception-class.md) questa classe definisce un'eccezione di ATL.
