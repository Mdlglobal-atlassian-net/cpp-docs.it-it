# <a name="active-template-library-atl-tutorial"></a>Esercitazione relativa alla Libreria ATL (Active Template Library)

ATL è progettato per semplificare il processo di creazione di controlli efficiente, flessibili e semplici. Questa esercitazione illustra la creazione di un controllo ActiveX, che illustra molte nozioni fondamentali su ATL e COM.

In questa esercitazione si apprenderà come aggiungere un controllo a un progetto ATL che consente di disegnare un cerchio e un poligono pieno. Si aggiungerà quindi una proprietà che indica il numero di lati il poligono e creare codice di disegno per l'aggiornamento del controllo quando la proprietà modificata. Il controllo verrà quindi visualizzato in una pagina Web tramite alcuni VBScript per renderlo di rispondere agli eventi.

L'esercitazione è suddivisa in sette passaggi. È consigliabile eseguire ogni passaggio nell'ordine come passaggi successivi dipendono le attività completate in precedenza. Prima di iniziare, è necessario verificare di disporre di privilegi necessari per registrare un componente ActiveX nel computer specifico. In genere si tratta di un'operazione necessaria se si esegue Visual Studio .NET su una connessione di servizi Terminal.
