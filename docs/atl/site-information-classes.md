# <a name="site-information-classes"></a>Classi di informazioni del sito
Le classi seguenti consentono a un oggetto comunicare con il sito:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) recupera e imposta un puntatore al sito di un oggetto. Utilizzato per gli oggetti che non sono controlli.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) recupera e imposta un puntatore al sito di un oggetto. Usata per i controlli.
