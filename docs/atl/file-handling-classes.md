# File Handling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Queste classi forniscono metodi per gestire i file, i file temporanei e i file mappati alla memoria.  
  
-   [CAtlFile](../atl/reference/catlfile-class.md) questa classe fornisce un thin wrapper la gestione dei file API di Windows.  
  
-   [CAtlFileMapping](../atl/reference/catlfilemapping-class.md) questa classe rappresenta un file mappato alla memoria, aggiungendo un operatore di cast ai metodi [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md).  
  
-   [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) questa classe rappresenta un file mappato alla memoria.  
  
-   [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) questa classe fornisce i metodi per la creazione e l'utilizzo di un file temporaneo.
