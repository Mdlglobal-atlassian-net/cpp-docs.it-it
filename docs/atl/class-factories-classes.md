# Class Factories Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le classi di seguito riportate implementano o supportano una class factory:  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) fornisce una factory della classe predefinita per la creazione di un oggetto.  
  
-   Creazione di un oggetto dei comandi di[CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) tramite una licenza.  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) consente oggetti da creare in apartment raccolti.  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) crea un singolo oggetto.  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) definisce il class factory dell'oggetto.
