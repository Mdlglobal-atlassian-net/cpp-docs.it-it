# <a name="atl-composite-control-fundamentals"></a>Nozioni fondamentali sul controllo composito di ATL
Un controllo composito è un tipo di controllo ActiveX che può contenere (simile a una finestra di dialogo) altri controlli ActiveX o Windows. Una volta creato il controllo composito, può essere inserito ovunque che un controllo ActiveX può essere ospitato.  
  
 La creazione guidata progetto ATL e **Aggiungi classe** la finestra di dialogo automatizzare il processo di creazione e implementazione di un progetto di controllo composito, simile al risultato di esecuzione della procedura guidata applicazione per creare un framework applicazione MFC. Il processo di sviluppo è costituita da cinque passaggi:  
  
-   [Creazione di un progetto ATL](../atl/reference/creating-an-atl-project.md)  
  
-   [Inserimento di un controllo composito](../atl/inserting-a-composite-control.md)  
  
-   [Modifica il progetto ATL](../atl/modifying-the-atl-project.md)  
  
-   [Aggiunta di funzionalità per il controllo composito](../atl/adding-functionality-to-the-composite-control.md)  
  
-   [Per creare e testare il progetto ATL](../atl/building-and-testing-the-atl-project.md)
