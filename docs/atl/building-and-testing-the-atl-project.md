# Building and Testing the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Come accennato in [Inserimento del controllo composito](../atl/inserting-a-composite-control.md), uno dei componenti del progetto è una pagina HTML predefinito che ospita il nuovo controllo composito.  Una volta terminata la modifica del controllo composito, fare clic **Compila soluzione** o **Ricompila soluzione** dal menu **Compilazione**.  Una volta di progetto compilazioni correttamente, verrà caricata la pagina HTML, posizionata nella directory radice del progetto del controllo composito, in Internet Explorer o in un altro browser e verificare la funzionalità del controllo.  
  
 È inoltre possibile testare il controllo composito mediante lo strumento di Test Container, o qualsiasi altra applicazione che può ospitare un controllo ActiveX.  Per ulteriori informazioni sulla modalità di accesso a Test Container, vedere [Verifica di proprietà ed eventi tramite Test Container](../mfc/testing-properties-and-events-with-test-container.md).
