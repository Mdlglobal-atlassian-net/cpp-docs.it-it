# Data Types Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Le classi seguenti eseguono il wrapping dei tipi di dati C\+\+:  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) esegue il wrapping del tipo di dati `BSTR`.  
  
-   [CComVariant](../atl/reference/ccomvariant-class.md) esegue il wrapping del tipo di dati **VARIANT**.  
  
-   [CComCurrency](../atl/reference/ccomcurrency-class.md) include gli operatori e i metodi per creare e gestire un oggetto `CURRENCY`.
