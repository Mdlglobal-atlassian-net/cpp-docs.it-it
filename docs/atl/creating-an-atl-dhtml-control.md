# <a name="creating-an-atl-dhtml-control"></a>Creazione di un controllo DHTML ATL

La creazione guidata controllo ATL automatizza il processo di creazione di un controllo DHTML. Genera i file di risorse necessari, tra cui un file HTML contenente codice di esempio.

## <a name="to-create-an-atl-dhtml-control"></a>Per creare un controllo DHTML ATL

1. Seguire i passaggi descritti in [creazione di un progetto ATL](../atl/reference/creating-an-atl-project.md).

1. Nella **Visualizzazione classi**, fare doppio clic sul nodo del progetto, scegliere **Add**, fare clic su **Aggiungi classe** dal menu di scelta rapida. Nel **Aggiungi classe** finestra di dialogo, fare doppio clic sul [Creazione guidata controllo ATL](../atl/reference/atl-control-wizard.md). Nel **Creazione guidata controllo ATL**, fare clic sui **opzioni** scheda e selezionare **controllo DHTML**.

Ora puoi [testare il controllo predefinito](../atl/testing-the-atl-dhtml-control.md).
