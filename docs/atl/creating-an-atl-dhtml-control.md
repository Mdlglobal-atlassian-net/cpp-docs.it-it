# <a name="creating-an-atl-dhtml-control"></a>Creazione di un controllo DHTML ATL
Creazione guidata controllo ATL automatizza il processo di creazione di un controllo DHTML. Genera i file di risorse necessari, tra cui un file HTML contenente il codice di esempio.  
  
#### <a name="to-create-an-atl-dhtml-control"></a>Per creare un controllo DHTML ATL  
  
1.  Seguire i passaggi in [creazione di un progetto ATL](../atl/reference/creating-an-atl-project.md).  
  
2.  In visualizzazione classi, fare clic sul nodo del progetto, scegliere **Aggiungi**, fare clic su **Aggiungi classe** dal menu di scelta rapida. Nel **Aggiungi classe** la finestra di dialogo, fare doppio clic su di [Creazione guidata controllo ATL](../atl/reference/atl-control-wizard.md). Nella creazione guidata controllo ATL, fare clic su di **opzioni** e selezionare **controllo DHTML**.
