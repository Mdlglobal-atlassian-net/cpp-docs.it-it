# &#200; possibile chiamare direttamente le API DAO o ODBC?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Come di consueto in MFC, se è necessario un controllo più specifico, è possibile chiamare direttamente DAO o ODBC oltre ad accedervi tramite le classi.  MFC è stato progettato per semplificare la programmazione per Windows, ma consente anche la massima libertà di intervento quando è necessario accedere alle API sottostanti.
