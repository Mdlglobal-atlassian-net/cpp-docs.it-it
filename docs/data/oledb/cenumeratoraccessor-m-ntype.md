# CEnumeratorAccessor::m_nType
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Una variabile che indica se la riga descrive un'origine dati o un enumeratore.  
  
## Sintassi  
  
```  
  
USHORT m_nType;  
  
```  
  
## Note  
 Vedere [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx)*in OLE DB Programmer's Reference* per ulteriori informazioni.  
  
## Requisiti  
 **Intestazione:** atldbcli.h
