# END_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Contrassegna la fine di una proprietà.  
  
## Sintassi  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parametri  
 `guid`  
 \[in\] la proprietà GUID.  
  
## Esempio  
 Vedere [BEGIN\_PROPSET\_MAP](../../data/oledb/begin-propset-map.md).  
  
## Requisiti  
 **Intestazione:** atldb.h
