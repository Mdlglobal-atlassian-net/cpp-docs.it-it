# CBookmark::GetBuffer
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Recupera il puntatore nel buffer del segnalibro.  
  
## Sintassi  
  
```  
  
virtual BYTE* GetBuffer( ) const throw( );  
  
```  
  
## Valore restituito  
 Un puntatore al buffer del segnalibro.  
  
## Requisiti  
 **Intestazione:** atldbcli.h
