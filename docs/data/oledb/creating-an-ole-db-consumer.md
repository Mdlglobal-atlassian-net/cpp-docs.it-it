# Creazione di un consumer OLE DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

È possibile creare un consumer basato sui modelli OLE DB utilizzando o meno la Creazione guidata consumer OLE DB ATL, che tuttavia rappresenta il metodo consigliato.  Un consumer è in grado di operare in qualsiasi applicazione in grado di supportare ATL.
