# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Contrassegna la fine delle voci della mappa delle colonne.  
  
## Sintassi  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## Note  
 Viene utilizzata con una singola funzione di accesso per un rowset.  La macro di `BEGIN_COLUMN_MAP` viene completata con la macro di `END_COLUMN_MAP`.  
  
## Esempio  
 Vedere [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md).  
  
## Requisiti  
 **Intestazione:** atldbcli.h
