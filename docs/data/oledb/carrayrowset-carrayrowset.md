# CArrayRowset::CArrayRowset
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Crea un nuovo oggetto `CArrayRowset`.  
  
## Sintassi  
  
```  
  
      CArrayRowset(   
   int nMax = 100000    
);  
```  
  
#### Parametri  
 `nMax`  
 \[in\] numero massimo di righe nel rowset.
