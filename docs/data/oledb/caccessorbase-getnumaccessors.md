# CAccessorBase::GetNumAccessors
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Recupera il numero delle funzioni di accesso create dalla classe.  
  
## Sintassi  
  
```  
  
ULONG GetNumAccessors( ) const;  
  
```  
  
## Valore restituito  
 Il numero delle funzioni di accesso create dalla classe.  
  
## Requisiti  
 **Intestazione:** atldbcli.h
