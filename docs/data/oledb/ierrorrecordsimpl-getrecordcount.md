# <a name="ierrorrecordsimplgetrecordcount"></a>IErrorRecordsImpl::GetRecordCount
Restituisce il numero di record nell'oggetto record OLE DB.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      STDMETHOD( GetRecordCount )(  
   ULONG *pcRecords   
);  
```  
  
#### <a name="parameters"></a>Parametri  
 Vedere [IErrorRecords::GetRecordCount](https://msdn.microsoft.com/en-us/library/ms722724.aspx) nel *di riferimento per programmatori OLE DB*.
