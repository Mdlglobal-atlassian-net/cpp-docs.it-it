# <a name="crowsetclose"></a>CRowset::Close
Rilascia le righe e corrente [IRowset](https://msdn.microsoft.com/en-us/library/ms720986.aspx) interfaccia.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp
void Close() throw();  
  
```  
  
## <a name="remarks"></a>Note  
 Questo metodo rilascia tutte le righe correntemente nel set di righe.  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** atldbcli.h
