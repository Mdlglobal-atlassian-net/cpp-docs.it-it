# <a name="icommandimplcancel"></a>ICommandImpl::Cancel
Annulla l'esecuzione del comando corrente.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp
STDMETHOD(Cancel)();  
  
```  
  
## <a name="remarks"></a>Note  
 Vedere [ICommand::Cancel](https://msdn.microsoft.com/en-us/library/ms714402.aspx) nel *di riferimento per programmatori OLE DB*.
