# <a name="caccessorrowsetclose"></a>CAccessorRowset::Close

## <a name="syntax"></a>Sintassi  
  
```  
  
void Close( );  
  
```  
  
## <a name="remarks"></a>Note  
 Rilascia la memoria associata.
