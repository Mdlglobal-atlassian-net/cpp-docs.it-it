# Rowset dello schema
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Per schema, o metadati, si intendono tutte le informazioni sulla struttura o sull'organizzazione di un database, quali le informazioni sul provider, sul rowset, sulla tabella, sulle colonne o altre informazioni estranee al contenuto effettivo del database.  OLE DB consente di ottenere tali informazioni senza aprire il rowset tramite i rowset dello schema.  
  
 Per informazioni sull'utilizzo dei rowset dello schema, vedere [Recupero di metadati tramite i rowset dello schema](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
