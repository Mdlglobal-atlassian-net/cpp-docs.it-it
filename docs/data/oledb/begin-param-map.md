# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Contrassegna l'inizio delle voci della mappa dei parametri.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Parametri  
 *x*  
 [in] Nome della classe di record utente.  
  
## <a name="remarks"></a>Note  
 I parametri vengono utilizzati da [comandi](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## <a name="example"></a>Esempio  
 Vedere l'esempio per il [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) (macro).  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** atldbcli.h
