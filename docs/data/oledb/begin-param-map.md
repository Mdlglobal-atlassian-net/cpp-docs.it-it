# BEGIN_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Contrassegna l'inizio delle voci della mappa dei parametri.  
  
## Sintassi  
  
```  
  
BEGIN_PARAM_MAP(  
x  
 )  
  
```  
  
#### Parametri  
 *x*  
 \[in\] Nome della classe di record utente.  
  
## Note  
 I parametri vengono utilizzati da [comandi](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## Esempio  
 Vedere l'esempio relativo alla macro [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md).  
  
## Requisiti  
 **Intestazione:** atldbcli.h
