# Classe CMultipleResults
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Se si desidera che un comando gestione di più set di risultati, utilizzare `CMultipleResults` per l'argomento di template `CCommand`*TMultiple*.  
  
## Sintassi  
  
```  
class CMultipleResults  
```  
  
## Note  
 Per gestire i più set di risultati, [CCommand](../../data/oledb/ccommand-class.md) deve ereditare dalla classe.
