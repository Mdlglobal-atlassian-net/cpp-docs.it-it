# CEnumeratorAccessor::m_szParseName
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Stringa da passare a [IParseDisplayName](http://msdn.microsoft.com/library/windows/desktop/ms680604) per ottenere un moniker per l'origine dati o enumeratore.  
  
## Sintassi  
  
```  
  
WCHAR m_szParseName[129];  
  
```  
  
## Note  
 Vedere [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) in *OLE DB Programmer's Reference* per ulteriori informazioni.

 **Intestazione:** atldbcli.h
