# <a name="beginschemamap"></a>BEGIN_SCHEMA_MAP
Indica l'inizio di una mappa di schema.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp
      BEGIN_SCHEMA_MAP(SchemaClass);  
```  
  
#### <a name="parameters"></a>Parametri  
 *SchemaClass*
