# Quando &#232; consigliabile utilizzare le classi di database?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Numerose applicazioni fanno uso di dati che devono essere memorizzati in un database e altre applicazioni potrebbero trarre vantaggio dall'utilizzo di un database.  Un database offre un repository di dati flessibile a cui possono accedere, in numerosi casi, più utenti e più applicazioni.  I database sono in grado di memorizzare grandi quantità di dati e garantiscono un rapido accesso ai dati in caso di query e aggiornamenti.
