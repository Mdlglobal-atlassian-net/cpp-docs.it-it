# Progettazione e creazione di una visualizzazione di record (accesso ai dati MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mediante la [Creazione guidata applicazione MFC](../mfc/reference/database-support-mfc-application-wizard.md) è possibile creare una classe per la visualizzazione di record.  Se si opta per una procedura guidata per la creazione di un'applicazione, verranno create sia la classe per la visualizzazione di record, sia la relativa risorsa modello di finestra di dialogo \(senza controlli\).  Per aggiungere i controlli alla risorsa modello di finestra di dialogo, usare l'editor finestre di Visual C\+\+.  Se invece si opta per **Aggiungi classe**, sarà necessario creare innanzitutto la risorsa modello di finestra di dialogo nell'editor finestre e quindi creare la classe per la visualizzazione di record.  
  
 Le informazioni fornite sono relative sia a `CRecordView` che a `CDaoRecordView`.  
  
#### Per creare una visualizzazione di record mediante la Creazione guidata applicazione MFC  
  
1.  Vedere [Supporto database, Creazione guidata applicazione MFC](../mfc/reference/database-support-mfc-application-wizard.md).  
  
#### Per progettare il form  
  
1.  Vedere [Editor finestre](../mfc/dialog-editor.md).  
  
#### Per creare una classe per la visualizzazione di record  
  
1.  Vedere [Aggiunta di un consumer ODBC MFC](../mfc/reference/adding-an-mfc-odbc-consumer.md).
