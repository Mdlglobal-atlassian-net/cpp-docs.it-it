# Controlli associati a dati (ADO e RDO)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'associazione dati è un meccanismo che consente agli utenti di utilizzare dati estratti da un database.  
  
 In Visual C\+\+ è disponibile un meccanismo di associazione dati riutilizzabile tramite controlli ActiveX in progetti MFC, grazie al quale è possibile sviluppare rapidamente applicazioni di database.  
  
 Questa sezione è composta dalle seguenti sottosezioni nelle quali viene illustrato come implementare l'associazione dati in un'applicazione Visual C\+\+:  
  
-   [Utilizzo dei controlli ActiveX](../../data/ado-rdo/using-activex-controls.md)  
  
-   [Associazione dati con controlli ActiveX in Visual C\+\+](../../data/ado-rdo/databinding-with-activex-controls-in-visual-cpp.md)  
  
-   [Creazione di connessioni a database](../../data/ado-rdo/creating-database-connections.md)
