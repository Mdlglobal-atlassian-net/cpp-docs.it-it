# Inserimento di un controllo in un&#39;applicazione Visual C++
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

È possibile inserire un controllo ActiveX in una finestra di dialogo dalla Casella degli strumenti o tramite la finestra di dialogo [Inserisci controllo ActiveX](../../mfc/insert-activex-control-dialog-box.md).  
  
### Per inserire un controllo ActiveX dalla Casella degli strumenti  
  
1.  Fare clic con il pulsante destro del mouse su una sezione vuota della Casella degli strumenti.  
  
2.  Scegliere **Personalizza Casella degli strumenti** dal menu di scelta rapida, quindi selezionare i controlli desiderati.  
  
3.  Trascinare i controlli nella finestra di dialogo all'interno dell'editor finestra.  
  
### Per inserire un controllo ActiveX dall'editor finestra  
  
1.  Fare clic con il pulsante destro del mouse sulla finestra di dialogo.  
  
2.  Scegliere [Inserisci controllo ActiveX](../../mfc/insert-activex-control-dialog-box.md) dal menu di scelta rapida.  
  
    > [!NOTE]
    >  Quando si inserisce un controllo ActiveX in un progetto da **Inserisci controllo ActiveX**, le [classi wrapper](../../data/ado-rdo/wrapper-classes.md) non vengono incluse nel progetto.  Per personalizzare le funzionalità del controllo, sarà necessario creare manualmente una classe wrapper.
