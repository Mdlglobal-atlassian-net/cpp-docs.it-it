# Installazione del supporto di ATL per database
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

L'installazione di base include tutti i componenti necessari per supportare la programmazione con OLE DB.  Le interfacce OLE DB vengono installate come parte dell'SDK di MDAC.  I modelli OLE DB, che supportano la programmazione con OLE DB in Visual C\+\+ .NET, vengono installati come parte della libreria ATL.  L'installazione di Visual C\+\+ .NET comprende sia l'SDK di MDAC 2.8 che la libreria ATL.
