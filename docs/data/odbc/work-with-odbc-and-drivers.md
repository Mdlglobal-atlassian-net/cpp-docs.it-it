# <a name="work-with-odbc-and-drivers"></a>Utilizzo di ODBC e driver

ODBC consente di scrivere applicazioni che non dipendono da un solo sistema di gestione di database (DBMS). È possibile scrivere un'applicazione in grado di operare su più DBMS per i quali gli utenti dispongono del driver ODBC specifico per DBMS.
