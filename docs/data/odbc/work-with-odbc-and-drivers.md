# Utilizzo di ODBC e driver
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ODBC consente di scrivere applicazioni che non dipendono da un solo sistema di gestione di database \(DBMS\).  È possibile creare un'unica applicazione in grado di funzionare con più DBMS per i quali gli utenti finali dispongono del driver ODBC corretto, specifico di ogni DBMS.
