# Origine dati: determinazione dello schema dell&#39;origine dati (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

L'argomento è relativo alle classi ODBC MFC.  
  
 Per impostare i membri dati negli oggetti `CRecordset`, è necessario conoscere lo schema dell'origine dati a cui si sta effettuando la connessione.  La determinazione dello schema di un'origine dati comporta il recupero di informazioni quali un elenco delle tabelle dell'origine dati, un elenco delle colonne di ciascuna tabella, il tipo di dati di ogni tabella e l'esistenza di eventuali indici.
