# Dove sono reperibili gli articoli della Microsoft Knowledge Base relativi ai database?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Per trovare gli articoli in lingua inglese della Knowledge Base relativi a ODBC MFC, effettuare una ricerca nella Knowledge Base specificando la parola chiave MFCDATABASE.  
  
 Per trovare gli articoli in lingua inglese della Knowledge Base relativi a DAO MFC, effettuare una ricerca nella Knowledge Base specificando la parola chiave MFCDAO.
