# <a name="program-and-linkage--c"></a>Programma e collegamento (C++)
Un programma è costituito da uno o più unità di conversione collegate tra loro. Esecuzione (concettualmente) inizia nell'unità di conversione che contiene la funzione **principale**. (Per ulteriori informazioni sulle unità di conversione, vedere [fasi di conversione](../preprocessor/phases-of-translation.md)nella *riferimenti al preprocessore*.) Per ulteriori informazioni sul **principale** funzione, vedere [avvio del programma: la funzione principale](../cpp/main-program-startup.md).)
