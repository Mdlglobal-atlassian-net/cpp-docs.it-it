# <a name="jump-statements-c"></a>Istruzioni di spostamento (C++)
L'istruzione di salto C++ esegue un immediato trasferimento di controllo locale.  
  
## <a name="syntax"></a>Sintassi  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
