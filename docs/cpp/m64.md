# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Sezione specifica Microsoft  
 Il **__m64** è di tipo di dati per l'uso con l'intrinseci MMX e 3DNow! oggetti intrinseci e viene definito in \<xmmintrin >.  
  
```cpp 
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Note  
 Non è necessario accedere il **__m64** direttamente ai campi. È comunque possibile visualizzare i seguenti tipi nel debugger. Una variabile di tipo **__m64** esegue il mapping ai registri MM [0-7].  
  
 Le variabili di tipo **_m64** vengono automaticamente allineate su limiti di 8 byte.  
  
 Il **__m64** tipo di dati non è supportato in [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)] processori. Le applicazioni che usano __m64 come parte di intrinseci MMX devono essere riscritte per usare intrinseci SSE e SSE2 equivalenti.  
  
**Fine sezione specifica Microsoft**
