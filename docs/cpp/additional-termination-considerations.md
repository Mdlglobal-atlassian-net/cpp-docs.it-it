# <a name="additional-termination-considerations"></a>Considerazioni di terminazione aggiuntive

È possibile terminare un C++ programma utilizzando `exit`, **Return**o `abort`. È possibile aggiungere l'elaborazione di uscita utilizzando la funzione `atexit`. Questi argomenti vengono approfonditi nelle sezioni successive.
