# Considerazioni di terminazione aggiuntive
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

È possibile terminare un programma C\+\+ utilizzando **exit**, `return` o **abort**.  È possibile aggiungere l'elaborazione di uscita utilizzando la funzione `atexit`.  Questi argomenti vengono approfonditi nelle sezioni successive.
