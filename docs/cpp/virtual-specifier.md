# <a name="virtual-specifier"></a>Identificatore virtual

La parola chiave [Virtual](../cpp/virtual-cpp.md) può essere applicata solo a funzioni membro di classe non statiche. Ciò significa che l'associazione delle chiamate alla funzione viene rimandata fino alla fase di esecuzione. Per ulteriori informazioni, vedere [funzioni virtuali](../cpp/virtual-functions.md).
