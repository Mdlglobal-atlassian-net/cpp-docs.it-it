# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Sezione specifica Microsoft  
 Il `__m128d` il tipo di dati, per l'utilizzo con gli intrinseci delle istruzioni Streaming SIMD Extensions 2, è definito in \<emmintrin >.  
  
```  
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Note  
 Non è necessario accedere direttamente ai campi `__m128d`. È comunque possibile visualizzare i seguenti tipi nel debugger. Una variabile delle mappe di tipo `__m128` nei registri XMM[0-7].  
  
 Le variabili di tipo **_m128d** vengono automaticamente allineate su limiti di 16 byte.  
  
 Il tipo di dati `__m128d` non è supportato nei processori ARM.  
  
**Fine sezione specifica Microsoft**
