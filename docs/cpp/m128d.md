# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Sezione specifica Microsoft  
 Il **__m128d** tipo di dati, per l'uso con gli intrinseci delle istruzioni Streaming SIMD Extensions 2, è definito in \<emmintrin >.  
  
```cpp 
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Note  
 Non è necessario accedere il **__m128d** direttamente ai campi. È comunque possibile visualizzare i seguenti tipi nel debugger. Una variabile di tipo **__m128** mappe per i registri XMM [0-7].  
  
 Le variabili di tipo **_m128d** vengono automaticamente allineate su limiti di 16 byte.  
  
 Il **__m128d** tipo di dati non è supportato nei processori ARM.  
  
**Fine sezione specifica Microsoft**
