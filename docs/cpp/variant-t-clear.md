# _variant_t::Clear
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Sezione specifica Microsoft**  
  
 Cancella l'oggetto incapsulato **VARIANT**.  
  
## Sintassi  
  
```  
  
void Clear( );  
  
```  
  
## Note  
 Chiama **VariantClear** sull'oggetto **VARIANT** incapsulato.
