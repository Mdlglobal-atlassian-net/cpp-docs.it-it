# <a name="m128"></a>__m128
## <a name="microsoft-specific"></a>Sezione specifica Microsoft  
 Il **m128 data** tipo, per l'utilizzo di Streaming SIMD Extensions e le funzioni intrinseche delle istruzioni Streaming SIMD Extensions 2, è definito in xmmintrin.  
  
```  
// data_types__m128.cpp  
#include <xmmintrin.h>  
int main() {  
   __m128 x;  
}  
```  
  
## <a name="remarks"></a>Note  
 Non è necessario accedere direttamente ai campi `__m128`. È comunque possibile visualizzare i seguenti tipi nel debugger. Una variabile delle mappe di tipo `__m128` nei registri XMM[0-7].  
  
 Le variabili di tipo **m128** vengono automaticamente allineate su limiti di 16 byte.  
  
 Il tipo di dati `__m128` non è supportato nei processori ARM.  
  
**Fine sezione specifica Microsoft**
