# <a name="m128"></a>__m128

## <a name="microsoft-specific"></a>Sezione specifica Microsoft

Il **dati __m128** tipo, da usare con il Streaming SIMD Extensions e intrinseci delle istruzioni Streaming SIMD Extensions 2, è definito in \<xmmintrin >.

```cpp
// data_types__m128.cpp
#include <xmmintrin.h>
int main() {
   __m128 x;
}
```

## <a name="remarks"></a>Note

Non è necessario accedere il **__m128** direttamente ai campi. È comunque possibile visualizzare i seguenti tipi nel debugger. Una variabile di tipo **__m128** mappe per i registri XMM [0-7].

Le variabili di tipo **__m128** vengono automaticamente allineate su limiti di 16 byte.

Il **__m128** tipo di dati non è supportato nei processori ARM.

**Fine sezione specifica Microsoft**
