# <a name="comerroroperator-"></a>_com_error::operator =
**Sezione specifica Microsoft**  
  
 Assegnare un oggetto `_com_error` esistente a un altro.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### <a name="parameters"></a>Parametri  
 `that`  
 Oggetto `_com_error`.
