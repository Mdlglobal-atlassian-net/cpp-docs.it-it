# _com_error::operator =
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Sezione specifica Microsoft**  
  
 Assegnare un oggetto `_com_error` esistente a un altro.  
  
## Sintassi  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### Parametri  
 `that`  
 Oggetto `_com_error`.
