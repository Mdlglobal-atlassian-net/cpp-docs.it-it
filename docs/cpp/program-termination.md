# Chiusura del programma
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In C\+\+ esistono vari modi per uscire da un programma:  
  
-   Chiamare la funzione [exit](../cpp/exit-function.md).  
  
-   Chiamare la funzione [abort](../cpp/abort-function.md).  
  
-   Eseguire un'istruzione [return](../cpp/return-statement-in-program-termination-cpp.md) da **main**.
