# <a name="c28022"></a>C28022

> avviso C28022: le classi di funzione \<classlist1 > in questa funzione non corrispondono alle classi di funzioni \<classlist2 > sul typedef utilizzato per definirlo.

Questo avviso viene segnalato quando si verifica un errore nelle annotazioni. Sia il typedef che la funzione sono `_Function_class_` annotazioni, ma non corrispondono. Se vengono usati entrambi, devono corrispondere.
