# <a name="c28282"></a>C28282

> avviso C28282: le stringhe di formato devono essere nelle precondizioni

Questo avviso indica che viene individuato un `__drv_formatString` annotazione, che non si trova in un'annotazione `_Pre_` (`__drv_in`) (per impostazione predefinita, i parametri della funzione sono `_Pre_`). Controllare se l'annotazione utilizzata in un blocco esplicito con un'annotazione `_Post_` (`__drv_out`). In tal caso, rimuovere l'annotazione da qualsiasi blocco di inclusione in cui Ã¨ stato inserito in un contesto di `_Post_`.
