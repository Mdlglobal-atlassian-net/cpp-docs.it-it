# <a name="c28275"></a>C28275

> avviso C28275: il parametro per \_valore\_macro\_ è null

Questo avviso indica che si è verificato un errore interno nel file del modello e non nel codice da analizzare. La funzione *macroValue* è stata chiamata senza un parametro.
