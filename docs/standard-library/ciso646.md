# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Include l'intestazione della libreria C standard \<iso646.h> e aggiunge i nomi associati allo spazio dei nomi `std`.

> [!NOTE]
> Uso di questa intestazione non Ã¨ necessaria in C++.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<ciso646 >

**Spazio dei nomi:** std

## <a name="remarks"></a>Note

L'inclusione di questa intestazione garantisce che i nomi dichiarati usando il collegamento esterno nell'intestazione della libreria C standard vengano dichiarati nello spazio dei nomi `std`.
