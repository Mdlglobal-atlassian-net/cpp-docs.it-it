# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Include l'intestazione \<della libreria standard C iso646. h > e aggiunge i nomi associati `std` allo spazio dei nomi.

> [!NOTE]
> Poiché questa intestazione di compatibilità definisce i nomi che sono C++parole chiave in, anche se non ha alcun effetto. L' \<intestazione di > iso646. h è deprecata C++in. L' \<intestazione ciso646 > viene rimossa nello standard bozza c++ 20.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<> ciso646

**Spazio dei nomi:** std

## <a name="remarks"></a>Note

L'inclusione di questa intestazione garantisce che i nomi dichiarati utilizzando il collegamento esterno nell'intestazione della libreria `std` standard C vengano dichiarati nello spazio dei nomi.
