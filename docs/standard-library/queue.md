# &lt; code &gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Definisce l'oggetto priority_queue delle classi modello, nonché la coda e diversi modelli di supporto.  
  
## <a name="syntax"></a>Sintassi  
  
```  
#include <queue>  
  
```  
  
### <a name="operators"></a>Operatori  
  
|||  
|-|-|  
|[operatore! =](../Topic/%3Cqueue%3E%20operators.md#operator_neq)|Verifica se l'oggetto queue a sinistra dell'operatore non è uguale all'oggetto queue a destra.|  
|[operatore <](../Topic/%3Cqueue%3E%20operators.md#operator_lt_)|Verifica se l'oggetto queue a sinistra dell'operatore è minore dell'oggetto queue a destra.|  
|[operatore \< =](../Topic/%3Cqueue%3E%20operators.md#operator_lt__eq)|Verifica se l'oggetto queue a sinistra dell'operatore è minore o uguale all'oggetto queue a destra.|  
|[operatore = =](../Topic/%3Cqueue%3E%20operators.md#operator_eq_eq)|Verifica se l'oggetto queue a sinistra dell'operatore è uguale all'oggetto queue a destra.|  
|[operatore >](../Topic/%3Cqueue%3E%20operators.md#operator_gt_)|Verifica se l'oggetto queue a sinistra dell'operatore è maggiore dell'oggetto queue a destra.|  
|[operatore > =](../Topic/%3Cqueue%3E%20operators.md#operator_gt__eq)|Verifica se l'oggetto queue a sinistra dell'operatore è maggiore o uguale all'oggetto queue a destra.|  
  
### <a name="classes"></a>Classi  
  
|||  
|-|-|  
|[Classe Queue](../standard-library/queue-class.md)|Classe di adattatori di contenitori di modelli che fornisce una restrizione di funzionalità, limitando l'accesso agli elementi anteriore e posteriore di un tipo di contenitore sottostante.|  
|[Classe priority_queue](../standard-library/priority-queue-class.md)|Classe di adattatori di contenitori di modelli che fornisce una restrizione di funzionalità, limitando l'accesso all'elemento superiore di un tipo di contenitore sottostante, che è sempre il più grande.|
