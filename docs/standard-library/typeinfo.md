# &lt;typeinfo&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Includere l'intestazione standard `<typeinfo>` per definire diversi tipi associati con l'operatore [Operatore typeid](../cpp/typeid-operator.md)tipo di autenticazione, che semplifica le informazioni sui tipi statici e dinamici.  
  
## Sintassi  
  
```  
  
#include <typeinfo>  
  
```  
  
## Note  
 Per informazioni sulle classi definite \<in\>typeinfo, vedere i seguenti argomenti:  
  
-   [Eccezione bad\_cast](../cpp/bad-cast-exception.md)  
  
-   [Eccezione bad\_typeid](../cpp/bad-typeid-exception.md)  
  
-   [Classe type\_info](../cpp/type-info-class.md)
