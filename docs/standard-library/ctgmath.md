# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;

Include in effetti il C++ intestazioni della libreria Standard \<complesso > e \<cmath >, che forniscono macro matematiche di tipo generico equivalenti a \<tgmath. h >.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<ctgmath >

**Spazio dei nomi:** std

## <a name="remarks"></a>Note

La funzionalit√† dell'intestazione della libreria C Standard \<tgmath. h > viene offerta dagli overload in \<complesso > e \<cmath >.

L'inclusione di questa intestazione garantisce che i nomi dichiarati usando il collegamento esterno nell'intestazione della libreria C standard vengano dichiarati nello spazio dei nomi `std`.
