# &lt;ctgmath&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Include in effetti le intestazioni della libreria C\+\+ standard \<ccomplex\> e \<cmath\> che forniscono macro matematiche di tipo generico equivalenti a \<tgmath.h\>.  
  
## Sintassi  
  
```  
  
#include <ctgmath>  
  
```  
  
## Note  
 Le funzionalit√† dell'intestazione della libreria C standard \<tgmath.h\> vengono fornite dagli overload in \<ccomplex\> e \<cmath\>.  
  
 L'inclusione di questa intestazione garantisce che i nomi dichiarati usando il collegamento esterno nell'intestazione della libreria C standard vengano dichiarati nello spazio dei nomi `std`.
