# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;
Include le intestazioni della libreria C++ standard \<ccomplex> e \<cmath> che offrono macro matematiche di tipo generico equivalenti a \<tgmath.h>.  
  
## <a name="syntax"></a>Sintassi  
  
```  
#include <ctgmath>  
  
```  
  
## <a name="remarks"></a>Note  
 La funzionalit√† dell'intestazione della libreria C standard \<tgmath.h> viene offerta dagli overload in \<ccomplex> e \<cmath>.  
  
 L'inclusione di questa intestazione garantisce che i nomi dichiarati usando il collegamento esterno nell'intestazione della libreria C standard vengano dichiarati nello spazio dei nomi `std`.
