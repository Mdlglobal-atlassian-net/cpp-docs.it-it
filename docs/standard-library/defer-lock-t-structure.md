# Struttura defer_lock_t
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Rappresenta un tipo che definisce un oggetto di [defer\_lock](../Topic/defer_lock%20Variable.md) utilizzata per selezionare uno dei costruttori di overload di [unique\_lock](../standard-library/unique-lock-class.md).  
  
## Sintassi  
  
```cpp  
struct defer_lock_t;  
```  
  
## Requisiti  
 **Intestazione:** mutex  
  
 **Spazio dei nomi:** std
