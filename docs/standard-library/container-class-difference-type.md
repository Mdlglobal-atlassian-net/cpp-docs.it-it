# <a name="container-classdifferencetype"></a>Container Class::difference_type
> [!NOTE]
>  Questo argomento è incluso nella documentazione di Visual C++ come esempio non funzionante dei contenitori usati nella libreria standard di C++. Per altre informazioni, vedere [C++ Standard Library Containers](../standard-library/stl-containers.md) (Contenitori della libreria standard di C++).  
  
 Descrive un oggetto che può rappresentare la differenza tra gli indirizzi di due qualsiasi elementi nella sequenza controllata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
 
typedef T1 difference_type;  
```  
  
## <a name="remarks"></a>Note  
 Viene descritta come un sinonimo del tipo non specificato **T1** (in genere **Alloc::difference_type**).
