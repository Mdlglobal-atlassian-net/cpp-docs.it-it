# &lt;forward_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Definisce il forward\_list della classe modello del contenitore e diversi modelli di supporto.  
  
## Sintassi  
  
```  
#include <forward_list>  
```
