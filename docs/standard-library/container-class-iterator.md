# <a name="container-classiterator"></a>Container Class::iterator
> [!NOTE]
>  Questo argomento è incluso nella documentazione di Visual C++ come esempio non funzionante dei contenitori usati nella libreria standard di C++. Per altre informazioni, vedere [C++ Standard Library Containers](../standard-library/stl-containers.md) (Contenitori della libreria standard di C++).  
  
 Descrive un oggetto che può essere usato come iteratore per la sequenza controllata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
 
typedef T5 iterator;  
```  
  
## <a name="remarks"></a>Note  
 Viene descritta come un sinonimo del tipo non specificato **T5**. Un oggetto di tipo **iterator** può essere convertito in un oggetto di tipo [const_iterator](../standard-library/container-class-const-iterator.md).
