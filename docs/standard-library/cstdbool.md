# <a name="ltcstdboolgt"></a>&lt;cstdbool&gt;

Include l'intestazione della libreria C standard \<stdbool.h> e aggiunge i nomi associati allo spazio dei nomi `std`.

> [!NOTE]
> Questa intestazione Ã¨ stata deprecata in C++ 17.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<cstdbool >

**Spazio dei nomi:** std

## <a name="remarks"></a>Note

L'inclusione di questa intestazione garantisce che i nomi dichiarati usando il collegamento esterno nell'intestazione della libreria C standard vengano dichiarati nello spazio dei nomi `std`.
