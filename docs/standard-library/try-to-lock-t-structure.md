# Struttura try_to_lock_t
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Rappresenta `struct` che definisce un oggetto di [try\_to\_lock](../Topic/try_to_lock%20Variable.md).  Utilizzata per selezionare uno dei costruttori di overload di [unique\_lock](../standard-library/unique-lock-class.md).  
  
## Sintassi  
  
```cpp  
struct try_to_lock_t;  
```  
  
## Requisiti  
 **Intestazione:** mutex  
  
 **Spazio dei nomi:** std
