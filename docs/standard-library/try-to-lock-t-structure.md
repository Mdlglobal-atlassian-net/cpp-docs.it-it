# <a name="trytolockt-structure"></a>Struttura try_to_lock_t

Rappresenta un **struct** che definisce un [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) oggetto. Usata per selezionare uno dei costruttori di overload di [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Sintassi

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Requisiti

**Intestazione:** \<mutex >

**Spazio dei nomi:** std
