# <a name="ltstackgt"></a>&lt;stack&gt;
Definisce lo stack della classe modello e due modelli di supporto.  
  
## <a name="syntax"></a>Sintassi  
  
```  
#include <stack>  
  
```  
  
### <a name="operators"></a>Operatori  
  
|||  
|-|-|  
|[operator!=](../standard-library/stack-operators.md#operator_neq)|Verifica se l'oggetto stack a sinistra dell'operatore non è uguale all'oggetto stack a destra.|  
|[operator<](../standard-library/stack-operators.md#operator_eq_eq)|Verifica se l'oggetto stack a sinistra dell'operatore è minore dell'oggetto stack a destra.|  
|[operator\<=](../standard-library/stack-operators.md#operator_lt_)|Verifica se l'oggetto stack a sinistra dell'operatore è minore o uguale all'oggetto stack a destra.|  
|[operator==](../standard-library/stack-operators.md#operator_lt__eq)|Verifica se l'oggetto stack a sinistra dell'operatore è uguale all'oggetto stack a destra.|  
|[operator>](../standard-library/stack-operators.md#operator_gt_)|Verifica se l'oggetto stack a sinistra dell'operatore è maggiore dell'oggetto stack a destra.|  
|[operator>=](../standard-library/stack-operators.md#operator_gt__eq)|Verifica se l'oggetto stack a sinistra dell'operatore è maggiore o uguale all'oggetto stack a destra.|  
  
### <a name="classes"></a>Classi  
  
|||  
|-|-|  
|[Classe stack](../standard-library/stack-class.md)|Una classe di adattatori di contenitori di modelli che fornisce una restrizione di funzionalità, limitando l'accesso all'elemento aggiunto più recentemente a un tipo di contenitore sottostante.|
