# <a name="ltstackgt"></a>&lt;stack&gt;

Definisce lo stack della classe modello e due modelli di supporto.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<stack>

**Spazio dei nomi:** std

> [!NOTE]
> Il \<stack > libreria Usa inoltre il `#include <initializer_list>` istruzione.
