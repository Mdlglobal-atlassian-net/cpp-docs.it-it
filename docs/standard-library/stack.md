# <a name="ltstackgt"></a>&lt;stack&gt;

Definisce lo stack del modello di classe e due modelli di supporto.

## <a name="requirements"></a>Requisiti

**Intestazione:** \<stack>

**Spazio dei nomi:** std

> [!NOTE]
> Anche la libreria \<stack > USA l'istruzione `#include <initializer_list>`.
