# &lt;bitset&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Definisce il bitset della classe modello e due funzioni di supporto del modello per la rappresentazione e la modifica delle sequenze di bit a dimensione fissa.  
  
## Sintassi  
  
```  
  
#include <bitset>  
  
```  
  
### Operatori  
  
|||  
|-|-|  
|[operator&](../Topic/operator&%20\(%3Cbitset%3E\).md)|Esegue un'operazione Bitwise\-AND tra due bitset.|  
|[operatore \<\<](../Topic/operator%3C%3C%20\(%3Cbitset%3E\).md)|Inserisce una rappresentazione di testo della sequenza di bit nel flusso di output standard.|  
|[operatore \>\>](../Topic/operator%3E%3E%20\(%3Cbitset%3E\).md)|Inserisce una rappresentazione di testo della sequenza di bit nel flusso di input standard.|  
|[operator^](../Topic/operator%5E%20\(%3Cbitset%3E\).md)|Esegue un'operazione Bitwise\-OR esclusiva tra due bitset.|  
|[operator&#124;](../Topic/operator%7C%20\(%3Cbitset%3E\).md)|Esegue un'operazione Bitwise\-OR tra due bitset.|  
  
### Classi  
  
|||  
|-|-|  
|[Classe bitset](../standard-library/bitset-class.md)|La classe modello descrive un tipo di oggetto che archivia una sequenza formata da un numero fisso di bit che forniscono un modo compatto di mantenere i flag per un set di elementi o condizioni.|
