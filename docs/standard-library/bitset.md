# <a name="ltbitsetgt"></a>&lt;bitset&gt;
Definisce il bitset della classe modello e due funzioni di supporto del modello per la rappresentazione e la modifica delle sequenze di bit a dimensione fissa.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
#include <bitset>  
  
```  
  
### <a name="operators"></a>Operatori  
  
|||  
|-|-|  
|[operator&](../standard-library/bitset-operators.md#op_amp)|Esegue un'operazione Bitwise-AND tra due bitset.|  
|[operator<\<](../standard-library/bitset-operators.md#op_lt_lt)|Inserisce una rappresentazione di testo della sequenza di bit nel flusso di output standard.|  
|[operator>> ](../standard-library/bitset-operators.md#op_gt_gt)|Inserisce una rappresentazione di testo della sequenza di bit nel flusso di input standard.|  
|[operator^](../standard-library/bitset-operators.md#op_xor)|Esegue un'operazione Bitwise-OR esclusiva tra due bitset.|  
|[operator&#124;](../standard-library/bitset-operators.md#op_or)|Esegue un'operazione Bitwise-OR tra due bitset.|  
  
### <a name="classes"></a>Classi  
  
|||  
|-|-|  
|[Classe bitset](../standard-library/bitset-class.md)|La classe modello descrive un tipo di oggetto che archivia una sequenza formata da un numero fisso di bit che forniscono un modo compatto di mantenere i flag per un set di elementi o condizioni.|
