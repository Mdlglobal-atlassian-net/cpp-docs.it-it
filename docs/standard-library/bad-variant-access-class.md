# <a name="badvariantaccess-class"></a>Classe bad_variant_access

Gli oggetti di tipo bad_variant_access vengono generati per gli accessi non valido di report per il valore di un oggetto variant.

## <a name="syntax"></a>Sintassi

```cpp
class bad_variant_access;
```
