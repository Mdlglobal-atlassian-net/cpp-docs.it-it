# Container Class::const_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Questo argomento è nella documentazione di Visual C\+\+ come esempio non funzionale di contenitori utilizzati nella libreria C\+\+ standard.  Per ulteriori informazioni, vedere [I contenitori STL](../standard-library/stl-containers.md).  
  
 Viene descritto un oggetto che può fungere da iteratore costante per la sequenza selezionata.  
  
## Sintassi  
  
```  
  
typedef T6 const_iterator;  
  
```  
  
## Note  
 Viene descritto qui come sinonimo del tipo specificato non **T6**.
