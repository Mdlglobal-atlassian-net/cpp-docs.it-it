# <a name="visual-studio-documentation-for-microsoft-c"></a>Documentazione di Visual Studio per Microsoft C++

Introduzione Questo repository contiene i file di origine per la documentazione tecnica di Microsoft C++ (MSVC o Visual C++) in continua evoluzione. Gli articoli vengono pubblicati nel [sito della documentazione di C++ in Visual Studio](https://docs.microsoft.com/cpp).

La documentazione di Visual Basic e Visual C# si trova in un repository separato all'indirizzo [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs) e la documentazione di Visual Studio si trova nel repository all'indirizzo [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs).

## <a name="contributing-to-the-documentation"></a>Collaborare alla stesura della documentazione

Gli utenti sono invitati a contribuire al miglioramento della documentazione di MSVC. Per una guida completa su come contribuire a migliorare docs.microsoft.com, vedere la [Panoramica della guida per i collaboratori di Microsoft Docs](https://docs.microsoft.com/contribute). Per informazioni su come contribuire alla documentazione di MSVC, vedere la [guida per i collaboratori](CONTRIBUTING.md).

All'interno di questo repository alcune cartelle sono dedicate ad aree funzionali specifiche di MSVC, ad esempio `standard-library` per gli articoli relativi alla libreria standard C++, `ide` per gli articoli relativi all'ambiente di sviluppo integrato (IDE) di Visual Studio e cos√¨ via. In ogni cartella, la sottocartella `/media` contiene i file delle immagini degli articoli. Per altre informazioni, vedere la [guida per i collaboratori](CONTRIBUTING.md).
