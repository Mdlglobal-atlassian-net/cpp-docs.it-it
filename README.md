# <a name="visual-studio-documentation-for-microsoft-c"></a>Documentazione di Visual Studio per Microsoft C++

Introduzione Questo repository contiene i file di origine per la documentazione tecnica di Microsoft C++ (MSVC o Visual C++) in continua evoluzione. Gli articoli vengono pubblicati nel [sito della documentazione di C++ in Visual Studio](https://docs.microsoft.com/cpp).

La documentazione per Visual Basic e Visual C# si trova in un repository [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs)separato in e la documentazione di Visual Studio si trova nel repository che si trova [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs)in.

## <a name="contributing-to-the-documentation"></a>Collaborare alla stesura della documentazione

Siamo lieti dei tuoi contributi per contribuire a migliorare la documentazione di MSVC. Per una guida completa per contribuire a docs.microsoft.com, vedere la panoramica della Guida per il [collaboratore Microsoft docs](https://docs.microsoft.com/contribute). Per informazioni su come contribuire alla documentazione di MSVC, vedere la [guida per i collaboratori](CONTRIBUTING.md).

All'interno di questo repository alcune cartelle sono dedicate ad aree funzionali specifiche di MSVC, ad esempio `standard-library` per gli articoli relativi alla libreria standard C++, `ide` per gli articoli relativi all'ambiente di sviluppo integrato (IDE) di Visual Studio e cos√¨ via. In ogni cartella, la sottocartella `/media` contiene i file delle immagini degli articoli. Per ulteriori informazioni, fare [riferimento alla guida](CONTRIBUTING.md) .
